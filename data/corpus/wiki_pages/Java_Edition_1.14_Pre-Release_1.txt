1.14 Pre-Release 1 is the first pre-release for Java Edition 1.14, released on April 10, 2019, which optimizes the game and fixes bugs. This is the first pre-release released in 2019.


== Changes ==


=== Mobs ===
Villagers

Fletchers can no longer sell Luck arrows.


=== General ===
Optimizations

Optimized gameplay.[more information needed]
Inside the optimize world screen there is now an option to remove cached world data (e.g., height maps and light), so that it is re-computed when the world is next loaded.
Tags

Removed all poi block tags, which were added in the previous snapshots.
