A version of Indev 0.31 was released on February 4, 2010, at 15:41 UTC.


== Changes ==


=== Mobs ===
General

Can now avoid dangers like cliffs, lava, etc.
Mobs now follow the player if they get close enough.
This is preparation for making mobs hostile.


== Bugs ==
2 bugs reported

Mobs spawn in large amounts and never despawn causing enormous amount of lag.
Major frame rate drops occur in certain conditions, making the game unplayable. This issue was not fixed until partially in Indev 0.31 20100205-1558 and fully in Indev 20100206-2034.


== Trivia ==
The al_version number of this version is 1.077, but the previous version's is 1.075.
It possibly means that a version is currently missing, or it might be due to an LWJGL update.
