The mountain edge or extreme hills edge was an edge variant of the windswept hills that generated before The Update that Changed the World, and was removed in Java Edition 1.18.


== Description ==
Before Java Edition 1.7.2, all extreme hills used to generate a mountain edge surrounding the whole biome. Similarly to the sparse jungle biome, it was a technical biome intended to provide a smooth transition from other biomes to the mountains. It was nearly identical to the extreme hills biome, but with gentler slopes.
With Vibrant Visuals, mountain edges use default environmental settings with a coolish color grading, making the biome appear colder blue. They use cold sunlight, which is slightly brighter at sunrise and sunset.
The mountain edge has never generated in Bedrock Edition, but remains unused.
