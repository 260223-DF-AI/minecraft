0.1.3 is a server version of Java Edition Alpha released for client version v1.0.16_02 on August 17, 2010.


== Additions ==
Added support for the /tell command, used to whisper something to another player.
Added support for the /list command, which will show a list of connected players to the op who issued it.
The server now logs what admin does what, and broadcasts any admin events to all connected ops.


== Changes ==
Ops can now build in the spawn area.


== Fixes ==
Fixed /save-off.
