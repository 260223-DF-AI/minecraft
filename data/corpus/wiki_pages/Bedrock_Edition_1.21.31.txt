1.21.31 is a hotfix to Bedrock Edition released on September 30, 2024, which fixes some bugs.
This version has been reuploaded on the Google Play Store with the internal version number 1.21.31.05 on October 10, after Bedrock Edition beta 1.21.50.20 was accidentally released to all users, even those not enrolled in the beta program.
This is the first native version of Bedrock Edition for PlayStation 5, which was released on October 22, 2024. Shortly after release, the game was updated to 1.21.40.


== Fixes ==
9 issues fixed
MCPE-183578  UI: D-pad positioning redo causing subluxation (metacarpophalangeal)
MCPE-185038  Transparent objects like water render as solid materials on Samsung Xclipse GPU devices
MCPE-185245  Various add-ons fail to load
MCPE-185282  World templates from the Marketplace no longer function and instead generate as normal worlds
MCPE-186676  Trading for emeralds using second slot causes game to crash
MCPE-186726  Entities (armor stands, villagers) drop down one block when reloaded
MCPE-187297  Minecraft entered beta 1.21.50.20 on Android on its own
Other

Fixed several crashes that could occur during gameplay.
An issue where some commands did not recognize certain block or item names has been fixed.
