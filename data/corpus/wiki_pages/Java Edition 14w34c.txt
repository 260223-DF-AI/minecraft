14w34c is the fifty-first snapshot for Java Edition 1.8, released on August 19, 2014,  which fixes bugs and crashes from 14w34a and 14w34b and removes the void fog.
This version was reuploaded to fix a crash with experience.


== Changes ==


=== Gameplay ===
Void fog

Removed void fog and the void particles.
