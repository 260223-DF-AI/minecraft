Beta 1.6.1 is a version of Java Edition Beta released on May 26, 2011, to fix an item bug in Beta 1.6.


== Fixes ==
1 bug fixed

Fixed an item bug when picking up an item with a full inventory caused an infinite number of item pickup animations to play.


== Trivia ==
After release, this version was only available for 12 minutes, becoming the shortest lived Java Edition Beta version.
