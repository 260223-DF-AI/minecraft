== Skipped version numbers ==
 
The following versions are assumed/implied to have been skipped – these versions have no proof of existence, other than the implicit proof of being part of a version number sequence. For instance, there is no proof for "0.0.14a_02" other than 0.0.14a_01 and 0.0.14a_03 both having been released.
Versions labeled "definite version skip" are confirmed to be skipped version numbers as opposed to the possibility of simply being an undocumented and unarchived version of the game.
