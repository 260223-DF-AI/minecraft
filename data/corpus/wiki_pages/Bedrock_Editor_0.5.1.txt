Bedrock Editor v0.5.1 is a minor alpha release for the Bedrock Editor released on January 4, 2024, which tweaks API and fixes bugs.


== Additions ==
World generation

Added Void World generation, which can be selected on world creation.


== Changes ==
API

Removed addSeparator function from IMenu API.
Added addEntityPicker function to IPropertyPane API for selecting an entity type.
Added interface IPropertyItemOptionsDataPicker for both entity and block picker optional properties, removed IPropertyItemOptionsBlocks.
Added toolGroupId optional property to IRegisterExtensionOptionalParameters API for organizing tools in custom groups.
User Interface

The Flat World toggle found on the Advanced tab of the Create New Project page has been replaced with a drop down, allowing new Editor projects to be created in a Vanilla, Flat, or Void world.


== Fixes ==
Fixed an issue where resource packs were not being retained during editing and testing.
