2.2.8908 (Windows), 2.2.8907 (Windows, via the Microsoft Store/Xbox app), 2.2.8909 (macOS), or 2.2.8906 (Linux), is an update to the launcher released on January 18, 2022.


== Changes ==
Updated how checkboxes within Installations tab work.
Updated user sign out flow for Mojang account users when they migrate to Microsoft account.


== Fixes ==
MCL-18431 - Skin preview doesn't show transparent pixels.
MCL-19984 - Java Edition shows Demo with active Xbox Game Pass subscription.
MCL-20247 - Can't change version.
MCL-20282 - Minecraft for Windows' uninstall warning is reused from Dungeons.
