Bedrock Editor v0.6.0, also known as the Visualization Update, is an alpha release for the Bedrock Editor released on April 10, 2024, which adds new summon tool and makes some changes to the UI and API.


== Additions ==

Added "Summon Tool" to Editor for spawning, moving, and deleting entities.
 Shift + CTRL + S to active the tool.
Click in an empty block to place the entity.
Click on an existing entity to select it.
CTRL + D to deselect the entity.
Mouse Wheel to rotate the entity.
DEL to delete the entity.
Keyboard U, D, L, R,  Page Up, or  Page Down to manually nudge selected entity in camera relative direction.


== Changes ==

Updates API class BrushShapeManager to provide APIs for cross-module brush shape extensions.
Supported block images on the editor UI will now have transparent background.
