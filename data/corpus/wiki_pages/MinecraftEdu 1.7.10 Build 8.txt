1.7.10 Build 8 is the eighth build for MinecraftEdu 1.7.10, released in 2014. It is based on Java Edition 1.7.10.


== Changes ==


=== General ===
Server window user control uses now only lowercase names.


== Fixes ==
6 bugs fixed

Some untranslated error messages regarding alias protection.
"This key is already in use" text sometimes showing behind buttons in control options.
No one being able to connect to a server if someone tried to connect with regular Minecraft client.
Giving in server window gave an error message.
Giving cobblestone, stone slabs etc. always gave regular stone.
Crowdin translation integration not working in client and server.
