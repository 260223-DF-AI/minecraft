0.984 is a version of MinecraftEdu, released on October 29, 2012, which changes server tools and fixes some bugs. It is based on Java Edition 1.3.2.
0.984 was reuploaded on November 9, 2012, to add some minor things.


== Changes ==


=== General ===
Open menu for servertool.
Tooltips to include main menu buttons.
Replaced big admin menu icons with smaller ones.
Translation fixes.
Fixed updating issue (does not anymore ask for updating).
