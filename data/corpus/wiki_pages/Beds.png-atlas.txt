beds.png-atlas is a procedurally-generated texture atlas used to store textures for beds.


== Format ==
beds.png-atlas is a fixed-width 512256-pixel file that stores textures for all of the bed block entities.


== History ==


=== Versions of the atlas ===


==== Mipmapped versions ====
Minecraft (until version 1.21.9) used to also generates mipmapped versions of this atlas, for unknown reasons. They are not used by the game at all - only blocks placed in the world which use the json model system use mipmapped textures.
