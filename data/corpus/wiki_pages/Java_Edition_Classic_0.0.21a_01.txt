0.0.21a_01 is a version of Java Edition Classic released on June 23, 2009, at 16:17 UTC to try to fix a "hang bug" by adding a max iterations to the clipping.


== Fixes ==
Attempted to fix a "hang bug" by adding a max iterations to the clipping.
However, the game still hangs. As the problem made worse, performance is now much lower than in previous versions.
