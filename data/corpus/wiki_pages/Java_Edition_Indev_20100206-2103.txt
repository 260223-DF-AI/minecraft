The second version of Minecraft Indev was released on February 6, 2010, at 21:03 UTC, to fix bugs in the previous version of Indev.


== Changes ==


=== Items ===
Bread

Is no longer stackable.[verify for Java Edition]


== Fixes ==
From Indev 20100206-2034

Fixed arrows crashing the game.


== Unconfirmed changes ==
The following changes happened in this version at the latest; as the immediately preceding versions have not been archived, there currently exists no way to confirm for certain whether the changes mentioned here were made in one of said versions.


=== User interface ===
The item forms of most blocks are now lit differently from before. The left and right side are once again shaded differently, making blocks appear brighter. If a block used a solid white texture, the top face would use the color  #FFFFFF, the left face  #9B9B9B, and the right face  #808080.
Farmland is not included in this table as this is the first archived version in which it exists, and whether this shading change occurred before or after the block's introduction is currently not known.
