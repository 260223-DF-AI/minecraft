1.17 Pre-release 2 (known as 1.17-pre2 in the launcher) is the second pre-release for Java Edition 1.17, released on May 31, 2021, which fixed bugs and crashes, including fixes related to smooth lighting.


== Changes ==


=== Block ===
Flowering azalea leaves

The namespaced ID was changed from minecraft:azalea_leaves_flowers to minecraft:flowering_azalea_leaves.


== Fixes ==
