26.1.1 is a hotfix for Java Edition released on April 1, 2026, which fixes an issue with chat reporting. It is compatible with 26.1 servers.
While it was not the last update to 26.1, it was the last one before snapshots for 26.2 began.


== Fixes ==
1 issue fixed
From 26.1

MC-307140  Chat messages can no longer be reported if the chat is enabled.
