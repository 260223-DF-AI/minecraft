Alpha v1.0.2_01 is a version of Java Edition Alpha released on July 6, 2010, which made changes to redstone-related items.


== Changes ==


=== Blocks ===
Redstone ore

Now emit red dust particles.
Redstone torch

Now only send redstone signals into blocks above or next to them.
Now emit red dust particles when lit, instead of flame particles.


=== Items ===
Redstone

No longer breaks when stepped on.[needs testing]
No longer connects through solid blocks diagonally down.
Now emit red dust particles.[needs testing]
The texture was slightly changed.


== Fixes ==
1 bug fixed

When trying to place a button on top of a block with a support block adjacent, the button will attach to the support block instead of floating.
