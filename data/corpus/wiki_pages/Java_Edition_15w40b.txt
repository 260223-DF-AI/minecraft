15w40b is the thirtieth snapshot for Java Edition 1.9, released on September 30, 2015 to fix a bug that made it impossible to connect to servers, among other bugs.


== Fixes ==
8 issues fixed
From released versions before 1.9

MC-48023  Can't trade with villagers that have a custom name after dying.
MC-53912  stacked minecarts with TNT not exploding properly.
MC-56648  Replace in Fill using /execute in a command block fails.
MC-58910  Bowsound loops, when Skeleton is not shooting.
From the 1.9 development versions

MC-89263  Blocks do not visually update unless looking downward.
From the previous development version

MC-89642  Server failure login.
MC-89643  Cant connect to myserver.
