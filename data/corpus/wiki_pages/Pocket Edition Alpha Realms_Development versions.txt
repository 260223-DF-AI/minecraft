This is a list of Pocket Edition Alpha Realms development versions.


== Realms alpha build 2 ==
Realms alpha build 2 is the second build version released for the Realms phase of v0.15.0 alpha. It was released to primarily fix crashes related to Realms.


=== Changes ===
Version number now reads "Realms alpha build 2" instead of "v0.15.0 alpha build 2".


=== Fixes ===
1 bug fixed
From released versions before v0.15.0

MCPE-11456 – Blocks above pressure plate do not update when block below pressure plate is destroyed.


== Realms alpha build 4 ==
Realms alpha build 4 is the third build version released for the Realms phase of v0.15.0 alpha.


=== Fixes ===
1 bug fixed
From released versions before v0.15.0

MCPE-13579 – No cocoa root displayed.


== Realms alpha build 1 ==

Realms alpha build 1 is the first build version released for the Realms phase of v0.15.0 alpha. Unlike the other first builds of a major update, where most of the planned stuff is added in, this build is for the Realms phase of 0.15, the major stuff will come "soon!".


=== Additions ===
Realms

Requires logging in to Microsoft Xbox Live account.
Free until the full release of v0.15.0.
Title screen

"Realms" button
Replaces the old "Options" button.
"Options" button
Is now an icon
Beside the "Languages" icon.


=== Changes ===
General

New grey inventory button design.
Title screen

Options button is now an icon
Moved to the bottom-right of the screen, next to the "Languages" button.
