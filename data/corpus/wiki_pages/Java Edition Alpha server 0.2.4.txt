0.2.4 is a server version of Java Edition Alpha released on November 10, 2010, for client version v1.2.2.


== Additions ==
New server-configuration setting: no-animals.


== Fixes ==
hellworld=true no longer overwrites old save chunks.
Boats are now visible to other players.
Players can now ride minecarts and boats.
Minecarts now move smoother.
Fishing now works.
Buckets can now pick up fluid.
Sheep and cows look like sheep and cows.
Now able to steer server-side boats.
Fixed stairs being difficult to destroy.
Increased the timeout before "took too long to log in" appears.
