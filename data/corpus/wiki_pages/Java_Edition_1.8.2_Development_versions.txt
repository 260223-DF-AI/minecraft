This is a list of Java Edition 1.8.2 development versions.


== 1.8.2-pre1 ==
1.8.2-pre1 is the first pre-release for Java Edition 1.8.2, released on December 18, 2014. This is the final pre-release released in 2014.


=== Additions ===


==== General ====
Debug screen 

New frame time graph.
Activated by pressing Alt+F3 to display frame time bar chart in lower left of screen.
Statistics

Added new statistics.
Cake slices eaten
Cauldrons filled
Water taken from cauldron
Armor pieces cleaned
Banners cleaned
Interactions with brewing stand
Interactions with beacon
Droppers searched
Hoppers searched
Dispensers searched
Noteblocks played
Noteblocks tuned
Plants potted
Trapped chests triggered
Ender chests opened
Items enchanted
Records played
Interactions with furnace
Interactions with crafting table
Chests opened


=== Changes ===


==== Blocks ====
TNT

Reverted explosion physics changes to those before the 14w31a snapshot.
Explosion force is no longer directionally biased.


==== Gameplay ====
Spectator mode

Mobs no longer follow/attack the player in Spectator mode.
