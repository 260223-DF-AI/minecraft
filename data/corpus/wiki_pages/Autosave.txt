Autosaving is a feature where the game is periodically saved so the player doesn't lose their progress.


== Description ==
In Java Edition, there is an autosave every 6,000 ticks (5 minutes), and every time the player opens the pause menu or a paused dialog screen. In multiplayer server, this can be toggled by /save-on and /save-off commands, and manually triggered by /save-all. In Bedrock Edition, the game autosaves much more often[more information needed], and configured by /save command.
The autosave indicator can be toggled in the video settings. In Java Edition, this is turned on by default and shows "Saving world" in the bottom right of the screen. In Bedrock Edition, this is turned off by default and shows an animation of an arrow pointing into a chest in the top-right of the screen.
