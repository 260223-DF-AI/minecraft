Spire armor trim is a type of smithing template used in smithing tables to add a trim to armor. It is consumed when used, but can be duplicated using an existing template, a purpur block, and diamonds.


== Obtaining ==
This item is found in End cities. Once obtained, it can be duplicated using the crafting recipe below.


=== Cloning ===

In Bedrock Edition spire armor trims can be duplicated with purpur blocks or purpur pillars.


=== Generated loot ===


== Usage ==
It is an ingredient in its own recipe, making it possible to craft copies after the initial item is found.


=== Crafting ingredient ===


=== Smithing ingredient ===

Using a smithing table, the spire armor trim can be applied to any armor piece, using any trim material.
