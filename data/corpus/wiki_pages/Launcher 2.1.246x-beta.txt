2.1.2467-beta (Windows), 2.1.2468-beta (macOS), or 2.1.2466-beta (Linux) is an update to the launcher released on February 14, 2019.


== Changes ==
The progress bar now updates properly in the "Prepare" phase.
Speed of the "Prepare" phase has been improved on multi-core machines.


== Fixes ==
Fixed a bug causing game launches to stall on Linux.
Fixed an error that would pop up when changing game directory in any configuration.
Fixed the launcher update progress bar not working on mac.
Fixed sounds not working in Minecraft versions prior to 1.6.
MCL-6280: No sound in older versions on native launcher, despite working in the Java launcher.
MCL-10486: minecraft-launcher 2.1.2339 hangs trying to load / run any version of minecraft.
