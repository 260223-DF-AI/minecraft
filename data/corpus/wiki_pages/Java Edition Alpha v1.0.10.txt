Alpha v1.0.10 is a version of Java Edition Alpha, released on July 22, 2010, which added initial support for multiplayer.


== Additions ==


=== Gameplay ===
Multiplayer

The only server is Mojang's server.
Accessible through a private invite only.


== Changes ==


=== General ===
Changed button "Single player" on the title screen to "Singleplayer".
Changed button "Multi player" on the title screen to "Multiplayer".
Removed the "Change location" and "Load location" settings in the controls menu.


== Bugs ==
2 bugs reported

Graphical glitch with mob in spawner block.
When the player dies, the death screen now displays "Score: &e0". This bug remained in the game until Java Edition Beta 1.9 Prerelease 2 and Java Edition 1.0.0.
