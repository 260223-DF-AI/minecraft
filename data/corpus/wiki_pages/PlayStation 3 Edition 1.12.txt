1.12  is a version of PlayStation 3 Edition released on December 20, 2014.  This version was released in Europe on December 20, 2014, while in North America the release was delayed until December 25, 2014. It is equivalent to Java Edition 1.6.4.
[edit]


== Additions ==
 


=== Blocks ===
Hardened clay, stained clay, block of coal, hay bale, activator rail, block of redstone, daylight sensor, dropper, hopper, minecart with hopper, minecart with TNT, redstone comparator, weighted pressure plate, beacon, trapped chest, multiple snow heights, command block


=== Items ===
Firework rocket, firework star, Nether star, lead, horse armor, name tag
Horse, donkey and mule spawn eggs
Version exclusive: All of the spawn eggs use the horse spawn egg texture.


=== Mobs ===
Wither, wither skeleton, witch, bat, horse, donkey, mule


=== Gameplay ===
Adventure mode
Added many new sounds
Mob naming


=== World generation ===
Witch hut
Replaced the Tutorial World with a new world
Nether fortress
Now generate loot chests


=== General ===
Interfaces for beacons, horses, hoppers and fireworks
New death messages

Options
Daylight Cycle, Keep Inventory, Tile drop, Regeneration option


== Changes ==


=== Blocks and items ===
Wood placement now depends on the surface it is placed on, rather than the direction of the player.
Redstone repeaters can now be locked.
Trapdoors can now be placed on upper half of blocks.
Bone meal no longer instantly grows crops to full size.
Dispensers can now face in any direction.
Snow can be crafted by 3 snow blocks.
Multiple snow layers can now be placed.
Golden apples give extra "absorption" health for a short period.
Updated textures which includes new textures for charcoal and woods.
Moved fences, iron bars, and cobblestone walls to the Building Blocks tab from the Miscellaneous tab in Creative.
Cobwebs no longer slow down players that are flying.
Swords can no longer be used to break blocks in creative mode.
Nether portals now teleport players in creative mode instantly.
Flowing water no longer pushes players that are flying.


=== Mobs ===
Slimes can now spawn in swamps at night.
Zombies can now pick up items.
Zombies and Skeletons can now spawn with different weapons and armor.
Mobs, items, and projectiles can now pass through portals.
Mobs named "Dinnerbone" or "Grumm" are now flipped upside down.
Spiders now have a chance to spawn with effects on Hard difficulty.
Ghasts and zombie pigmen no longer visibly catch fire when going into fire or lava.
Skeletons now have a new death sound instead of using the damage sound when dying. Unlike in Java Edition, the damage sound remains unchanged.


=== Gameplay ===
The longer the player remains in an area, the harder the monsters that spawn in that area will be.
Pressing use while sneaking now uses the held item instead of the currently selected block.
When riding an entity, dismounting has changed from pressing use on the entity to using the sneaking control.
Camera direction is no longer vertically reversed in third person front view to match Java Edition.


=== Bugs ===
The panorama was not changed to reflect the new tutorial world, it was fixed in 1.13.


== Fixes ==
Fixed two duplication glitches where the player could obtain infinite amount of diamonds, emeralds, gold, iron, or lapis lazuli.
