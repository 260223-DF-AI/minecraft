Beta 1.2_01 is a version of Java Edition Beta released on January 14, 2011, which reduced mob spawn rates, fixed bugs from Beta 1.2, and improved performance.


== Additions ==


=== Items ===
 Paintings

Added code for a new painting, but not the painting itself.
As such, when placed, this painting uses the painting placeholder texture.

=== Mobs ===
General

Reduced spawn rates a bit.


=== General ===
Splash

Text on the main menu is no longer always showing the "Finally beta!" splash.


== Technical ==


=== General ===
Performance

Improved performance.


== Fixes ==
3 bugs fixed

Fixed various blocks acting weird when placed (rails, torches, etc.).
Fixed lighting issues.
Added a temporary fix to get rid of chunk visibility errors.
