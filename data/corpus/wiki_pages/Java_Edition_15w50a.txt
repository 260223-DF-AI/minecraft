== Changes ==


=== Blocks ===
Hopper

Now support loot tables, using the same NBT data as chests.


=== Mobs ===
General

Improvements to pathfinding with respect to cacti and passable block.
Skeletons

No longer seek shade when wearing a helmet.


== Fixes ==
8 issues fixed

MC-50468  Skeleton AI ignoring helmet - skeleton still stays in shadow.
