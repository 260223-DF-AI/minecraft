Alpha v1.0.2_02 is a version of Java Edition Alpha released on July 6, 2010.


== Changes ==


=== Blocks ===
Water

Flowing water has its "level" data value directly set to 0 only if there is an opaque block or another water block down.


== Technical ==


=== General ===
Debug screen

The game output log now prints a "Loading texture" (+ <web link>) string when the player's skin is loading.
