13w01b is the second snapshot for Java Edition 1.5, released on January 4, 2013 to fix some of the bugs and crashes in the 13w01a snapshot.


== Changes ==


=== Blocks ===
Chests

All containers output a redstone signal when a comparator is placed against it directly.
The strength of it depends on how full the container is.
This works with trapped chests as well, meaning it can provide two redstone signals with different causes at the same time.
Hoppers

Rails can now be placed on top of hoppers, which allows emptying chest minecarts.
