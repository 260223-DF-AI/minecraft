== Skipped version numbers ==

Versions labeled "definite version skip" are confirmed to be skipped version numbers as opposed to the possibility of simply being an undocumented and unarchived version of the game.
Bedrock Edition minor version numbers from 1.14 onwards do not increment entirely sequentially; each digit has semantic meaning, so 1.14.1 can be followed by 1.14.10 which can be followed by 1.14.20, with no 'skips' involved.
