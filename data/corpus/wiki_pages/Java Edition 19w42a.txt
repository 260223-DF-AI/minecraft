19w42a is the tenth snapshot for Java Edition 1.15, released on October 16, 2019, which makes slime blocks and honey blocks no longer stick together and fixes many rendering bugs, as well as several other bugs.


== Additions ==


=== General ===
Tags

Added lectern_books item tag.
Contains written book and book and quill.


== Changes ==


=== Blocks ===
Dispensers

Now eject empty glass bottles "properly" (likely not, due to MC-114111) when not facing water.
Fire charges launched from a dispenser now light campfires.
Can now place armor stands.
Honey blocks and slime blocks

No longer stick to each other.


=== Gameplay ===
Elytra

Can now start gliding even rise in immediately after the jump up, not only during the descent after the jump.


=== General ===
Textures

Removed wax block and crystallized honey textures.
