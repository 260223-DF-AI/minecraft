2.2.11698 (Windows), 2.2.11696 (Windows, via the Microsoft Store/Xbox app), 2.2.11699 (macOS), or 2.2.11697 (Linux), is an update to the launcher released on March 22, 2022.


== Additions ==
Added improved timeout to login process to avoid forever spinning during login failure.


== Changes ==
Some in-Launcher FAQ content had been pushed to Crowdin for localization.
Various additional stability and reliability improvements.


== Fixes ==
Fix to bug that left processes running after Launcher close, making Launcher unable to re-launch.
Fix to Dungeons install dialog.
Fixes to offline mode issues, including adding updated error/fail state during login attempt while offline.
MCL-18076 - Snapshots option disabled in patch notes menu.
MCL-20088 - Several mentionings of "Minecraft: Java Edition" within the new launcher strings are spelled inconsistently.
MCL-20196 - Cyrillic "" and "" characters are not supported by the pixelated font.
MCL-20703 - "Patch notes" in Java Edition has a lowercase 'n'.
MCL-21080 - Launcher does not allow quoted paths in java path.
MCL-21094 - Inconsistency in usage of hypen in "Third-party licenses".
