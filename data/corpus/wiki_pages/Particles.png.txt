particles.png and related particle atlases are texture atlases found inside the game's internal resource packs used to load the majority of the game's particle textures, excluding those derived from other textures.


== Usage ==
These texture atlases are used for particle textures. The main file, particles.png, is used for the majority of particles. Some particles with dynamic textures, such as campfire smoke and falling leaves have their own texture atlases. Texture atlases can be found in Content/data/resource_packs/vanilla_version/textures/particle or .../particles, where the version number corresponds to the latest version the texture atlas has been modified. Each particle defines which texture atlas it uses in its JSON file in the same resource pack. Each particle atlas has an associated texture set which defines the MERS values used in Vibrant Visuals.

== Format ==
particles.png may have changed width with time. The file is split into different squares, usually 88 but bigger for some other cases, which correspond to the texture of many particles found in-game. Like with items.png, unused sections of the texture are completely transparent, unlike with terrain.png and the paintings sheet where such regions are colored magenta.
