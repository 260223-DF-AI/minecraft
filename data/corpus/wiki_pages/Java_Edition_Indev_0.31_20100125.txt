A version of Indev 0.31 was released on January 25, 2010, at 21:54 UTC.


== Additions ==


=== General ===
Particles

Added a particle flame.
Added a particle lava.


== Changes ==


=== Blocks ===
Lava

Lava exposed to air becomes stone, similar to how water exposed to air would evaporate.
Now emits particles lava.
Torch

Now emits particles flame.
TNT

No longer damages entities or destroys dropped items.
Fuse is now 4 seconds (raised from 2).
Now flashes when lit.


=== World generation ===
Starting house

Added one more chest, containing a stack of each color of cloth.
Cloth is no longer found in the main blocks chest.


=== Gameplay ===
Damage

Drowning damage has now been reduced from 2HP per half-second to 2HP per second.
Drowning now creates particles around drowning entities.


== Fixes ==
1 bug fixed

Fixed the bug causing item switching to make a block unbreakable and cycle through all block textures on it.


== Bugs ==
1 bug reported

Making sand or gravel fall onto the player's head creates a temporary X-ray effect.
