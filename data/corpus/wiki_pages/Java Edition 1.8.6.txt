1.8.6 is a minor update to Java Edition that was released on May 25, 2015, to fix an issue that caused Creative mode players with modified clients to be able to create signs that could run any command. 1.8.6 is compatible with servers running 1.8 – 1.8.5.


== Changes ==


=== Blocks ===
Dispensers

No longer place command blocks.


== Fixes ==
1 issue fixed
Private issues

MC-75630 – Exploit with signs and command blocks.


== Trivia ==
This was the last 1.x.6 version for nearly nine years until the release of 1.20.6.
