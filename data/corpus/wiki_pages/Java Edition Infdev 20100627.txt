A version of Minecraft Infdev released on June 27, 2010.
This is the final Infdev version to be released onto www.minecraft.net/infdev. It is also retroactively the final Infdev version, as Notch mentioned "the version that was called “infdev” up until yesterday" on June 30, implying Infdev 20100629 to be the start of Alpha.


== Changes ==


=== Blocks ===
Monster spawner

Now have a unique tile entity ID, as they previously shared their IDs with signs which led to various problems.
Water

Flowing water now emits a sound effect.


=== Mobs ===
General

Passive mob spawning drastically reduced.


== Technical ==
Debug screen

Changed Free memory to Used memory.
Now displays MB in parentheses right next to the percentage.


== Fixes ==
3 bugs fixed

Fixed the minecart rotating bug.
Fixed game crashing when signs were nearby.
Fixed game crashing when window's height changed to 0.
However, the game will still crash if the window is initially launched with height 0.
