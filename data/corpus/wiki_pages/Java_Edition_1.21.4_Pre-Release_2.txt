1.21.4 Pre-Release 2 (known as 1.21.4-pre2 in the launcher) is the second pre-release for Java Edition 1.21.4, released on November 25, 2024, which fixes bugs.
This version has a bug where players could take damage when falling from 1 block, this was fixed in Java Edition 1.21.4 Pre-Release 3.


== Changes ==


=== General ===
Data pack

The pack format version is now 61.
Renamed CookTime to cooking_time_spent.
Renamed CookTimeTotal to cooking_total_time.
Renamed BurnTime to lit_time_remaining.
Added the field lit_total_time representing the total time the furnace should be lit.
