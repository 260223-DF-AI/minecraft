1.7.3 (displayed as release 1.7.3 in the launcher)  is the first pre-release for Java Edition 1.7.4, released on December 6, 2013.


== Fixes ==
11 issues fixed
From released versions before 1.7.4

MC-5002 – Some texts cannot be translated in a language file.
MC-11129 – Unicode font text shadow in fullscreen mode.
MC-13046 – Certain supported characters glitch out, clearing out lines of text.
MC-32322 – World loading screen uses wrong font.
MC-33035 – Loading text in Polish (and other languages with Unicode font) isn't showing properly.
MC-34573 – Font file of Unicode page C2 (Korean) shows character '숼'(U+C23C) at the position of '쉴'(U+C274).
MC-41135 – Players become invisible and immune for PvP damage after being killed and relogging instead of respawning.
From the 1.7.4 development versions

MC-40587 – Broadcast server list giving same values.
From the previous development version

MC-41528 – No taskbar workbench icon in Windows.
MC-41537 – Hopper don't suck items from above if all the slots have an item.
MC-41566 – When the zombie chicken jockey is killed, chicken still attacks.


== Trivia ==
This version is mislabeled as a release on the launcher.
This is the last pre-release to take the form of full releases.
This is the last pre-release released in 2013.
This is one of a few development versions that weren't removed from the launcher before September 2018. Other such versions are 1.6.1 development versions.
