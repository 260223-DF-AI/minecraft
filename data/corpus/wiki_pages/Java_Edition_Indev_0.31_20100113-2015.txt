A version of Indev 0.31 was released on January 13, 2010, at 20:15 UTC. It was announced on The Word of Notch at 22:43 UTC.
This version has a Creative mode variant.


== Changes ==


=== Blocks ===
Water

Is now finite.
Always drains from highest remove location.
Water does not move on the surface it is on.
Doesn't spread out over unfilled water surfaces, if it has an empty space next to it.
Evaporates (23 chance) or copies (13 chance).


=== World generation ===
General

Oceans still have infinite water.


== Fixes ==
1 bug fixed
From Indev 20100110[verify for Java Edition]

Dropping an item from a stack greater than 1 now drops 1 of that item again.[verify for Java Edition]


== Bugs ==
2 bugs reported

TNT removes water
Game degrades in performance after world generation


== Creative variant ==
  
A variant of this version of Indev 0.31 was released with Creative mode enabled. There are a few bugs with this version due to some Creative mode code that was broken by the time this version came out; for example, blocks are not infinite.
This was one of the few versions between 0.30 and Beta 1.7.3 to get a Creative release, the only other versions being the first, second and third December 23, 2009, versions of Indev 0.31.
