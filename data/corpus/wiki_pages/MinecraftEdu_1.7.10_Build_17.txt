1.7.10 Build 17 is the seventeenth build for MinecraftEdu 1.7.10, released on April 17, 2015. It is based on Java Edition 1.7.10.


== Changes ==
Removed "built in" WorldEdit. You can now get WorldEdit functionalities by downloading a server side mod from the online mod section of the Launcher.


== Fixes ==
3 bugs fixed

Toggleable stuff like levers not working since forge update in b16.
"Export to zip" not getting all necessary mod files for example when exporting a saved world to zip.
Not being able to hide the Server Launcher button.
