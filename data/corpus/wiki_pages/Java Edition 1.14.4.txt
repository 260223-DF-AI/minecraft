1.14.4 is a minor update to Java Edition released on July 19, 2019, which made minor changes to villagers and server permissions, and fixed bugs not fixed in 1.14.3. It is not compatible with 1.14.3 servers.


== Additions ==


=== Command format ===
/debug

Added /debug report.
Used to get more detailed information while debugging performance, such as lists of chunks and block entities in each dimension.
Saves this information as a .zip file in the .minecraft/debug folder.
server.properties

Added function-permission-level to server.properties which controls which permission levels functions have.


=== General ===
Splashes

Added "Ping the human!"


== Changes ==


=== Items ===
Suspicious stew

Suspicious stews made from poppies now gives the player Night Vision instead of Speed.


=== Mobs ===
Pillagers

Changed arm texture.
Villagers

Now voluntarily pick up items.
Trades can now be completed more times before locking.
Now remember their gossip after becoming a zombie villager.
Gossip about players who converted a zombie villager now lasts longer.
Can now work without also restocking at the same time.
This has no effect other than playing the work sound.
Improved performance of villager pathfinding.


=== Command format ===
Permissions

/reload and /forceload are now available to permission level 2.


=== General ===
Camera

Removed the camera pivot offset when in first-person view.
client.jar

Re-integrated Realms code into the client, in package com.mojang.realmsclient.dto.
Re-added Realms assets to the client.
Performance

Improved general performance.
Improved leaf rendering performance.
Improved chunk loading when traveling at high speeds.
Minecraft Realms

Removed the "Player Activity" button from the Realms screen.
