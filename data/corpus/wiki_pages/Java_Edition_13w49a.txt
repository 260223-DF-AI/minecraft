13w49a is the eighth and final snapshot for Java Edition 1.7.4, released on December 5, 2013. This is the final snapshot released in 2013.


== Additions ==


=== Mobs ===
 Chicken jockeys

Consist of a baby zombie riding a chicken.
The zombie controls the chicken instead of using the chicken AI.
Because the zombie is riding a chicken, it cannot take fall damage.
Extremely rare to find.


== Changes ==


=== General ===
Mojang logo

Changed from 512512 to 256256.


=== Music discs ===
All music discs are now stored in an indexed hash, rather than stored directly in the .minecraft folder.


== Fixes ==
4 issues fixed
From released versions before 1.7.4

MC-31622  Wrong render distance on 9 and more chunks settings.
MC-41165  SetSlot packet can no longer set item under cursor.
MC-41407  Minor optimization to world generation.
MC-41508  Hopper: java.lang.IllegalStateException: Not a JSON Object.


== Trivia ==
13w49a was previously removed from the launcher for some time, but was added back in September 2018.
