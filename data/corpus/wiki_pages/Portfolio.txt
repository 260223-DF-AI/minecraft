A portfolio is an item that can save photos taken with a camera.


== Obtaining ==
Portfolios can be obtained in the Creative inventory or given through the /give command.


== Usage ==

Once a photo has been taken using the camera, it appears in the portfolio. Once equipped, pressing the Use Item control brings up a two-page book of photos the player has taken, in chronological order. Captions can be added below each picture. Each picture can be deleted or saved as a photo item in the player's inventory. The "Add Photo" button allows the player to add pictures from photo items in their inventory, while the "Export Portfolio" button creates a PDF file in a specified folder containing all photos stored in the portfolio.
