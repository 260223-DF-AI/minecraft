1.5.1 Build 16 is the sixteenth and final build for MinecraftEdu 1.5.1, released on July 12, 2013. It is based on Java Edition 1.5.1.


== Additions ==
Students can now use the build tools keybindings.
Moved open current map folder button from setting to world information in server GUI.


== Fixes ==
Fixed TNT ignition with fire arrows when TNT was disabled.
Pistons can't push blocks over border blocks or build disallow blocks.
