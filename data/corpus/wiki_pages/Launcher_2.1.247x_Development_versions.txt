== 2.1.233x-beta ==
2.1.2337-beta (Windows), 2.1.2338-beta (macOS), or 2.1.2339-beta (Linux) is an update to the launcher released on February 6, 2019.


=== Additions ===
Added a beta update switch in the Settings page.
New game log window with improved performance.


=== Fixes ===
Fixed bugs related to links and UNC folder paths.
Fixed missing Bootstrap version information on the version page.
8 issues fixed
MCL-5569  Launcher does not open a window after being opened
MCL-5642  UNC path game directory not supported
MCL-8418  Won't launch minecraft on already existing .minecraft
MCL-10094  Profiles are in wrong order if there are more than ten entries in the list
MCL-10135  Launcher does not sort profiles correctly
MCL-10279  Launcher can't start if the .minecraft folder in appdata is a symlink
MCL-10476  Can't use hard link in custom java executable path
MCL-10479  Beta Launcher breaks on 404 Error for non-Mojang-hosted jars

== 2.1.236x-beta ==
2.1.2366-beta (Windows), 2.1.2365-beta (macOS), or 2.1.2364-beta (Linux) is an update to the launcher released on February 11, 2019.


=== Additions ===
The launcher will now display a notification of new launcher versions, even if they release while the launcher is already running.


=== Fixes ===
Fixed a bug that caused the launcher to soft-lock when there was no launcher_profiles.json file on disk.
MCL-5436: Cannot launch directory junction files.
MCL-10280: Launcher can't save the game directory in profiles.


== 2.1.246x-beta ==
2.1.2467-beta (Windows), 2.1.2468-beta (macOS), or 2.1.2466-beta (Linux) is an update to the launcher released on February 14, 2019.


=== Changes ===
The progress bar now updates properly in the "Prepare" phase.
Speed of the "Prepare" phase has been improved on multi-core machines.


=== Fixes ===
Fixed a bug causing game launches to stall on Linux.
Fixed an error that would pop up when changing game directory in any configuration.
Fixed the launcher update progress bar not working on mac.
Fixed sounds not working in Minecraft versions prior to 1.6.
MCL-6280: No sound in older versions on native launcher, despite working in the Java launcher.
MCL-10486: minecraft-launcher 2.1.2339 hangs trying to load / run any version of minecraft.
