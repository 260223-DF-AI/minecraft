Beta 1.7.3 is a version of Java Edition Beta released on July 8, 2011, to fix some of the bugs and crashes in Beta 1.7.2.


== Fixes ==
9 bugs fixed

Corrected a block duplication bug when using pistons.
Corrected a redstone torch duplication bug when using pistons.
Corrected a client crash when placing a sign in front of the piston, powering the piston and then removing the block beneath the sign.
Ice blocks are now pushed without causing water streams to break everything.
Powered rails are no longer being powered magically without a power source.
Pistons connected to the end of a piston transistor via redstone are now properly closed when the power goes out.
Doors no longer create purple particles.
Modified clients can no longer edit texts of placed signs in multiplayer.
Changed so that paintings pushed by pistons will pop off.


== Trivia ==
This was the last version to:
be released before the game started getting official, public pre-releases
include a dirt background in the title screen
use the unlicensed copy message
have a hotkey for controlling fog
not include Creative mode accessibly
feature the Far Lands, as the bug that caused them was patched on Beta 1.8
