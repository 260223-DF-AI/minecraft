Beta 1.21.90.28 (ChromeOS, Android), or Preview 1.21.90.28 (Windows, iOS, iPadOS, Xbox, PlayStation) is the seventh and final beta/Preview version for Bedrock Edition 1.21.90, released on June 2, 2025, which fixes bugs and makes it so Vibrant Visuals is no longer used by default on some consoles.


== Changes ==


=== Mobs ===
Happy Ghast

Players riding happy ghasts no longer cause nearby happy ghasts to stop.
This prevents players from blocking each other when flying happy ghasts too close together.
Happy ghasts can now be pushed by other mobs even while being ridden, except when made immobile by a player standing on it.


=== Graphical ===
Vibrant Visuals

Turned off Vibrant Visuals by default on some console devices (Xbox One, Xbox One S, PlayStation 4).


=== Technical ===
Entity Filters

Added the is_vehicle_family entity filter, which returns true when the subject entity is riding a vehicle with the specified family.


== Fixes ==
3 issues fixed
From the 1.21.90 development versions

MCPE-220333 – Vibrant Visuals: The screen no longer turns bright white upon leaving the End in Vibrant Visuals
MCPE-220606 – Ghast is not centered vertically on its hitbox
Other

Fixed a crash that could occur when viewing the friends drawer.
