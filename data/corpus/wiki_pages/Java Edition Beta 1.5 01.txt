Beta 1.5_01 is a version of Java Edition Beta released on April 20, 2011, to fix some of the bugs and crashes in Beta 1.5.


== Changes ==


=== Blocks ===
Snow layer

Now included in replaceable blocks category.


=== Mobs ===
Wolf

Spawn rate slightly reduced.


== Fixes ==
8 bugs fixed

Blocks can now be placed on snow.
Rain no longer goes through translucent blocks.
Pumpkins are no longer worn backward on the player. This was also fixed (likely backported) in the PC Gamer Demo.
The game no longer crashes when rails are placed next to detector rails, or vice versa.
Ladders are no longer climbable when spaced every other block.
Wolves are less common than in Beta 1.5, but still more common than in Beta 1.4.
Blocks and items can no longer be placed into the output slot of furnaces, thus fixing the statistic exploit.
The game attempts to recover the world when a broken save file is detected.


== Bugs ==
Minecarts can no longer be extinguished with water.
Taking an item out from the player's inventory crafting grid caused the game to crash.
