1.6.4 Build 14 is the fourteenth build for MinecraftEdu 1.6.4, released on March 20, 2014. It is based on Java Edition 1.6.4.


== Additions ==
Added visual aids to indicate what specific building mode the player is in.
Added an online mod repository.


== Changes ==
The launcher now continues to run in the background when you are in the game by default.
