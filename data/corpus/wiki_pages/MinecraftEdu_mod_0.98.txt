0.98 is a major update to MinecraftEdu, which adds a new block and a new tab for the teacher menu. It is based on Java Edition 1.1.[more information needed]


== Additions ==


=== Blocks ===
 Teleport Block

Teleport blocks Teleport the player to the location that the player set in the world.
Teacher permissions:
Use & set visibility for teleport blocks anytime.
Break teleport blocks anytime.
Use teleport blocks if there is a build allow block below the teleport station.
