13w24a is the thirteenth snapshot for Java Edition 1.6.1, released on June 13, 2013.


== Additions ==


=== Command format ===
doDaylightCycle

Added new gamerule for daylight cycle.
Stops time relative to day changing from incrementing, pausing the sky in place.


== Changes ==


=== Items ===
Spawn eggs

Now work on water.
Skis

Removed the skis texture, which was a red herring added by Dinnerbone in 12w50a.


=== Mobs ===
Spiders

Now have a chance to spawn with potion effects.

=== General ===
Resource packs

The game now uses resource packs instead of texture packs.
The Texture Ender was released to convert 1.5-compatible texture packs into resource packs.
Game directory

Renamed the game directory parameter for the game's main class (net.minecraft.client.main.Main): from --workDir to a more adequate --gameDir.
