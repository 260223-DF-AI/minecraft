1.9.3-pre3 is third and final pre-release for Java Edition 1.9.3, released on May 3, 2016, which removes the server blocking test from the previous pre-release and fixes bugs.


== Changes ==


=== General ===
Networking

The game no longer checks https://sessionserver.mojang.com/blockedservers, as it had begun to in the previous pre-release.


== Fixes ==
2 issues fixed
From the previous development version

MC-101545 – Problem with the new pre-release.
Other fixes:

Fixed a potential crash with spreading water or lava.
