1.6.4 Build 18 is the eighteenth build for MinecraftEdu 1.6.4, released on May 22, 2014. It is based on Java Edition 1.6.4.


== Additions ==


=== General ===
Keep Inventory button to Player Settings tab. When enabled, users don't lose their inventory on death.
Custom screenshot location. You can define your own file path for screenshots in minecraftedusettings.ini
Students can't move their mouse when frozen.
Comments to launchersettings.ini, serverwizardsettings.ini, eduworldsettings.ini and minecraftedusettings.ini
Ability to send one on one messages from the Player Management tab.


== Changes ==
MOTD text area changed to textfield.
Allow Nether & End changed to Allow Other Dimensions.


== Fixes ==
3 bugs fixed

Students were able to place and dig when frozen.
The mod fix in build 17 was not working properly. This time it should be fixed.
Block inspector text was misplaced in some gui scaling situations.
