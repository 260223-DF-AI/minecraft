1.21.01.0 is the fourth and final Preview version for Minecraft Education 1.21.03, released on June 18, 2024.


== Additions ==

Added features from Armored Paws and Tricky Trials.


== Changes ==


=== Items ===
Balloons and glowsticks

Chemistry items like balloons and glowsticks are now available in the Creative inventory.


=== General ===
The "Manage Account" button has been moved to the bottom right of the home screen so all the player's profile management is in one place.


== Fixes ==
Spectator mode has been accidently included as an available game mode.
Some functions while coding in Python Notebooks will result in a Syntax Error: Unexpected 0 error message.
Improvements to performance and product stability.
