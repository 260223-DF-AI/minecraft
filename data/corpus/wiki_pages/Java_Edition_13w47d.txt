13w47d is the fourth snapshot for Java Edition 1.7.4, released on November 22, 2013 to fix some of the bugs and crashes in the 13w47a, 13w47b and 13w47c snapshots.


== Fixes ==
4 issues fixed
From the 1.7.4 development versions

MC-35296  Draw order-related graphical errors involving stained glass.
MC-35561  Players can see a stained glass pane clearer through stained glass.
MC-35694  Stained glass texture different while in Creative-"Survival Inventory".
From the previous development version

MC-40255  Falling invisible slime causes "blue screen".


== Trivia ==
13w47d was previously removed from the launcher for some time, but was added back in September 2018.
