Void may refer to:


== Minecraft ==
Void, empty space external to the buildable area in any dimension.
The Void, an empty biome generating only in buffet world and a superflat preset called "The Void".
Void start platform, the only structure that generates in that superflat preset.
Void world, a world type in Bedrock Editor.
Void fog, a removed world feature that produced thick black fog near the bottom of the world.
Void Air, a Java Edition-exclusive variant of air.
Structure Void, a block used to prevent existing blocks to be replaced by structures loaded by the structure block.
A 2x2 painting by Kristoffer Zetterstrand named "The void".
Bottle of void, a joke item introduced in 23w13a_or_b.
