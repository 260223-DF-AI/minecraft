1.0.1 is a server update for Minecraft (Java Edition) released on November 24, 2011, to fix some bugs and crashes in 1.0.0 servers.
1.0.1 is the final server-only version of Java Edition and the final version released in 2011. As a server-only update, 1.0.0 remained the current client version.


== Fixes ==
2 bugs fixed

Invalid server key error fixed when attempting to log into servers.
Null pointer exception error fixed when logging into servers.
