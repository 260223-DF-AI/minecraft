1.18 Release Candidate 3 (known as 1.18-rc3 in the launcher) is the third release candidate for Java Edition 1.18, released on November 26, 2021, which fixes a bug.


== Fixes ==
1 issue fixed
From the 1.18 development versions

MC-242859  Blocks losing the loot inside them after dying.
