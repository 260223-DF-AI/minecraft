Beta 1.7 is a version of Java Edition Beta released on June 30, 2011, which adds pistons and shears among other changes.


== Additions ==


=== Blocks ===
 Pistons

Can push blocks when powered by redstone.
Crafted with wooden planks, cobblestone, iron ingot and redstone.

 Sticky pistons

Can push and pull blocks when powered by redstone.
Crafted with 1 piston and 1 slimeball.

Stone bricks

The tooltip has been added to the language file (en_US.lang). However, the block itself isn't yet present in the game.


=== Items ===
 Shears

Can be used to collect leaves and obtain wool from a sheep without harming it.
Has 238 uses.
Crafted with 2 iron ingots placed diagonally.


=== Mobs ===
 Silverfish

Texture was added. However, the mob itself isn't yet present in the game.

 TNT

Can no longer be primed by hitting it with an empty hand or while holding any item other than a flint and steel.
Instead, the block is destroyed and dropped as an item.
The old behavior isn't removed completely, and it's hidden behind the data value of 1 which is accessible using map editors.
It is activated under the influence of powered by redstone.


=== Mobs ===
 Sheep

No longer drop 1-3 wool from punching them.
They now drop 2-4 when being sheared.
Now always only drop one wool when killed while not sheared.


=== General ===
Controls

When pressing F3 in-game, a value ("F") from 0 to 3 appears to indicate direction.
Textures

The bed and cake side textures have been shifted downward in the terrain.png file.


== Fixes ==
3 bugs fixed

Fixed clay generation.
Squid can now despawn.
⇧ Shift+ clicking in a container while having a full inventory no longer crashes the game.


== Bugs ==
4 bugs reported

Right-clicking on a sheep causes the game to crash.
Some blocks crash the game when attempted to be pushed or pulled. This also softlocks the world.
Pushing a piston onto the same redstone line that is powering the piston causes a crash. This also softlocks the world.
Powered redstone crashes the game when pulled or pushed by a piston. This also softlocks the world.


== Trailer ==

An official video documenting some of Beta 1.7's major features was released on Mojang's official YouTube channel on June 30th, 2011. The video was made by Hat Films. Made before Java Edition was officially released (and long before 1.7 came out), the trailer omits “Beta” when referring to the version number.
