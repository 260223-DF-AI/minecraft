26.1 Pre-Release 2 (known as 26.1-pre-2 in the launcher) is the second pre-release for Java Edition 26.1, released on March 13, 2026.


== Additions ==


=== General ===
Options

Added a new display option called "Exclusive Fullscreen" which controls if fullscreen mode should take full control of a monitor.
Defaults to false (i.e. non-exclusive mode).
Any change will be applied only after the game restarts.


== Changes ==


=== Items ===
Spear

The dismount speed thresholds have been updated to match Bedrock Edition.


=== Mobs ===
Axolotl

Fixed issue with Baby Axolotl's right hind leg not moving correctly when swimming.
Cat

Baby cats are now scaled up to be the same size as baby ocelots.
Horse

Baby horses are no longer scaled up slightly.


=== Gameplay ===
Trading

The master librarian will now have three available trades to ensure that an enchanted book trade is always offered when using the Villager Trade Rebalance experiment.


=== General ===
Text component format

Components in server status messages (MotD) nested more than 16 times will now be discarded and replaced with an ellipsis.
