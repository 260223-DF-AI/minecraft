A version of Indev 0.31 was released on December 31, 2009, at 18:56 UTC.


== Additions ==


=== Gameplay ===
Inventory

It can be accessed by pressing I.
Player statistics are added as a placeholder in the inventory screen.
Rana is shown in the inventory screen as the player's character as a placeholder.


=== General ===
Controls

Pressing F7 takes an isometric screenshot.


== Changes ==


=== Blocks ===
Brown mushroom

Can no longer be eaten.
Grass blocks

Grass blocks can now exist in slightly darker areas, meaning that there is no more dirt around trees.
Red mushroom

Can no longer be eaten.
  Torches

Texture changed from  .
Changed length from 13 pixels to 10 pixels.
Texture in the inventory changed from  to .


=== Gameplay ===
Hotbar

The player now spawns with 99 stone, TNT, bookshelves and leaves on the hotbar.
Survival mode

The game is now in Survival mode by default.
The player now has 10 air bubbles again instead of 1.[verify for Java Edition]


=== General ===
Lighting

Changed shading of blocks in the hotbar.
Dynamic lighting no longer affects inventory and hotbar items.[verify for Java Edition]


== Removals ==


=== Gameplay ===
Creative inventory

Is no longer accessible, as the key-bind for it has been removed, and there is no way to set a key-bind for it in the options menu. Its class file has also been removed.
No Creative build of this version was released, so this change goes unused, and can only be found via modifying the game.


=== Command format ===
Commands

Removed all commands.


== Bugs ==
1 bug reported

This version had an issue with mouse controlling.
This is caused by having an outdated LWJGL version


== Trivia ==
Until Beta 1.8, all versions, with the exception of Indev 0.31 20100113-2015 (Creative) are in Survival mode only.
Note that most Creative mode functionality was still present in the code even through Beta 1.7.3. (Creative mode was re-added in Beta 1.8 Pre-release, 618 days later). As a result, Creative mode can still be enabled in these versions via mods.
