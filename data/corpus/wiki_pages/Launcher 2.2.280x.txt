2.2.2806 (Windows), 2.2.2804 (macOS), or 2.2.2803 (Linux), is an update to the launcher released on May 25, 2021.


== Changes ==
Fixed build breaks on Unix-like systems.
Fixed an issue with Windows not logging 'proper looking' paths in logs.
Various fixes on MacOS builds to improve reliability.
Updated bootstrapper to remove dependency on vcruntime140.dll.
Added additional logging to catch CDN problems.


== Fixes ==
MCL-17433 - First three columns of launcher log are displayed darker than they were previously.
MCL-17472 - Several search fields in the launcher are not selected when clicking on the search icon.
MCL-17281 - Launcher won't load.


== Trivia ==
The changelog of this version is same with the 2.2.27xx's, only additionally fixed 1 bug.
Changelog of 2.2.27xx disappeared when this version released.
