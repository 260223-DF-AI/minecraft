1.21.114 is a hotfix to Bedrock Edition released on October 16, 2025, which fixes some bugs.


== Fixes ==
8 issues fixed
MCPE-181653  Pink and black square appears at the top of the screen in the Camp Enderwood map
MCPE-217410  "Game is paused" is translated as "Game dijeda" in Indonesian
MCPE-226297  Duplicate Incorrect JSON File for Copper Golem Texture in Resource Pack
MCPE-228420  Input delay / unresponsive controls after 1.21.111 update on mobile devices
MCPE-228432  PlayStation: Game crashes when loading world after version 1.21.111 release
Other

Fixed several crashes that could occur.
Fixed various issues related to player skins and the Character Creator.
Fixed a bug that prevented Summon tool from spawning custom entities in Bedrock Editor.
