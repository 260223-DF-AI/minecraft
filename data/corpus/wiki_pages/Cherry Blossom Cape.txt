== Appearance ==
The cape design resembles the pink leaves of cherry blossom trees.
The elytra features a four-tone gradient pattern from white from the bottom to pink to the top.


== Distribution ==


=== Java Edition ===
In Java Edition, this cape was given to players who voted on the 2023 Mob Vote through minecraft.net, the Minecraft Launcher, or the 2023 Mob Vote Event server.
Joining the server but not voting did not grant the cape to the player in Java Edition.


=== Bedrock Edition ===
In Bedrock Edition, this cape was given to players who joined the 2023 Mob Vote Event server.
Voting but not joining the server did not grant the cape to the player in Bedrock Edition.
