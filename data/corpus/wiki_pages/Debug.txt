== Minecraft ==
Debug Stick, item used to change various block states.
Debug mode, world type used to test block states.
Debug mode grid, all blocks in the debug mode, see the navbox.
F3 Debug screen, screen shown when F3 is pressed.
Debug hotkey, hotkey involving the F3 key that shows hidden game information and settings.
Debug renderer, graphical tool showing technical information about the game.
/debug[JE  only], command used to start and stop debug sessions.
Debug property, flags that enable debugging features.
debug.png and debug2.png, debugging assets found in the game's texture files.
 debug_fourj_item, an item once used in Legacy Console Edition for unknown debugging purposes.
