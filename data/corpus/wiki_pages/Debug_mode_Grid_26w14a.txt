This page lists all the rows in the debug mode block grid in 26w14a. 
There are 173 lines with up to 174 block states. The last line has 126 block states. Of these 172, those that appear at the end of a line are repeated at the beginning of the next. Therefore, there are 30,054 block states in total, with 29,882 unique block states.
The first block is located at coordinates (1, 70, 1) and each block is 2 blocks away from its adjacent blocks, thus the block in row r, column c is at coordinates (2r-1, 70, 2c-1).


== New blocks ==
Nine blocks were added, all at the end of the version 26.1.1 grid, and each has only one block state.
They are all in row 173, between columns 118 and 126.

Iron Trap
Copper Trap
Exposed Copper Trap
Weathered Copper Trap
Oxidized Copper Trap
Waxed Copper Trap
Waxed Exposed Copper Trap
Waxed Weathered Copper Trap
Waxed Oxidized Copper Trap
