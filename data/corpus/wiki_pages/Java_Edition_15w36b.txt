15w36b is the twentieth snapshot for Java Edition 1.9, released on September 2, 2015 to fix some bugs and crashes from 15w36a.


== Changes ==


=== Items ===
Snowballs and eggs

Now produce particles when thrown at an entity.


== Fixes ==
5 issues fixed
From released versions before 1.9

MC-669  Failing to place blocks in creative mode causes an animation.
MC-43601  Hunger is lost when standing in close proximity to mobs/stat.walkOneCm increases.
MC-67573  Snowballs and eggs do not produce particles when hitting entities.
From the 1.9 development versions

MC-87128  Minecraft does not create world icon if it is missing.
From the previous development version

MC-87869  Crash after stepping off pressure plate.
