1.21.93 is a hotfix for Bedrock Edition released on July 3, 2025, which fixes some bugs, and adds the "Lava Chicken" music disc.


== Additions ==


=== Items ===
Music Disc

 Added the "Lava Chicken" music disc by Hyper Potions.
Obtained by killing a zombie while it is riding a chicken (a chicken jockey).
If the chicken is killed before the zombie is, the music disc is not dropped.
Has a comparator output of 9 when played in a jukebox.


== Changes ==


=== Technical ===
API

Added a new condition passenger_of_entity that allows checking which vehicle entity type is being ridden by the damaged entity that is dropping loot.
Like damaged_by_entity, allows specifying an entity_type field to check against when deciding which entry in a loot table to select.
Added a new condition is_baby that allows checking if the entity that is dropping loot is a baby.


== Fixes ==
8 issues fixed
MCPE-220353 – Player can no longer move the camera while using the attack, crouch, jump, and eat buttons on touch devices.
MCPE-222020 – Vibrant Visuals: Auto exposure does not react to handheld emissive items, making them uncomfortably bright.
MCPE-222048 – Joining a world in splitscreen causes the secondary player to use a guest account.
MCPE-222147 – Happy ghast causes extreme lag.
MCPE-222298 – Boats lose forward momentum abruptly when falling down.
MCPE-222906 – Marketplace world resource/behavior packs disappear permanently after closing/re-opening game.
Other

Fixed parent_setup not running with the correct context, which caused animation issues in some packs.
Glowing text on signs no longer gets overly bright when looked at directly in Vibrant Visuals.
