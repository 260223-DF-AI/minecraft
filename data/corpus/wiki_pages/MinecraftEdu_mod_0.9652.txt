0.9652 is the first major update to MinecraftEdu. It is based on Java Edition Beta 1.8.1, which adds new blocks, and the teacher menu.


== Additions ==


=== Blocks ===
 Information Sign

Decorative block that can be used to inform nearby students about something.
 Build allow block

Allows students to build.
It can be used to set up areas where students can build, even if student building is disabled in the world.
 Build disallow block

Disallows students to build.
It can be used to set up areas where students cannot build, even if student building is enabled in the world.
 Border block

Used to setup borders that students cannot cross. Borders can be created using build tools.
 Information block

Stores messages inside it. It is used to show information, messages, and links.
Text can be stored or edited in this block by students if a build disallow block is directly underneath it.
Teachers and students can read the text by right clicking this block.
Can be broken by students.


== Changes ==


=== General ===

Teacher menu

The teacher menu is the in game control panel for teachers that allows the teacher to manage the world as well as the players in it.
The player can use the teacher menu when they are logged in to the server as a teacher. Only teachers can use it.
The teacher menu has had a redesign, now using green buttons for the tabs.
Teacher menu now has five tabs: (The previous 0.6 version only had 3 tabs)
Personal Settings (User settings at the time)
Changing these settings will only affect the teacher: Change their movement speed, turn their creative mode on/off, and teleport themselves to spawn/surface.
World Settings
Settings that alter the world including: Change the gamemode and difficulty for everyone, enable/disable weather effects; day/night cycle; disallow fire and dynamite; and enable/disable animals, monsters, and villagers.
Player Settings
Manage settings that affect all students such as: Freeze/mute all students and allow/disallow student building.
Player Management
Manage individual users: Teleport a student to the teacher/the teacher to a student, freeze/mute individual students, change the gamemode of a single student, and give students the ability to use build tools.
Assignments
Add assignments to a list that is visible to the students. The teacher can track who has completed the assignments and what the completion ratio is. Students will be able to view the assignments by pressing M
Give
Give items and blocks to all players or a specific player.
