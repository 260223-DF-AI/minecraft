1.7.10-pre1 is the first pre-release for Java Edition 1.7.10, released on June 3, 2014, containing Realms changes.


== Changes ==


=== General ===
Minecraft Realms

All Realms-related modules were moved into a separate library.
Future additional features to Realms will not require new releases of Minecraft.
New features
Player activity monitor.
Access to more server settings.
PVP: On/Off.
Difficulty: Peaceful/Easy/Normal/Hard.
Spawn protection: Off/1..16.
Gamemode: Creative/Adventure/Survival.
Spawn animals: On/Off.
Spawn monsters: On/Off.
Spawn NPCs: On/Off.
Improvements in Realms UI.


== Fixes ==
1 issue fixed
MC-55580  Minecraft Realms: User will still have Operator rights after re-inviting them.


== Trivia ==
This version was previously removed from the launcher for some time, but was added back in September 2018.
This pre-release released on the same day as 1.14.3 Pre-Release 1, but five years earlier.
