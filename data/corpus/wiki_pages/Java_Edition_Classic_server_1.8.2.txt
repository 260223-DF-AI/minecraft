1.8.2 is a server version of Java Edition Classic released on June 20, 2009, at 21:09 UTC.
It is compatible with all versions between 0.0.20a_01 and 0.0.23a_01. Versions 0.24 through 0.27 can connect, but they do not function due to the lack of Survival mode support on the server at this time, as well as the addition of new blocks.


== Fixes ==
Fixed players being able to break unbreakable blocks.
