15w49b is the forty-fifth snapshot for Java Edition 1.9, released on December 3, 2015 to fix some bugs and crashes from 15w49a.


== Additions ==


=== Command format ===
/time query 

Now accepts the value day, which returns the in-game day number.
Sound events

entity.leashknot.break


== Changes ==


=== Blocks ===
Cobwebs

Removed ability to catch thrown projectiles (added in 15w49a).
End portal block

Hitbox height changed to 0.75 (was 0.0625).


=== Items ===
Melons

Melons and glistering melons are once again oriented in opposite directions (they were facing the same direction in 15w49a).


=== Non-mob entities ===
Bottles o' enchanting, eggs, ender pearls, snowballs, thrown potions

Removed ability to be caught by cobwebs (added in 15w49a).
Again collides with non-opaque blocks.


=== Command format ===
Sounds events

Removed entity.hanging.place and entity.hanging.pop.
