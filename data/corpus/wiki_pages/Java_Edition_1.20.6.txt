1.20.6 is a minor hotfix update to Java Edition released on April 29, 2024, which fixes a data loss bug involving trader llamas. It is compatible with 1.20.5 servers.


== Fixes ==
1 issue fixed
From released versions before 1.20.6

MC-271109  Trader llama inventory shifted, partially lost during upgrade.
