Beta 1.6 Test Build 3 is a version used to test Java Edition Beta 1.6 the day before its release. It was never officially released to the public; however, both the client and the server are packed in an archive that can be found on the legacy update server. This is the only archived Test Build for Java Edition Beta.


== Additions ==


=== Blocks ===
 Dead bush

Found in deserts.
 Shrub

Can only be obtained by inventory editing and uses the grass texture in the inventory.
 Grass

Found in certain biomes.
Seeds are now found in grass; using a hoe on the ground no longer yields seeds.
 Fern

Has the same characteristics as grass.
 Trapdoor

Acts as a ground door.


=== Items ===
 Maps

Lets the player scope out different biomes.


=== General ===
Splashes

Added "It's groundbreaking!"
Added "Let our battle's begin!"
Added "The sky is the limit!"


=== Command format ===
Server.properties

Added new settings:
view-distance.
allow-nether.


=== Gameplay ===
Multiplayer

Added Nether support.


== Changes ==


=== Blocks ===
General

Blocks destroyed by other players in multiplayer now show the breaking block particle effect.
It is no longer possible to build solid blocks on the top layer of the map.
Blocks that do not change appearance when the data changes do not send block updates when their data changes now.
Beds

Trying to sleep in the Nether now causes the bed to explode.
Dispensers

Now make sounds and particles in multiplayer when activated.
Doors

Now make sound for other players in multiplayer.
Fire

Has been severely nerfed so it spreads slower and does not spread infinitely.
Ice

Now regenerates regardless of whether it is snowing or not
Mushrooms

Now spread, albeit slowly.
Powered rails

Now speedier to offset the loss of glitch-boosters.
Stairs

The inventory icon for stairs has been reversed to show a more recognizable figure.
Water

Any water below a layer of source blocks no longer behaves like source blocks.


=== Items ===
Arrows

When shot by one player, can now be picked up by other players.


=== Non-mob entities ===
Boats and minecarts

Players are now dismounted when they ride a boat or minecart through a Nether portal.


=== Gameplay ===
Lighting

The lighting on block icons in the inventory has been changed to appear brighter.
Players

When stuck in walls, slide towards the nearest gap if there is one.


=== General ===
Controls

 Shift+clicking while crafting makes as much as possible and moves it straight to the player's inventory.
 Shift+clicking something in the player's inventory pushes it directly in the player's hotbar and vice-versa.
Network traffic

Compressed network traffic more aggressively.
Texture packs

Added support for misc\grasscolor.png and misc\foliagecolor.png images.

== Trivia ==

Advanced OpenGL is enabled by default in this version. In Beta 1.6 Advanced OpenGL was removed temporarily until Beta 1.6.5 due to bugs.
Advanced OpenGL crashes Minecraft on some computers.
Picking up multiple stacks of items takes a few ticks to finish, unlike the final version where it only takes one.
The fix for the crash when riding a vehicle or animal through a nether portal is done differently.[verify for Java Edition]
Session requests are performed over HTTP instead of HTTPS.
Session requests fail on any code other than HTTP 200, instead of only with HTTP 400.
Connection to the server is closed differently; no thread is used in this build.
GL state when rendering items is not properly reset, which may lead to graphical artifacts.[verify for Java Edition]
