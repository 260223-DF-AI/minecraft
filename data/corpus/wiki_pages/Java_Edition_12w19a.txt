12w19a is the fifth snapshot for Java Edition 1.3.1, released on May 10, 2012.


== Additions ==


=== Blocks ===
 Cocoa

Added cocoa pods to jungles.
Generate on the side of jungle wood.
Have 3 stages of growth, and the number of cocoa beans dropped depends on stage of growth.
Sandstone and  smooth stone variants

Now have their proper names (e.g. "Smooth Sandstone").


=== World generation ===
World type

Added "Large Biomes" type.


== Changes ==


=== Items ===
 Cocoa beans

Changed texture.
 Cookies

Now recover 2 points of hunger rather than 1.


=== General ===
 Bonus chest

Torches now spawn around bonus chests if each side has a block at the same height around it.
Server jar

The server jar now accepts the command-line argument --bonusChest.


=== World generation ===
Biomes

Increased the scale and depth of some biomes:


== Fixes ==
6 bugs fixed

Enchanting bows and swords work again (Other tools now have tooltips, etc. once more).
The End and the Nether generate normally again.
Item repair works again.
Starting a game in Creative no longer starts in Survival mode.
Bonus chests work correctly again.
Particles no longer activate wooden pressure plate with an arrow on it.
Door opening and closing sound effects work correctly again.


== Trivia ==
In this version, when loading a singleplayer world, one needs to put minecraft.zip/server/minecraft_server.jar into .minecraft/server manually to continue loading.
