Alpha v1.0.17_04 is a version of Java Edition Alpha released on August 23, 2010, to fix a bug relating to the player getting stuck. This version is available in the launcher.


== Fixes ==
1 bug fixed

Fixed some instances of the player getting stuck.[more information needed]


== Trivia ==
The original Brocraft Herobrine livestream took place in this version. The world seed in which the Herobrine stream took place is 3609313613745973624.
