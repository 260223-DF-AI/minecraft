1.20.4 is a minor hotfix update to Java Edition, released on December 7, 2023, which fixes a critical bug. It is compatible with 1.20.3 servers.
This is the last release of Java Edition to run on 32-bit operating systems and Java from 17 to 20.
This is the final version released in 2023.


== Fixes ==
1 issue fixed
From 1.20.3

MC-267185 – Decorated pots can delete items when reloading the world.
