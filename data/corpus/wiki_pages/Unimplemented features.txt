Unimplemented features are features which are not naturally obtainable in the game. They may refer to:

Unused features, which are still in the game but can't be obtained without using commands or third-party tools.
Mentioned features, which have been mentioned by Minecraft's developers but their implementation has not been confirmed yet.
Removed features, which were once in the game but since have been removed.
Scrapped features, which were officially considered but never implemented in a build.
Shelved features, where were originally going to be part of a released update but were removed to be possibly added in a later update.
