The second version of 0.25 SURVIVAL TEST is a version of Survival Test released on September 3, 2009, at 16:15 UTC.


== Changes ==


=== Non-mob entities ===
Arrows

Now deal 7HP damage instead of 3HP.


== Trivia ==
This version was released 2 minutes after 0.25 SURVIVAL TEST, making that version the shortest lived Minecraft version across all editions.
