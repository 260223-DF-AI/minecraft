v0.9.0 alpha build 1 is the first build version released for v0.9.0 alpha, and the first build released in 2014.


== Additions ==


=== General ===
Added immersive mode for Android.
Only Android 4.4 KitKat.
Available on other versions with 3rd party-apps.


=== Gameplay ===
Superflat world type.
Updated create world screen - now includes Superflat, Infinite and 'old' world types.
Unlimited worlds.
New falling mechanics for sand and gravel.
New insta ticking of water and lava.
Grass and foliage colors for different biomes.
Water color for swamps.
Added more spawn eggs in creative.
New interact button (to make the game easier for new players).

== Changes ==


=== General ===
The version watermark visible in development versions has been moved from the bottom of the screen to the top.
Prettier clouds
New culling of clouds.
The fog is darker underground.
Textures
New leaves rendering.
Inventory/GUI
New crafting screen.
Stonecutter UI Changed.


=== World generation ===
Removed generated structures:
Oceans
Added as a biome instead
Gravel beaches
Frozen Oceans
Will no longer naturally generate


=== Biome ===
Removed biomes:
Shrubland
Seasonal forest
Rainforest
Tundra
Legacy frozen ocean


=== Blocks and items ===
Correct name for stairs, slabs and spawn eggs.
Beds no longer give full health in normal difficulty.
Smooth lighting added for:
Beds
Fences
Walls
Cobweb
Chests
Leaves
Smooth lighting removed for:
Glowstone
Glass
Jack 'o' lanterns
Improved performance of TNT.


=== Mobs ===
"New" mob spawning.
Passive mobs now spawn naturally in creative mode without the use of spawn eggs.
Tripod cameras have been removed.


=== AI ===
Spiders.
Now spawn from leaves, creating a higher amount of spiders in tree-rich biomes at night.
Animation is now twice as fast.
Improved AI.


=== Others ===
New culling algorithm.
Tile entities are saved differently.
New leveldb based saving format.
New entity handling.
Updated light generation.
Remade network code (mostly the same packages but new system).
