0.1.2 is a server version of Java Edition Alpha released along with client version v1.0.16_01 on August 12, 2010.
0.1.2 was reuploaded on the same day to add server support for the /save-on command.


== Additions ==
Added support for new commands:
/tp <player1> <player2> moves player1 to player2's location.
/save-all forces a full save to disk.
/save-off disables all saving (except save-all).
/save-on re-enables saving.


== Reupload additions ==
Added support for /save-on.
