rd-20090515 is a pre-Classic version made on May 15, 2009, around 22:25 UTC, which adds new blocks, block picking, support for fullscreen, introduces hills and more. This version was never released. While there is a version mislabeled as "rd-20090515" in the launcher, loading it will just run rd-161348, which is after rd-20090515 and the last version of pre-Classic.
This version is the first to be made under the new name of the game, Minecraft: Order of the Stone.


== Additions ==


=== Blocks ===
 Cobblestone

Recycles the previous texture for stone, although slightly modified.
 Dirt

New block.
Generated under grass blocks.
Transforms to grass when is next to a grass block.
 Planks

A new wooden block.


=== Gameplay ===
Controls

Crosshair (+).
In this version, it was very thin.
Block picking using the number keys from 1 to 4.
The currently selected building block now displays in the top-right corner of the screen.
Fullscreen mode.
Pressing G will spawn mobs.


=== General ===
Support for different block types and dynamic blocks.
A subtle layer of fog can now be seen in the distance of lit block surfaces.


== Changes ==


=== Blocks ===
  Grass block

Texture changed from .
Grass blocks revert to dirt over time when they have no light or when a block is on top of it.
Removed the ability to place grass blocks.
 Stone

Texture changed from .


=== Mobs ===
  Mob

Reduced the number of mobs who spawn from 100 to 10.
Mobs who fall below level Y = -100 are now removed from the world.
Mobs now appear darker when in shade.


=== Non-mob entities ===
Player

Changed player's physics.
Now they can jump farther, walk and fall faster.
Moving is much slower when in midair.


=== World generation ===
General

Bumps may now appear, making hills and sometimes cliffs.


=== General ===
Light

Shadows have been brightened.


== Technical ==


==== Level generation ====
Unless a level.dat from an older version is launched, the world is no longer flat.
Loading a world from older pre-Classic versions causes all blocks to become stone.


== Trivia ==
This version is the version shown in the "Minecraft progress." video.
The "rd" before the version number stands for RubyDung, a game Notch was working on before Minecraft, whose codebase was later reused for Minecraft.
A duplicate of rd-161348 in the launcher is labeled "rd-20090515", however, rd-20090515 is not on the Minecraft download servers, and attempting to download it downloads rd-161348 instead, a later version.
