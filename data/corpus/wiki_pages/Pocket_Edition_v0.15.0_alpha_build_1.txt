v0.15.0 alpha build 1 is the first build version released for v0.15.0, also known as the Friendly Update.


== Additions ==


=== General ===
Achievements for iOS and Android
Version exclusive: More achievements.
Is accessed from the home screen
Online multiplayer with Xbox Live friends
New UI for the main menu, shared with the Windows 10 Edition
New UI for the skins menu, it has a skin selection carousel that is similar to Console Edition


=== Gameplay ===
Pig riding
Version exclusive: New sound when accelerates.
Interact buttons
Boost, used when riding a pig with a carrot on a stick.
Name, used for name mobs using name tags.


=== Blocks ===
Pistons and sticky pistons
Version exclusive: New animation; piston arm is visibly thicker at the bottom.
Version exclusive: Can push block entities such as chests.
Version exclusive: Blocks that stick to walls (such as levers) can now be placed on pistons or sticky pistons.
Observer
BUD block
