Minecraft Preview is an application that can be used by players to test out beta features from Bedrock Edition. It functions as a separate application, instead of the previous system of opt-in beta content inside of the Bedrock Edition game itself. 
As of 2025, Minecraft Preview is available for Windows, Xbox and PlayStation, as well as limited users on iOS and iPadOS.


== Development ==
On January 31, 2022, it was announced that the beta program would be expanded to other platforms, starting with iOS and iPadOS, in the form of Minecraft Preview app, separate from the normal Bedrock Edition app. The initial iOS and iPadOS launch was only limited to 500 users on TestFlight, which was later expanded by adding 500 more later on.
Later on February 7, 2022, Minecraft Preview was released for Xbox players via Xbox Insider Hub first, to test the migration of player worlds to the new app. The testing period was divided into two phases, for which users who don't own Minecraft can participate in, before it is officially released only to those who have bought Minecraft on Xbox.
After the Minecraft Preview releases on iOS, iPadOS, and Xbox, the app released on Windows on February 9, 2022 and on PlayStation on February 15, 2024. Only players who own and bought the game can download and play Minecraft Preview.


== Differences from Bedrock Edition ==
Minecraft Preview has some different features and functionalities from Bedrock Edition that are not planned to be implemented in the base game in the next update:

The top of the screen always shows some debug information, similar to the information on the first few rows of the debug screen in Java Edition. The first row shows the version, build, specifications about RenderDragon, and some device specifications. The second row shows information related to performance, such as FPS, memory usage, but also the screen resolution and GUI scaling, and the third row shows some more performance specifications. When playing on Realms, the Realm ID is also shown here. It can be disabled in options.txt.
A render of the currently selected skin is shown in each player's profile screen.
Minecraft Preview is not compatible with regular Bedrock Edition servers, but uses a separate server version. This also applies to Realms.
Players owning a Realms subscription on Bedrock Edtion have access to a separate Realms server on Minecraft Preview.
Achievements or trophies are not available on Minecraft Preview.
Several menu screens get updated with an Ore UI design preview. As of 26.20.22, these include the Realms Hub, Realms edit world screen, world upload to Realms, and Editor main menu. Ore UI versions of these menu screens get implemented in Bedrock Edition in a later update, but not always the update where the Preview version is made for.
The "Render Dragon Features for Creators" experiment is only available in Minecraft Preview.
This includes point light features and vanilla light colors for Vibrant Visuals.
Texture streaming video setting is currently exclusive to Preview.
Realms and Marketplace Pass subscriptions can't be managed from Minecraft Preview.
Exclusive loading tips are shown in the loading world screen.
The only splash that can be seen in game is "Preview!!!".
An exclusive panorama is shown when Vibrant Visuals is disabled.
Closed captions settings and parties are currently exclusive to Minecraft Preview.
All other features from Bedrock Edition are still available, plus features that will be included in the update where the Preview version is made for.


== Installation ==


=== Windows ===
Minecraft Preview is available on the Microsoft Store on Windows if the user owns Bedrock Edition on Windows or has an active subscription to Xbox Game Pass Premium, Xbox Game Pass Ultimate, or PC Game Pass on their account.


=== iOS & iPadOS ===
Installing Minecraft Preview on iOS and iPadOS requires the user to be enrolled on TestFlight, a beta enrollment application. The application is currently available to a limited amount of users. Spots are freed on the first day of each month.


=== Xbox ===
Minecraft Preview is available on the Microsoft Store on Xbox if the user owns a digital copy of Bedrock Edition on Xbox or has an active subscription to Xbox Game Pass Premium or Xbox Game Pass Ultimate on their account. Minecraft Preview is not available for users who own a physical disc copy of Bedrock Edition for Xbox.


=== PlayStation ===
Minecraft Preview is available on the PlayStation Store on PlayStation 4 and PlayStation 5 if the user owns Bedrock Edition on PlayStation.


=== Other platforms ===
It is mentioned in the FAQ that Minecraft Preview is planned to be released on other platforms, with it being stated that the FAQ will be updated once the app is released on said platforms.
