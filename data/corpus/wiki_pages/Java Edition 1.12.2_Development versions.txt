This is a list of Java Edition 1.12.2 development versions.


== 1.12.2-pre1 ==
1.12.2-pre1 is the first pre-release for Java Edition 1.12.2, released on September 13, 2017, which includes minor optimizations and a new game logo.


=== Changes ===
Title screen

New game logo with Minecraft Java Edition


=== Fixes ===
6 issues fixed
From released versions before Java Edition 1.12.2

MC-15152 – Standing in a nether portal and opening a chest will render the chest open until another chest is opened
MC-79154 – Hoppers occasionally duplicate items (within chunks)
MC-86850 – Ender Pearling through a nether portal in the over-world tp's you to the portals nether coordinates but in the overworld
MC-91803 – Cave / ambiance  [sic] music can play far away from the player and is not audible
MC-120444 – Almost any entity inside farmland gets teleported when it turns into dirt
Private issues

MC-120567 – Private security issue


== 1.12.2-pre2 ==
1.12.2-pre2 is the second and final pre-release for Java Edition 1.12.2, released on September 15, 2017, which fixed some bugs. This is the final pre-release released in 2017.


=== Fixes ===
4 issues fixed
From released versions before Java Edition 1.12.2

MC-104259 – Player / mob stuck on farmland while it changes to dirt
MC-118976 – Landing on farmland gets the player stuck in flight mode
MC-120611 – Player stuck in grass path when it changes to dirt
From the previous development version

MC-120645 – Portals kick players for "Timed out" upon use in multiplayer
