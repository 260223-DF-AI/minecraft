End Portal may refer to:


== Minecraft ==


=== Blocks ===
End Portal (block), the block that is created when an end portal is activated
End Portal Frame, the blocks that make up the frame of end portals
End Gateway (block), the block that makes up end gateway portals


=== Structures ===
End portal, a structure found in strongholds that can be used to travel to the End
Exit portal, a structure that returns players back to the Overworld from the End
End Gateway, the portal created after defeating the ender dragon
