This is a list of Pocket Edition v0.11.0 alpha development versions.


== v0.11.0 alpha build 1 ==
v0.11.0 alpha build 1 is the first build version released for v0.11.0, and the first build released in 2015.


=== Additions ===


==== General ====
Skins
Death messages
Localization
Text formatting codes
Show server status and message of the day on server entries screen


==== Gameplay ====
New World edit screen.
Added mob effects:
 Poison
Can be obtained by eating rotten flesh and pufferfish, or being attacked by a cave spider on normal and hard difficulty.
 Nausea.
Can only be obtained by eating Pufferfish.
 Hunger
Can only be obtained by eating Raw Chicken.
Does not affect the player yet.
Fishing.
Change gamemode.
Lock/unlock day cycle.
Rename worlds.
Version exclusive: Make old worlds infinite.


==== Blocks ====
Redstone Block
Currently its only purpose is to store redstone dust.
Grass Path block
Grass path blocks that are lower than normal grass blocks (15/16th of a block).
To create, tap a grass block with a shovel.
Replaces the gravel paths in villages.


==== Items ====
Boats
Version exclusive: Emits smoke when destroyed.
Version exclusive: Crafting recipe is 5 wood planks and a wooden shovel.
Larger in size.
Color is dependent on the planks used.
Sprite color is always the same; it doesn't change based on actual boat color.
When in a boat, two buttons for steering appear. The right button steers to the left, the left button steers to the right. Pressing both buttons moves the boat forward.
Are more resistant to damage - when colliding with another block, the boat will not break.
Can support two riders, including passive and hostile mobs.
Passive and hostile mobs can ride in a boat without the player.
Fishing Rod
Rotten Flesh
Fish
Raw Cod
Cooked Cod
Raw Salmon
Cooked Salmon
Pufferfish
Tropical Fish

Version exclusive: Spawn only in ocean biomes.

Cave Spider
Version exclusive: Cave spider jockeys
Bat
Baby Zombie
Baby zombies have a 5% chance of spawning as a chicken jockey. (In turn, zombies have a 0.2% chance of being a chicken jockey)
Magma Cube
Drop magma cream
Can only be spawned from spawn eggs.
Ghast
Only available through spawn eggs.
Spawn egg can only be received through inventory editing.
