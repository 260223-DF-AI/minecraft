2.4.36 is an update to the launcher released on April 17, 2023.


== Additions ==
Minecraft Legends is now available in the launcher.


== Changes ==
Improved the process for launcher updates.


== Fixes ==
MCL-22245 - "Learn more" buttons within "Where is the play button!?" messages do not display their full content.
MCL-23013 - "Play" button for Minecraft Dungeons is not consistently aligned in the launcher.
MCL-23130 - Minecraft Error message strings displayed within the Launcher are untranslatable.
MCL-23171 - Minecraft Launcher string used for the GamePassUpsell is missing a colon for the respective game names.
MCL-23172 - Minecraft Launcher string used for the GamePassUpsell is missing a variable.
MCL-23176 - The string "Buy now!" within the upsell section for Minecraft Dungeons and Java Edition is untranslatable.
