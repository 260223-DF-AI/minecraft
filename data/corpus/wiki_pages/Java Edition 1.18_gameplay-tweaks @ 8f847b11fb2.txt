1.18/gameplay-tweaks @ 8f847b11fb2 is an unreleased developer version of Java Edition 1.18 experimental snapshot 6, recorded in the morning of September 1, 2021, and uploaded to Henrik Kniberg's YouTube channel two days later on September 3.


== Changes ==
This version appears to contain all of the changes introduced in experimental snapshot 6.


== Trivia ==
The menu screen says that this version is modified.
At 2:59 in the video, the debug screen reads "MOJANGSTAAAAAA" at the top alongside the version number.
