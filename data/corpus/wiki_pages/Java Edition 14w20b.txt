14w20b is the twenty-fourth snapshot for Java Edition 1.8, released on May 15, 2014, to fix some of the bugs and crashes in 14w20a.


== Fixes ==
3 issues fixed
From released versions before 1.8

MC-51758 – Some users have multiple UUID records for agent:minecraft.
From the previous development version

MC-55204 – Mob entities appear to spawn then despawn rapidly.
MC-55220 – Scoreboard objectives no longer show in list.


== Trivia ==
14w20b was previously removed from the launcher for some time, but was added back in September 2018.
