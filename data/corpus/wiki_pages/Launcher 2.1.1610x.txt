2.1.16101 (Windows and Dungeons), 2.1.16100 (macOS), or 2.1.16102 (Linux) is an update to the launcher released on July 22, 2020.


== Fixes ==
Fix Dungeons install location validation when determining if chosen location is under the Launcher installation path.
MCL-1440: Ignore a few system environment variables that could potentially restrict Minecraft resource utilization.
