13w41b is the eleventh snapshot for Java Edition 1.7.2, released on October 11, 2013 to fix some of the bugs in the 13w41a snapshot.
13w41b was reuploaded later in that day, to fix crashes related to the server icon.

== Trivia ==
13w41a has the same protocol version as 13w41b, but they are not compatible with each other.
13w41b was previously removed from the launcher for some time, but was added back in September 2018.
