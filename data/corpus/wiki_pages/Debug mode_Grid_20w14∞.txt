This page will list all the rows in the debug mode block grid in 20w14∞. 
There are 126 lines with up to 127 block states. The last line has 22 block states. Therefore, there are 15.897 block states in total.
The first block is located at coordinates (1, 70, 1) and each block is 2 blocks away from its adjacent blocks, thus the block in row r, column c is at coordinates (2r-1, 70, 2c-1).


== New grid order ==
In 20w14∞, the block grid has been completely shuffled, with all blocks (after the first) having an apparently random order. Unlike the standard grid where similar blocks are close together and there are rarely changes in order between versions (an exception was the pumpkin). Probably the randomness is intentional, as are shuffled the creative inventory and the randomness of dimension generation.
Shuffled only affects the order of blocks by ID; all states within the same block remain together, and the order of the states remains the same.
The order of the blocks does not depend on the seed or dimension, being the same in different seeds and different dimensions. The new dimensions continue to generate the block grid and barrier platform and do not generate most other features. Although it can generate certain entities (such as ender crystals) and liquids. Light effects from the new dimensions can affect the rendering of the blocks. In some dimensions, blocks are rendered only when interacted with, or after interacting with, a nearby block.
