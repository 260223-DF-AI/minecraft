0.24_SURVIVAL_TEST_03 is a version of Java Edition Classic released on September 1, 2009, at 17:55 UTC. This version is not available in the launcher, but has been archived elsewhere.


== Fixes ==
3 bugs fixed

Liquids no longer are dropped as items when blown up by creepers.
Players can no longer copy blocks into their hotbar.
Fixed most other bugs from 0.24 SURVIVAL TEST.​[more information needed]


== Bugs ==
2 bugs reported

Hitting a mob near a sign causes the text to turn white.
Placing a sapling on sand makes the sapling disappear.


== Trivia ==
This is the oldest archived version of the Survival Test.
