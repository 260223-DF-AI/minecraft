2.6.10 is an update to the launcher released on July 21, 2023.


== Changes ==
Added Realms tabs to Java & Bedrock product pages.
Updates to Gamertag spacing.
Updated "Bedrock Preview" logo.


== Fixes ==
5 issues fixed
MCL-23380 – Launcher patch notes spell macOS incorrectly.
MCL-23526 – The Minecraft for Windows game selection drop-down menu is untranslatable.
Other
Various accessibility and UI fixes, improvements & updates.
Bug fix to partially address "failed to create profile" bug; if you're running into this, ensure you're logged in to the Launcher and the Microsoft Store with the same account.
Fix to "Third-party licenses" page in Launcher.
