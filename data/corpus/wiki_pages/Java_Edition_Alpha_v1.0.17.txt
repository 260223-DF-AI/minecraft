Alpha v1.0.17, also known as Seecret Friday 8, is the eighth Seecret Update, released on August 20, 2010, at 13:05 UTC which added fences, spider jockeys, and removed a special splash.


== Additions ==


=== Blocks ===
 Fences

Added oak fences, the first fence to be added into the game.
Fences can only be placed on top of a solid block.
The player spawns with 64 oak fences.


=== Mobs ===
 Spider jockeys

The first jockey to be added into the game.


== Changes ==


=== Items ===
Milk bucket

an now be emptied.


=== Mobs ===
Chickens

No longer take fall damage and now slowly glide down using their wings.


=== Gameplay ===
Hotbar

The player starts with 64 fences in the inventory
Right clicking on usable blocks makes the player's arm swing to match the third person animation for right clicking.
Multiplayer

Chickens, slimes[dead link] and pigs added in multiplayer.
Working day/night cycle in multiplayer.
Third person view

Is now centered again.


=== General ===
Splashes

Removed "Happy birthday! I love you! Alex x" special splash.


== Bugs ==
1 bug reported

Items in the inventory are invisible.


== Trivia ==
Alpha v1.0.17 has the highest minor version number since Classic, at "17". The highest in Bedrock Edition is "221", with Bedrock Edition 1.16.221, and the highest in the full release of Java Edition is "11", with 1.21.11.
After release, this version was only available for 8 minutes, becoming the shortest lived Java Edition Alpha version.
