1.6.4 Build 9 is the ninth build for MinecraftEdu 1.6.4, released in 2014. It is based on Java Edition 1.6.4.


== Additions ==


=== General ===
Missing translations to minimized Launcher.
World Downloader window always open to top left corner.
Option to hide or not hide to Launcher options and the mini Launcher itself.


== Changes ==


=== General ===
Materials and Screenshots are not downloaded from World Library.
Improved World Downloader functionality.
Welcome message and Message of the Day text fields next to each other in server window.


== Fixes ==
2 bugs fixed

Downloading the same world from World Library didn't overwrite the old world.
Wrong world type in server world information tab.
