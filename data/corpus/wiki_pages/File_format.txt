File format may refer to:


== Minecraft ==
Anvil file format  the binary file format currently used in Java Edition since 1.2.1.
Region file format  the binary file format formerly used in Java Edition from Beta 1.3 to 1.1.
Zone file format[unofficial name]  the binary file format used only in Java Edition Infdev 20100624.
