18w20a is the thirty-fifth snapshot for Java Edition 1.13, released on May 15, 2018.


== Additions ==
Command format

Teams can now have a prefix and a suffix, specified as raw JSON text.
/team option <team> prefix <prefix>
/team option <team> suffix <suffix>
