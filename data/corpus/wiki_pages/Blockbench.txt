Blockbench is a free and open source 3D-modeling application, designed for creating low-polygon models. Its development is supported by Mojang Studios, who use the application internally for model and asset animation. It is also recommended as tool for model editing in the Bedrock Edition creator documentation. Mojang Studios also provides official plugins for the software  namely the Minecraft Block Wizard, Minecraft Entity Wizard, and Minecraft Item Wizard  to help creators with add-on development. It is also commonly used to make item models on Java Edition.


== Features ==
Blockbench allows users to make custom models, as well as text in the style of the logo text on the Title Screen (with the help of a plugin), among other things. One can change their current model's appearance by changing its composition of voxels, and the texture file that maps onto them.
While it originally exclusively supported models that were only made of cuboids, more versions allow it to support and edit meshes, splines, billboard sprites, and armatures.


=== Plugins ===


==== Minecraft Block Wizard ====
The Minecraft Block Wizard  is a Blockbench plugin created by Mojang designed to help the player create block models for Bedrock Edition. It has a variety of options and presets, such as the Lectern preset.


==== Minecraft Entity Wizard ====
The Minecraft Entity Wizard  is a Blockbench plugin created by Mojang designed to help the player create entity models for Bedrock Edition. It has most of the entities in vanilla Bedrock as presets, for the user to edit and animate to their liking.


==== Minecraft Item Wizard ====
The Minecraft Item Wizard is a Blockbench plugin created by Mojang Studios designed to help the player create item models for Bedrock Edition. It has the apple, the iron sword, the iron pickaxe, and iron armor as preset.


==== Minecraft Title Generator ====
The Minecraft Title Generator is a Blockbench plugin created by Ewan Howell designed to assist in creating text images with the Minecraft font. It has a variety of options, including fonts, colors, and position.
