Beta 1.6 is a version of Java Edition Beta released on May 26, 2011, which adds maps, trapdoors, the Nether in multiplayer as well as other changes and many bug fixes.


== Additions ==


=== Blocks ===
 Dead bush

Generates naturally only in deserts.
Can only be placed on sand.
 Shrub

Doesn't generate anywhere.
Only obtainable via inventory editing.
 Grass

Generates in the form of random patches, on grass blocks in most biomes with grassy soil, like plains or forests.
Seeds are now found in grass.
Its color change depends on the humidity.
 Fern

Generates naturally only in swamps and rainforests.
Seeds are now found in fern.
Its color depends on the temperature of the biome it’s in.
 Wooden trapdoor

It can be opened like a door.
Crafted with planks.


=== Items ===

 Maps

Allows the player to see a section of the world from a top-down perspective.


=== Command format ===
Server.properties

Added new settings:
view-distance.
allow-nether.


=== Gameplay ===
Multiplayer

Added Nether support.


== Changes ==


=== Blocks ===
General

Blocks destroyed by other players in multiplayer now show the breaking block particle effect.
It's no longer possible to build solid blocks on the top layer of the map.
Blocks that don't change appearance when the data changes don't send block updates when their data changes now.
 Beds

Trying to sleep in the Nether causes the bed to explode.
 Dispensers

When activated, it makes sounds and triggers particles in multiplayer.
 Doors

Now make sound for other players in multiplayer.
 Fire

Has been severely nerfed so it spreads slower, and doesn't spread infinitely.
 Ice

Now regenerates regardless of whether it's snowing or not.
 Jukebox

Now supports more than eleven different music discs.
 Leaves

Leaves with the graphics level set to fast no longer act like opaque solid blocks, preventing blocks such as Torches, Doors and Levers to be placed on them and preventing mobs from naturally spawning on them.
 Mushrooms

Now spread, although slowly.
 Powered rail

Now speedier to offset the loss of glitch-boosters.
 Water

Below a layer of source blocks no longer behaves like source blocks.


=== Items ===
 Arrows

When shot by one player, can now be picked up by other players.
 Seeds

Using a hoe on grass no longer yields seeds.


=== Non-mob entities ===
 Boats

Players are dismounted when they ride a boat or minecart through a Nether portal.
 Minecarts

Players are dismounted when they ride a boat or minecart through a Nether portal.


=== Gameplay ===
Combat

Removed the 10 tick cooldown when missing a hit (hitting the air).
Controls

⇧ Shift+ clicking while crafting makes as much as possible and moves it straight to player's inventory.
⇧ Shift+ clicking something in player's inventory pushes it directly in player's hot bar and vice-versa.
Lighting

The lighting on block icons in the inventory has been changed to appear brighter.
Players

When stuck in walls, slide towards the nearest gap if there is one.


=== General ===
Network traffic

Compressed network traffic more aggressively.
Options

Temporarily disabled Advanced OpenGL.
Texture packs

Added support for misc\grasscolor.png and misc\foliagecolor.png images.
