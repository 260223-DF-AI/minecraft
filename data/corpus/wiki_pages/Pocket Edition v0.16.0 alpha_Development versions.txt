This is a list of Pocket Edition v0.16.0 alpha development versions.


== v0.16.0 alpha build 1 ==
v0.16.0 alpha build 1 (labelled v0.15.90 alpha build 1 in-game and 0.15.90 internally) is the first build version released for v0.16.0, also known as the Boss Update. This build accidentally included several features from Education Edition which were quickly removed or disabled in the following builds.
VR bug fixes in this update were later added to v0.15.10.

== v0.16.0 alpha build 5 ==
v0.16.0 alpha build 5 (labelled 0.15.90.8 internally and v0.15.90.1 alpha build 1 in-game) is the fifth and final build version released for v0.16.0, also known as the Boss Update. This is the final build released in 2016.


=== Additions ===
New command: /enchant
Added sounds for strays, husks and minecarts.
Added texture packs from v0.15.7 and v0.15.8 versions.
Added MineCon skin pack from v0.15.9.
Added elder guardian spawn egg to the creative inventory.
Added a texture for the elder guardian spawn egg to be shown instead of an empty spawn egg.
Added a new button when the player sneaks when a villager is highlighted.


=== Changes ===
Add-ons are now named Behavior Packs.
Some icons for settings are now colored.
Re-added the Old world type which was removed in build 1.
A chest now generates in big houses of cold taiga and ice plains villages.
