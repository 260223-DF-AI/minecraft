2.3.136 is an update to the launcher released on April 12, 2022 as a beta version for Windows, macOS and Linux, on April 20, 2022 as a beta version for Windows (New Launcher), and on April 22, 2022 as a non-beta version for all these platforms.


== Additions ==
Added Setting that lets user review the Launcher on the Microsoft Store from within the Launcher.


== Changes ==
Updates to some error messages to make them more useful.
Various Launcher bug fixes, stability improvements and updates to distribution pipeline.


== Fixes ==
MCL-21100 - Game output log cuts off messages and does not show scroll bar.
MCL-21109 - Some selected buttons and fields can't be interacted with using both the Space and  Enter keys.
