1.8.2-pre4 is the fourth pre-release for Java Edition 1.8.2, released on January 16, 2015.


== Changes ==


=== World generation ===
Dark oak trees and oak trees

Oak and dark oak branches no longer replace blocks.


== Fixes ==
4 issues fixed
From released versions before 1.8.2

MC-10176 – Oak/dark oak tree branches can replace all solid blocks (including bedrock and barrier).
MC-51804 – Corner fences/cobblestone wall with a block next to them turn black (Smooth lighting off).
MC-71163 – Bed can overlap when placed on grass.
From the previous development version

MC-76562 – Hoppers do not work.


== Trivia ==
This version was previously removed from the launcher for some time, but was added back in September 2018.
