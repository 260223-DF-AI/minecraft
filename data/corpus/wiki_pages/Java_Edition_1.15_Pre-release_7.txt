1.15 Pre-release 7 (known as 1.15-pre7 in the launcher) is the seventh and final pre-release for Java Edition 1.15, released on December 9, 2019, which fixes some debug screen related bugs.


== Fixes ==
4 issues fixed
From the 1.15 development versions

MC-167061  FPS/TPS graphs on F3 screen are empty in spectator mode.
MC-167105  Debug screen renders behind hotbar.
MC-167106  Debug screen FPS/TPS graphs are not rendered correctly.
MC-167108  Lines on the debug screen are taller than before.
