Since the beginning of the development of Legacy Console Edition, there have been a number of features that were removed from the game. These features may have been replaced, or a developer decided against the feature later on.
Note that this page only documents game features that were removed; features of a particular game element that were removed are noted in that element's history.


== Biomes ==

In TU5, world generation was updated to match Java Edition Beta 1.8, causing several biomes to be removed.


== Game Chat ==
PlayStation 4 Edition 1.83 removed the Game Chat option in sound settings, which allowed voice chat from within the game. The console's party chat must now be used if players want to use voice chat.
