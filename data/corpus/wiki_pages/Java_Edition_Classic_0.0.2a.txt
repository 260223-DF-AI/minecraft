0.0.2a is a version of Java Edition Classic released on May 16, 2009, at 16:52 or 17:05 UTC, which added in game version number, frame rate and chunk update counter, and fixed some issues with the mouse look.
This is the first version of Minecraft to have a version number; version 0.0.1a has been skipped.


== Additions ==


=== General ===
Debug screen

Version number
Very large text.
Located in the top-left corner.
Added a frame rate and chunk update counter.
Located in the top left corner, below the version number.
Before this, the game would output the current FPS and any chunk updates to the developer console.
Font

Added a Minecraft font.
Filename: default.gif.


== Fixes ==
1 bug fixed

Fixed more issues with mouse looking.


== Bugs ==
2 bugs reported

When a key is pressed down and the game is blurred without being clicked out of, the game still registers that key being pressed. This still occurs in modern Minecraft, making this the game's oldest bug.
Some issues reported were low sensitivity, crashing, jerky mouse looking, and low FPS.


== Trivia ==
This was deemed "relatively stable" by Notch, and the URL for the game was posted to other users in the IRC channel.
