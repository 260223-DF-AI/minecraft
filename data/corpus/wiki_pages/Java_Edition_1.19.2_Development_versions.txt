== 1.19.2 Release Candidate 1 ==
1.19.2 Release Candidate 1 (known as 1.19.2-rc1 in the launcher) is the first release candidate for Java Edition 1.19.2, released on August 4, 2022, which fixes an issue causing players to get disconnected with secure chat.
