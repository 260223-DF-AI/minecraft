A quiz block was a block used to set up student quizzes.


== Obtaining ==
The quiz block was not available in the creative inventory. It was only obtainable through being given in the teacher menu or by being distributed by console command, such as with a command block.


== Usage ==

The quiz block is a block that teachers can place in their world. Right-clicking on it as a teacher will open a menu in which teachers can create quizzes for students to answer.


== Data values ==
The last ID of this block was 3730.
