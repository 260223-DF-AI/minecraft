A version of Minecraft Indev was released on February 11, 2010, at 23:27 UTC. It was announced on TWoN at 23:37 UTC, by which time the next version had already been released.


== Additions ==


=== Gameplay ===
Daylight cycle

Each cycle lasts about 15 minutes each.
7.5 minutes of sun and 7.5 minutes of moon each cycle.
Dictates when certain mobs spawn.


== Changes ==


=== Mobs ===
Pigs

Added sounds.

 Sheep

Sheep now spawn with cloth again.
They now drop 13 cloth when punched, each piece being either white, light gray, or gray.
Added sounds.


== Bugs ==
3 bugs reported

The light level often would not match the time of day.
In some instances, performance would deteriorate and the game would become near-unplayable as a result.
Due to the addition of daylight cycle, there is now a bug where during sunset and sunrise changes of the light level can cause extreme amount of chunk updates and temporary game lags. This bug remained in the game until Beta 1.8.


== Trivia ==
The al_version number of this version is 1.096, but the number of the previous version is 1.093. The two numbers between them might correspond to currently missing or LWJGL updates. One of these updates may have been released on February 8.
