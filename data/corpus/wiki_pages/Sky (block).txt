Sky is a joke block added in Java Edition 25w14craftmine.


== Obtaining ==
Sky cannot be broken naturally without Creative mode or commands. It spawns naturally in the hub biome.


== Usage ==
It serves no purpose and is purely a decorative block. It cannot be destroyed in Survival mode.

== Issues ==
Issues relating to "Sky (block)" are not maintained on the bug tracker because it is an April Fools' joke, and is therefore not in the newest stable version or snapshot. Issues reported there are closed as "Invalid".


== Trivia ==
Sky is rendered the same way as dimension control and shimmering door but doesn't flash because the model has a tintindex.
