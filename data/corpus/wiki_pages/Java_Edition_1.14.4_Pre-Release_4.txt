1.14.4 Pre-Release 4 (known as 1.14.4-pre4 in the launcher) is the fourth pre-release for Java Edition 1.14.4, released on July 10, 2019, which modifies server permissions and fixes bugs.


== Additions ==


=== General ===
server.properties

Added function-permission-level to server.properties which controls which permission levels functions have.


== Changes ==


=== Mobs ===
Pillagers

Changed arm texture.


=== Command format ===
/forceload and/reload 

Now available up to permission level 2.
