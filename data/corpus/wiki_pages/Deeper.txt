"Deeper" is a song composed by Lena Raine that plays in the deep dark, dripstone caves.


== Locations ==


== Track description ==
"Deeper" is a song written by Lena Raine to bring a creepy vibe to the deep dark or dripstone caves. It also samples the leitmotif of Infinite Amethyst.


== Data values ==


=== ID ===
Java Edition and Bedrock Edition:


=== Raw music files ===
In Java Edition, the music files can be found in .minecraft/assets/objects.
In Bedrock Edition and Minecraft Education, it can be found in:

Mobile: com.mojang/resource_packs/music/vanilla_music/sounds/music/game/
Windows: %PROGRAMFILES%\WindowsApps\Microsoft.MinecraftUWP_<version>_x64_8wekyb3d8bbwe\data\resource_packs\vanilla_music\sounds\music\game
All music files are in Ogg Vorbis.
