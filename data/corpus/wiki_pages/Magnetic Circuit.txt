"Magnetic Circuit", also known as "calm4", is a song by Markus Persson (aka Notch). It was accidentally included as music in Java Edition Alpha v1.1.1.


== Track description ==

The track was created by Notch himself, and is 3 minutes and 13 seconds long. It consists of an upbeat synth, battle-like tune. At 1:36, a slowed-down sample of Notch saying "Mojang Specifications" can be heard.


== History ==
Notch accidentally included calm4.ogg in Java Edition Alpha v1.1.1 when he used it to test the music system.
The track is believed to have appeared and disappeared in the downloaded game files multiple times, but exact dates or versions are unknown.
After Java Edition Alpha v1.1.2, calm4.ogg was removed.
With the introduction of the 1.6.1 launcher, playing older versions with the track calm4.ogg does not allow the track to be heard, since music is downloaded separately from the jar files.[dubious]
On February 1, 2020, calm4.ogg was reuploaded by Notch to his SoundCloud page under the name "Magnetic Circuit" after he was questioned about it.
