2.8.2 is an update to the launcher released on August 29, 2023.


== Fixes ==
Improvements & bug fixes for Bedrock & Java installation flow & messaging.
Fixed bugs for Java Quick Play.
Fixed some Java skins bugs.
MCL-23496 - Crowdin link in language progress toast no longer works.
MCL-23547 - 'Soothing scene' video/audio will abruptly end when returning to the "Play" tab.
MCL-23662 - The word "Owned" within a launcher string used for the Realms tab is untranslatable.
MCL-23675 - Quick Play labels used for the Quick Play feature are untranslatable.
MCL-23677 - The "QUICK PLAY" string in the Minecraft: Java Edition tab is untranslatable.
