22w16b is the seventh snapshot for Java Edition 1.19, released on April 20, 2022, which fixes a bug related to a crash.


== Fixes ==
2 issues fixed
From the 1.19 development versions

MC-249786 – Allay has trouble throwing items onto targets at different heights.
From the previous development version

MC-250312 – Game crashing when clicking singleplayer: java.lang.NullPointerException: Cannot invoke "java.lang.Comparable.compareTo(Object)" because "pivot" is null.
