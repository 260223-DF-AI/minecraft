1.8.1-pre1 is the first pre-release for Java Edition 1.8.1, released on October 15, 2014.


== Additions ==


=== Command format ===
Game rules

doEntityDrops: Works like doTileDrops but applies to entities instead.


== Changes ==


=== Blocks ===
Jack o'lanterns

Can be used to create iron golems and snow golems again.


=== Items ===
Bows

Bows are no longer upside down when held in the inventory.
Maps

Changed some colors.
Potion of Leaping

Can now be extended with redstone, like any other potion.


=== Mobs ===
General

No longer run from creepers.
Armor stands

Added Marker tag.


=== World generation ===
Witch huts

Newly generated witch huts can spawn witches on Y 64 to 71.
This allows for three spawning floors in witch farms.


=== Command format ===
Game rules

doTileDrops no longer affects entities.


=== Gameplay ===
Credits

Ryan Holtz (TheMogMiner) and Michael Stoyke (Searge) are now mentioned in the credits.


=== General ===
Options

Renamed "Multiplayer Settings" to "Chat Settings" from 1.5.x and lower.
Performance

Various improvements.
