1.2 is a launcher version released on August 1, 2013.


== Additions ==
Added a "Time Machine" feature to use Beta versions.
An option for Alpha has been included but the versions are not available yet.
Also allows players to play versions 1.2.5-1.4.7


== Fixes ==
3 issues fixed
MCL-63 – Launcher does not recognize already-installed versions.
MCL-105 – 1.5.x shows default Java icon in the taskbar when started with new launcher.
MCL-861 – Default profile picking up local modded version.
