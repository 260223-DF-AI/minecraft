1.7.10 Build 15 is the fifteenth build for MinecraftEdu 1.7.10, released on February 27, 2015. It is based on Java Edition 1.7.10.


== Additions ==
Now informs the user if they try to spawn animals, monster or villagers in Peaceful difficulty or when disabled.
Added "Students can always fly" option.


== Changes ==
Teacher's personal creative mode not tied to the server gamemode anymore.


== Fixes ==
Debug folder sometimes appearing to the parenfolder of minecraftedu.
Server.log saying "debugging for server..." instead of "debugging for client..."
Teachers being frozen if they got frozen as a student.
