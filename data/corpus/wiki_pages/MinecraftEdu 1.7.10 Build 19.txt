1.7.10 Build 19 is the nineteenth build for MinecraftEdu 1.7.10, released on May 29, 2015. It is based on Java Edition 1.7.10.


== Additions ==
Golems are now affected by the monsters setting.


== Changes ==
Teacher menu and some other menus appearances redone.


== Fixes ==
3 bugs fixed

Exporting a world to zip not working properly.
Huge lag when using creative inventory search in other than English language.
Students being able to use /msg when muted.
