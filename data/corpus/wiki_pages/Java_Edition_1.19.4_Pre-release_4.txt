1.19.4 Pre-release 4 (known as 1.19.4-pre4 in the launcher) is the fourth and final pre-release for Java Edition 1.19.4, released on March 8, 2023, which updates textures for cherry leaves and cherry signs, and fixes bugs.


== Additions ==


=== General ===
Options

Added "Credits & Attribution" button in the options menu.
Minecraft's credits, attribution, and licenses can be accessed.
Tag

Added following damage type tag:
#bypasses_shield: contains #bypasses_armor, falling_anvil, and falling_stalactite.


== Changes ==


=== Blocks ===
Sculk sensor

Is now activated upon breaking shulker bullets.


=== Non-mob entities ===
Display entity

When a new interpolation is started, it now starts from the current state instead of the final state.


== Experimental ==
These changes only take effect when the 1.20 experimental data pack is enabled.


=== Changes ===


==== Blocks ====
Cherry leaves and cherry sign

The textures have been updated.[more information needed]
