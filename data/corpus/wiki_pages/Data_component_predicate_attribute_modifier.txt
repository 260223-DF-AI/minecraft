[String][NBT List / JSON Array] attribute: (Optional) Any number of attribute(s) (an [String] ID, or a [String] tag with #, or an [NBT List / JSON Array] array containing [String] IDs)
[NBT List / JSON Array] id: (Optional) Checks the namespaced ID of the attribute modifier.
[Double] amount: (Optional) Checks the amount from the attribute modifier. Matches an exact [Int]value, or checks if the value is between a range.
[NBT Compound / JSON Object] amount: (Optional) Another format.
[Double] max: (Optional) The maximum value.
[Double] min: (Optional) The minimum value.
[String] operation: (Optional) Checks the operation of the attribute modifier. Must be one of add_value, add_multiplied_base, and add_multiplied_total.
[String] slot: (Optional) Checks the slot of the attribute modifier. Must be one of any, mainhand, offhand, hand, head, chest, legs, feet, armor, and body.
