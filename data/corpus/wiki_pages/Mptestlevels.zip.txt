mptestlevels.zip is a pack of Java Edition worlds uploaded by Notch on minecraft.net and shared in the #minecraft chat on irc.esper.net on June 8, 2009. It contains the worlds used in Notch's test server during the Classic Multiplayer Test, before the public release of the Classic server software. The levels contained within appear to be numbered from oldest to latest used. The exact contents of this pack are unknown, as the file has not been archived and is currently lost.
