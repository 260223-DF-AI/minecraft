1.6.4 Build 8 is the eighth build for MinecraftEdu 1.6.4, released on January 23, 2014. It is based on Java Edition 1.6.4. This is the first build in 2014.


== Additions ==


=== General ===
Worlds are downloaded directly from World Library site.
Launcher doesn't close when you run something from it. Instead it minimizes.
Option to set "Message of the Day" from the Server windows' "World information" tab.
Added setting to always show xp bar (no matter what gamemode is selected). always-show-xpbar=true/false to EduWorldSettings.ini
Spanish translations.
Dutch translations.


== Changes ==


=== General ===
Online Worlds and Local Worlds merged to one World Library.
Renamed Servertool to Server Launcher.
World Template creation removed from servertool. World Templates are now created at World Library.
Moved welcome message option to "World Information" tab in server window.
Build crosshairs improved.


== Fixes ==
3 bugs fixed

Launcher component hiding not working.
Launcher spamming "you have files in your mods folder".
Translations not working in client and server.
