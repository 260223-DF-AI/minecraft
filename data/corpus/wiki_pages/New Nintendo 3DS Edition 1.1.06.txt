1.1.06 is an update for the New Nintendo 3DS Edition released on November 8–9, 2017, which improved performance and fixed bugs and crashes.


== Changes ==
Improved save and load times.
Expanded functionality for quick move through the UI.
Various UI tweaks and fixes.
Experience can now be obtained by smelting and trading.
Smaller, refined font for Roman alphabet languages.
Various stability and performance improvements.


== Fixes ==
Fixes to item stacking in inventory and hotbar.
Fixed crash issues related to various entities overpopulating.
Fixed crash that occurred when using bottles o' enchanting.
