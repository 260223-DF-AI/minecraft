Bedrock Editor v0.7.6 is a minor alpha release for the Bedrock Editor released on October 23, 2024.


== Additions ==
Locate tool

Added a tool to allow player to find specific biomes within player's world and teleport to them easily.
This tool can be found under "World Options"  "Locate".


== Changes ==
Time of day is fixed to noon at Editor startup.
Improved viewport focus visibility with an animated outline on focus and added focused state as a step for toggle mode (CTRL +  Tab).
Added focusToolInputContext function to ModalToolContainer which will try to set modal input focus (e.g. viewport for tool rail).
