22w44a is the third snapshot for Java Edition 1.19.3, released on November 2, 2022, which adds new spawn eggs, tweaks the Creative inventory,  and fixes bugs.


== Additions ==


=== Items ===
Spawn eggs

 Ender dragon spawn egg
 Iron golem spawn egg
 Snow golem spawn egg
 Wither spawn egg
Ender dragon and wither spawn eggs are only available through commands to prevent accidental destruction of player builds.


=== Gameplay ===
Game rules

Added blockExplosionDropDecay, mobExplosionDropDecay and tntExplosionDropDecay.
When set to false, all blocks drop loot.
When set to true, blocks drop loot randomly depending on how far from the explosion center.
Defaults to false for TNT, true for block and mob.
Added snowAccumulationHeight.
When it is snowing, this game rule determines the maximum number of snow layers that can be accumulated in each block.
Defaults to 1.
Set to 0 makes no snow form at all.
Set to 8 or above lets snow form up to the level of a full block.
Added waterSourceConversion and lavaSourceConversion.
When set to true, allows new sources of that fluid to form.
Defaults to true for water and false for lava.
Added globalSoundEvents, controlling whether certain gameplay moments are heard by all players regardless of location, defaults to true.


=== General ===
Tags

Added the following item tag:
#fence_gates: acacia_fence_gate, birch_fence_gate, dark_oak_fence_gate, jungle_fence_gate, oak_fence_gate, spruce_fence_gate, crimson_fence_gate, warped_fence_gate, and mangrove_fence_gate.


== Changes ==


=== Blocks ===
Spawners

Renamed to "Monster Spawner" to match Bedrock Edition, and removed purple text color.
No longer have a default mob spawn type when placed by a player, which was a pig previously.
Don't emit fire particles when a mob spawn type has not been defined.
Pick-block now works for spawner blocks.
The mob type is now displayed in the hover description of a spawner item stack.
If a mob type has not been defined yet, the hover description will describe how to set it.


=== Items ===
Spawn eggs

Changed the colors of polar bear spawn egg (to ), to distinguish it from the ghast spawn egg.


=== Mobs ===
Turtles

Now have a 5 minute cooldown after breeding.


=== Non-mob entities ===
Minecart with spawners

Don't emit fire particles when a mob spawn type has not been defined.
Renamed to "Minecart with Monster Spawner".


=== Gameplay ===
Creative inventory

Operator-only blocks and items can now be found in the "Redstone Blocks" tab when have the required permissions. This includes the barrier, all variants of command blocks, debug stick, jigsaw block, light block (level 15), minecart with command block and structure void, but does not include the structure block.
In "Functional Blocks" tab:
Added bee nest and tinted glass.
Moved respawn anchor before beds.
Reordered pressure plates by functionality.
Moved end portal frame and infested blocks here from Natural Blocks tab.
In "Redstone Blocks" tab:
Added barrel, cauldron, chest, composter and furnace.
In "Building Blocks" tab:
Added block of amethyst and chain.
Moved block of redstone and block of coal here from the Natural Blocks tab.
In "Crafting" tab:
Added ancient debris.
Reordered the following tabs for better consistency:
Building blocks
Consumables
Natural blocks


=== General ===
Sounds

Step sounds can now be heard when walking on carpets, lily pads and small amethyst buds.
Step sounds can now be heard when walking through crimson roots, glow lichen, Nether sprouts and warped roots.
Tags

Removed #non_flammable_wood block tag.
Added crimson roots, glow lichen, lily pad, moss carpet, Nether sprouts, small amethyst bud, warped roots and #wool_carpets to the #inside_step_sound_blocks block tag.
Removed crimson hanging sign and warped hanging sign from the #non_flammable_wood item tag.
