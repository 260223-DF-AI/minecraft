1.5.1 Build 6 is the sixth build for MinecraftEdu 1.5.1, released in April 2013. It is based on Java Edition 1.5.1.


== Fixes ==
6 bugs fixed

We do not distribute stock options.txt anymore with Minecraft. We now distribute options_orig.txt that we copy as options.txt if it is missing.
Fix to show the disconnected reason for user (in kicks / bans)
When closing serverGUI, user will be prompted if wanting to close it.
Can now go back to servertool from server.
Setting disable-startmenu to true in minecraftedusettings.ini no longer crashes client when entering main menu.
Updated icons in launcher.
