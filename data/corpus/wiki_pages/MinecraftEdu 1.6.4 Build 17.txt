1.6.4 Build 17 is the seventeenth build for MinecraftEdu 1.6.4, released in 2014. It is based on Java Edition 1.6.4.


== Additions ==


=== Blocks ===
 Home Block

Individual teleport station block in which the player can teleport to.


=== Items ===
 Block Inspector

A special item used to identify blocks in the world.


== Changes ==
Moved "Teleport Self to Spawn" and "Teleport to Surface" buttons from the Assignments menu to the Personal menu on Teacher.


== Fixes ==
1 bug fixed

Some mods like Galacticraft not working properly. Some GUIs were not opening.
