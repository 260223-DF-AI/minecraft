1.21.123 is a hotfix to Bedrock Edition released on November 13, 2025, which fixes some bugs.


== Fixes ==
7 issues fixed
MCPE-167489  Textures turn pink (after playing for long periods of time).
MCPE-179496  Memory leak when the "Improved Input Response" option is enabled.
MCPE-230558  When pressing any of the number keys on an item in a stonecutter, the game crashes.
Other

Fixed several crashes that could occur during gameplay.
Fixed an issue where IME Candidates would appear in the wrong monitor or misplaced in relation to text field.
Fixed IME input when application lost and regained focus (switching from the app and back while editing).
Blocks with the tick component tick when placed by Jigsaw Structure during world generation.
