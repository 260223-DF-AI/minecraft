0.25_05 SURVIVAL TEST is a version of Survival Test released on September 3, 2009, at 17:11 UTC. This version is not available in the launcher, but has been archived elsewhere.


== Changes ==


=== General ===
Skins

Custom skins now display on the player's arm.


== Fixes ==
1 bug fixed

Monsters no longer chase after dead monsters.


== Trivia ==
Versions _01 through _03 were not released under those names, and version _04 was skipped. Each patch version of 0.25 SURVIVAL TEST did not change the version number, which led to a wiki admin mistakenly changing the latest version listed on the wiki from "0.24_SURVIVAL_TEST_3" to "0.24_SURVIVAL_TEST_4", a version that does not exist. Notch saw this and thought that he was telling him that the unlabelled patches for 0.25 should have labels and that the current version should be "_04", so Notch made the next version "0.25_05", even though under Notch's version format used at the time the version should technically have been "0.25_04".
