1.14 Pre-Release 4 is the fourth pre-release for Java Edition 1.14, released on April 17, 2019, which decreases health of foxes, fixes bugs and makes changes to the default font.


== Additions ==


=== General ===
Font

Added more characters to the default font.
Corrected several glyphs.


== Changes ==


=== Mobs ===
Foxes

Now has 10HP health opposed to 20HP  10 health like the player in previous snapshots.
