Beta 1.3_01 is a version of Java Edition Beta released on February 23, 2011, to fix some of the bugs and crashes in Beta 1.3.


== Changes ==


=== General ===
Java

Removed all dependencies on Java 1.6.


== Fixes ==
3 bugs fixed

Fixed the recent issues on macOS in which the game would crash when the "Singleplayer" button was clicked.
Able to connect client to Minecraft server 1.3.​[more information needed]
Fixed an item duplication glitch.
