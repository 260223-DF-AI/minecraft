The second version of Indev 0.31 was released on December 23, 2009 at 14:57 UTC. This version was not playable as it cached the previous version due to an incorrect al_version, and it was reuploaded two minutes later to fix this.


== Additions ==


=== Blocks ===
  Torches

Emit light.
Have a height of 13 pixels.
 If the player places a torch on the ground next to a wall, the torch attaches to the wall using a different model.
It is possible to create a flying torch by placing it on the ground, then by breaking the ground.
It is also possible for flowers, mushrooms and saplings except that the torch stays in the air forever.


== Changes ==


=== Blocks ===
Grass blocks

Can now grow in slightly lit areas.
Lava

Is now luminous.
Saplings

Do not decay in slightly lit areas.
Leaves

Now create a shadow.
The grass under leaves is now dirt due to this.


== Trivia ==
This version as well as 0.25 SURVIVAL TEST were both only available for 2 minutes, making them the shortest lived versions across all Minecraft editions.
