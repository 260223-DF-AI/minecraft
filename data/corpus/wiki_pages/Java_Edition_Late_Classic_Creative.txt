Late Classic Creative refers to six versions of Java Edition Classic released from October 27 to November 10, 2009 (0.28 through 0.30). These versions are all in Creative mode, but Survival mode can be enabled in all versions by modifying Minecraft.class in minecraft.jar. The last version in this phase, 0.30, has a Survival mode variant.


== Additions ==

Building
Left click removes blocks and right click places blocks.
New block: Obsidian
Mobs
Sheep
Multiplayer
Players now push each other when standing next to each other.
Opping or deopping a player makes changes instantaneously instead of having to reload.
Players can build through other players.
Player tags don't appear over other players unless the mouse is hovered over them.
Options
Added rain toggle to options.
Added tree growing switch to server.properties.
General
Re-added Creative mode.
The ability to save levels locally as a .mine file.
