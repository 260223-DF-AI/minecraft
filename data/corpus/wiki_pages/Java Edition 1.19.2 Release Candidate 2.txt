1.19.2 Release Candidate 2 (known as 1.19.2-rc2 in the launcher) is the second and final release candidate for Java Edition 1.19.2, released on August 5, 2022, which fixes a crash issue in the social interactions screen.


== Fixes ==
1 issue fixed
Other

Fixed a crash in the social interactions screen.


== Trivia ==
After this release during the same day, 1.19.2 was released making the release date for the version which the release candidate is for, the shortest.
