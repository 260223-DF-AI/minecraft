18w30a is the first snapshot for Java Edition 1.13.1, released on July 25, 2018, which adds a search bar in the world list and a few other things, and fixes many bugs.


== Additions ==
Blocks

Re-added punch-to-explode TNT using the unstable state (instead of the explode state in 1.12).
Gameplay

Added a search bar in the world list.
Debug screen

Added  Control + F3 + C to force a JVM crash, instead of a regular crash. However, with some keyboards it only triggers with right control.
