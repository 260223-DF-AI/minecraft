This page is about planned versions of all editions of Minecraft. These updates either have major sources to their existence or have received development versions.
Only editions with known planned updates are listed here; if no known planned updates exist, the edition is left out.


== Java Edition ==


=== 26.2 ===

26.2, the release of Chaos Cubed, is a game drop with no set release date, which adds the sulfur caves biome, including sulfur cubes and sulfur and cinnabar blocksets.


== Bedrock Edition ==


=== 26.20 ===

26.20 is a minor update with no set release date, which adds experimental features for Chaos Cubed and fixes bugs.


=== 26.30 ===

26.30, the release of Chaos Cubed, is a game drop with no set release date, which adds the sulfur caves biome, including sulfur cubes and sulfur and cinnabar blocksets.
