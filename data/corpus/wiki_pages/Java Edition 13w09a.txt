13w09a is the eleventh snapshot for Java Edition 1.5, released on February 26, 2013.


== Additions ==


=== Mobs ===
General 

A way for mobs/entities to ride other mobs/entities.
They no longer dismount upon reload.
Mobs can now be spawned riding other mobs.


== Changes ==


=== Entities ===
Player riding on an entity keeps it when logging off (the entity disappears along with player) and is still riding the entity upon logging back on.


=== General ===
Fonts

Fonts are now changeable.
Added HD font support.
Language

Updated language files.
Lighting

Smooth lighting can now be changed to Minimum, Maximum & Off.
Minimum uses the old smooth lighting, and maximum fixes a bug with stairs.
Texture packs

Animated blocks and items in texture packs now require the animation definition.
