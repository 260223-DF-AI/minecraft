1.20.2 Pre-Release 4 (known as 1.20.2-pre4 in the launcher) is the fourth and final pre-release for Java Edition 1.20.2, released on September 13, 2023, which fixes bugs.


== Additions ==


=== General ===
Tag

Added the following damage type tag:
#always_kills_armor_stands: contains arrow, trident, mob_projectile, fireball, and wither_skull.
Damage types included in this tag should always kill an armor stand.


== Fixes ==
2 issues fixed
From the 1.20.2 development versions

MC-265239 – Error in logs: java.lang.IllegalStateException: recursive call to sendBlockUpdated.
MC-265289 – Error in log: java.lang.IllegalStateException: onTrackingStart called during navigation iteration.
