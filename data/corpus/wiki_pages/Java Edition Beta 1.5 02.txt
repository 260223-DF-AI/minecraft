Beta 1.5_02 is a server-only version of Java Edition Beta released on April 20, 2011, to fix a few issues with Beta 1.5_01. Beta 1.5_01 clients can connect to servers running Beta 1.5_02.


== Fixes ==
2 bugs fixed

The game no longer crashes from "internal server error" created by bugs in Beta 1.5_01 server in multiplayer.
The "Saving Chunks" screen is no longer shown when taking an item out of a crafting table.
