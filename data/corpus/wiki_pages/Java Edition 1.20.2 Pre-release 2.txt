1.20.2 Pre-release 2 (known as 1.20.2-pre2 in the launcher) is the second pre-release for Java Edition 1.20.2, released on September 7, 2023, which reverts changes to recipe book searching, and fixes bugs.


== Changes ==


=== Gameplay ===
Recipe book

Reverted the changes of recipe book searching made in the previous pre-release.


=== General ===
Resource pack

The resource pack version is now 18.


== Experimental ==
These changes only take effect when the "Villager Trade Rebalance" experimental data pack is enabled.


=== Changes ===


==== General ====
Tags

Moved the #on_desert_village_maps, #on_jungle_explorer_maps, #on_plains_village_maps, #on_savanna_village_maps, #on_snowy_village_maps, #on_swamp_explorer_maps, and #on_taiga_village_maps structure tags into the experimental trade rebalance data pack.
