Beta 1.20.50.23 (Android) or Preview 1.20.50.23 (Windows, iOS, iPadOS, Xbox) is the fourth beta/Preview version for Bedrock Edition 1.20.50, released on November 2, 2023, which brings more parity with Java Edition and fixes bugs.


== Additions ==


=== General ===
Marketplace

Added the ability to deep link users within a server to dressing room offers.


== Changes ==


=== General ===
Create New World and Main Menu

Added an altered main menu and world creation experience.


== Experimental ==
These additions and changes are accessible by enabling the "Update 1.21" and "Beta APIs" experimental toggles.


=== Changes ===


==== Blocks ====
Copper Door and Copper Trapdoor

Updated their block and item textures for all variants.
Crafter

Its texture has been updated.
Tuff Bricks, Tuff Brick Slab, Tuff Brick Stairs, and Tuff Brick Wall

Updated their block and item textures.


==== Technical ====
API

Moved Entity.remove from beta to 1.7.0.
DataDrivenEntityTriggerAfterEvent
Renamed property id to eventId for clarity.
Changed getModifier to work in read-only mode.
Changed DefinitionModifier from a class to an interface with properties.
Made EntityTypes.get properly reflect that it can return EntityType | undefined instead of just EntityType.
Dynamic Properties
NBT change: Dynamic properties are now stored using the behavior pack manifest UUID, rather than the module UUID. Existing worlds using dynamic properties will continue to work, and will be migrated to the new format when properties are read or modified.
JSON

Deprecated on_dig event triggers from minecraft:digger in format versions 1.20.50 and higher.


== Fixes ==
14 issues fixed
From released versions before 1.20.0

MCPE-31014 – 100% hotbar opacity isn't opaque enough.
MCPE-41898 – Exporting MCWORLD or MCSTRUCTURE file to a OneDrive folder fails silently.
MCPE-168598 – Some in-game UIs have no transparent black background in Pocket UI.
MCPE-169617 – Creator icon hover border is missing.
From released versions before 1.20.50

MCPE-167218 – Unable to crouch underwater/in water.
MCPE-175159 – Recipes menu defaults to "All" items.
MCPE-176184 – Firework rockets can't be used to break chorus flowers.
MCPE-176198 – Updating to 1.20.50.21 ejects all mobs too large for boats, even if they were put there prior to the update.
MCPE-176371 – Calcite can no longer be obtained by creative inventory or /give command.
Others

Added the missing crafting recipes for chiseled copper (all variants), tuff blocks, copper doors, trapdoors, and grates (all waxed variants).
Fixed possible soft locking when performing different actions within Marketplace.
Fixed a regression that caused flying players to slow down significantly after touching ground.
The classic skins info pop now sends you to the correct settings page.
Fixed a crash when joining a Realm after doing other things for a while.
