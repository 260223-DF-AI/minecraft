0.982 is a version of MinecraftEdu, released on May 22, 2012. It is based on Java Edition 1.2.5.
0.982 was reuploaded on July 25, 2012, to change the logo, main menu, position of buttons, add WorldEdit, add new skins, and more.


== Additions ==


=== General ===
Added WorldEdit plugin & support for it.
All skins improved, 8 new skins!
Added custom keybindings menu for menus and commands.
Added serverside custom texturepack support.
Added helpful Information box to the server.
Greatly improved performance.
Tweaks and fixes.


== Changes ==


=== General ===
