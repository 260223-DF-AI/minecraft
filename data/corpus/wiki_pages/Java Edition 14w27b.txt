14w27b is the thirty-third snapshot for Java Edition 1.8, released on July 2, 2014, to fix some bugs and crashes in the 14w27a snapshot.
14w27b was reuploaded as the original version featured an incorrect version label of '14w27a'.


== Changes ==


=== General ===
Block models

The block state files now support an array of models allowing for random models.
As a result of this, grass blocks, dirt, sand, red sand, stone, netherrack, bedrock, and TNT all have their top texture randomly rotated.
