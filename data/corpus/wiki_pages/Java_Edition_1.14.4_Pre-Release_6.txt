1.14.4 Pre-Release 6 (known as 1.14.4-pre6 in the launcher) is the sixth pre-release for Java Edition 1.14.4, released on July 15, 2019, which improves performance, removes the camera pivot offset in first-person mode, and fixes some bugs.


== Changes ==


=== General ===
Camera

Removed the camera pivot offset when in first-person view.
Performance

Improved performance.
Improved chunk loading when traveling at high speeds.


== Fixes ==
4 issues fixed
From 1.14

MC-149018  High idle CPU usage on server.
From 1.14.3

MC-154271  Rolling shutter issue on macOS.
From the 1.14.4 development versions

MC-156451  Velocity resets when breaking blocks occasionally.
Other

Fixed a memory leak.
