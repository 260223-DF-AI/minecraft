3.5.17 is an update to the Launcher released on December 6, 2024.


== Additions ==
Added new "Editor Mode" tab in Bedrock.
Added new "Marketplace Pass" tab in Bedrock.


== Changes ==
Removed "Java vs Bedrock" comparison chart.


== Fixes ==
5 issues fixed
From released versions before 1.21.4

MCL-22433  Capes show no texture in edit skin menu when using a right-to-left language
MCL-24212  Minecraft Launcher profiles do not update "LastUsed" data value in json file.
Other
Bug fixes to some controls and settings.
More improvements & fixes to Offline Mode.
Various other bug fixes and under the hood improvements.
