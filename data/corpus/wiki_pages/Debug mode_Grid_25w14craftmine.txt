This page lists all the rows in the debug mode block grid in 25w14craftmine. 
There are 167 lines with up to 168 block states. The last line has 114 block states. Therefore, there are 28,002 block states in total.
The first block is located at coordinates (1, 70, 1) and each block is 2 blocks away from its adjacent blocks, thus the block in row r, column c is at coordinates (2r-1, 70, 2c-1).


== Access debug mode ==
In this version, it's not possible to create a world directly in debug mode. To access a world in debug mode, you need to use an external editor to modify the level.dat file. Defining the generator type of a dimension of a 25w14craftmine world for debug.

When loading the world, some blocks may not render; however, simply the player interacting with them will render them.
