Minecraft: China Edition (Simplified Chinese: 我的世界; "My World"; Pinyin: wǒ de shìjiè; Bopomofo: ㄨㄛˇ ˙ㄉㄜ ㄕˋ ㄐㄧㄝˋ) refers to localizations of Java Edition and Bedrock Edition for the mainland of China, developed by Mojang Studios, Xbox Game Studios and NetEase. China Edition's variants have most of the same parity issues and differences between editions as their non-China equivalents. China Edition has 800 million registered accounts as of April 2025.


== Variants ==
China Edition includes multiple different variants of Minecraft. The Mobile Edition is available as a variant of Bedrock Edition, and the Desktop Edition has both Bedrock Edition and Java Edition variants.


== Mobile Edition ==

The Mobile Edition is designed for mobile devices and offers a modified version of Bedrock Edition.

It offers a launcher to enter the game, which hides the main menu of the vanilla game and directly opens the world. In addition to launching the game, the Mobile Edition also features functions such as group social, community forum, homeland, and resource center.
The launcher has a built-in browser to display announcements and news webpages of China Edition, some of which detect the environment to make the content visible only in-game. The launcher also has a network environment detection feature similar to vanilla Bedrock Edition, which prevents players from logging in when using Network Agent at OS level.


=== Platform ===
Minecraft China Edition of the mobile game supports the following platforms: Android, iOS, iPadOS, and macOS.
Minecraft China Edition is run both on official Netease servers and third-party servers. The official Netease-server-run game version is mainly downloaded from China Edition's official website, or on the Apple App Store for iOS/iPadOS/macOS. Players can only receive invitations for beta versions of China Edition by playing on the official server. Third-party servers, only available on Android devices, are mainly downloaded from third-party app stores or game platforms.
Minecraft China Edition can be run on macOS as a "designed for iPad" compatible. Unlike iOS/iPadOS, complete game files and local cache directories can be visited using Finder on macOS.

=== Development cycle ===
Since the release of China Edition, the mobile game version has undergone 5 major updates per year, usually in January, April, June (or July), September, and November. The Bedrock Edition version is usually updated to the next stable version with significant changes, with an average annual update of the main version number and the secondary version number. During September and November, there are two main updates: anniversary celebration and gratitude season. In April, there is a fixed secondary update without a specific theme, and a weekly hotfix. The main update requires downloading a new version launcher. The release time of the Android official server is may be earlier than the Android channel server and iOS/iPadOS/macOS platforms on the same day. The update is usually on the first or fourth Friday of the month. Within 2 weeks prior to the update, the Android official server invites players to download the development version of the mobile game to test the stability of the new version. The development version of the mobile game can play multiplayer games with players who are also using the development version, but can use the new version of the game content, features, and components in advance, and can receive trial rewards within the game.
Clearing the cache on the main interface removes all accumulated hotfix resource packages and cache files. After restarting, it automatically updates to the latest version.
China Edition versions are identified in this format:X.Y.Z:ABCDEF. 

X is the main version number, which increases by 1 only when there is a major update to the mobile game launcher.
Y is the minor version number, which increases by 1 for each major update and returns to zero after the major version number is updated.
Z is the minor version number, which is displayed as a hotfixed resource version on the game side, and returns to zero after the minor version number is updated. The minor version number for the development version of the mobile game is fixed at 100.
ABCDEF is the development ordinal number, and the larger the number, the newer the version. The iOS/iPadOS/macOS platform is usually equal to or 1-2 different from the Android platform. The development version uses this value to refer to the relationship between the relative game side.


=== Game content ===


==== Singleplayer ====

Singleplayer worlds are Minecraft games played locally, and world save folders may be viewed through file management applications, similar to Bedrock Edition. 
In the game menu, players can directly enable online multiplayer for singleplayer worlds, appearing in the online lobby.

Vanilla

Players can create Vanilla worlds on Minecraft China Edition similar to players on Bedrock Edition. World generation on Minecraft China Edition is identical to Bedrock Edition.
Players may add and remove any number of resource packs downloaded from the in-game Resource Center to one world, but may only use a maximum of three shaders/texture packs at once, also purchased from the in-game Resource Center. Three official resource packs created by Netease, namely, "My Friends", "My Companion", and "Magical Commands" are enabled by default in all vanilla worlds. 
Experiments can be enabled on Vanilla worlds of Minecraft China Edition, similar to worlds on Bedrock Edition. Villager Trade Rebalancing and Upcoming Creator Features are available on China Edition as in Bedrock Edition'. Custom Biomes and Holiday Creator Features are exclusive to China Edition.

Cloud Saves
Singleplayer worlds on Minecraft China Edition can be synchronised and saved to the cloud. Uploaded worlds can be downloaded and accessed on any device logged in with the associated account. Singleplayer worlds can be set to automatically upload to the cloud, or can be manually uploaded. By purchasing a paid subscription service, the amount of storage and save slots available significantly increases. Uploaded worlds are deleted 30 days following a subscription's expiration. The subscription can be set to renew automatically upon expiration.

Free: 1 save slot, 20MB storage space
Basic: 15 save slots, 200MB storage space, price 100 diamonds/per month.
Intermediate: 20 save slots, 1024MB storage space, price 300 diamonds/per month.
Premium: 25 save slots, 2048MB storage space, price 600 diamonds/per month.
Minecraft China Edition Club Members, based on their membership level, are provided with either Intermediate and Premium as a free benefit, as long as their membership is active.

Other Gamemodes
Officially categorized as, simply, "Other Gamemodes," such gamemodes are modified versions of vanilla Minecraft, usually with defined storylines and mostly based upon Chinese folk culture, or science fiction. Most were created and released by Netease with major seasonal updates, with 10 gamemodes released from Q1 2020 to Q2 2022. They are no longer being updated, but all gamemodes remain available and compatible with the latest game version.
Regular versions of "Other Gamemodes" are available for free, which includes the full flow of the game and its main mechanics. Supplementary content for each gamemode can also be obtained by either purchasing advanced editions, or for free by having a valid Minecraft China Edition Club Membership.
"Other Gamemodes" have unique mobs, biomes, dimensions, items, achievements, etc., and can be played in multiplayer, with achievements credited to the player's game account. Game progress from such gamemodes usually serves as the quest objective for the current campaign.


==== Multiplayer ====
Online games
Online games are public, Bedrock Edition servers approved by Netease. Game content is provided and maintained by the server developer. Relevant resource packs are automatically downloaded when entering the server for the first time.
Online games include survival, mini-games, themed RPGs, etc. Online Games servers allow for microtransactions; select products must be purchased with the premium currency "diamonds" for real currency.

Local multiplayer
In the "Local Multiplayer" section of the online lobby, players can join multiplayer rooms created by others, or host their own using their own local world. The player can also set a local singleplayer world to automatically launch to the online lobby whenever the player joins. Hosts automatically gain operator status upon starting a room. Cross-platform play with China Edition PC players can be enabled or disabled. LAN multiplayer can also be enabled or disabled.
"Local Multiplayer" supports resource packs downloaded from the in-game Resource Center. Upon joining a multiplayer room, the resource packs used in the room's world automatically downloads for the player. 
Every online room has a unique 6 digit identification code. Players may join specific rooms by searching the corresponding code. The room host may set a room password. Players in a room can generate a QR code that can be scanned using the launcher "Scan" function. Upon a successful scan, the player automatically joins the corresponding multiplayer room. Players may advertise their multiplayer room on the World Chat. Players may also send link invitations to in-game friends, or to in-game group chats. To assist players in locating rooms of a particular genre, hosts may also advertise their room by labeling the room with two "room themes" of eight: Survival, Creative, Adventure, Simulation, Action, Architecture, PvP, and Leisure. Hosts may also advertise their rooms with a "room slogan." There are 15 preset slogans; custom slogans are available only to Minecraft China Edition Club Members.
"Local Multiplayer" rooms usually accommodate a minimum of 2 players, and a maximum of 10 players. Minecraft China Edition Club Members can host rooms of up to 30 players, depending on the player's membership level. The room host can set an arbitrary room player limit. If the number of players exceeds the limit, others cannot enter. Similar to Bedrock Edition, the host may set player access (Invited players/Friends/Public) and default player permissions (Visitor/Member/Operator/Custom).
Players can report "Local Multiplayer" rooms for violations in-game, or directly in the online lobby of the launcher.

Team Lobby
The entrance to the team lobby is located on the launcher homepage, next to the player nickname. In the team lobby, players can create or join a team. Unlike a room, a team has a separate launcher chat room, and the room owner can change the online gameplay or local online map of the current team. Players can join other players' teams by searching for their names.
The room owner can set a team slogan, which is displayed on the team page. There are 10 preset slogans in total.

Multiplayer Games
Multiplayer Games (Not to be confused with "Local Multiplayer" and "Online Games") is an official free online gameplay component with an in-app purchase store and progress system. The game progress of the quarter game will be used as the mission goal of the current activity. A total of 4 were launched from the third quarter of 2022 to the second quarter of 2023. Updates no longer occur, but all games continue to be available.


=== In-game interface ===
The Mobile Edition has made minor modifications to the interface of the Bedrock Edition. The interface of resource-pack-enabled worlds, online games, and online play may be different from the default interface. Developers can use resource packs to add new buttons or change existing buttons.


==== Controls ====

Players are prompted to select their preferred controls when first entering a singleplayer world; settings can also be altered afterwards in the settings menu, similar to Bedrock Edition. The "new touch control" of Bedrock Edition cannot be enabled at this time.


==== HUD ====

The modified HUD of the Mobile Edition includes five buttons on the top and a action mode button on the right. The six buttons on the top are menu, chat, fold, report, emotes, voice from left to right, and the right side is the action mode switch button.


==== Inventory ====
The Mobile Edition of the inventory has added a mount page, located on the right side of the inventory GUI, where you can view the 3D preview model of the current mount and summon the mount.


==== Menu screen ====
The Mobile Edition has made some changes to the Bedrock Edition menu screen and is highly integrated with the launcher-related functions.

Pause Menu:

Removed the "Marketplace" and "Invite to Game" buttons.
A new "Component" button has been added to view the component information enabled in the current game. It is highly integrated with the launcher interface and supports viewing component details and comment area content.
In the player list, the level system icon is displayed to the left of the player name (in the old version of the level system style).
The lower right corner displays shortcut buttons for online game-related functions, which can quickly switch between single-player games and local connections.
The "Dressing Room" button will jump to the launcher backpack page to change the skin. This function is temporarily closed due to a vulnerability.
Settings Menu:
Removed several options from the Account page.
A new "Video Quality Level" option has been added to the video page, including "Smooth", "Standard" and "HD", which can quickly modify the remaining options on the page to the preset style.
The "Screen Animation" option on the video page cannot take effect immediately and needs to be restarted before taking effect.
A new "Cross Key Operation" option has been added to the touch screen page to switch the operation mode.
There have also been some adjustments to other pages​[more information needed].


=== Social Communities ===
Including Novelties, News and Community, which are official account/player game dynamics publishing area, game news publishing area, and in-game forum.


==== Friends ====

Friend system including adding friends, chatting with friends, playing with multiplayer game, etc. Players can add up to 300 friends.

Intimate Relationships:
Intimate relationships is an evaluation mechanism for the interactive relationship between friends. You can gain intimacy by interacting with friends in the game, with a daily limit of 40 points.
Intimate relationships include: Close Friend, Sister, Brother, and Best Friend. Only one Close Friend can be added, and the total number of Sisters, Brothers, and Best Friends cannot exceed 20. Intimate relationships can be upgraded to higher levels, and intimate benefits can be used when reaching the corresponding level.


==== Chat Box ====

There are five types of chats available in the launcher, namely General, Playground, Friends, Groups, and Server.

General: The server's public multiplayer chat group can be used for communication between stranger players.
Playground: Only available in Block Park, for communication between players in the same main city server.
Friends: Have a one-on-one private chat with your friends.
Group: Created by the group owner, who can set the group avatar and invitation permissions. Players with permissions can invite friends.
Server: Available only in online game servers, used as a group for real-time communication among players in the server.


==== Homeland ====

An in-launcher mini-game that allows players to customize the interior of their own virtual house. Minecoins (not to be confused with Bedrock Edition Minecoins), a unique in-game currency, was used to purchase furniture, decorations, walls, and additional floors from the in-game mall. Minecoins could be earned through solving crafting puzzles. Players can visit other players' houses while playing "Homeland" by either clicking on a friend's portrait, or entering the "Homeland" Plaza to visit other players. Players can "Like" other players' houses when visiting.


=== Resource Center ===

The resource center, similar to the Marketplace in Bedrock Edition', is an in-game platform in which players can purchase and download user-created content packs, such as resource packs, shaders, texture packs, skins, custom worlds, etc.  
Each content pack in the resource center has an independent page, which displays the pack's creator, details, ratings, cost, comments, download count, etc. A pack's rating is evaluated based on the weighted usage behavior of players such as game time. Players meeting requirements may apply for refunds after a purchase. Content packs can be wishlisted, or saved to a player's "Shopping Cart". Content packs can also be gifted to or by in-game friends. Minecraft China Edition Club Members may download and test any content pack for up to 10 minutes in a designated world, free of charge.

Play Pass is a paid subscription service within the Resource Center allows the player to experience content packs designated by the subscription, within the active period of the subscription. Play Pass also provides various levels of the Cloud Save service.

Proof of Courage to Explore: 200+ content packs. Basic Cloud Save Subscription included. Price: ¥15/month.
Proof of your Love for Exploration: 400+ content packs. Intermediate Cloud Save Subscription included. Original price: 3000 diamonds per month.
Proof of Peak Exploration: 500+ content packs. Premium Cloud Save Subscription included. Price: ¥50/month.


=== Store ===

Used to purchase cosmetic items, such as player skins, weapon/armor skins, Footprints or Mounts. All items available in the store are officially created by the Netease team.

Recommended: Displays newly-released skins.
Themed skins: Themed bundles of player skins. Players can purchase three skins at random from a selected theme for 300 Bounded Diamonds, or purchase all skins in the entire themed bundle for 900 Bounded Diamonds.
Bundles: cosmetic packs, lottery tickets and other cosmetic-related bundles purchased with diamonds. Usually discounted.
Appearance: premium and standard skins for purchase, some for Emeralds and Diamonds.
Exchange: Comfortable Wool or Premium Leather currencies can be exchanged for designated skins.
Purchases: For purchasing Diamonds (see below) with real money. Additional cosmetic or currency rewards are provided with additional purchases.
Club Membership: Provides additional benefits. Players can obtain a Minecraft China Edition Club Membership by obtaining "Membership Points," which are obtained by spending diamonds in purchasing Resource Packs from the Resource Center.


=== Other ===


==== Currency ====
The resource center utilizes a two-tier currency system using "emeralds" and "diamonds" (not to be confused with their in-game equivalents). Both currencies use their Programmer Art textures.
 Emeralds are a free currency that can be obtained through methods such as daily logins, monthly gifts, finishing tasks, and occasionally from the mailbox.
 Diamonds are a premium currency that is purchased directly with renminbi (CN¥), and persists even after the end of the month. 1 diamond is equivalent to 1 fen, or 0.01 CN¥. Diamonds can occasionally be obtained in events. Bounded Diamonds are a variant of the Diamond currency that cannot be purchased directly with real money. Bounded Diamonds are rewarded for purchases of standard Diamonds, and occasionally can be obtained in events.

In addition to diamonds and emeralds, there are also other special currencies such as Rubies, Amethyst, Ender Coins, Skin Fragments, Partner Fragments, Homeland Minecoins, Premium Leather.

Events are held from time to time in the launcher, and can be viewed on pages such as the store, events/benefits, hot sales, adventure guides, and seasonal events.

Events/benefits: This is the entrance to most events. The visibility of some sub-pages is related to the content of the current season's events.
Recommendation: Display the event recommendation page, which can be quickly jumped to.
Set sail: Includes limited-time sign-in, daily missions, skin gift packs and other types of events.
Daily: Includes longer-term events.

Adventure Handbook: Exclusive pages for themed season events.
Some of the events are as follows:

Theme Season: A permanent event located on the Adventure Handbook page. The theme is updated every quarter, along with a series of related appearance components. Players can upgrade their Adventure Handbook by completing online missions, and can receive prizes for each upgrade. Extra prizes can be received by purchasing advanced or legendary Adventure Handbooks. The exchange coins obtained can be used to redeem limited event prizes.

Crystal Secret Treasure: A permanent event located on the event page. You can use amethyst currency to draw prizes. The prize pool is updated every week.
Event gameplay: It will be released irregularly starting from the second quarter of 2023, all in the form of events. You can start the game with one click on the event page. Although the gameplay has the same event page, there are also different game types, including local gameplay (also known as local connection) that is a local mashup package component after downloading and online gameplay in the online lobby. The gameplay has multiple missions, it will count the number of times the missions are completed in the game and increase the progress. After the missions are completed, you can receive rewards. After the event, the gameplay is usually removed from the shelves. If there is still a resource center page, you can play it permanently. The removed gameplay can still be downloaded and used through the link shared by others to enter the store page.


==== Level System ====

Every China Edition player is ranked using "Levels" from 1-50 based on their playtime, as well as the amount of Diamonds purchased. A player's level is displayed on their Personal Homepage, in various comment areas, etc. A lower "Level" applies a time limit on how often a player can comment, or post on their Personal Homepage; this limit is reduced up to 1 second as the player's level increases. Players of Oak Rank (level 1-5) are unable to comment or post on their Personal Homepage.
The level icons (ordered low to high) are:  Oak,  Acacia,  Sand,  Cobblestone,  Iron,  Emerald,  Gold,  Diamond,  Purpur, and  Bedrock, each representing 5 consecutive levels. The Gold and Diamond ranks use representations of their Programmer Art textures.


==== Personal Homepage ====

The personal homepage is a page that displays the player's personal information, including Personal Information, Gameplay and Modules, Achievements, and Chat.

Personal Information: Displays player nickname, player ID, portrait and portrait frame, skin, "persona labels", "personal signature" (a signature remark), time played in total/creative/survival/multiplayer, cosmetics owned, location, and resource packs owned. Players can modify the content/visibility of personal information.
Gameplay and resource packs: Displays resource packs owned, duration of resource pack usage, and Resource Center Wishlist
Achievements: (See below)
Posts: Players may share in-game screenshots with text to their Posts board. Posts may be "Liked," shared to both in-game platforms and social media, or commented on.
Message Board: Players may leave comments on others' message boards.
Collection: Used to view a player's cosmetic item collection


==== Achievements ====

The Achievements page is located on the Personal Homepage and data is bound to player's account, similar to Bedrock Edition.
Achievements include five categories: Career, Game, Social, Collection, and Paradise, with a total of 48 achievements. "Achievement Points" are rewarded after attaining an achievement. The "Hero's Road" portrait frame is rewarded for obtaining 3000 "Achievement Points. Each achievement has 4 qualities: beginner, advanced, excellence, and legendary, worth 10, 30, 50, and 100 "Achievement Points" respectively. The game displays a notification at the bottom of the screen when any achievement is attained.

The "Desktop Edition" is the Windows version of China Edition, including Java Classic Edition and Bedrock Cross-Platform Edition, and provides specific editions of Java Edition and Bedrock Edition games.

The Java Classic Edition runs on Windows 7 and later, and the launcher comes with the required Java environment (64 bit JRE8,JDK17 and JDK22). It cannot be installed or run on ARM-based devices. Some versions may experience exceptions on earlier versions of Windows, and the "ray tracing version" runs normally only on supported devices on Windows 10 and Windows 11 platforms.
The Bedrock Cross-Platform Edition runs on Windows 10 and later.

==== Game content ====

The "Start Game" in the bottom left corner of the launcher is the starting point for the game. Players can play various games based on Java and Bedrock versions in the Windows version. The Java version used in the Chinese version has a small amount of text that is slightly different from Standardization of Translations. The content of the base bedrock version is basically the same as that of the mobile game, and it can be used for interactive gaming.
The "Start Game" section prioritizes displaying players' recent games, and new game content can also be searched and downloaded by category. The launcher skips the main menu and directly enter the game.


===== Single player game =====
The single player game section allows players to manage all local game saves across versions. By selecting save, players can view detailed information including version, game duration, installed components, and more. Single player game saves can be exported, and when creating a world, saves can be imported and a world with the same version can be created. When starting the Java version save, multiple components developed by NetEase are forced to be built-in. At the same time, you can choose to open the "Command Suspension Window" built-in component. After opening, a button is displayed in the upper left corner of the game, and you can use commonly used commands with just one click.
Players can choose the following game versions when creating a world:

Java Edition:  1.7.10,  1.11.2,  1.12.2,  1.16.4,  1.18.1,  1.20.1,  1.20.6 , 1.21, (with Forge as the built-in module loader).

In the launcher, the names of Java Edition 1.15.2 and above omit the minor version number, such as 1.16.4 will be displayed as 1.16.

Versions removed from the create world page include: 1.8, 1.8.8, 1.9.4, 1.10.2, 1.13.2, 1.14.3, 1.15.2.
Single player game saves can be uploaded to the cloud for cross device gaming, and players have 5 save slots and 1GB of storage space; Members have an additional 2 archive slots and 2GB of storage space. The archive slots of members are locked after the membership expires and cannot be uploaded or downloaded.


===== Multiplayer games =====
Network Game
Online games are officially certified Java customized servers in the Chinese version of games, with Apollo as the core. The server host is located in NetEase's data center. The game content is provided and maintained by developers, who can list products in the server's online store. These products must be purchased using the launcher currency "Diamond". Players can choose not to display the in-game mall. The Java version of online games has the same range as single player games.
For the first time entering the server, you need to download a client other than the game version. For a single online game, players can create up to 3 character profiles, and any ban imposed due to game account violations also takes effect on all character profiles.

Online Lobby
The online lobby of the Windows version is the same as that of the mobile version, both of which are real-time bedrock version gameplay servers. The online lobby of the Windows version is not compatible with the mobile version.

Local Online
Local online allows players to create any single player game version of the game room, including Java, Rock, and Light Chase versions. With the homeowner's device as the host, any version of the room can be configured with components. When members enter, they download them on the startup interface, and the game can be played directly after it starts.

Rental Servers
Rental server is a Java version of server hosting service, which has two types: pure rental service and custom rental service. Pure rental server is a fully original game server, with customizable components, plugins, and other additional content to provide higher server performance. The custom rental service channel is currently under maintenance and cannot be created.

==== Game Center ====

The Game Center is an online game platform within the PC version of the game launcher, which allows you to quickly classify and filter online games and play them.


==== Component Center ====

The Component Center is a game add-on content distribution platform within the Windows version launcher, similar to Market.
The configurable components of single player games in the Windows version are divided into two categories based on their impact on the game: functional and visual. Visual is further divided into two subcategories: material sic and light and shadow. The above components can be retrieved in the Component Center by version, price, theme, etc. Online games, online gameplay, and more can also be found and purchased in the component center. Slightly different from the international Java version skin, the 128/256 pixel skin component can also be used for Java games.


===== Component Package =====
A component package is composed of several functional and visual components combined to achieve specific game effects, similar to an integration package. Component packages are mainly provided by the community, and any player can filter, edit, and submit their purchased components. Excellent component packages can be recommended by the community.
