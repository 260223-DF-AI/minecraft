13w10a is the fourteenth snapshot for Java Edition 1.5, released on March 4, 2013.


== Additions ==


=== Blocks ===
Detector rails

Added texture for activated rails.


== Changes ==


=== Blocks ===
Command block

Pressing Esc in a command block exits its GUI.


=== Mobs ===
Slimes

Spawn rates in swamps now depend on the current moon phase.
