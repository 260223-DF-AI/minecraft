1.15.1 is a minor update to Java Edition released on December 17, 2019, which improves performance and fixes a few bugs in 1.15. This is the final version released in 2019.


== Changes ==


=== Mobs ===
Dolphin

Now can not spawn in bubble columns.


=== General ===
Performance

Optimized chunk rendering performance, especially for chunks with many different block states.
Network handling

Improved network handling of invalid biome IDs.


== Fixes ==
10 issues fixed
From released versions before 1.15

MC-96174  Highlighted slot darkening.
MC-126856  When auto-jump is enabled, the game checks an incorrect block to decide whether the player should jump or not.
MC-135050  NullPointerException while tessellating block model.
MC-150508  Resource pack format was not incremented to 5.
MC-152516  Dolphins can spawn in soul sand bubble columns.
MC-163571  Walls are lit too dark when connecting to ice.
From 1.15

MC-167482  Corrupt chunk causes force upgrading a world to fail.
MC-167530  Anvils causing java.lang.StackOverflowError.
Other

Fixed an error that was spammed to the game's log files caused by custom monster spawners spawning villagers.
Fixed a crash that occurred in the Realms screen.
