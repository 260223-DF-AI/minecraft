0.29_01 is a version of Java Edition Classic released on October 29, 2009. This version is not available in the launcher but has been archived elsewhere.


== Changes ==


=== General ===
Hitbox

At times, it is possible to build or destroy through other entities, including players.
Multiplayer

Players' names only appear when pointing at them with the crosshair.
Names are no longer hidden through terrain.


== Trivia ==
0.29_01 can be changed to Survival mode by editing the Minecraft.class file in minecraft.jar.
This version was originally rediscovered in a modified state in March 23, 2019.
An unmodified copy was later found by Twitter user lunasorcery on June 25, 2021.
In the blog post for this version, Notch states "You can now build through players". However, this feature does not work like intended.​[more information needed]
