17w31a is the first and only snapshot for Java Edition 1.12.1, released on August 1, 2017.


== Changes ==


=== General ===
Game credits

Updated game credits to include more Mojang staff.
Credits scroll 50% faster when viewed directly from the title screen than through the End Poem.
Splashes

Removed "Better than Prey!"
Skeleton

No longer picks up items based on their enchantments.


== Fixes ==
1 issue fixed
Private issues

MC-119011 – Security issue with recipe book.

== Trivia ==
17w31a was previously removed from the launcher for some time, but was added back in September 2018.
Unlike most of the first-of-the-week snapshots, this snapshot was released on a Tuesday instead of a Wednesday.
