Alpha v1.2.1 is a version of Java Edition Alpha released on November 5, 2010, to fix bugs. Although the launcher claims to have this version available, the version listed is just a duplicate of Alpha v1.2.1_01.


== Changes ==


=== Blocks ===
Leaves

Particle colors now reflect the biome instead of having the grayscale texture.


=== Mobs ===
Hostile mobs

Now only spawn at light level 7 or lower regardless of height.


=== Gameplay ===
Multiplayer

Music now plays on multiplayer.


== Fixes ==
3 bugs fixed

Signs on the sides of cacti no longer crash the game.
Optimized CPU usage for mob spawning.
Music now plays in multiplayer.


== Trivia ==
Excluding pre-reupload versions, Alpha v1.2.1 is the most recent Java Edition version that is currently lost.
