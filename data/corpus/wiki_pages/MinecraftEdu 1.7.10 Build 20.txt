1.7.10 Build 20 is the twentieth build for MinecraftEdu 1.7.10, released on July 2, 2015. It is based on Java Edition 1.7.10. This is the final build in 2015.


== Additions ==
ComputerCraftEdu is now bundled with MinecraftEdu.
"Turtles can build" setting to ComputerCraftEdu teacher menu.
ComputerCraftEdu settings to Server.
German translation.


== Fixes ==
2 bugs fixed

Not being able to spawn animals if monsters were disabled.
Users not being able to run and potions not increasing movement speed.
