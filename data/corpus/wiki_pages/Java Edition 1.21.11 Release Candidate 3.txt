1.21.11 Release Candidate 3 (known as 1.21.11-rc3 in the launcher) is the third and final release candidate for Java Edition 1.21.11, released on December 8, 2025.
A separate unobfuscated version of 1.21.11 Release Candidate 3 was released, titled 1.21.11 Release Candidate 3 Unobfuscated (or 1.21.11-rc3_unobfuscated when in the launcher). This is an experimental version released in preparation for obfuscation being removed from Java Edition clients and servers.
This is the final release candidate released in 2025.
It is also the last release candidate and development version to use the old 1.x.y version format.


== Fixes ==
3 issues fixed
From the previous development version

MC-304790 – Transparent item models are rendered incorrectly.
MC-304797 – Zombie nautiluses generated in ocean ruins are not persistent.
MC-304810 – Zombies, skeletons and zombie horses can catch fire from light sources when exposed to the sky in an arid biome.
