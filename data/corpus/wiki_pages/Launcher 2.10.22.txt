2.10.22 is an update to the launcher released on September 21, 2023.


== Additions ==
Added Lo-Fi Minecraft Melodies.


== Changes ==
Improvements to how Launcher handles game update availabilities.


== Fixes ==
MCL-23699 - Minecraft launcher cannot load after being closed.
MCL-23679 - Long installation names causes the quick play summary for it to become extremely long.
MCL-23680 - Quick Play does not use custom scroll bar.
MCL-23709 - Quick Play icon tooltips appear below other launcher content.
MCL-23727 - New option "Skip java runtime version check" not working.
