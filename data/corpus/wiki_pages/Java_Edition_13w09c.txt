13w09c is the thirteenth snapshot for Java Edition 1.5, released on March 1, 2013 to fix some of the bugs and crashes in the 13w09a and 13w09b snapshots.


== Additions ==


=== Gameplay ===
NBT

Added SeeFriendlyInvisibles tag.


== Changes ==


=== Items ===
Potions

All potions are now categorized in Creative mode.


=== Command format ===
/effect

Amplifier is no longer capped at 4.
The player can now remove effects by running /effect <player> <effect> 0.
