18w47b is the eighth snapshot for Java Edition 1.14, released on November 23, 2018, which improves world generation performance and changes the textures of more blocks and items.


== Additions ==


=== General ===
Client.jar

Added a version.json file to the root of client.jar, which lists information such as data version, protocol version, data pack version, parent version, build date and time, and whether it is a stable build.


== Changes ==


=== Blocks ===
Barrel

facing=south block state texture has been changed.
Impulse and repeating command blocks

Textures have been changed.
Logs 

Top texture of normal and stripped logs have been changed.
Spruce door and trapdoor

Textures have been changed.


=== Items ===
Raw cod

Reverted the texture changes back to the 1.13 version.
Cooked cod

Changed the texture to be similar to the 1.13 texture.


=== Mobs ===
Adjusted mob spawning behavior to be closer to Java Edition 1.13.2 behavior.
Cat

The tuxedo cat's texture has been changed.


=== General ===
Performance

Numerous world generation performance improvements.
Textures

Changed the textures of more blocks and items.
