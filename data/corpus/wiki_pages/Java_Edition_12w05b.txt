12w05b is the fourth snapshot for Java Edition 1.2.1, released on February 3, 2012, which makes small changes to mob AI and fixes a few bugs.


== Changes ==


=== Mobs ===
 Creepers

Have a slightly improved AI.
Now run away from tamed cats and ocelots.


=== General ===
Death screen

Has a slight delay before buttons become active, to avoid accidentally clicking buttons when killed.


== Fixes ==
2 bugs fixed

Fixed shift-clicking the result from crafting only giving half the items and consuming all leftover materials.
Fixed cats being unable to sit when right-clicked with an empty hand.


== Trivia ==
This is the first ever "b" snapshot.
