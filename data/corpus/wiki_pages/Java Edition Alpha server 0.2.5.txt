0.2.5 is a server version of Java Edition Alpha released with client version v1.2.3 on November 24, 2010, without .exe file.
This version was reuploaded at 10:04 UTC (41 minutes after release) to fix broken leaf decaying.​[more information needed]


== Additions ==
New server configuration settings:
pvp <true/false> (true by default). If this is set to false, players cannot directly attack other players (i.e. by hitting them or shooting them with a bow; things like lava still damage them).
spawn-monsters <true/false> (true by default). If this is set to false, all hostile mobs are removed and they cannot spawn.
spawn-animals <true/false> (true by default). If this is set to false, all passive mobs are removed and they cannot spawn.


== Changes ==
The fire on players and mobs who are burning is now visible in multiplayer.
Damage is now correctly given for falling, drowning, and burning.
Eating food restores health.
Players can now respawn.


== Fixes ==
Fixed a block respawning bug in multiplayer.
