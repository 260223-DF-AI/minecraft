Different panoramas have been used in the menu screens in both Java Edition and Bedrock Edition, but they have been replaced for various reasons, such as a new update with a different theme. This page documents all such panoramas.


== Prototype ==

Dinnerbone had stated that the menu panorama was originally going to be an in-game render of a live world, with custom world generation with the seed 100. He also mentioned another panorama that came before the one seen in Beta 1.8, which depicted a plains biome near a beach.


== Original panorama ==

This panorama was introduced to Java Edition in Beta 1.8's early development and used for over six years before its replacement in 1.13-pre1.
The panorama used in Bedrock Edition was identical to the one used in Java Edition. When this panorama was first introduced in Bedrock Edition, it had used the version which fixed the white line inside of it.
This panorama was also in the Early Access builds of Minecraft Education.
This panorama had been taken on the seed 2151901553968352745 or 8091867987493326313 (both are considered valid seeds) at coordinates X=61.48 / Y=75 / Z=-68.73. The seed 2151901553968352745 spawns the player in a plains biome at -156, 244. The panorama distance direction is 380 blocks SW. These seeds work in versions from Alpha v1.2.0 to Beta 1.7.3 (except for population differences). The actual version that was used to generate this panorama must have been a development version between Beta 1.6.6 and Beta 1.7, based on the file dates. Dinnerbone noted that the panorama was taken by Notch while he was travelling, around June 115, 2011.
In Java Edition, this panorama is used from Beta 1.8 Pre-release to 18w22c (includes 2.0, 15w14a, and 1.RV-Pre1).
In Bedrock Edition, this panorama is used from v0.7.3 alpha to 1.1.7. The constituent image files were blurred.
In Minecraft Education, this panorama is used from its initial release to 1.0.1. 
This panorama was never used in Legacy Console Edition, as that edition had its own set of panoramas. Usage of this panorama in other editions is unknown.

== Caves & Cliffs: Part II ==

The panoramas used in Java and Bedrock editions are identical.
This panorama had been taken on the seed 2151901553968352745 at coordinates X=4.83 / Y=130.15 / Z=-7294.28. This seed spawns the player in a windswept hills biome at 0, 0. Distance direction is 7294 blocks S in a snowy taiga biome. This seed generates mostly correct on snapshot 21w40a. Interestingly, this seed is exactly the same seed as the one used in the original panorama that was present in the game from Beta 1.8 until the Update Aquatic. This was an easter egg put in the game by Jappa, thought to be inspired by a tweet. He later confirmed that this was indeed an homage that put emphasis on how much Minecraft had changed over the years.
In Java Edition, this panorama was introduced in 21w41a, currently used in 21w37a to 1.18.2 (includes Deep Dark Experimental Snapshot 1), and formerly used in 22w11a through 22w14a (includes 22w13oneBlockAtATime).
In Bedrock Edition, this panorama is used in versions from beta 1.18.0.21 to beta 1.19.0.20.
In Minecraft Education, this panorama is used in versions from 1.18.32 to 1.18.45.

== Chase the Skies ==

The panorama for both Java and Bedrock editions was taken on the seed 8819392414030687460 at coordinates X=1266 / Y=105 / Z=-125in Java Edition. The world spawn of this seed is in a bamboo jungle biome at coordinates X=0 / Y=102 / Z=0 and the coordinates X=1266 / Y=105 / Z=-125 are located in a birch forest 1272 blocks to the west.
In Java Edition, this panorama was introduced in 25w20a, currently used in 25w15a through 1.21.8, and formerly used in 25w31a to 25w33a.
In Bedrock Edition, this panorama was introduced in Preview 1.21.90.28 and was used until Preview 1.21.110.24.
A separate panorama was introduced in Bedrock Edition Preview 1.21.90.26 to be exclusively used when Vibrant Visuals are turned on. The Vibrant Visuals panorama still exists in the 1.21.90 vanilla resource pack, and can briefly be seen in-game by loading a world locked in 1.21.90 or 1.21.100.
Two panoramas only used in Bedrock Edition Previews were taken in Bedrock Edition on the same seed around the same coordinates. The first panorama is located in coordinates X=1266, Y=109, Z=-126, and the second one is in coordinates X=1266, Y=108, Z=-127. The first panorama was used only for version Preview 1.21.90.25, and the second panorama was used for versions Preview 1.21.90.26 and Preview 1.21.90.27 while Vibrant Visuals was disabled.


== The Copper Age ==

In Java Edition, this panorama was introduced in 25w34a, currently used in 25w31a to 1.21.10, and formerly used in 25w41a to 25w44a.
In Bedrock Edition, this panorama was introduced in Preview 1.21.110.25 and was used until Preview 1.21.130.25. A different panorama shows when Vibrant Visuals is enabled. The Vibrant Visuals panorama still exists in the 1.21.110 vanilla resource pack, and can briefly be seen in-game by loading a world locked in 1.21.110 or 1.21.120. 
The standard panorama was taken in Java Edition, whereas the Vibrant Visuals panorama was taken in Bedrock Edition. Both panoramas were taken in worlds using the seed 7904846446833094596 at coordinates X=119 / Y=65 / Z=-6271. The world spawn of this seed is in an old growth birch forest at X=16 / Y=67 / Z=-32 and the coordinates X=119 / Y=65 / Z=-6271 are located in a plains biome 6240 blocks to the south.
