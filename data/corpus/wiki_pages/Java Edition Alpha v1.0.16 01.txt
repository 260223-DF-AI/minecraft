Alpha v1.0.16_01 is a version of Java Edition Alpha released along with server version 0.1.2 on August 12, 2010, which added new commands for servers.


== Additions ==


=== Command format ===
/tp <player1> <player2>

Moves player1 to player2's location.
/save-all

Forces a full save to the data storage device.
/save-off

Disables all saving (except /save-all).
/save-on

Re-enables saving.


== Changes ==


=== General ===
Can press ↵ Enter on the IP entry screen.


== Fixes ==
1 bug fixed

Fixed a bug with potentially invisible players.
