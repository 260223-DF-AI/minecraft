v0.6.2 alpha was a development testing version for what would become v0.7.0 alpha. It was never officially released to the public, however, there are screenshots and pictures of its existence.


== Additions ==


=== General ===
Pocket Realms (alpha stage)

An official subscription-based server hosting service that allows players to create and manage their own private Minecraft servers.
Chat

Players can press the  button at the top-right of the screen to open the chat and talk with other players.
Pause

A dedicated pause button is also added at the top-right of the screen next to Chat button.


== Changes ==


=== Menus ===
Main Menu

While there is a lack of information on what changes were made in this version. There are a few noticeable changes from Pocket Edition v0.6.1 alpha that later made it to the v0.7.0 alpha.

The background is now a blurred image of a Minecraft world instead of a dirt background.
The version number and Mojang copyright are both now in the bottom-left instead of the top and the bottom-right respectively.
The buttons have been redesigned of a horizontal rectangle instead of square.
The Join Game button has been removed.
The options button has been moved to the bottom-right.
The Start Game button has been renamed to the Play button and remove the block icon.
Added the Play on Realms button in the middle.
Select World

Redesigned
Multiplayer now has a Realms & Local button
Settings

Replaced Android GUI with one rendered in game
Now uses options.txt to store settings.
Removed the Lower graphics quality toggle.
Is now split into four subsections, with some of them being split into further subsections:
Log-in
Game
Inputs (Controls)
Graphics
View Distance
Fancy Graphics
Beautiful skies
Animated water
Experimental:
Sound


== Changes to v0.7.0 alpha ==
There are a few changes that didn't make it to the release build of Pocket Edition v0.7.0 alpha

The Button Texture uses the same darker colored gradient style present in 0.6.1 alpha and prior.
The Settings Button uses the same texture present in 0.6.1 alpha and prior, though it's position has been moved to the bottom right corner and it lacks any text on the button itself.
The Pause button present in this build was removed in the release version of 0.7.0 alpha.
Many other UI elements remain unchanged from prior releases.
