Alpha v1.1.2_01 is a version of Java Edition Alpha released on September 23, 2010. This version is available in the launcher.


== Changes ==


=== General ===
Minecraft now downloads resources from Amazon S3 cloud storage service, instead of directly from Minecraft.net.


== Bugs ==

3 bugs reported

Placing a sign on a cactus crashes the game.
Placing a sign at the world height limit crashes the game.
Torches can be placed on leaves in "Fast Graphics" mode.


== Trivia ==
Due to the change in how the game downloads its resources, this is the oldest version in which sounds work using the modern Minecraft Launcher.
This was the last version to feature the old terrain generator along with the bright green grass, which was an iconic staple of Alpha.
In addition, this was the last version in which the Far Lands would simply not render beyond 32,000,000 blocks out from the center of the map.
This was also the last version to have winter mode.
This was also the last version to generate monoliths on the terrain.
