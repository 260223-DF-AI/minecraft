MinecraftEdu has been in development since November 5, 2011, when it was simply named MinecraftEdu Beta. What is now known as MinecraftEdu refers to the versions of MinecraftEdu during the months from November 2011 to March 2013. The next and final phase, what was called MinecraftEdu, lasted from March 2013 to April 2016 and here MinecraftEdu was discontinued and replaced with Minecraft Education.


== Full release ==
The full release stage of MinecraftEdu's development started in March 2013, and has continued until April 5, 2016 and was discontinued and replaced with Education Edition until the release of Education Edition 0.14.2. MinecraftEdu was the same as Java Edition, it shares most of the code the versions that were used in Java Edition and also in MinecraftEdu were Java Edition Beta 1.6.6, Java Edition Beta 1.7.3, Java Edition Beta 1.8.1, Java Edition 1.0.0, Java Edition 1.1, Java Edition 1.2.5, Java Edition 1.3.2, Java Edition 1.4.7, Java Edition 1.5.1, Java Edition 1.5.2, Java Edition 1.6.4, Java Edition 1.7.10 and Java Edition 1.8.9.

Hosting versions is only available in the 1.7.10 and 1.8.9 of MinecraftEdu.
