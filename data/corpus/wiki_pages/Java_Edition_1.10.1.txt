1.10.1 is a minor update to Java Edition released on June 22, 2016, which changes the hitbox of farmland and fixes a few bugs. It is compatible with 1.10 servers.


== Changes ==


=== Blocks ===
Farmland

Now have a 1516 block tall hitbox (was formerly a full block).
Are now 1516 blocks tall visually.


== Fixes ==
7 issues fixed
From released versions before 1.10

MC-85162  Slight player/mob offset when on farmland block.
MC-99914  DataWatcher/EntityMetadata ID is based on the class load order.
MC-100333  Entities from old ( 1.5.2) worlds are killed when loaded in current versions due to a typo in the health data fixer.
MC-102867  Chat key bound to Unicode character causes crash in Creative inventory.
From 1.10

MC-102072  Structure block  keyboard navigation: Unable to navigate in the "structure name" field of structure blocks, using the arrow-keys, pos1, end or the delete key.
MC-103177  Missing subtitles for polar bears, strays, husks, and wither skeletons.
MC-103497  Ender dragon does not respawn.
