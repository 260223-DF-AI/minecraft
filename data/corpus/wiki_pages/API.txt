Application programming interface (API) may refer to several software interfaces for game customization, online services, and more:


== Minecraft ==
Script API – the JavaScript API used to interact with the game in Bedrock Edition.
Mojang API – Minecraft online services for Java Edition, used for game authentication, player profile query, skin retrieval, and etc.
Microsoft authentication – Microsoft online services for Java Edition and Bedrock Edition, used for Xbox account authentication, game ownership, and etc.
