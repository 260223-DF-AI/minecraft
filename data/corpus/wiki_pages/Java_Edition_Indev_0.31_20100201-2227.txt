A version of Indev 0.31 was released on February 1, 2010 at 22:27 UTC.


== Changes ==


=== Blocks ===
 Bedrock

Hardness now equals -1.
 Log

Now breaks faster than stone.
 Ores

Now need a pickaxe to break the block.
No longer possible to mine ores without the correct tier pickaxe.
Ores drop only 1 of their respective mineral when smelted or mined.
 Sponge

Now soaks up water spawner and lava spawner blocks.
 Stone

Now need a pickaxe to break the block.
No longer possible to mine stone without the correct tier pickaxe.


=== Items ===
 Pickaxes

Decreased mining speed of pickaxes.
Tools

Harder materials now require better tools.


=== User interface ===
The item forms of most blocks are now lit differently from before. The right side is now shaded identically to the left side, which remains unchanged from the previous version. If a block used a solid white texture, the top face would use the color  #FFFFFF, and both the left and the right face  #666666.


== Fixes ==
1 bug fixed

Reverted bug fix with crashing the game during world generation offline from previous version.


== Bugs ==
2 bugs reported

When generating a new level any grass block below flower turns into dirt.
When inside non-solid blocks the fog appears within two block radius.
