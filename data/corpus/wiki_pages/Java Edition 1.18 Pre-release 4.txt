1.18 Pre-release 4 (known as 1.18-pre4 in the launcher) is the fourth pre-release for Java Edition 1.18, released on November 17, 2021, which fixes bugs and crashes.


== Fixes ==
4 issues fixed
From the 1.18 development versions

MC-241172 – Rare Datafixer error occurred in old world.
From the previous development version

MC-241774 – Crash when going nether portals or creating a single biome world with nether biomes // java.lang.IndexOutOfBoundsException: Index 0 out of bounds for length 0.
MC-241775 – Crash when going through end portals or creating a single biome world with end biomes // java.lang.NegativeArraySizeException: -5.
MC-241778 – Game crashes or freeze when loading and upgrading the 1.17.1 version of the Superflat world // java.util.concurrent.CompletionException: z: Biome decoration.

== Trivia ==
This Pre-release was released around two hours after the previous one released.
This was the first time two Java Edition developmental versions were released on the same day since 19w38a and 19w38b, which were released more than two years before.
