The Official Minecraft Server List is a website launched by Mojang Studios and GamerSafer, that allows players to find servers to play in for Java and Bedrock editions. It was launched in November 2023, but was in development for about 1 year.


== Overview ==
The website features a list of servers for the player to choose from, where each server contains its own categories based on game modes, features, server type, and other factors, while displaying the available badges on the server box.
According to Minecraft.net's article, all listed servers must comply with the Minecraft Usage Guidelines (MUG), and they must provide official contact information and state the purpose of the server.
Any server owner with a significant amount of players may be able to apply to the website, through the Add Your Server page and following the listed Basic Eligibility Requirements.


== Badges ==
All servers listed on the Official Minecraft Server List have a badge, which according to the Badges page are awarded for servers who showcase their work and commitment to safe and enjoyable experiences.
