13w37b is the fourth snapshot for Java Edition 1.7.2, released on September 13, 2013 to fix some of the bugs and crashes in the 13w37a snapshot.


== Fixes ==
3 issues fixed
From released versions before 1.7.2

MC-15547 – Structures (witch huts, nether fortresses etc.) don't get saved with the world file.
From the previous development version

MC-30796 – 2 portals overlaying each other does not render correctly.
MC-30806 – Exception with /give command.


== Trivia ==
13w37b was previously removed from the launcher for some time, but was added back in September 2018.
This is the first snapshot to be released on Friday the 13th.
This snapshot was released on the same day as the 1.6.3 pre-reupload.
