Build 0016 (or Build 1.66.0016.0) is the earliest known development version of Xbox 360 Edition. It was showcased at MINECON 2011 and is based on Java Edition Beta 1.6.6.


== Features ==
Everything from and before Java Edition Beta 1.6.6, written in C++ instead of Java.
Console-exclusive controls and menus.
A tutorial world.
Exclusive graphics specifically relating to MINECON 2011, such as an overlay displaying "Work In Progress" and a small MINECON 2011 logo at the top left.


== Differences ==
The chat box in this version more closely resembled Java Edition's, due to the larger font size and the background being smaller, not casting all the way to the right like the final version does.
		
			
			
		
		
			
			
		


== Trivia ==
This is the first ever version of Minecraft to be shown on the Xbox 360.
This build features a tutorial world completely unique to this build. It is unknown when the tutorial level was changed to the final revision during development.
