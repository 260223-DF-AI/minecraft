1.4.5 is a minor update to Java Edition that was released on November 19, 2012, to fix some of the bugs and crashes in 1.4.4. It is compatible with 1.4.4 servers.
A pre-release version of 1.4.5 was released on November 16, 2012. This pre-release was later reuploaded, and this reupload proceeded to become the full release of 1.4.5.


== Changes ==


=== Items ===
 Map

Now showing all of the players correctly in multiplayer.


=== Mobs ===
 Wither Skeleton

Now spawns in Nether Fortresses that generated in old versions.
