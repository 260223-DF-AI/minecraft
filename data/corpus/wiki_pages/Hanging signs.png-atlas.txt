hanging_signs.png-atlas was a procedurally-generated texture atlas used to store textures for all hanging signs.


== Format ==
It contained all hanging sign textures. However, it is unknown if it was actually used for rendering hanging signs; in the same version in which this atlas was introduced, all of the existing hanging sign textures were also added to signs.png-atlas.​[more information needed]


== History ==


=== Versions of the atlas ===


==== Mipmapped versions ====
Minecraft would also generate mipmapped versions of this atlas, for unknown reasons. They were not used by the game at all - only blocks placed in the world which use the json model system use mipmapped textures.
