This is a list of Minecraft Education 1.21.91 development versions.


== Preview 1.21.40.1 ==
1.21.40.1 is the first Preview version for Minecraft Education 1.21.91, released on November 13, 2024, which adds features from the Bundles of Bravery drop.


=== Additions ===

Added features from Bedrock Edition 1.21.20 to Bedrock Edition 1.21.40, Bundles of Bravery.
Except Hardcore mode.


== Preview 1.21.40.2 ==
1.21.40.2 is the second Preview version for Minecraft Education 1.21.91, released only for Windows on November 27, 2024.​[more information needed]


== Preview 1.21.50.1 ==
1.21.50.1 is the third Preview version for Minecraft Education 1.21.91, released for ChromeOS and Android on December 17, 2024, and for Windows on December 22, 2024, which adds features from The Garden Awakens drop.


=== Additions ===

Added features from Bedrock Edition 1.21.50, The Garden Awakens.


=== Changes ===


==== General ====
Cloud storage

Support for larger world sizes.
Lower error rates and better upload and download performance.
Multiplayer and Coding

Multiplayer sign-in and connecting updated for better reliability.
Coding infrastructure updated for better stability & performance.


=== Fixes ===
Improvements to performance and product stability.


== Preview 1.21.51.0 ==
1.21.51.0 is the fourth Preview version for Minecraft Education 1.21.91, released on January 28, 2025.


=== Fixes ===
Improved client stability.


== Preview 1.21.80.1 ==
1.21.80.1 is the fifth Preview version for Minecraft Education 1.21.91, released on May 7, 2025, which adds features from the Spring to Life drop.


=== Additions ===

Added features from Bedrock Edition 1.21.60 to Bedrock Edition 1.21.80, Spring to Life.
Except Ore UI, new Superflat presets, and worlds pausing in the pause menu.
Re-added the "Only Allow Trusted Skins" option.


=== Changes ===
Cloud storage system

Now supports larger world sizes.
Lowered error rates and better upload and download performance.
Multiplayer and Coding

Coding infrastructure updated for better stability and performance.
Improved performance and product stability.


=== Fixes ===
Improved client stability.
Multiplayer sign-in and connecting updated for better reliability.


== Preview 1.21.90.1 ==
1.21.90.1 is the sixth and final Preview version for Minecraft Education 1.21.91, released on June 16, 2025, which adds features from the Chase the Skies drop.


=== Additions ===

Added features from Bedrock Edition 1.21.90, Chase the Skies.
Except Vibrant Visuals.
Added Spectator mode from Bedrock Edition 1.19.50.
Added new superflat changes from Bedrock Edition 1.21.80.
Added Ore UI for new world screen from Bedrock Edition 1.19.20.
Added read/write file functionality with MakeCode.


=== Changes ===
Code Builder IDE changes

Tynker has been removed as a supported IDE.
In-game notifications were added so everyone is aware of the upcoming changes.
