== 26.1.2 Release Candidate 1 ==
26.1.2 Release Candidate 1 (known as 26.1.2-rc-1 in the launcher) is the first and only release candidate for Java Edition 26.1.2, released on April 8, 2026, which tweaks UI checkboxes and fixes an exploit.


=== Changes ===


==== General ====
UI

The checkbox message tooltip is now displayed only when text overflows past 2 rows.
The confirmation checkbox in the Report Player screen now displays a tooltip with the same text as the message.


=== Fixes ===
Fixed players in Spectator mode being able to attack other players.
