Beta 1.19.60.27 (Android) or Preview 1.19.60.27 (Windows, iOS, iPadOS, Xbox) is the seventh and final beta/Preview version for Bedrock Edition 1.19.60, released on January 19, 2023, which fixes bugs.


== Fixes ==
Gameplay

Fixed a bug where enemies from "Spellrune" by Gamemode One didn't perform any melee attacks.
This also fixed a bug which was present in older worlds which caused ravagers not to attack.
Fixed a bug that could sometimes cause players to fall through the world after using End Gateways.
Entities no longer disappear when sent through End Gateways. (MCPE-164985)
Performance and Stability

Fixed a crash that could occur when sending entities through End Gateways. (MCPE-165496)
Fixed an issue that occurred when trying to place items into a container output slot.
Player Skins

Fixed a bug that prevented capes from appearing correctly on classic skins. (MCPE-165986)
User Interface

Fixed a bug where players sometimes can only see a black screen after switching back to the game from running other applications on iOS and iPadOS.
