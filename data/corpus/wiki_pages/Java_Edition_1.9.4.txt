1.9.4 is a minor update to Java Edition released on May 10, 2016, which fixes bugs. It is compatible with 1.9.3 servers.


== Changes ==


=== General ===
Warnings

Removed incorrect snapshot version warning when loading a world saved in an older version.


== Fixes ==
5 issues fixed
From released versions before 1.9

MC-92057  Particles are dark at Y256.
From 1.9

MC-94670  Bat subtitle is misspelled ("squeeks").
MC-95574  Minecraft freezes when window loses focus.
MC-100116  [r=x] selector is not selecting mobs or players in the full radius when ran in command blocks.
From 1.9.2

MC-100313  Sweep attack sometimes does not work after switching slot while standing still.
