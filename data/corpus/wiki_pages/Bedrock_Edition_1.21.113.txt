1.21.113 is a hotfix to Bedrock Edition released on October 9, 2025, which fixes some bugs.


== Changes ==


=== Technical ===
The all_slots_empty, any_slot_empty, and has_equipment filters now support main_hand as an equipment location.


== Fixes ==
4 issues fixed
MCPE-226250  Hopper destroys bundle contents
MCPE-228459  Copper Golems do not drop item when right clicked
Other

Fixed a crash when using %.<number>s specifier with a length longer than the multiplied string.
Fixed a potential crash if an Actor query in script was happening during world load.
