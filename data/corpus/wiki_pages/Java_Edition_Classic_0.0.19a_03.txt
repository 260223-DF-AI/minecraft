0.0.19a_03 is a version of Java Edition Classic released on June 20, 2009, at 00:09 UTC to fix bugs.


== Fixes ==
4 bugs fixed

Tried to fix a bug where the client incorrectly throttles players that get kicked for running too fast.
Fixed mirroring.
Fixed the default player stance, but mobs still have glitched skins.
Fixed rotation interpolation bugs.


== Bugs ==
1 bug reported

Mobs have glitched skins.


== Trivia ==
This version has identical gameplay to 0.0.19a_04 due to that version's only change being a bug fix that did not work.
