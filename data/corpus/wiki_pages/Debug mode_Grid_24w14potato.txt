This page lists all the rows in the debug mode block grid in 24w14potato. 
There are 172 lines with up to 173 block states. The last line has 19 block states. Therefore, there are 29,602 block states in total.
The first block is located at coordinates (1, 70, 1) and each block is 2 blocks away from its adjacent blocks, thus the block in row r, column c is at coordinates (2r-1, 70, 2c-1).
