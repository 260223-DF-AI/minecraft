Potatofication is a joke enchantment added in 24w14potato.


== Usage ==
A tool enchanted with Potatofication causes blocks mined by it to drop either regular potatoes or poisonous potatoes. Block must be mineable by that tool to drop potatoes, for example, mining potone with a pickaxe enchanted with Potatofication doesn't drop anything, but a hoe enchanted with Potatofication does drop both the block and potatoes. 
The number of potatoes dropped is calculated in the following way:

Split the level into two non-negative integers n and m.
For Potatofication I, only valid splits are 1+0 and 0+1. For Potatofication V, valid splits include 5+0, 4+1, 3+2, 2+3, 1+4 and 0+5.
Drop n2 poisonous potatoes and m2 regular potatoes.
A poisonous potato plant is added to the drops 5% of the time.


== Issues ==
Issues relating to "Potatofication" are not maintained on the bug tracker because it is an April Fools' joke, and is therefore not in the newest stable version or snapshot. Issues reported there are closed as "Invalid".
