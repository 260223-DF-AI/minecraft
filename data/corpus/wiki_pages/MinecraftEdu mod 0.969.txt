0.969 is a minor update to MinecraftEdu. It is based on Java Edition Beta 1.8.1, which adds the logo for the game, a new block, minor changes, and gameplay news.


== Additions ==


=== Blocks ===
 Spawn Block

Would update the world spawn point to that location. If a spawn block already existed in the world, the spawn point would always be overwritten in favor of the newest block.
Spawn points set by the spawn block behaved identically to world spawn points set using commands.
Student players also had the ability to teleport themselves to the current spawn block from their student menu.
