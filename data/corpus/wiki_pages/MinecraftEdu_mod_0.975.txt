0.975 is a major update to MinecraftEdu, which adds a new block, new building tools, and a new Teacher menu tab. It is based on Java Edition 1.0.0.[more information needed]


== Additions ==


=== Blocks ===
 Foundation block

Foundation blocks measured distances to other foundation blocks.
Allows measurement between foundation blocks.
When equipped, measures distance to other foundation blocks in 3 dimensional space.
Can only be used when in build mode, where the player has an outlined box shown.


=== General ===
New advanced Build Mode for quick world editing
Teleport blocks for public and private use
Better support for sharing long texts in game
More fine-grained control of student powers
Many other little fixes and improvements[more information needed]


== Changes ==


=== General ===
Teacher menu 

Added a new tab for building tools.
Build Tools
Use the MinecraftEdu build tools to create and modify the world faster. These tools include:
Build mode
Changes the way the player moves to be more precise
The player can move through solid blocks
Enables Long Distance Building which allows the player to place blocks up to 60 blocks away
Fill/Clear tool
Allows the player to fill or clear large areas by selecting start and end positions
Custom Place and Dig amounts
Set how many blocks are placed or dug with a single mouse click.
Undo options
Undo latest events if the player does not like what they did.
