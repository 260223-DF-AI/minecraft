Rib armor trim is a type of smithing template used in smithing tables to add a trim to armor. It is consumed when used, but can be duplicated using an existing template, netherrack, and diamonds.


== Obtaining ==
This item is found in nether fortresses. Once obtained, it can be duplicated using the crafting recipe below.

== Usage ==
It is an ingredient in its own recipe, making it possible to craft copies after the initial item is found.


=== Crafting ingredient ===


=== Smithing ingredient ===

Using the smithing table, the rib armor trim can be applied to any armor piece, using any trim material. The rib armor trim is intentionally darker than other armor trims.
