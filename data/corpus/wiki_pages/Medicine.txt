The medicine is a type of potion that removes the specified effect instead of applying it.


== Obtaining ==


=== Brewing ===
Cures are brewed from awkward potions using different elements. These drinks remove the specified effect when drank, and cannot be modified into splash, lingering, extended or enhanced versions.


== Usage ==
Each type of medicine has an associated effect that it can cure (see  Brewing for details). Players can drink the medicine only if they have the corresponding effect. Drinking the medicine eliminates the effect. Although medicine does not resemble potions, the player still gets the glass bottle back.
