This is a list of Launcher 2.2.1032x development versions.


== 2.2.9699 ==
2.2.9716 (Windows), 2.2.9699 (Windows, via the Microsoft Store/Xbox app), 2.2.9709 (macOS), or 2.2.9700 (Linux), is an update to the launcher released on February 1, 2022.


=== Changes ===
Updated Preview images on the Bedrock tab to more beautiful and less blurry versions.


=== Fixes ===
Fixed several crashes and under-the-hood user issues.
Fixed loading problem that sometimes occurred during a game installation.
MCL-17395 - The EN_US strings, "Credits and Third party licenses" and "Third party licenses", are both missing a hyphen between the words "Third" and "party".
MCL-20855 - Using Tab key in Game Output will select two invisible "buttons".


== 2.2.1027x ==
2.2.10271 (Windows), 2.2.10272 (Windows, via the Microsoft Store/Xbox app), 2.2.10274 (macOS), or 2.2.10273 (Linux), is an update to the launcher released on February 7, 2022.


=== Changes ===
Fixed some links from the Launcher which were pointing to the wrong places.
Updated textures of the ore icons in Installation screen.


=== Fixes ===
Fixed some user launch issues, crashes, and other critical bugs.
MCL-13519 - Launcher fails to launch game and then incorrectly claims that the game is already running.
MCL-17983 - Outdated ore textures in installation icon dropdown.
MCL-21061 - Instead of using saved password users now have to enter two passwords.
