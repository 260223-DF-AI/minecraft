Achievements may refer to:


== Minecraft ==


=== Bedrock Edition ===
Achievement – the achievement system in Bedrock Edition.
Trophies – the same system under a different name for the PlayStation consoles.


=== Java Edition ===
Achievement/Java Edition – the outdated system used in older versions of Java Edition.
Advancement – the successor of the previous achievement system.


==== Commands ====
Commands/achievement – a command used to grant or remove the player an achievement in older versions of Java Edition.
Commands/advancement – a command used to grant or remove the player an advancement.


=== Legacy Console & New Nintendo 3DS editions ===
Achievement/Legacy Console and New Nintendo 3DS editions – the achievement system for Legacy Console Edition and the New Nintendo 3DS Edition.


=== Tutorials ===
Achivement guide – a guide from the Minecraft Wiki to help players obtain achievements in Bedrock Edition and Legacy Console Edition.


== Minecraft: Story Mode ==
Achievement – the achievement system in Minecraft: Story Mode.
Trophy – the same system under a different name for the PlayStation consoles.


== Minecraft Earth ==
Challenges - a mechanic used to guide the player into features of the game while earning experience simultaneously.


== Minecraft Dungeons ==
Achievement – the achievement system in Minecraft Dungeons.
Trophy – the same system under a different name for the PlayStation consoles.


== Minecraft Legends ==
Achievement – the achievement system in Minecraft Legends.
Trophy – the same system under a different name for the PlayStation consoles.
