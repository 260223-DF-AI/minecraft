0.0.14a_01 is a version of Java Edition Classic released on May 27, 2009, at 22:36 UTC, to try to fix a bug preventing players from loading saved worlds.


== Fixes ==
Tried to fix a bug from the previous version that prevented players from loading saved worlds.


== Bugs ==
4 bugs reported

The player can no longer place blocks in liquids.
This version had a problem with some players getting NullPointerException errors.
Some users could not load the game and got a black or white screen instead.
There was an issue with mouse control.


== Trivia ==
Many features erroneously said by Mojang to be implemented in this version were actually added in 0.0.14a.
