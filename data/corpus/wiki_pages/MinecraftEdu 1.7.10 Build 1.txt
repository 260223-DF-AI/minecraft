1.7.10 Build 1 is the first build for MinecraftEdu 1.7.10, released on July 2, 2014. It is based on Java Edition 1.7.10.


== Additions ==


=== 1.7.10 ===


==== General ====
Minecraft Realms

Player activity monitor.
The ability to upload one's own worlds.
Access to more server settings.
PVP: On/Off.
Difficulty: Peaceful/Easy/Normal/Hard.
Spawn protection: Off/1-16.
Game Mode: Creative/Adventure/Survival.
Spawn Animals: On/Off.
Spawn Monsters: On/Off.
Spawn NPCs: On/Off.
Command Blocks: On/Off.


== Changes ==


=== 1.7.10 ===


==== General ====
Minecraft Realms

All Realms-related modules were moved into a separate library.
Future additional features to Realms will not require new releases of Minecraft.
Improvements to the user interface.


=== 1.7.9 ===


==== General ====
Naming System

Improved name conversion speed.
Skin system

Added support for the new skin system.
Loosened restrictions so NPCs and similar are still possible for now.
Servers still have to send signed blobs to show skins, but there are no more timeout & "is it for this player" checks.


=== 1.7.8 ===


==== General ====
Servers

Client will auto-kick the player from the server after 23 hours, 59 minutes.
Twitch streaming

Now asks for confirmation before starting streaming to avoid accidental broadcasts.


== Removals ==
Worldedit is removed temporarily.
