0.2.0 is a server version of Java Edition Alpha released with client version v1.1.0 on September 10, 2010.


== Additions ==
Added server-side inventory.
Made the server save player locations when they leave the server.


== Changes ==
Made chests, signs, and furnaces work in multiplayer.
Added monsters=false to server.properties.
It can be turned on for experimental monsters, but they cannot be hurt directly.
