1.6.4 Build 4 is the fourth build for MinecraftEdu 1.6.4, released in October 2013. It is based on Java Edition 1.6.4.


== Additions ==


=== General ===
Launcher

Launcher and Server tool now creates a "mods/mcVersion" folder if it doesn't exist.
Usually used for mods.


== Fixes ==
8 bugs fixed

Animals setting sent Villagers chat message.
Logging in with invalid credentials hanging in login screen.
Auto login works if remember me was ticked in launcher.
Students being able to always edit signs.
Students being able to place blocks to signs even if student building was disabled.
Command blocks not handling edu commands.
When switching creative gamemode to minecraftedu player will now fall to ground if in air.
Visual fill tool not rendering blocks.
