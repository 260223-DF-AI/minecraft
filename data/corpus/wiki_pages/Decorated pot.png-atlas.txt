decorated_pot.png-atlas is a procedurally-generated texture atlas used to store textures for decorated pots.


== Format ==
decorated_pot.png-atlas is a generated 128×64-pixel file that stores textures for decorated pot elements.


== History ==


=== Versions of the atlas ===


==== Mipmapped versions ====
Before 25w31a, Minecraft generated mipmapped versions of this atlas, for unknown reasons. They were not used by the game at all - only blocks placed in the world which use the json model system use mipmapped textures.
