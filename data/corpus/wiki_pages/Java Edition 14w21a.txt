14w21a is the twenty-fifth snapshot for Java Edition 1.8, released on May 22, 2014.


== Changes ==


=== Entities ===
General

Tags CustomName and CustomNameVisible now work on all entities.


=== Command format ===
/clone

Mode now can additionally be move or force.
move moves the selection from the first location to the second location, which may overlap.
force forces the cloning operation, even if the areas overlap; to use this option, either replace or masked must be specified.
