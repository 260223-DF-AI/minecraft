1.14.1 Pre-Release 1 is the first pre-release for Java Edition 1.14.1, released on May 7, 2019, which fixes several bugs and improves performance.


== Additions ==


=== General ===
Controls

Pressing Esc while holding F3 will pause the game without bringing up the pause menu.
Splashes

Added "Awesome game design right there!"


== Changes ==


=== Blocks ===
Crafting table

Updated texture to fix incorrect planks texture.
Iron ore

Updated texture to fix miscolored pixels.


=== Items ===
Leather horse armor

Changed texture to be more consistent with the other types of horse armor.
Iron horse armor

Tweaked colors slightly.


=== World generation ===
Villages

Many structures are tweaked to have better lighting.


=== General ===
Controls

Pressing F3 + N when in Survival mode now changes the player into Creative mode, if cheats are enabled.
