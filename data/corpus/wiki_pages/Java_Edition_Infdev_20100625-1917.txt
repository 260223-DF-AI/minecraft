A version of Minecraft Infdev was released on June 25, 2010, adding dungeons, saddles, and monster spawners, marking the second Seecret Friday update.


== Additions ==


=== Blocks ===
 Monster spawner

Found in dungeons.
Spawn one of the following mobs: zombie (50%), skeleton (25%), or spider (25%).


=== Items ===
 Saddle

Can be found in dungeons.
Used to ride pigs.


=== Generated structures ===
 Dungeon

The chests may contain sulphur, string, wheat, bread, iron ingots, buckets, saddles, and rarely golden apples.
Moss stone is now obtainable in normal gameplay due to it spawning in dungeons.


== Changes ==


=== World generation ===
 Trees

The appearance density has decreased.


== Technical ==


=== Player format ===
[Byte] OnGround tag.
0 if the player is in the air, 1 if on ground.


== Bugs ==
5 bugs reported

The Far Lands now lag and have positioning errors, allegedly different from those introduced in Infdev 20100624.[verify for Java Edition]
Being 15 blocks near previously saved signs crash the game.
If a minecart is used by the player and the player breaks it, it crashes the game.
The game crashes when changing window's height to 0.
