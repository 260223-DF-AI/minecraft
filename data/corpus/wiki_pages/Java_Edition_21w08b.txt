21w08b is the twelfth snapshot for Java Edition 1.17, released on February 25, 2021, which fixes two issues with ore distribution.


== Fixes ==
2 issues fixed
From the 1.17 development versions

MC-216101  Redstone ore generates with a flat distribution below Y=12.
From the previous development version

MC-216744  Iron ore does not generate above Y=0.
