16w15a is the second snapshot for Java Edition 1.9.3, released on April 11, 2016, which fixes a bug related to mob pathfinding which caused performance issues.


== Fixes ==
1 issue fixed
From released versions before 1.9.3

MC-94438 – Server: Mob pathfinding induced massive lag in any versions 15w49a and later.
