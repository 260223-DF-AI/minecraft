0.27 SURVIVAL TEST is a version of Survival Test released on October 24, 2009. This version is not available in the launcher, but has been archived elsewhere.


== Changes ==


=== Blocks ===
 Lava

Texture changed from .
Lava texture side now has a bit of the water texture on top (best seen when blowing up TNT).
 Water

Changed texture from .
The water texture side now has a bit of the placeholder block texture on top (best seen when blowing up TNT).


=== Gameplay ===
Controls

The swing speed of the player's arm is now slowed down to one swing per second when held, from 4.
This is a hard limit. The arm will no longer swing faster if the player clicks faster.
Score points

Changed score.
Zombies now give 80 points upon death.
Skeletons now give 120 points.
Creepers now give 200 points.
Spiders now give 105 points.


== Fixes ==
1 bug fixed

Killing a spider during rain no longer crashes the game.[verify for Java Edition]


== Bugs ==
1 bug reported

The games performance randomly drops excessively under certain conditions.
