A test instance block is a technical block that is used for game tests.


== Obtaining ==
Test instance blocks are available using the /setblock, /fill, /give, or /test commands, and are available in the Creative inventory under the Operator Utilities tab. When placed, test instance blocks are unbreakable in Survival and have the same blast resistance as bedrock.


== Usage ==
Test instance blocks cannot be placed in Survival. When placed, using the test instance block opens its GUI. The GUI opens only if the player is in Creative mode and has permission level 2 or higher. Based on the status of the test, the block produces a beam similarly to a beacon, with the color being  #808080 for a running test,  #00ff00 for a successful test,  #ff0000 for a failed required test, and  #ff8000 for a failed optional test.
