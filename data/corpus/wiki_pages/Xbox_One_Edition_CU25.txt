CU25  is a version of Xbox One Edition released on June 21, 2016.  
[edit]


== Additions ==
Added the Battle mini game.
Added an option for Safe Sprint in the Controls menu.

Removed the Logo in the controls menu.
Depending on the controls layout selected, this option moves either the "Change Camera" mode or "Sneak/Dismount" away from the stick button, since many press the stick by accident when they start sprinting.
Added caves and ravines to the Custom Superflat options.
Added distribute-items functionality via a toggle button (start and confirm distribution).
Allows a stack of items to be distributed by dragging the stack across multiple slots in the Inventory.


== Changes ==
Enabled Battle Map Pack 1 and Battle Map Pack Season Pass DLC.
Updated the "Skin Select" menu.
Switching the selected item in the hotbar now shows the first 3 rows of the item's lore such as attributes, enchantments, or effects from potions instead of just the name. It also shows the correct rarity color.
Equipping armor using the quick equip functionality now plays a sound, which is unused in Java Edition
The player's hand now disappears from the death screen in first person view after 1 second.


== Fixes ==
13 issues fixed
MCCE-1854  Grass positioning is perfect
MCCE-1903  Spawn Below World Glitch
MCCE-1933  Potion of Leaping Color is Wrong
MCCE-1971  Crash when 2 liquids break skulls
MCCE-2005  Slime-chunk-based slime farm no longer working
MCCE-2007  Slime Chunks Not Spawning ANY Slimes
MCCE-2021  "Tie Dye Outfit" achievement is buggy.

Fixed a crash when a client player entered a game where the maximum number of item frames had opened maps.
Fix for slimes not spawning in slime chunks.
Fixed an issue allowing players to use the armor stand when the "Trust Players" option is disabled.
Fixed an issue where items could not be moved from the furnace to the main inventory using quick move.
Fixed an issue in the Superflat world generation menu with the stem mushroom block.
