1.20.13 is a minor update to Minecraft Education, released on November 8, 2023, which fixes some bugs.


== Changes ==


=== Gameplay ===
Game mode

Changed default game mode for new worlds back to Creative.


=== General ===
Multiplayer

Improved some multiplayer error messages to make them easier to understand.


== Fixes ==
Some sign-in failures will be automatically retired.
