0.0.20a is a version of Java Edition Classic released on June 20, 2009, at 20:53 UTC, which added inventory and cloth blocks.


== Additions ==


=== Blocks ===
  Block of gold
  Cloth 

There are 16 different colors.
  Dandelion

Decay when in darkness.
Referred to internally as "flowerYellow".
  Rose

Decay when in darkness.
Referred to internally as "flowerRed".
  Brown mushroom

Decay when in darkness.
  Red mushroom

Decay when in darkness.


=== Gameplay ===
Inventory

Accessed by pressing B and lists all available blocks.
All available blocks previously used to be on the hotbar.


=== Command format ===
/solid

Toggles between placing stone and placing bedrock.


== Changes ==


=== Blocks ===
Bedrock

Players can no longer destroy bedrock, although admins can.
Cobblestone

Is now available again through the new inventory.
Sand

Is now available again through the new inventory.


=== Gameplay ===
Controls

The scroll wheel for block selection was inverted.


=== General ===
Skins

Made the player and human skin darker.


== Technical ==


=== General ===
Removed the FPS limiter because it would cause the computer's clock to speed up.


== Fixes ==
2 bugs fixed

Human mobs no longer have glitched skins.
Water texture no longer overlaps, causing z-fighting.[verify for Java Edition]


== Bugs ==
3 bugs reported

Players can no longer connect to some servers.
Players can no longer break bedrock in singleplayer.
Performance has reduced.
