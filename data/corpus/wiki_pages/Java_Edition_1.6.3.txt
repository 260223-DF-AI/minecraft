1.6.3 is the first and only pre-release for Java Edition 1.6.4. It was released to provide a fix to allow structures to be saved with the world file.
1.6.3 was reuploaded on September 17, 2013.[more information needed]


== Changes ==
The game mode selector in the Create New World screen no longer has a colon in its text (i.e. it reads "Game Mode Survival" instead of "Game Mode: Survival").


== Fixes ==
2 bugs fixed

MC-15547  Structures (witch huts, nether fortresses etc.) do not get saved with the world file.
In order for current structures (witch huts, nether fortresses, etc.) to be saved to the world file, the player must load the appropriate chunks within the game manually. The game will not automatically scan worlds and update structure data globally. After this is completed, structures will be compatible in all snapshots starting with snapshot 13w37b.
Fixed a handshake exploit.
Removed browser compatibility. (The .jar file cannot be ran on a browser JVM.)


== Trivia ==
The pre-reupload of this version was released on the same day as 13w37b.
