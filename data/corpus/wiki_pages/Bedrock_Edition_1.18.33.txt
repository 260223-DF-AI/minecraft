1.18.33 is a hotfix to Bedrock Edition released only for the Nintendo Switch on May 13, 2022, which fixes bugs. This version is compatible with all Bedrock platforms running on 1.18.30 until 1.18.32.


== Additions ==


=== General ===
Settings

Added "Clear Marketplace Cache" button to the storage settings.
Used to clear local cache data but not delete save game data.


== Fixes ==
Enabled configuration changes of some back-end services.
