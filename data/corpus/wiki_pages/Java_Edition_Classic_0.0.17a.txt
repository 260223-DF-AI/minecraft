0.0.17a is a version of Java Edition Classic released on June 10, 2009, shortly after 19:16 UTC, which added /setspawn command for admins, made changes to multiplayer and fixes bugs. This version is not available in the launcher, but has been archived elsewhere.


== Additions ==


=== Gameplay ===
Controls

List logged-in players when  Tab is pressed in multiplayer.


=== Command format ===
/setspawn

Sets the current location as the default spawn location for new players.
Only for admins.


== Changes ==


=== Gameplay ===
Multiplayer

Server-side client verification in server.properties.
Set this to false would allow players to connect directly via http://www.minecraft.net/play.jsp?ip=<ip>&port=<port>.
Added logged-in.txt on the server.
Players can no longer spam ping requests.
Private server addresses changed behavior.
The server generates an externalurl.txt when run now.
Chat

Log appears when typing.
Chat is disabled in singleplayer.


=== General ===
Font

The characters 0 and O now display differently.


== Fixes ==
4 bugs fixed

Fully fixed /banip (it was working, but not intuitive.)
Fix spawn location to default at log in.
Server flood exploits prevention.
Players now get kicked if they place a block not available through legitimate gameplay.
Players now get kicked if they place a block further than allowed through legitimate gameplay.
No more lava floods.


== Trivia ==
Client.jar file of this version is compiled at 20:14 UTC, but the server version was compiled at 19:45 UTC.
It might mean that 0.0.17a is actually released at same time with server version and later reuploaded, or it was intentionally released later for unknown reasons.
