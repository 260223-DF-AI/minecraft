15w45a is the thirty-ninth snapshot for Java Edition 1.9, released on November 5, 2015. The development team's aim had been to have the 1.9 release feature-complete by the end of October 2015, followed by a period of bug-fixing, and this snapshot was the first snapshot during that planned time period of bug-fixing.


== Additions ==


=== General ===
Realms

Added a "Realms Notifications" toggle to the options menu.


== Changes ==


=== Blocks ===
Lily pads

Collision box was raised from 1⁄4 of a pixel to 1 pixel in height.


=== Items ===
Maps

Are crafted at a zoom level of 1:1, as was the case before 15w34a.
Can no longer be zoomed in.
This reverts maps to 1.8 behavior.
Shields

Placing a pattern on a shield now consumes the banner.


=== Mobs ===
Elder guardians

Rare drops now only provide fish, as they had in 1.8 versions, eliminating fishing treasure and fishing junk that had been seen in snapshots.
Shulker

Added sounds for shulkers getting damaged.


=== Non-mob entities ===
Boats

The dark oak boat can now be crafted.


=== Command format ===
General

Tab-completed coordinate arguments now become tildes if no block is in focus.
NBT format

The Linger byte tag on ThrownPotion entities has been removed.
Instead, the potion lingers if the stored item has an ID of minecraft:lingering_potion.
The following command creates a ThrownPotion lingering potion: /summon ThrownPotion ~ ~5 ~ {Potion:{id:"minecraft:lingering_potion",tag:{Potion:"minecraft:swiftness"}}}.
The ThrownPotion renders as its stored item, no matter the item.
For instance, if the stored item has an ID of minecraft:diamond_sword, the potion appears as a diamond sword while being thrown.


=== Gameplay ===
Achievements

Overpowered is changed from "Build a Notch Apple" to "Eat a Notch Apple".
Jumping

Jump height increased from 1 3⁄16 blocks to 1 4⁄16 blocks.


=== General ===
Optimizations

Tweaks to pathfinding and monster AI.
Some performance optimizations.
