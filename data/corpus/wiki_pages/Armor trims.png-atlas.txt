armor_trims.png-atlas is a procedurally-generated texture atlas used to store textures for armor trims.


== Format ==
armor_trims.png-atlas is a generated 2048×1024-pixel file that stores textures for armor trim elements.


== History ==


=== Versions of the atlas ===


==== Mipmapped versions ====

Minecraft also originally generated mipmapped versions of this atlas, for unknown reasons. They were not used by the game at all – only blocks placed in the world which use the json model system use mipmapped textures.
