A version of Minecraft Indev was released on February 7, 2010, shortly before 16:47 UTC.


== Changes ==


=== Blocks ===
Glass

Now allows light and rain to go through it again.


== Technical ==
Attempted an LWJGL update.


== Trivia ==
The al_version number of this version is 1.092, but the number of the previous version is 1.086. This means there are five spaces for either lost versions or attempted LWJGL updates or fixes in this timeframe (likely moreso the latter, given this version did indeed try update LWJGL).
