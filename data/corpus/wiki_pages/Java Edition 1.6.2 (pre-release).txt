Pre-release 1.6.2 is the only pre-release for Java Edition 1.6.2, released on July 5, 2013. Despite the version number being identical, this version differs in build from the full release.
This version was reuploaded twice later on July 5, 2013:

The first time was to fix the server.jar and to fix an issue with paintings and item frames.
The second time was to fix more issues with the server.jar, as well as the vulgar language from British English translation of sponge.
This second reupload of the server became the full release of server version 1.6.2. The full release client is different from the pre-release client.


== Changes ==


=== Mobs ===
 Horses

Now remember who tamed them.
 Baby Zombies and Baby Zombie Pigmen

Now spawn naturally.
Baby zombies and baby zombie pigmen can also be rarely spawned from a spawn egg.
