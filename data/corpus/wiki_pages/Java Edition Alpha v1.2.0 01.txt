Alpha v1.2.0_01 is a version of Java Edition Alpha released on October 30, 2010, at 21:47 UTC to fix some of the bugs and crashes in Alpha 1.2.0.


== Changes ==


=== Blocks ===
Furnaces

Now face towards the player when placed.
Gravel

Now naturally generates in the Nether.
This generally replaces the location of some soul sand.


=== Items ===
Armor

No longer completely blocks damage.


== Fixes ==
2 bugs fixed

Fixed an inventory crash.
The game sometimes no longer crashes when the player creates a new world.
