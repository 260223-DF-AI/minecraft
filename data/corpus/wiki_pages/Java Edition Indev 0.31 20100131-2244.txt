A version of Indev 0.31 was released on January 31, 2010, at 22:44 UTC, which changed tool and weapon damage values.


== Changes ==
Tools and weapons

Changed damage values.​[more information needed]
Shovel: 1 damage.
Pickaxe: 2 damage.
Axe: 3 damage.
Sword: 4 damage.
Changed the speed of mining.
Wood: 1.
Stone: 2.
Iron: 3.
Diamond: 4.


== Unconfirmed changes ==
The following changes happened in this version at the latest; as the immediately preceding versions have not been archived, there currently exists no way to confirm for certain whether the changes mentioned here were made in one of said versions.


=== User interface ===
The item forms of most blocks are now lit differently from before. The right side remains unchanged, but the top is now brighter and the left is now darker. If a block used a solid white texture, the top face would use the color  #FFFFFF, the left face  #6D6D6D and the right face  #666666.
