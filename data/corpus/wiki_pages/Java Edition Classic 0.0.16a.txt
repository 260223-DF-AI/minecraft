0.0.16a is a version of Java Edition Classic released on June 7, 2009, at 17:07 UTC, which added commands.


== Additions ==


=== Command format ===
/op

Grants operator status to player.
/deop

Revokes operator status from player.
/ban

Adds player to banlist.
/unban

Removes a banned player from the banlist.
/banip

Adds IP address to banlist.
Cannot be undone in-game, banned-ip.txt must manually be changed.
/kick

Kicks a player off a server.
/broadcast

Displays a message to multiple players.
