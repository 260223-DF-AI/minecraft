1.7.10-pre2 is the second pre-release for Java Edition 1.7.10, released on June 4, 2014, which fixes some minor bugs and a crash in 1.7.10-pre1 related to being kicked from a server.
1.7.10-pre2 was reuploaded shortly after release to fix the version number still saying "1.7.10-pre1".


== Fixes ==
1 issue fixed
Fixed a crash related to being kicked from a server.


== Trivia ==
This version was previously removed from the launcher for some time, but was added back in September 2018.
This version came out exactly six years before 1.16 Pre-release 1.
