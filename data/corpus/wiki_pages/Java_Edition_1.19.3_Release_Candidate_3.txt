1.19.3 Release Candidate 3 (known as 1.19.3-rc3 in the launcher) is the third and final release candidate for Java Edition 1.19.3, released on December 6, 2022, which fixes bugs. This is the final release candidate released in 2022.


== Fixes ==
3 issues fixed
From released versions before 1.19

MC-108707  Potion effects, particles, and fire animations persist client-side after death in server environments.
From 1.19.2

MC-258279  Particle groups aren't cleared on resource reload.
From the 1.19.3 development versions

MC-258258  The EntityTag NBT data on armor stands is no longer applied upon placement.
