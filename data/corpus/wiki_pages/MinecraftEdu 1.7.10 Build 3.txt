1.7.10 Build 3 is the third build for MinecraftEdu 1.7.10, released on August 7, 2014. It is based on Java Edition 1.7.10.


== Additions ==


=== General ===
Students can't do anything if they are frozen.
Disabled Server Window save button when save is in progress.


== Removals ==


=== General ===

5 bugs fixed

Custom world save folder not working.
@p, @r and @a now work in command blocks with /eduuser commands like togglefreeze and togglemute of an individual student.
Students not being able to check assignments.
Chat not working.
Some mods like Custom NPCs not working properly.
