1.21.7 Release Candidate 2 (known as 1.21.7-rc2 in the launcher) is the second and final release candidate for Java Edition 1.21.7, released on June 26, 2025. This version fixes bugs and updates the pack format versions due to there having been breaking changes in the previous release candidate.


== Changes ==


=== General ===
Data pack

The pack format version is now 81.
Resource pack

The pack format version is now 64.


== Fixes ==
4 issues fixed
From 1.21.6

MC-298879 – The game crashes upon startup on some AMD graphics
MC-299110 – Force upgrading does not write new DataVersion to converted data
MC-299126 – The server crashes when certain mobs ride a turtle
From the previous development version

MC-299130 – Pack formats were not incremented in 1.21.7 rc1 after breaking changes
