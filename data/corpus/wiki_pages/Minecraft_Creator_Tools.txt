Minecraft Creator Tools, or mctools.dev, is an official open-source website that hosts web-based tools for Bedrock Edition creators. It currently hosts one tool.


== Project Inspector ==

The Project Inspector, also known as Validation and Analytics, lets users upload add-ons, packs or worlds to be verified by the website. The tool separates checks into three categories. Checks include, but are not limited to:

All
Invalid JSON
Invalid manifests and dependencies
Script version numbering
Deprecated JSON components
Platform Versions
Minimum engine version
World template version
Add-on Best Practices
World data validation
Product type in manifest
