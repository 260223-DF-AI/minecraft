1.20.12 is a major update to Minecraft Education, released on August 8, 2023, which introduces new features from Trails & Tales.


== Additions ==

Added features from Trails & Tales.
Menu

Added "My Templates" button.


== Changes ==


=== Mobs ===
Agent

Now has no range restrictions, it can move and build far away from the player.
Now has an animation for unsuccessful actions.
Now chooses which item to detect or interact with in a “line of sight” in the direction the code specified, up to 1 block away.


== Fixes ==
Increased the reliability of multiplayer connections and decreased the number of disconnections during a multiplayer session.
