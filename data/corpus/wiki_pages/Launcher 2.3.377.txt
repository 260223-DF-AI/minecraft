2.3.377 is an update to the launcher released on August 23, 2022.


== Additions ==
Added ability to copy error message text with single click.
Additional translations applied to Launcher.


== Changes ==
Minecraft Preview is available in the Launcher.
Improvements to Launcher installer functionality.
Improvements to installation flow, logic & access for all games and Game Pass.


== Fixes ==
Fixes to sign-in process.
Fix to ensure videos in Launcher are refreshable.
Bug fixes around 1.19+ Java versions not loading.
MCL-21324 - Could not delete a file error.
MCL-22073 - Launcher doesn't say "Waiting on install" while scrolling down on Bedrock launcher section.
