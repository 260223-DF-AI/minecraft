A version of Indev 0.31 was released on January 28, 2010, at 23:04 UTC, which added arrow crafting and fixed bugs from previous version.


== Additions ==


=== Gameplay ===
Crafting

Pressing B opens the "Crafting" menu.
Unlike with the inventory, the player can't move while crafting (just like versions newer than Java Edition Alpha v1.0.14).
The maximum item stack size is only 1 in a box of the "Crafting" menu.
Only available recipe being an apple in the top left-hand corner box to an arrow.
As long as an apple is in the top left slot, other items can be added in any other slot of the crafting grid, as the game only checks the top left slot and ignores the content of all other slots. All the items added to the crafting grid are consumed when taking the arrow out from the output slot.
