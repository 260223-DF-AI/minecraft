This is a list of Pocket Edition v0.15.0 alpha development versions.


== v0.15.0 alpha build 1 ==
v0.15.0 alpha build 1 is the first build version released for v0.15.0, also known as the Friendly Update.


=== Additions ===


==== General ====
Achievements for iOS and Android
Version exclusive: More achievements.
Is accessed from the home screen
Online multiplayer with Xbox Live friends
New UI for the main menu, shared with the Windows 10 Edition
New UI for the skins menu, it has a skin selection carousel that is similar to Console Edition


==== Gameplay ====
Pig riding
Version exclusive: New sound when accelerates.
Interact buttons
Boost, used when riding a pig with a carrot on a stick.
Name, used for name mobs using name tags.


==== Blocks ====
Pistons and sticky pistons
Version exclusive: New animation; piston arm is visibly thicker at the bottom.
Version exclusive: Can push block entities such as chests.
Version exclusive: Blocks that stick to walls (such as levers) can now be placed on pistons or sticky pistons.

BUD block

== v0.15.0 alpha build 2 ==
v0.15.0 alpha build 2 is the second build version released for v0.15.0, also known as the Friendly Update.


=== Changes ===

Players no longer spawn in tree leaves.
Restored GUI scale options.
Tweaked grass path texture.
Tweaked rose bush & peony textures.
Fixed enchantment table UI close button.


=== Fixes ===
Fixed some crashes happening in specific situations.
Walking past buttons no longer trigger suffocation overlay.
Existing dye color not overwritten when mixing colors in a cauldron.
Torches and redstone torches fixed so that they don't drop when a block is placed or removed by it.
Restrict hosting or joining games while on your mobile network (instead of being limited to wifi) to prevent huge bills on metered networks.
Riders no longer riding in circles if horse has been idle for a while.
Fixed spider jockeys in multiplayer.
Fixes to feed button & feeding animals on touch devices.
Bug fixes for sticky pistons.
You can chat while sleeping in a bed again.
Fixed graphics glitches while riding a horse.
