1.21.1 is a minor hotfix update to Java Edition released on August 8, 2024, which fixes critical exploits including block entity swap, and adds two new languages. Servers running this version are compatible with 1.21 clients.


== Additions ==


=== General ===
Languages

Added support for Tzotzil and Belarusian (Latin) languages.


== Fixes ==
4 issues fixed
From released versions before 1.21

MC-254899  Sitting/standing a tamed cat does not trigger minecraft:player_interacted_with_entity
MC-260689  Arm spins when swimming with a loaded Crossbow in your offhand
Private issues

Fixed block entity swap based update suppression as shown in this video.
Fixed an exploit which allows modified clients to crash the server by requesting command completion for a malicious target selector input, causing a crash due to stack exhaustion.
