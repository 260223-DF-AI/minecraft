2.2.8116 (Windows), 2.2.8117 (Windows, via the Microsoft Store/Xbox app), 2.2.8115 (macOS), or 2.2.8114 (Linux), is an update to the launcher released on December 7, 2021.


== Changes ==
Improved post-Microsoft Account migration flow.


== Fixes ==
Now New Windows Launcher will load or save configs which used by the Standalone Launcher. (MCL-19986)
Now user will not receive an error message when clicked "Add Account" then quickly close the login window.
Now users new to Java will now be prompted to create a Java gamertag when they navigate to the Skins Java tab. (MCL-20219)
