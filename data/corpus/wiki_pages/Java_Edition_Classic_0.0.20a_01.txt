0.0.20a_01 is a version of Java Edition Classic released alongside server version 1.8.2 on June 20, 2009, at 21:11 UTC to fix bugs.


== Changes ==


=== Gameplay ===
Hotbar

Cobblestone positions are now switched to second slot in the hotbar.
Dirt positions are now switched to third slot in the hotbar.


== Fixes ==
3 bugs fixed

Fixed a bug that caused clients to build unbreakable blocks.
Fixed a protocol error.
Fixed FPS issues.


== Bugs ==
5 bugs reported

Water transparency is bugged.
Sometimes, the water looks opaque.
Water tiles have light blue bits in their borders.
Usernames are now harder to read.
If the player aims at a certain angle while placing a block, it appears to go inside an existing block, due to a bug caused by using OpenGL picking.
Specifically, looking at glass or leaves causes this bug to happen.
