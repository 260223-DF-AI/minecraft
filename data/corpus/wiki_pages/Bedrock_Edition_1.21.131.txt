1.21.131 is a hotfix to Bedrock Edition released on December 16, 2025, which fixes some bugs. This is the final version released in 2025.


== Changes ==


=== Items ===
Spear

Increased the duration during which a spear can apply knockback while charging.
Charge attack animations now align more closely with the spear's gameplay phases (engaged, tired, disengaged) for improved visual feedback.


== Fixes ==
4 issues fixed
MCPE-156784  Adventuring Time achievement unlocked without visiting 17 biomes.
MCPE-232316  Escape key and Menu Cancel button stop working in the Marketplace home page after using the search bar.
MCPE-232978  Add-Ons preventing worlds from being opened (Block error code).
Other

Fixed UI textures looking incorrect on Nintendo Switch when loading add-ons or connecting to servers.
