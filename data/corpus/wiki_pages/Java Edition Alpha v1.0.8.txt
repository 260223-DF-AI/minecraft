Alpha v1.0.8 is a version of Java Edition Alpha released on July 19, 2010, which added cows, milk, and leather. It also changed the crafting recipe for leather armor.


== Additions ==


=== Items ===
 Leather

Drops when a cow is killed.
 Milk

Currently unobtainable.
Has no function.


=== Mobs ===
 Cows

Drop 0–2 leather upon death.
Have 10HP.


== Changes ==


=== Items ===
Leather armor

Now crafted using leather, rather than cloth.
