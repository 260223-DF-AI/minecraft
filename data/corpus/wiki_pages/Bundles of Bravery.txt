Bundles of Bravery is a game drop that was released on October 22, 2024, as Java Edition 1.21.2 and Bedrock Edition 1.21.40. It adds bundles to the game. It also adds Hardcore mode to Bedrock Edition and makes it available to Realms on Java Edition. 
The game drop was announced at Minecraft Live 2024, though the name was accidentally revealed on Minecraft.net as part of a summary of the event posted a few days before it happened.


== Additions ==


=== Items ===
 Banner patterns 

Two new banner patterns have been added in Java Edition to achieve parity with Bedrock Edition:
 Field masoned banner pattern
 Bordure indented banner pattern
  Bundle

Crafted from a string and a piece of leather.
Can hold a mixed stack of items at a time.
Can be dyed into 16 colors.
The specific item to be removed from a bundle can be selected using the scroll wheel.
Moved from experimental status to full release after an extensive development period (2020-2024), completing the implementation of all content that had been postponed from Caves & Cliffs.


=== Game modes ===
 Hardcore for Bedrock Edition and Java Edition Realms

A variant of Survival mode in which the world is locked to Hard difficulty.
Players cannot respawn after dying, but can continue to observe the world in Spectator mode.
Fully featured world creation screen available to create Hardcore Realms.


== Changes ==


=== Blocks ===
Loom in Java Edition

Now requires field masoned banner pattern and bordure indented banner pattern to craft corresponding banners.
 Redstone torch in Java Edition

The texture and model have been updated.
  Redstone comparator and Repeater in Java Edition

The powered textures and models have been updated.


=== Items ===
 Banner patterns in Java Edition

Added unique item textures for each banner pattern.
Each banner pattern item now includes their pattern in their name instead of as a subtitle.
Smithing templates

Are now named after their template type instead of the generic "Smithing template" name.
They instead specify that they are a smithing template through a subtitle in the hover text.


=== Mobs ===
Dolphin, glow squid and squid in Java Edition

Now have baby variants, matching Bedrock Edition.
Salmon in Java Edition

Now have large and small variants, matching Bedrock Edition.
