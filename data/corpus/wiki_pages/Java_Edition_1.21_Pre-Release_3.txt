1.21 Pre-Release 3 (known as 1.21-pre3 in the launcher) is the third pre-release for Java Edition 1.21, released on June 5, 2024, which fixes bugs.


== Changes ==


=== General ===
Data pack

The pack format version is now 48.
Enchantment definition

Renamed replace_disc entity effect type to replace_disk.


== Fixes ==
22 issues fixed

MC-272194  Empty attribute modifiers lost during upgrade.
MC-272996  Using "/execute in..." to switch dimensions causes the player to fall when flying.
