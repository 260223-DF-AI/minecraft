Beta 1.2_02 is a version of Java Edition Beta released on January 21, 2011, that mainly changed where skins are downloaded from.
The version of Beta 1.2_02 in the launcher is missing its META-INF file, but is otherwise identical.
There is a later instance of this version, from May 17, 2011, which features many secret developer features.


== Additions ==


=== General ===

Controls

Pressing  Shift + F2 generates a 3645017700 TGA format screenshot (which ends up being 1.8 GB, and no UI included). While the file is being generated, the game freezes and experiences major graphical distortions. This is unfinished test code, as confirmed by Notch.


== Changes ==


=== Blocks ===
Lapis lazuli ore

Now drops 48 lapis lazuli per block mined (increased from 1) on single player only (servers have not been affected yet).
Can now be found at bedrock level.


=== Items ===
Ink sac

Renamed "Ink Sack" to "Ink Sac".
Cocoa beans

Renamed "Coco Beans" to "Cocoa Beans".


=== General ===
Skins

Now downloaded from Amazon S3, instead of Minecraft.net.


== Trivia ==
Prior to the 2018 skin update, Beta 1.2_02 was the oldest version in the launcher to have working skins.
However, if the player changed their skin after 1.8, the last skin the player used prior to 1.8 would be displayed. This occurred in all versions after Beta 1.2_02 until 1.3.1. It is currently unknown why this happened and why 1.3.1 was the oldest version to display post-1.8 skins.
