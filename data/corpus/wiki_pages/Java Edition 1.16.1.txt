1.16.1 is a hotfix update to Java Edition released on June 24, 2020, which fixes stability issues with Realms. It is not compatible with 1.16 servers.


== Fixes ==
4 issues fixed
From released versions before 1.16

MC-110792 – Slime blocks no longer launch arrows.
MC-146743 – Mainhand third person animations incorrect while loaded crossbow in offhand.
MC-186546 – Endermen that have been turned hostile by other means than looking at them will turn neutral when looked at.
From 1.16

MC-191138 – Crash when clicking on realms info – java.lang.ArrayIndexOutOfBoundsException: -1

== Trivia ==
This version is popular with speedrunners since the following update (1.16.2) decreased ender pearl drop rates when bartering and made bastions more dangerous with the addition of piglin brutes.
1.16.1 was the last version without any development versions until 1.21.3.
Before 1.16.1, the last version without any development versions was 1.11.2.
