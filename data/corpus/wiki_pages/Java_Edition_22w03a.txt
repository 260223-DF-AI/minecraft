22w03a is the first snapshot for Java Edition 1.18.2, released on January 19, 2022, which adds a new command and fixes bugs. This is the first snapshot released in 2022.


== Additions ==


=== Command format ===
/placefeature

Used to place a configured feature at a given location.
Syntax: /placefeature <feature> [<pos>]
feature: The namespaced ID of a configured feature to place.
pos: The position to use as the origin for the feature placement.
If omitted, ~ ~ ~ is used.


=== General ===
Debug screen

Now contains the chunk position in the region file, and the name of this region file which the player is currently in, in the end of the "Chunk" row.
Languages

Added Friulian.
Tags

Added the #fall_damage_resetting block tag.
Contains the #climbable block tag, sweet berry bush, and cobweb.


== Changes ==


=== Blocks ===
Ender chest

No longer gift wrapped around Christmas.


=== World generation ===
Eroded badlands

Tweaked generation to generate more cohesively with the rest of the surrounding badlands.
Seeds

A seed of the number zero is no longer handled as a special case.
Any spaces before/after an inputted seed are now trimmed.


=== Command format ===
/locate

Now uses a namespaced ID parameter.


=== General ===
Debug screen

Moved the location of the player within a chunk from "Chunk" row, into the new square brackets at the end of the "Block" row.
Logging

Upgrades log4j to 2.17.0.
Tags

Powder snow is no longer part of the #goats_spawnable_on block tag.
Snow blocks and powder snow are now part of the #azalea_root_replaceable block tag.
