v0.11.0 alpha build 4 is the fourth build version released for Alpha 0.11.0.


== Additions ==
Implemented milk drinking to remove mob effects.


== Changes ==
Version name now includes the beta number.


== Fixes ==
13 bugs fixed

Sleeves are now properly rendered in the first-person mode.
Fixed potion duration rendering for high values.
Fixed some gamepad menu issues.
Fixed spawning high up in the air.
Fixed wrong player spawning rotation and velocity.
Fixed chat background rendering when using bold.
Changed the way skins are loaded and saved.
From released versions before 0.11.0

MCPE-7889 - Game crashes when pressing 'Quit to title'.

MCPE-8163 - Fishing bobber gets stuck in boat.
MCPE-8218 - Existing old world type game crashes.
MCPE-8220 - Fishing pole bobber glitch.
MCPE-8275 - Drinking milk does not remove potion effects.
MCPE-8277 - Enderman angry animation only on client side.
