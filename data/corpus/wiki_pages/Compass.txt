A compass is an item that points in the horizontal direction of the Overworld's spawn point. 
A lodestone compass is a compass variant that is obtained by using a compass or lodestone compass on a lodestone, and instead points toward that lodestone's horizontal position.
If the target position of either type of compass is in a different dimension, or is a lodestone that has been broken, it spins in random directions.

== Usage ==
Both types of compass normally point in the horizontal direction of a specific position in the world and do so when viewed in most ways; including in the inventory, when held by any player or entity or displayed in an item frame. They also point toward their target position when viewed in the crafting table preview menu, even without being crafted.[Bedrock Edition  only] The direction of the needle is relative to the player viewing it.
Normal compasses point toward the world spawn point in the Overworld. Using the /setworldspawn command to change the world spawn can change where normal compasses point.
If a compass or lodestone compass is used on a lodestone, it becomes a lodestone compass that points toward that lodestone. All lodestone compasses have an enchantment glint to distinguish them from normal compasses.
A compass or lodestone compass spins and points in random directions if it cannot find its target position. There are two causes for this:

If the compass is in a different dimension to its target position (e.g. a lodestone compass is in the Overworld but its target lodestone is in the Nether), its needle spins randomly. Because of this, normal compasses only work in the Overworld (as their target position is the Overworld spawn point) and always spin randomly in the Nether and the End.
If a lodestone compass's target lodestone is broken, it becomes 'disconnected' and its needle spins in random directions. A disconnected lodestone compass can be manually re-connected to any lodestone by using the compass on the lodestone. It will not re-connect automatically.
Compasses and lodestone compasses do not stack with each other, and only lodestone compasses that point toward the same lodestone may stack together. In Java Edition, all disconnected lodestone compasses can stack with each other, regardless of which lodestones they originally pointed to. In Bedrock Edition, disconnected lodestone compasses can only stack together if they stacked together before they became disconnected, such as if they pointed to the same lodestone. 
In Java Edition, if a lodestone compass is placed in a storage block, the lodestone can be broken and replaced without disconnecting the compass, as long as the compass remains in storage while the lodestone is missing. In Bedrock Edition, lodestone compasses in storage disconnect instantly when the lodestone is broken.
In Java Edition, uncrafted compasses viewed in the crafting table preview always point straight forward and never at any particular position in the world.
