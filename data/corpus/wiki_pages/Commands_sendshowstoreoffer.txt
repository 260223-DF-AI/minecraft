Send a request to show players the store offer.


== Syntax ==
sendshowstoreoffer <player: target> <redirectType: RedirectLocation> <offerId: string>
sendshowstoreoffer <player: target> server


== Arguments ==
player: target: CommandSelector<Player>

Must be a player name or a target selector. 
redirectType: RedirectLocation: enum

Specifies the redirect type, values are:
marketplace
character
offerId: string: string

Specifies the offer ID.
Must be a string. And it must be a single word that has no space or a double-quoted string (When quoted, \ can be used to escape characters).
