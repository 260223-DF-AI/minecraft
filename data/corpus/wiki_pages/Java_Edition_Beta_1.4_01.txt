Beta 1.4_01 is a version of Java Edition Beta released on April 5, 2011, which partially removed locked chests and fixed a respawning issue with beds.


== Changes ==


=== Blocks ===
Locked chests

No longer randomly generate on April 1.
Can now be broken like normal chests.
Now "decay" like leaves.


== Fixes ==
1 bug fixed

Beds now act as a respawn point as intended.
