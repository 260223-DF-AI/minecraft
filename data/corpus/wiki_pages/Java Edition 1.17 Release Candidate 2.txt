1.17 Release Candidate 2 (known as 1.17-rc2 in the launcher) is the second and final release candidate for Java Edition 1.17, released on June 7, 2021, which fixes bugs and crashes.


== Changes ==
Authlib was updated from 2.2.30 to 2.3.31.
Legacy skin handling now has improved skin validation.
The spawn egg dispenser behavior now catches any exceptions that could be thrown when spawning the entity.

2 issues fixed
Other

2 private issues were fixed.
