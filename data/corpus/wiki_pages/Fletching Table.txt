The fletching table is a fletcher's job site block that can generate naturally in villages. It is used to turn an unemployed villager into a fletcher.


== Obtaining ==


=== Breaking ===
Fletching tables can be broken by hand, but using an axe is faster.

Fletching tables drop themselves when broken with any tool.


=== Natural generation ===
Fletching tables can generate naturally inside fletcher houses in villages.


=== Crafting ===


== Usage ==
The fletching table currently has no right-click functionality or GUI; Its functionality was intended to be added in upcoming updates as of 2019.


=== Placement ===
Regardless of the player's position while placement, the different textures of the block are always facing in the same position (the textures with the target are facing west and east, while the one with the knife and feather is facing up).


=== Profession ===

If a fletching table has not been claimed by a villager, any nearby unemployed villager has a chance to change their profession to fletcher and claim the fletching table as their job site block.


=== Fuel ===
Fletching tables can be used as a fuel in furnaces, smelting 1.5 items per block.

== Trivia ==
Although the fletching table has no player interaction in the base game, in the "Farmer's Market" expansion to the board game "Minecraft: Builders & Biomes", it allows the player to reveal an additional weapon when fighting a mob.
