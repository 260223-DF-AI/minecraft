A version of Indev 0.31 was released on January 22, 2010 at 22:51. It was announced on The Word of Notch at 22:57 UTC.


== Additions ==


=== Blocks ===
  Lava spawner

When placed, the block would create lava blocks which would replicate infinitely to fill up specific volumes.
Added to the inventory.


=== Items ===
 Bow

Can now be used to fire arrows.
Added to the inventory.
Shooting arrows into lava creates flaming arrows.


== Changes ==


=== Blocks ===
General

Blocks can now be placed on resource items, pushing them out.
Lava works like water now, but slower.


=== Items ===
General

Readded the pulse effect when an item gets picked up in the hotbar.
Picking up items with a full inventory after reloading a level sends them to hidden inaccessible slots.
If the level is not reloaded beforehand, any extra items go to fill up the armor slots. Once they're full no more items can be picked up.
Stack size reduced to 50 from 99.
Water and lava source blocks are now given to the player in stacks of 5.
 Arrows

Added as an item.
Now require a bow to fire.
Deal no damage to players or mobs yet.
Added to the inventory.
Have a shake effect when hitting a block, and slight deviations in where they land.[verify for Java Edition]
Now fall down when the block they landed on is removed.
 Axes

Half of the axe head has been removed.


=== World generation ===
General

Liquid spawner is now in the level generator.
New lighting pass, preventing stack overflows on huge deep floating maps.
Caves are less flooded.
Readded flowers to terrain generation.
The arrow counter above the hotbar was removed.


=== Gameplay ===
Explosions

Now have a higher blast radius.
Now destroy stone, among other changes.[more information needed][verify for Java Edition]
Now destroy dropped items.
Saving

Players can now save and load the game locally using the Indev level format.
Other

Players have many times longer combat reach.
Dropped items and TNT no longer bounce when hitting the ground


== Fixes ==
1 bug fixed
From Indev 0.31 20100114

Picking up items that a player already has a full stack of creates a new stack. One item stacks are no longer created.


== Bugs ==
3 bugs reported

Rana now spawns in large numbers, causing lots of lag as they accumulate.
Flammable blocks no longer ignite in lava.
Armor sprites only show up after saving and reloading a level.[verify for Java Edition]
