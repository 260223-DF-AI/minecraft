3.15.8 is an update to the Minecraft Launcher that was released on May 21st, 2025.[verify]


== Changes ==
Slightly adjusted the text to be to the bottom/right.
The launcher now has a new theme, featuring darker colors, and the color standardization has been improved.
The hexadecimal code of those colors are:
 #323233 to  #171714
 #1f1f1e to  #302d2d
 #1f1f1e to  #262423
Currently, it is not possible to remove this theme.


== Additions ==
Added new slides on the main play section.


== Fixes ==
Fix to make it easier & more reliable to get new Bedrock updates when they are released.
A bunch of stability & UX improvements throughout the Launcher.
MCL-24939 - Armor misspelled "armour" in Dungeons game description (EN_US)
