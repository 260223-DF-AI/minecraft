The action bar is a space just above the player's hotbar where text can be displayed. Some gameplay elements, such as beds, use this to give contextual hints to the player. Text can also be displayed here using the /title command.


== Behavior ==
The action bar is displayed above where item names are shown when an item is selected in your hotbar, so an item name and action bar text can be shown on the screen at the same time.
When text is displayed here, it instantly appears and remains on the screen for approximately 1.5 seconds[more information needed] before fading out. If another action bar message is sent, it instantly replaces the previous text without any transition/animation and remains there for the usual 1.5 seconds it normally would if it was not replacing anything.
In Bedrock Edition, the opacity of the black background on the action bar can be adjusted in the accessibility settings.


== Gameplay uses ==
In Bedrock Edition, certain messages are shown in the chat instead of the action bar.


== Commands ==

A custom text message can be displayed on the action bar using the /title command:

JE: title <targets> actionbar <title>
BE: title <player: target> actionbar <titleText: message>
