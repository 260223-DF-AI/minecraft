The fourth version of 0.25 SURVIVAL TEST is a version of Survival Test released on September 3, 2009, at 16:58 UTC.


== Changes ==


=== Non-mob entities ===
Arrows

Now stay around longer before despawning.


=== Gameplay ===
Controls

Players can no longer right-click to deal damage.


== Technical ==


=== General ===
Loading a world via the URL now works.

== Trivia ==
Each subsequent patch version of 0.25 SURVIVAL TEST did not change the version number, leading to one user mistakenly changing the wiki version from "0.25_SURVIVAL_TEST_3" to "0.25_SURVIVAL_TEST_4", a version that does not exist, upon the release of this version. Notch saw this and thought that this user was telling him that the unlabelled patches for 0.25 should have labels and that the current version should be "_04", so Notch made the next version "0.25_05 SURVIVAL TEST", even though under Notch's version format used at the time the version after this one would technically be "_04".
