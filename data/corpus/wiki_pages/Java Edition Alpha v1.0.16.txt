Alpha v1.0.16 is a version of Java Edition Alpha released along with server 0.1.1 on August 12, 2010, that made commands usable again in servers, and added unused files for additional music discs.


== Additions ==


=== Command format ===
/banlist

Displays the server's blacklist.
/home

Teleports a player to the world spawn.
/pardon

Removes entries from the banlist.
/pardon-ip

Specifies the IP address to remove from the banlist.
/stop

Stops a server.
/op

Re-added.
Promotes a user to operator.
/deop

Re-added.
Demotes a user from operator.
/kick

Re-added.
Makes a user disconnect from the server.
/ban

Re-added.
Prevents a user from rejoining the server.


=== Gameplay ===
Multiplayer

Added server operators.
Operators are able to send server commands by typing a slash followed by a server command.
Operators can build and destroy blocks near the spawn area.
