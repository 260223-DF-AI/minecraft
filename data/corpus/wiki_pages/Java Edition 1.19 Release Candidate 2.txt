1.19 Release Candidate 2 (known as 1.19-rc2 in the launcher) is the second and the final release candidate for Java Edition 1.19, released on June 3, 2022, which fixes an unspecified crash.


== Fixes ==
3 issues fixed
From released versions before 1.19

MC-228944 – Minecraft 1.17 crashes upon opening new world on Apple M1, even with Rosetta.
MC-249558 – All versions crash on M1 Mac with JVM crash when attempting to launch.
Other

Fixed a crash.
