18w08a is the twenty-first snapshot for Java Edition 1.13, released on February 21, 2018, which modified ocean generation and split the ocean into several different biomes.


== Additions ==


=== World generation ===
Underwater caves and Underwater ravines

Magma blocks spawn at the bottom of ocean canyons at lava levels, creating downward bubble columns.
Biomes

Warm Ocean
Lukewarm Ocean
Cold Ocean
Deep Warm Ocean
Deep Lukewarm Ocean
Deep Cold Ocean
Deep Frozen Ocean
Frozen Ocean


=== Command format ===
/locate

Added a clickable teleport link to the /locate command output.


=== General ===
Added a new sound effect for squid shooting ink.


== Changes ==


=== Items ===
 Seagrass

The inventory item no longer has an animated texture.


== Fixes ==
5 issues fixed
From released versions before 1.13

MC-94487 – Ravines don't naturally cut through sand, sandstone, or terracotta upon generation, but other cave types do.
MC-122000 – Items get deleted when the inventory is overflowed by using the recipe book to return items from a crafting table.
From the 1.13 development versions

MC-124299 – /give command: Items on ground for a long time not able to pick up.
MC-124948 – /enchant can be used by non-ops.
MC-125124 – Crash generating mansion (Exception generating new chunk "Not yet implemented")
