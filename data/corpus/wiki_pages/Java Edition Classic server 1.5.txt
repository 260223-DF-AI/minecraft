1.5 was a Java Edition Classic server version released with client 0.0.19a on June 19, 2009.
Server version 1.5 was reuploaded to fix the version number.


== Additions ==
Allow players to send most admin commands via stdin.
Added chat throttling.
Players who spam the chat get muted for 8 seconds.
Added movement throttling.
The server doesn't process more than a single move command per tick.
Stand still to catch up if it's lagging.
Added max-connections to server.properties.
This is the maximum number of connections per IP to the server.
Defaults to 3.
