Alpha v1.0.5 is a version of Java Edition Alpha released on July 13, 2010, which adds snowballs, snow blocks, and fixes bugs.
Alpha v1.0.5 was reuploaded at 21:49 UTC (16 minutes after release) to fix a bug with snow.


== Additions ==


=== Blocks ===
 Snow

Drops 6 snowballs when destroyed.
Crafted with 9 snowballs.


=== Items ===
 Snowballs

Can be thrown, but deal no damage.
Stack to 64.


== Changes ==


=== Blocks ===
Signs

No longer require a free block above them when placing them.
Ice

Increased hardness from 0.1 to 0.5.
Snow

Drops snowballs when broken by any means other than replacing with another block.


=== General ===
Crashes

Game no longer saves on crashes.


== Fixes ==
3 bugs fixed

Fixed building on snow bug.
Fixed ice rendering bug.
Third person view no longer causes water particles to surround the player at water level.


== Bugs ==
1 bug reported

Entity shadows are glitched.


== Reupload fixes ==
Snow

No longer goes through glass.
