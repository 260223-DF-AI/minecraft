1.21.91, dubbed Chase the Clouds, is a minor update to Minecraft Education, released on July 22, 2025, which adds features from the Bundles of Bravery, The Garden Awakens, Spring to Life, and Chase the Skies game drops.


== Additions ==

Added features from Bedrock Edition 1.21.20 to Bedrock Edition 1.21.90, Bundles of Bravery, The Garden Awakens, Spring to Life, and Chase the Skies.
Except Vibrant Visuals, worlds pausing in the pause menu, and Hardcore mode.
Re-added the only allow trusted skins.
Added Spectator mode from Bedrock Edition 1.19.50.
Added Ore UI for new world screen from Bedrock Edition 1.19.20.
Added read/Write file functionality with MakeCode.


== Changes ==


=== General ===
Code Builder IDE changes

Tynker has been removed as a supported IDE.
In-game notifications were added so everyone is aware of the upcoming changes.
Cloud storage

Further general stability improvements.
Contains many of the below mentioned stability improvements and bug fixes, but does not contain the Bedrock updates listed below.
Cloud storage system

Now support for larger world sizes.
Lowered error rates and better upload and download performance.
Multiplayer and coding

Coding infrastructure updated for better stability and performance.
Improvements to performance and product stability.
World creation screen

The create new world and edit world screen is being updated with improved navigation.


== Fixes ==
Improved client stability.
Multiplayer sign-in and connecting updated for better reliability.
MacOS Login freeze fix.
