1.19.83 is a hotfix to Bedrock Edition released on May 17, 2023, which fixes bugs.


== Fixes ==
4 issues fixed
From released versions before 1.19.80

MCPE-169176 – Inventory cursor jumps all over the place.
MCPE-170184 – Trade Tables no longer support custom entity Spawn Eggs.
Other

Fixed several crashes that could occur during gameplay.
Fixed custom textures that override ‘minecraft’ blocks with aux metadata as a list of textures.


== Trivia ==
This version was released 14 years after Classic 0.0.11a, the first publicly released version of Java Edition.
