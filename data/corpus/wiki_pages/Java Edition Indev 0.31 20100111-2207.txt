A version of Indev 0.31  was released on January 11, 2010, at 22:07 UTC, which added fullscreen mode. It was announced on The Word of Notch at 22:25 UTC, by which time the next version had already been released.


== Additions ==


=== General ===
Controls

Fullscreen mode accessible via F11.
The game now rescales if the window it is in rescales.


== Bugs ==
1 bug reported

Fullscreen mode is randomly inaccessible.
