26.1 Release Candidate 1 (known as 26.1-rc-1 in the launcher) is the first release candidate for Java Edition 26.1, released on March 19, 2026. This is the first release candidate released in 2026 and the first release candidate to be released with only an unobfuscated build.
This version contains no gameplay changes from its preceding pre-release.


== Technical ==
The only difference between this version and 26.1-pre-3 is the transition from pre-releases to release candidates. Therefore, the only code change in this version is that the CRASH_EAGERLY field in net.minecraft.SharedConstants has been changed from true to false, as with every transition to release candidates, which results in the following effects:

Removed a packet error handler.
Changed how chunk load failures are internally reported.
