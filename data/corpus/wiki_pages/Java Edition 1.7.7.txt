1.7.7 is a minor update to Java Edition released on April 9, 2014, to fix some crashes in some multiplayer servers from 1.7.6. It is compatible with 1.7.6 servers.
1.7.7 was reuploaded a day later on April 10, 2014, to remove the message Outdated server! pretending to be 1.7.7.


== Fixes ==
Fixed some crashes for some multiplayer servers.
