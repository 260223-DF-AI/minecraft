A quadruple compressed poisonous potato block is a compacted block made of triple compressed poisonous potato blocks in 24w14potato.


== Obtaining ==


=== Breaking ===
A quadruple compressed poisonous potato block can be broken using anything, but a pickaxe is the fastest. Quadruple compressed poisonous potato blocks always drop themselves when mined.

== Trivia ==
One quadruple compressed poisonous potato block is made up of 59,049 poisonous potatoes (approximately 923 stacks of 64 or 34.2 shulker boxes).
On average (with a drop rate of 2%), 2,952,450 potatoes have to be farmed to obtain all of the poisonous potatoes.
A stack of these blocks would be made up of 3,779,136 poisonous potatoes, which could fill 2,187 single chests.
The quadruple compressed poisonous potato block is the block with the most hardness in the game with 1,312.2.
The highest hardness in the game prior to 24w14potato was 100.
Unbreakable blocks use a hardness of -1.
The hardness of the quadruple compressed poisonous potato block is 6,561 times more than the hardness of the poisonous potato block as 6,561 poisonous potato blocks make up one quadruple compressed poisonous potato block.
This block has the longest name of any block in Minecraft's history at 43 characters.
For comparison, the longest block name in any release version is the waxed weathered copper golem statue at 35 characters, and the longest block name in any non-joke development version is the waxed lightly weathered cut copper stairs at 41 characters.
This block would have a mass of 7,127.1 imperial tons or ~7.24 gigagrams if it existed in real life.
