15w47c is the forty-third snapshot for Java Edition 1.9, released on November 20, 2015 to fix some issues with 15w47b.


== Changes ==


=== Blocks ===
Cobweb

Now drops when broken with shears.
Silk Touch is no longer required.

== Trivia ==
15w47c was previously removed from the launcher for some time, but was added back in September 2018.
15w47c features a glitch with the Ender Dragon respawn code. Because of this, bedrock became obtainable in Survival mode.
