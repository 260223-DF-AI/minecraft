1.8 was a server version of Java Edition Classic released along with client version 0.0.20a on June 20, 2009, at 20:53 UTC.


== Additions ==
Added server support for the new blocks added in 0.0.20a.
Added support for the /solid command, used to toggle between placing normal stone and unbreakable bedrock.


== Changes ==
The server software no longer allows regular players to destroy bedrock, but admins can.
