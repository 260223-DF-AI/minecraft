1.7.10 Build 14 is the fourteenth build for MinecraftEdu 1.7.10, released on February 17, 2015. It is based on Java Edition 1.7.10.


== Additions ==
Individual usercommands like togglefreeze and togglemute now accept true or false as a parameter.
More mods to the online mods repository.


== Changes ==
Signs are now saved with UTF-8 encoding.


== Fixes ==
Students being able to get edu blocks from the creative intentory if mods were installed.
Translations not defaulting to English if current language was not supported in edu.
