Build 0035 (or Build 1.66.0035.0) is a development version of Xbox 360 Edition. This version is based off Java Edition Beta 1.6.6.


== Removals ==
Removed the option to refresh the online games list in the world menu as it crashed the game when pressed.​[more information needed]


== Fixes ==
Fixed the gamma slider being stuck at 0.​[more information needed]
Fixed for the blocks randomly appearing after destroying it.​[more information needed]


== Bugs ==
There may be a memory leak as the game lags increasingly after playing for a while.
The spawn point is reset after saving and reloading a world.
There may be other bugs not yet documented.


== Trivia ==
This build, and all builds prior, are internally referred to as "Minecraft" instead of "Minecraft: Xbox 360 Edition" when released.
