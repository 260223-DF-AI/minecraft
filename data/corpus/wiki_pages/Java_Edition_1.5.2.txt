1.5.2 is a minor update for Java Edition, released on May 2, 2013, to improve performance and fix some of the bugs and crashes in 1.5 and 1.5.1.
A pre-release version of 1.5.2 was released on April 25, 2013. This pre-release was later reuploaded, and this reupload became the full release of 1.5.2.
This is the last official Minecraft version that runs on PowerPC Mac computers and Java 5, and the last version to be playable without any login required.


== Additions ==
Added banner that displays if 1.6 had been released.
