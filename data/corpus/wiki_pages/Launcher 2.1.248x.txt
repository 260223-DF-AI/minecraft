2.1.2481 (Windows), 2.1.2480 (macOS), or 2.1.2482 (Linux) is an update to the launcher released on February 21, 2019.


== Additions ==
Added a function, which checks the file when a startup fails. (macOS)
Updated translations.


== Fixes ==
MCL-4934: Launcher forgets login token when switching Wi-Fi networks.
MCL-9071: Launcher requires youtube.com.
MCL-10527: .exe won't start the launcher if .minecraft folder is moved using a symbolic link.
MCL-10588: Launcher .deb dependency conflict.
