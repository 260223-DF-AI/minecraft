Alpha v1.0.16_02 is a version of Java Edition Alpha released on August 13, 2010. Corresponding server version 0.1.3 was released on August 17, 2010.


== Additions ==


=== Command format ===
/msg <player> <message> (aliases: /tell and /w)

Sends a private message to another player.
/list

Displays a list of connected players to the operator who issued it.


== Changes ==


=== Gameplay ===
Multiplayer

Operators can now build in the spawn area.
The server logs which admin does what, and broadcasts any admin events to all connected operators.


== Fixes ==
2 bugs fixed

Fixed /save-off.[more information needed]
Fixed a bug relating to the deaths of players and mobs.[more information needed]
