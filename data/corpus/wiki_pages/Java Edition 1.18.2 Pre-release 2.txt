1.18.2 Pre-release 2 (known as 1.18.2-pre2 in the launcher) is the second pre-release for Java Edition 1.18.2, released on February 21, 2022, which adds a density function, and fixes bugs.


== Additions ==


=== World generation ===
Custom world generation

Added spline density function: general-purpose building block that allows user to express almost any function using a cubic spline.


== Fixes ==
7 issues fixed
From released versions before 1.18

MC-214138 – Placing block after jumping from an enchantment table causes desync.
MC-234390 – The minecraft:ui.button.click sound isn't played when clicking on the credits button in the main menu.
From 1.18

MC-243766 – Unable to put focus on "Copyright Mojang AB. Do not distribute" using ↹ Tab key.
From the 1.18.2 development versions

MC-248638 – Strongholds can generate in The Void biome.
From the previous development version

MC-248681 – Superflat worlds cannot be created without using presets.
MC-248694 – An empty generator-settings string crashes the server when starting up.
MC-248717 – Fortress mobs can spawn outside of fortresses.
