Minecraft features a multitude of menu screens, which allow the player to navigate to worlds and servers, and change how the game looks.


== Usage ==


== List of menu screens ==
Title Screen  the menu shown upon game startup allowing for access to worlds.
Server list  a menu which allows players to join and save the servers they visit.
Game Menu  a menu accessible during gameplay which allows players to quit to title or disconnect from a server, access options, enable LAN  and etc.
Select World  allowing one of multiple worlds to be accessed.
Create New World  a menu allowing for the configuration of multiple aspects for a new world.
Seed Templates  a menu to choose a preset seed with specific features or terrain in Bedrock Edition.
World Options  a menu to manage the world settings and management.
Options  which are configurable via a series of menus in Java Edition.
Settings  which are configurable via a series of menus in Bedrock Edition.
Controls  a subset of options, also given a dedicated menu.
Super Secret Settings  which applies in-game shaders, removed after Java Edition 1.9.
How to Play  a menu which shows built-in guides designed to help new players to Minecraft.
Loading world screen  the menu shown while entering or leaving a world.
Realms Stories  a social hub for members of a Realm in Bedrock Edition.
Chat  a messaging system to communicate with players and execute commands.
