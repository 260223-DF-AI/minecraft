Bedrock Editor v0.3.2 is a minor alpha release for the Bedrock Editor released on March 22, 2023, which fixes bugs.


== Fixes ==
Fixed issues with UI not scaling properly when the resolution or game window size is updated.
Item drops have been disabled in Editor.
This allows for Undo/Redo and Cut/Copy/Paste to work properly.
Items can’t be dropped in Tool mode or Crosshair mode.
Fixed a bug where Tool mode would use the third person camera if it was the active camera when switching from Crosshair mode (issue #991927 (#56), issue #992927 (#92)).
Crosshair mode will still remember the user's state and return to the previous setting when switch back to Crosshair mode.
Fixed issues where camera control was disabled when using a Nether or End portal in Crosshair mode (issue #991931).
Removed the old stonecutter and Nether reactor core from the block picker.
