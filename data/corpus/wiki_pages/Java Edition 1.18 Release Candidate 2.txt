1.18 Release Candidate 2 (known as 1.18-rc2 in the launcher) is the second release candidate for Java Edition 1.18, released on November 26, 2021, which fixes a bug related to world generation.


== Fixes ==
1 issue fixed
From the 1.18 development versions

MC-242547 – In large caves, the cave generation can't reach Y=-54, not allowing to generate large lava lakes.
