This page lists all the rows in the debug mode block grid in 22w13oneBlockAtATime. 
There are 151 lines with up to 152 block states. The last line has 56 block states. Of these 150, those that appear at the end of a line are repeated at the beginning of the next. Therefore, there are 22,856 block states in total, with 22,706 unique block states.
The first block is located at coordinates (1, 70, 1) and each block is 2 blocks away from its adjacent blocks, thus the block in row r, column c is at coordinates (2r-1, 70, 2c-1).


== Change blocks states ==
This version has 2233 more blocks states than the previous version.


=== New itens ===
How did we get here?

Attach the grid between row 1, column 2, to row 15, column 97.
It has 2210 block states:
item with the values 0, 1, 2, 3, ..., and 1104.
The number display the item ID on it. See How did we get here? (block)/Display.
waterlogged with the values true, and false


=== Change blocks states ===
Soul Fire

Attach the grid between row 28, column 142, to row 29, column 16.
It has a new block state variable, with 26 values:
age with the values 0, 1, 2, 3, ..., and 25.
Similar to normal fire, but with more values.
Previously, it had no block state variables, so the number of new block stats is 25.
End Portal Frame

Attach the grid between row 51, column 37, to row 51, column 42.
Added two new values and ordering for the block state facing.
Now the values are north, east, south, west, up, and down.
Previously the values were north, south, west, and east.
The boolean variable eye has been removed.
Previously, there were 8 block states in total, now there are only 6, so the number of block stats less is 2.
