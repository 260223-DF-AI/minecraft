2.2.1164 (Windows), 2.2.1166 (macOS), or 2.2.1165 (Linux), is an update to the launcher released on January 5, 2021 and is the first beta released in 2021.


== Changes ==
Redesigned how news articles are displayed.
News for each game is now under its own page
All news can still be found in the "News" tab.
The play button is now sticky.
Moved the Java Edition profile name back next to the play button.
Made the 'No cape' string translatable.
The version manifest fetching now triggers on reconnection.


== Fixes ==
Deleting skins with very long names no longer escapes modal
MCL-15652 - Disabling Wi-Fi while the Microsoft login page is open without any information entered and pressing the Back button results in -106 error message
MCL-16222 - Launcher "fails to respond" (swirling symbol)
