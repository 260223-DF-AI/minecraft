23w51b is the second snapshot for Java Edition 1.20.5, released on December 18, 2023, which fixes a crash bug. This is the final snapshot released in 2023.


== Fixes ==
1 issue fixed
Fixed a crash that would occur when the name tag of a player was visible while sleeping, swimming, or dying.
