A version of Minecraft Indev was released on February 13, 2010, at 22:58 UTC.


== Changes ==


=== Mobs ===
Skeletons

Now burn in daylight.
 Spiders

No longer attack the player in daylight, unless provoked.
Eyes now glow.
Zombies

Now burn in daylight.


=== World generation ===
 Starting house

Now made of wooden planks and stone rather than mossy cobblestone.
Now only spawns in "nice" (flat and unobstructed) areas.


=== Gameplay ===
General

The game now kicks the player to the title screen after generating a world offline.


=== General ===
Music and sounds

Now play once again.
Options

Allocated memory and free memory are only displayed if Show FPS is set to on.
The memory displayer is now formatted with the §7 code (light gray).


== Fixes ==
From Indev 20100207-1703

Slabs no longer cause lighting issues.
From Indev 20100211

Hell worlds no longer start off fully bright before getting darkened.


== Bugs ==
1 bug reported

When the game cannot find a suitable place to spawn a starting house, it simply does not spawn.
Commonly occurs in Small, Deep maps and Small, Long maps.
This bug lasts until the end of Indev.
