1.3.2 is a minor update to Java Edition released on August 16, 2012, to fix various bugs and crashes in 1.3.1.
A pre-release version of 1.3.2 was released on August 14, 2012, for players to test, though the full release of 1.3.2 is identical to this pre-release.


== Changes ==


=== World generation ===
Trees

Large oak trees now generate with sideways logs.


=== General ===
Server list

The player can now toggle showing server addresses in the server list by pressing F1.


== Fixes ==
4 bugs fixed

Fixed players sometimes falling out of the world when entering the Nether if the player has been there before in the same session.
Fixed players taking damage when water or players push them into corners.
Fixed various uncommon crash bugs.
Fixed players not getting fall damage when falling from a height of 4 blocks.
