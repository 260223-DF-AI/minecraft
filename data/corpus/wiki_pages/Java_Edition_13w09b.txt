13w09b is the twelfth snapshot for Java Edition 1.5, released on February 27, 2013 to fix some of the bugs and crashes in the 13w09a snapshot. It was also the first version to include Minecraft Realms.


== Additions ==


=== Command format ===
/effect

Command: /effect <player> <effect> [<seconds>] [<amplifier>].
Maximum number of seconds is 1,000,000.
Amplifier increases level of the chosen status effect.
Maximum number for the amplifier is 4.


=== General ===
Minecraft Realms

Only added to few randomly chosen players.
Exactly 100 players were chosen.
Feature is accessed through button in the title screen.


== Changes ==


=== Items ===
Apples

The Fortune enchantment now increases the chance of dropping apples.
