Illageralt is an unused font in Java Edition, which is based on the font of the same name from Minecraft Dungeons.


== Definition ==
This is a "secret" font not used anywhere in the game. It is a reference to Illageralt, a Minecraft Dungeons rune-like alphabet. It can be used only with the "font" JSON text key.
It references asciillager.png.


== Supported characters ==
asciillager.png
This is a near-exact mirror of the rune-like font used in Minecraft Dungeons.
