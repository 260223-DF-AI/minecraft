15w41b is the thirty-second snapshot for Java Edition 1.9, released on October 7, 2015 to fix a crash that affected some users on launch.


== Fixes ==
7 issues fixed
From released versions before 1.9

MC-856 – Stepping sound loops when 2 or more players are standing on the same block.
MC-16435 – Boats still break on lily pads.
MC-59261 – Villagers with some blocks in head slot either display incorrectly or are not able to be placed.
MC-65421 – Open Lan unlock difficulty world if is locked.
MC-66557 – Boat velocity changes in 14w32b.
From the 1.9 development versions

MC-86166 – No bow animation on Multiplayer.
MC-89894 – Mobs riding slimes causes the game to crash.
