v0.1.3 alpha is a Pocket Edition version which makes various minor changes and bug fixes. This is the final version released in 2011.


== Android ==


=== Additions ===
Added  info_update and info_update2 to be used as placeholders for not yet supported block types.


=== Changes ===
Better tablet support.
Less oversized in game UI elements.
Performance improvements, especially in Non-Fancy graphics mode.
Added cactus to deserts.
Added Turn Sensitivity setting.
Improved network chunk loading.


=== Fixes ===
Fixed a crash when connecting to some iOS server maps.
Fixed blocks that were placed on snow were added incorrectly in multiplayer.


== Internal version ==
There are two different revisions of both the Android 2.3+ full and demo versions. They contain unknown differences in libminecraftpe.so, as well as a minor change to terrain.png.
Version: 0.1.3(1035), Size: 1.58 MB
Version: 0.1.3(1036), Size: 1.58 MB


== iOS ==


=== Additions ===
Added an "Update game!" block used as placeholder for not yet supported block types.


=== Fixes ===
Improved network chunk loading.
Fixed bug with opening dialogs after closing the options menu.


== J-version ==
v0.1.3j alpha was released to the Android Market on an unknown release date, providing support for older Android devices, and added more sounds. The internal version is 0.1.3(1033) and has a size of 2.00 MB.


== Demo ==
A free "lite" demo version of v0.1.3 alpha was released to the Android Market on December 3, 2011. In mid-2014, it was removed from the Play/App stores. The internal version of the first revision is 0.1.3(1035), while the second revision's internal version is 0.1.3(1036).


== J-version demo ==
A free J-version "lite" demo of v0.1.3 alpha was released to the Android Market on an unknown release date. In mid-2014, it was removed from the Play/App stores. The internal version is 0.1.3(1033).
