2.1.14931 (Windows and Dungeons), 2.1.14932 (macOS), or 2.1.14930 (Linux) is an update to the launcher released on May 26, 2020.


== Changes ==
Change Dungeons default product name to Minecraft Dungeons.
Update path validation for setting Minecraft Dungeons install location and update default path.
Display error message in Minecraft Dungeons install dialog if the launcher fails to fetch product manifest.
Fix minor issue with polling dynamic content.
Updated translations.
Apply fix for always showing CTA button for Dungeons, both in Minecraft launcher and in Minecraft Dungeons launcher.
Handle fetching errors that happen in the background smoother.
Handle errors realated to invalid install locations for Minecraft Dungeons in install dialog.
Remove Pause and Cancel buttons in progressbar when repairing Minecraft Dungeons.
UI improvements and copy updates.


== Reupload ==
This version was reuploaded to hotfix MCL-14107 – "Launcher wipes entire install location when uninstalling".
