2.1.5127 (Windows), 2.1.5128 (macOS), or 2.1.5129 (Linux) is a launcher beta version released on June 19, 2019, which fixed bugs.


== Changes ==
Added more logging for connectivity issues.
Enabled selecting of text for copy-paste in various parts of the interface.
Spell checking is now disabled in input fields.
Clicking outside a create/edit installation dialog no longer closes it.
Updated translations.


== Fixes ==
4 bugs fixed
Fixed animation bug in new installations dialog.
Various accessibility fixes.
Fixed a bug causing the Launcher to start in offline mode despite being online.
MCL-10817  Crash after fastly closing and restarting the launcher.
MCL-10994  Some characters are unsupported by the "Play" button font.
MCL-11026  Beta launcher cannot establish connection to the internet.
