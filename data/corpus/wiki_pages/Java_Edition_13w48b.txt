13w48b is the seventh snapshot for Java Edition 1.7.4, released on November 26, 2013 to fix some of the bugs and crashes in the 13w48a snapshot.


== Changes ==


=== Mobs ===
Sheep

Naming a sheep "jeb_" (with a name tag) makes the sheep's wool fade between the different dye colors in order (like a rainbow effect).
Shearing produces the color wool the sheep had before it was named.


== Fixes ==
3 issues fixed
From released versions before 1.7.4

MC-35897  Damaged armor, tools & weapons placed in certain inventory slots cause other slots to go black.
MC-36482  Upon exiting (shift-click) minecart, the range and criteria is drastically off, up to 8 blocks away.
From the previous development version

MC-40947  Opening Broadcast settings crash.


== Trivia ==
13w48b was previously removed from the launcher for some time, but was added back in September 2018.
