1.0.8 is a launcher version released on July 2, July 8, and July 9, 2013. The first release created an issue in which Java Edition 1.6.2 was unplayable on Mac OS X. The second release attempted to fix the issue, but was not successful. The third release was successful in fixing the issue.


== Changes ==
The launcher now has a high-definition icon. 


== Fixes ==
5 issues fixed
MCL-90 – Unable to play on osx 10.5.8 using new launcher (Invalid memory access of location)
MCL-111 – Pressing play in launcher, and quickly closing it leaves java process hanging and throws an exception
MCL-515 – Text can Overlap in Launcher Resize
MCL-633 – 1.0.4 breaks the authentication system
MCL-756 – New launcher won't launch previous versions (1.5.2) on OSX (newest version works fine).
