1.19.52 is a major update to Minecraft Education, released on April 12, 2023, which introduces new features from The Wild Update.


== Additions ==

Added features from The Wild Update.


=== General ===
Library
Added computer science folders.


== Changes ==


=== General ===
Library
Improved Search featuring tag search, better non-English searching, and more.
OS minimum requirement
Changed OS minimum requirement, as following:
macOS Big Sur (11.0)
Windows 10
iOS 11
Sign-In Experience
Switching from Azure Active Directory Authentication Library (ADAL) to Microsoft Authentication Library (MSAL) for Microsoft authentication.
User Experience
Easier to use tooltips and control hints in-world.


== Fixes ==
Fixed some bugs.
