2.3.228 is an update to the launcher released on June 8, 2022.


== Additions ==
Added code to make the new Java + Bedrock bundle work.
Work to support M1 Mac devices.


== Changes ==
Various entitlements, offline mode & stability improvements.


== Fixes ==
there are three bugs plus other game instalation fixes present in this version of the launcher.

Game installation bug fixes.
MCL-19213 - Cannot open relative game directories in system file picker.
MCL-21218 - Play button size doesn't match across the 3 game pages.
MCL-21425 - Unable to update Launcher.
