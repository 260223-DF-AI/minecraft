0.0.16a_02 is a version of Java Edition Classic released on June 7, 2009, at 18:41 UTC, which released multiplayer and fixes bugs.
0.0.16a_02 was reuploaded for the first time at 10:27 UTC on June 8, 2009, to do some testing. This version had a bug with an unavailable input stream from minecraft.jar.
0.0.16a_02 was soon reuploaded for the second time at 10:36 UTC to fix the unavailable input stream bug, but this fix was not successful.
0.0.16a_02 was reuploaded for the final time at 10:47 UTC to fix a bug with an unavailable input stream from the minecraft.jar file.


== Additions ==


=== Gameplay ===
Multiplayer

== Fixes ==
4 bugs fixed

Fixed a connection problem.
Partially fixed /banip.
Fixed flickering on the login screen.
Fixed being able to spawn mobs in multiplayer.


== Bugs ==
2 bugs reported

Unavailable input stream from minecraft.jar.
This bug was fixed by reupload 3.
It was reported that a player swimming diagonally towards a corner where the right side is higher than the left side wouldn't be able to get onto the land until they look away from the block.
