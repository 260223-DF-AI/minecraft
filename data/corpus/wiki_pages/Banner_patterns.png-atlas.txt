banner_patterns.png-atla is a procedurally-generated texture atlas used to store textures for banners and all banner patterns.


== Format ==
banner_patterns.png-atlas is a fixed-width 512512-pixel file that store textures for all banner patterns.


== History ==


=== Versions of the atlas ===


==== Mipmapped versions ====

Minecraft also previously generated mipmapped versions of this atlas, for unknown reasons. They are not used by the game at all  only blocks placed in the world which use the json model system use mipmapped textures. The mipmapped version no longer generates since 1.21.9.
