The nerd creeper is a hostile creeper-based mob present in Java Edition 3D Shareware v1.34.


== Spawning ==
Nerd creepers spawn when the player uses the cheat code "NEEEERD".
Creepers in this version have an NBT tag called [Boolean] d20 1 or 0 (true/false) that determines whether the creeper is a nerd.
A nerd creeper can be summoned with the following command: /summon creeper ~ ~ ~ {d20:1}


== Drops ==
When killed, nerd creepers drop: 

5XP experience points, if killed by the player or a tamed wolf.
1 3D item.
1 music disc at random, if killed by a skeleton or stray.
1 creeper head, if killed by a charged creeper.


== Behavior ==
Nerd creepers behave the same way as creepers except when killed, upon which they drop the 3D item in addition to gunpowder.
