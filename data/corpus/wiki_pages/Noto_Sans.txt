Noto Sans is a font used by Bedrock Edition for long text that might be difficult to read when using the default pixel font, Mojangles. It is also used by the Minecraft.net website and the Minecraft Launcher.
The version of Noto Sans bundled with Bedrock Edition is 2.000, released in 2017. The same version is used by the Minecraft.net website as a webfont if Noto Sans is not installed locally. The launcher comes with the 1.04 version, released in 2012. For comparison, the latest version available is 2.015, released in 2024.
Noto Sans can also refer to a font subfamily which also includes fonts covering non-Latin writing scripts, such as Noto Sans CJK (also known as Source Han Sans), which is used for Chinese characters, kana and hangeul.
smooth/ is a version of Noto Sans rendered into PNG character atlases.
