1.21.9 Release Candidate 1 (known as 1.21.9-rc1 in the launcher) is the first and only release candidate for Java Edition 1.21.9, released on September 25, 2025.


== Fixes ==
2 issues fixed
From the 1.21.9 development versions

MC-302243  Creakings can spawn with the spawnMonsters game rule set to false.
From the previous development version

MC-302278  The texture in the middle of conduits is upside down.


== Trivia ==
This version has the same protocol version number as 25w41a. The two versions are not actually compatible; 1.21.9-rc1 throws a "Network Protocol Error" message when attempting to join a 25w41a world, and 25w41a crashes when attempting to join a 1.21.9-rc1 world.
