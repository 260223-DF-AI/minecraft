0.30 refers to 2 versions of Java Edition Classic released on November 10, 2009, with the Creative variant at 18:21 UTC and Survival at 18:49 UTC. They were announced on TWoN at 19:05 UTC, by which time they had already been reuploaded. The only difference between these two versions is that one version is in Creative mode, while the other is in Survival mode. The Creative version was the one featured for free on the Minecraft website until its removal in September 2015. Server versions 1.10 and 1.10.1 were released for 0.30 on December 11, 2009. 0.30 is the final version of Classic development to be released. The Survival variant of 0.30 is the final version of Survival Test, while the Creative variant is the final version of Late Classic Creative.
The Survival variant was reuploaded at 18:58 UTC and the Creative at 19:00 UTC.[more information needed]
A 2011 recompile of the Creative variant containing renewed signature files is available in the launcher, mislabeled as 0.30_01c. Gameplay is identical to the original 2009 build. A hash for "0.30_01a" was included in a list of version hashes compiled by Searge, but it was actually for a texture pack for 0.30, included in Mojang's archives likely by mistake.


== Changes ==


=== Blocks ===
Grass block

Attempting to place grass blocks by means of hacking one's inventory in  multiplayer now causes the server to automatically kick the player.
TNT

Hitting a primed TNT (Survival only) defuses it and lets it be picked up.
Explosions no longer produce smoke particles.


=== Mobs ===
Sheep

Now can be spawned legitimately in Survival, as previous versions required modifying the game to be in Survival to allow them to spawn.


=== General ===
Loading and saving

Drastically improved world generation time.
Previously, generating a new world could take up to 2 minutes. From 0.30 onward, generating a new world only takes a few seconds.
Added local file saving.
Online saving is no longer available for free accounts.
A "Premium only!" message is displayed if a player tries to save a world online with a free account.


== Fixes ==
5 bugs fixed

Blocks now drop themselves again.
Fixed the sheep hitbox so it no longer has areas the player can hit through.
In Survival mode, mobs other than sheep can now spawn naturally.
TNT no longer disappears upon being hit in Survival.
Multiplayer lag decreased.
