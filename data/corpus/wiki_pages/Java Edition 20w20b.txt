20w20b is the seventeenth snapshot for Java Edition 1.16, released on May 14, 2020, which fixes some bugs.


== Fixes ==
6 issues fixed
From released versions before 1.16

MC-44936 – Extending a potion effect that you already have can cause the timer to display wrong.
MC-134057 – Recipe book preferences don't save.
MC-145102 – "--server" client launch argument may crash the client on the loading screen.
From the 1.16 development versions

MC-172737 – Crafting table menu forgets 'show crafting' setting.
MC-176755 – Server logs error on start: Unable to bootstrap registry 'minecraft:chunk_generator_type'.
From the previous development version

MC-183769 – NPE in server initialization.

== Trivia ==
This snapshot was released on slicedlime's 40th birthday.
This is the last version of Minecraft that uses the old Mojang logo from 2013 (first is 13w47a).
This snapshot changed the main class of the server from net.minecraft.server.MinecraftServer to net.minecraft.server.Main .
