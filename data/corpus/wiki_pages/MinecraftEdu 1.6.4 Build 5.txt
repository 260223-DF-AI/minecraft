1.6.4 Build 5 is the fifth build for MinecraftEdu 1.6.4, released in October 2013. It is based on Java Edition 1.6.4.


== Additions ==


=== General ===
Launcher

Mute all students & Freeze all students buttons back to Player Settings menu.
Added a lot of hidden settings to launchersettings.ini that can be used to hide certain components from launcher starting screen:
disable-launcher-top-menu=true
disable-launcher-top-menu-file=true
disable-launcher-top-menu-open=true
disable-launcher-top-menu-settings=true
disable-launcher-top-menu-change-language=true
disable-launcher-minecraftedu-button=true
disable-launcher-minecraft-button=true
disable-launcher-minecraft-custom-button=true
disable-launcher-servertool-button=true
disable-launcher-updates-button=true


== Fixes ==
3 bugs fixed

Custom skins not rendering to other players.
Netherrack on fire trying to spread if fire/tnt was disabled.
When starting mcedu / mc in offline mode, it now checks if the corresponding jar file is not found, not that both mc & mcedu jar files are not found.
