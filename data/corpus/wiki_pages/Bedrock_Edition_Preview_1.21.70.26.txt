Beta 1.21.70.26 (ChromeOS, Android) or Preview 1.21.70.26 (Windows, iOS, iPadOS, Xbox, PlayStation) is the sixth and final beta/Preview version for Bedrock Edition 1.21.70, released on March 5, 2025,  which fixes bugs.


== Changes ==


=== General ===
Title screen

The panorama was changed to showcase the Spring to Life drop.
Spawn event

Pigs, cows, and chickens can now be spawned via spawn events, and /summon or /event can be used to spawn the cold and warm variants.


== Fixes ==
5 issues fixed
From released versions before 1.21.70

MCPE-190093  Bundles lose contents when given to armor stand
MCPE-190747  Server IP Address Visible
From the previous development version

MCPE-190466  Invincible mobs created with the old glitch disappear
MCPE-190710  Shulker bullet gives 20 times longer levitation
MCPE-190760  New client entity definition for zombie_villager_v2 has the wrong file name, making the compiled Bedrock samples misleading
