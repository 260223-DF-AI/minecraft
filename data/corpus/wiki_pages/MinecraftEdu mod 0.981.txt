0.981 is a minor update to MinecraftEdu, which adds support to 1.2.5, WorldEdit, improved skins, new panel information, shortcuts menus, and more. It is based on Java Edition 1.2.5.


== Additions ==


=== General ===
Performance improvements. "Can’t keep up" errors should be mostly fixed.
WorldEdit integration & latest WorldEdit is now shipped with MinecraftEdu.​[more information needed]
Added new shortcuts menu, so the player can bind some commands & menus to be opened with certain keys.
Custom texturepack support for maps.​[more information needed]
Improvements to graphical user interfaces.
New skins & all the old skins have been improved.
