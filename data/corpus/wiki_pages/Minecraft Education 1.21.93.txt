1.21.93 (Windows) or 1.21.93.1 (iOS), is a minor update to Minecraft Education, released on October 30, 2025, which fixes some bugs.


== Changes ==
Updates to MakeCode:
Added more events.
Code does not run automatically anymore when closing.


== Fixes ==
Fixed switching touch controls on Android.
Behavior and resource packs are correctly included in the world again when opened from My Templates.
Fixed a login error.
