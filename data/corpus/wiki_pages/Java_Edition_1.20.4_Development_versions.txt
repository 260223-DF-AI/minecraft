== 1.20.4 Release Candidate 1 ==
1.20.4 Release Candidate 1 (known as 1.20.4-rc1 in the launcher) is the first and only release candidate for Java Edition 1.20.4, released on December 6, 2023, which fixes a critical bug. This is the final release candidate released in 2023.
