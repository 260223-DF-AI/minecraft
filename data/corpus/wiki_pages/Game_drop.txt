A game drop (or simply drop) is a type of named update containing a number of features related to a theme. Three to four game drops are released each year.


== Description ==
Game drops are loosely planned up to two years at most in advance, give or take, depending on community feedback. A game drop can be small, like Bats and Pots, or large, like Mounts of Mayhem. Game drops can have any number of features, and can tackle any feature types.
The game drop system was first announced with the release of Armored Paws. Bats and Pots was then retroactively called a drop. Due to being released after the announcement of the game drop system, Tricky Trials would also qualify as a drop but is not officially considered one.
For the most part, one game drop is released in each business quarter of a year.

== Game drops by season ==
Game drops can be classified into four types, based on which Northern Hemisphere season they are released in:

Spring Drop
Summer Drop
Fall Drop
Winter Drop
Although it is no longer the primary classification for game drops, Mojang Studios has used this nomenclature on several occasions:

The Garden Awakens has explicitly been referred to as the "Winter Drop" during its experiment phase in Java Edition.
Spring to Life and The Copper Age were respectively referred to as "Spring drop" and "Fall drop" in the manifest.json file of their respective experimental packs in Bedrock Edition.
Chase the Skies was named "Summer drop" by a Mojang employee.
Some files on Minecraft.net use season to refer to drops in their names; for example :
MCV_SpringDrop_testing_Jan8_Wildflowers_8a_1280x720.jpg uses "SpringDrop".
MCV_Su25_VibrantVisuals_April2_Overworld_2_1280x720.jpg uses "Su25" for "Summer 2025".
MCV_FA25_July22_02_Feature_CopperGolemStatue_C_DotNet_1280x720.jpg uses "FA25" for "Fall 2025".
