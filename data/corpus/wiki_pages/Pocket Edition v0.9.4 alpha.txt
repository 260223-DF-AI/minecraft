v0.9.4 alpha is a minor update to Minecraft: Pocket Edition released on July 17, 2014 for Android and on July 19, 2014 for iOS. It fixes some of the bugs and crashes in v0.9.3 alpha. For iOS devices, this version includes changes from v0.9.3.


== Changes ==
When returning to the game after locking the device, going to the home screen or switching apps, the game opens up in the pause menu
Zombies now have 2 () armor points, preventing 8% of damage from other mobs or the player.


== Fixes ==
2 issues fixed
From released versions before 0.9.0

MCPE-6187 – Baked Potatoes missing name and description
Others

Fixed a crash which occurred after locking the device
