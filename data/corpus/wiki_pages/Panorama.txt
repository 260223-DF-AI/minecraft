The panorama (also known as the cubemap) is a slowly panning wide-angle view of a Minecraft world displayed on the title screen and most other menu screens. It is themed after the latest named update and can be customized with a resource pack.


== Description ==
The panorama is shown as a background in all menu screens that do not cover the entire background. This includes all menu screens in Java Edition and in Bedrock Edition the main menu, loading world screen (except Realms), pop-up windows, all Ore UI menus, and the Dressing Room. Non-Ore UI error screens display a static background instead.
A separate panorama is shown when Vibrant Visuals is enabled. This consists of HDR image files which are rendered with the post-processing effects from Vibrant Visuals; the default data-driven color grading settings, and the adjustable gamma, upscaling mode, and resolution. Exclusive panoramas are also shown for Minecraft Preview with Vibrant Visuals disabled and Minecraft Education.
In the options, the panorama scrolling can be adjusted with the "Panorama Scroll Speed" accessibility setting in Java Edition and the "Screen Animations" video setting in Bedrock Edition. A blur overlay can be adjusted in the options in Java Edition.
The panorama files in Java Edition are found in the indexes directory as multiple square images (panorama_<0-5>.png) and are extracted in the same way as sound files, as denoted in Tutorial:Sound directory. In Bedrock Edition, the regular panorama is stored in the regular vanilla resource pack in  textures/ui. Each Vibrant Visuals panorama (including discontinued) is stored in the internal resource pack of the respective version as .hdr files.
With resource packs, the vanilla panorama may be overwritten with custom images. Six square (1080x1080) images named panorama_<0-5>.png (or panorama_<0-5>.hdr for Vibrant Visuals) need to be placed in the  assets/<namespace>/textures/gui‌[JE  only][verify]/ textures/ui‌[BE  only] directory. The first four panoramas are used horizontally from left to right, panorama_4 is rendered above panorama_0 and panorama_5 is rendered below the first panorama. Custom panoramas reverse the animation in Bedrock Edition by default (moving right instead of left). Several Marketplace add-ons and Minecraft Education events contain custom panoramas.

Launching a world created from a world template locked in older versions with Vibrant Visuals enabled briefly switches the panorama to a historical panorama from that version, or the missing texture before 1.21.90.


== Current panoramas ==


=== Tiny Takeover ===

The panoramas in Java Edition and Bedrock Edition are different.
In Java Edition, this panorama was introduced in 26.1 Pre-Release 1, currently used in 26.1 Snapshot 2 onward. In Bedrock Edition, this panorama was introduced in Preview 26.10.25. A different panorama shows when Vibrant Visuals is enabled.
The Java Edition panorama was taken in Java Edition, whereas both Bedrock Edition panoramas were taken in Bedrock Edition. All three panoramas were taken in worlds using the seed -1701815224735573849 at X=-256 / Y= 126 / Z=-2007. The world spawn of this seed is located in a desert biome at the coordinates X=266 / Y=78 / Z=-534 and the coordinates X=-256 / Y= 126 / Z=-2007 are in a cherry grove biome 1563 blocks to the north.


=== Copper, Collaborate & Compete ===

In Minecraft Education, this panorama was introduced in 1.21.132. It is the same as the one used in The Copper Age, with agents and balloons edited into the image.


=== Minecraft Preview ===

This panorama is exclusively available in Previews of Bedrock Edition and Minecraft Education after Preview 1.18.20.22 and Preview 1.21.40.1 respectively, and in betas after Bedrock Edition beta 1.21.80.21. It is the only panorama that the player can see in Previews and betas when Vibrant Visuals is turned off.
This panorama was taken on the seed 2510333169515186510 at coordinates X=979 / Y=71 / Z=-11, in Java Edition.
Although this panorama is exclusive to Bedrock Edition and Minecraft Education, it was actually taken in Java Edition 1.17.1, which can be inferred from the color of water, the way in how clouds fade at high distances, the back side of leaves not rendering, and the side texture of the dirt path.

== Trivia ==
In the Legacy Console Edition 0.66.0054.0 panorama, a second castle can be seen in the far distance. This was part of a development version of the tutorial world, which was scrapped before Minecraft's release on Xbox 360.
