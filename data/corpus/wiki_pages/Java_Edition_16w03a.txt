16w03a is the fiftieth snapshot for Java Edition 1.9, released on January 20, 2016.


== Changes ==


=== Blocks ===
Beacons

No longer immediately activate when placed on valid pyramids, as they had in the previous snapshot.


=== Non-mob entities ===
Minecarts

All types of minecarts now display their CustomName when CustomNameVisible is set.


=== Command format ===
/achievement

No longer affects statistics.


=== General ===
Sounds

More new sounds.[more information needed]
Some updated sounds.[more information needed]
