1.14.2 is a minor update to Java Edition released on May 27, 2019, which fixes bugs left over in 1.14.1. It is not compatible with 1.14.1 servers.


== Additions ==


=== General ===
Font

Added a default Vietnamese font.
Changed the character U+0540 (Ȝ) in the default Armenian font.
Added missing glyphs in the default Hebrew and Armenian fonts.
Added Runic alphabet to the default font.


== Changes ==


=== Blocks ===
Campfire

Flaming arrows can no longer light waterlogged campfires.
Shulker boxes

Empty shulker boxes no longer drop themselves when destroyed in Creative mode.


=== Mobs ===
Parrots

Can now spawn on grass blocks instead of only grass.


=== Gameplay ===
Raids

Raiders now only spawn in fully loaded chunks.


=== General ===
Debug screen

Now contains a server-side chunk count.
Error messages

Errors encountered while loading regions now print more diagnostics to the log file.
Lighting

All light is now re-calculated the first time a world saved in a previous version is opened in this version.
Splashes

Removed "Woo, minecraftforum!" as the forum announced it would soon be archiving.
