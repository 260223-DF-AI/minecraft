v0.8.2 alpha was a minor update to Minecraft: Pocket Edition released on April 12, 2014 exclusively for Fire TV. This is the first version released for the Fire TV, though it was available previously for the Fire. This is the first version released in 2014.


== Additions ==
Added controller support for the Fire TV's controller
Added interface for controller features


== Changes ==
Has a reduced number of options
