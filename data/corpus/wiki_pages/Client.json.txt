client.json is the file that accompanies client.jar in .minecraft/versions/<version> and lists the version's attributes. When using the latest version of the Minecraft Launcher, it is named <game version>.json (e.g., 1.21.7.json). The JSON file for specific versions is located in the version_manifest.json file.
