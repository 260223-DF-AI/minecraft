This is a list of Launcher 2.1.596x development versions.


== 2.1.547x ==
2.1.5478 (Windows), 2.1.5476 (macOS), or 2.1.5477 (Linux) is a launcher beta version released on June 28, 2019, which fixed bugs.


=== Changes ===
The arrow on drop-downs now dims and moves up a little bit when the drop-down is not being hovered on.
The Help screen now has a link to the Minecraft Launcher Feedback site.
Various small styling improvements.
Updated translations.


=== Fixes ===
Fixed an issue where the multiple game instance warning did not show for profiles that had previously changed folders.
Fixed switch account screen not being scrollable with many accounts added.
MCL-6629  Launcher floated out of desktop boundaries when taskbar is at the top.
MCL-6544  Launcher opens up on nonexistent monitor.
MCL-10230  Erases all launch settings after computer hard-locks.
MCL-11067  "<language> is x% translated!" popup is never translated.
MCL-11117  View crash report button does nothing.


== 2.1.55xx ==
2.1.5560 (Windows), 2.1.5559 (macOS), or 2.1.5558 (Linux) is a launcher beta version released on July 3, 2019, which fixed bugs.


=== Changes ===
Added support for Dark mode on OS X.
Updated log window with new styling.
Removed weird pixels from login background image.
Updated translations.


=== Fixes ===
Fixed a bug causing error messages while logging in to immediately disappear.
Fixed a bug in how version file fields for downloads and logging are treated for versions which inherit other versions.
Fixed the downloads field of versions not being treated as optional.
Fixed empty lines sometimes appearing in the launcher log file.
Fixed the drop down arrow in the installation selection sometimes covering up the text.
MCL-8389 - Log output not auto-scrolling.


== 2.1.581x ==
2.1.5819 (Linux), 2.1.5816 (macOS), or 2.1.5815 (Windows) is a launcher beta version released on July 16, 2019, which fixed bugs.


=== Changes ===
More information about how the launcher starts the game is now logged.


=== Fixes ===
MCL-8702 - Launcher black screen - ERROR:gl_surface_egl.cc(711)] eglCreateWindowSurface failed with error EGL_BAD_ALLOC
MCL-10036 - minecraft launcher does not appear on main display if closed on secondary display which is then disconnected before launch
Fixed a bug that meant the log files were missing information on macOS.
Fixed an error where the system language was not properly used by the launcher.
Fixed a bug causing the Launcher Log window to use up performance when open.
Fixed a bug where the --workDir argument did not work on certain systems.


== 2.1.590x ==
2.1.5908 (Linux), 2.1.5907 (macOS), or 2.1.5906 (Windows) is a launcher beta version released on July 22, 2019, which fixed bugs.


=== Fixes ===
MCL-10463 - Minecraft freezing on startup
MCL-10963 - Application Data directory is not changed with --workDir argument
MCL-11268 - Native launcher does not set current working directory correctly when launching the game
MCL-11315 - Relative paths in Game Directory are used twice
MCL-11393 - On startup, the launcher freezes at around 80-90% and then crashes.
MCL-11565 - When I press play the minecraft launcher doesn't load and an error appears: "Game Crashed"
Fixed a bug that caused the game to start with the wrong arguments on some systems.
Fixed a bug that caused the game to start with the wrong directory setting when running with a custom directory.
