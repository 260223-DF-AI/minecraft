1.20.2 Release Candidate 1 (known as 1.20.2-rc1 in the launcher) is the first release candidate for Java Edition 1.20.2, released on September 15, 2023, which fixes bugs.


== Changes ==


=== General ===
General

The positions that all entities ride on minecarts have been adjusted to make more sense.

From the previous development version

MC-265388  Mobs now sit too high in minecarts.
