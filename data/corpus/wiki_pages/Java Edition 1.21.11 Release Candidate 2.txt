1.21.11 Release Candidate 2 (known as 1.21.11-rc2 in the launcher) is the second release candidate for Java Edition 1.21.11, released on December 5, 2025, which fixes a crash.
A separate unobfuscated version of 1.21.11 Release Candidate 2 was released, titled 1.21.11 Release Candidate 2 Unobfuscated (or 1.21.11-rc2_unobfuscated when in the launcher). This is an experimental version released in preparation for obfuscation being removed from Java Edition clients and servers.


== Fixes ==
Fixed a crash issue affecting some players.
Fixed by adding safety checks for block and chunk positions while generating or loading terrain.
