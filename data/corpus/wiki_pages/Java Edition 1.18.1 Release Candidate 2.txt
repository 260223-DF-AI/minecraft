1.18.1 Release Candidate 2 (known as 1.18.1-rc2 in the launcher) is the second release candidate for Java Edition 1.18.1, released on December 8, 2021, which fixes a bug that caused chunk loading issues.


== Fixes ==
1 issue fixed
From the previous development version

MC-245010 – Sometimes certain chunks will never load.
