This is a list of Java Edition 1.7.10 development versions.


== 1.7.10-pre1 ==
1.7.10-pre1 is the first pre-release for Java Edition 1.7.10, released on June 3, 2014, containing Realms changes.


=== Changes ===


==== General ====
Minecraft Realms

All Realms-related modules were moved into a separate library.
Future additional features to Realms will not require new releases of Minecraft.
New features
Player activity monitor.
Access to more server settings.
PVP: On/Off.
Difficulty: Peaceful/Easy/Normal/Hard.
Spawn protection: Off/1..16.
Gamemode: Creative/Adventure/Survival.
Spawn animals: On/Off.
Spawn monsters: On/Off.
Spawn NPCs: On/Off.
Improvements in Realms UI.


=== Fixes ===
1 issue fixed
MC-55580  Minecraft Realms: User will still have Operator rights after re-inviting them.


== 1.7.10-pre2 ==
1.7.10-pre2 is the second pre-release for Java Edition 1.7.10, released on June 4, 2014, which fixes some minor bugs and a crash in 1.7.10-pre1 related to being kicked from a server.
1.7.10-pre2 was reuploaded shortly after release to fix the version number still saying "1.7.10-pre1".


=== Fixes ===
1 issue fixed
Fixed a crash related to being kicked from a server.


== 1.7.10-pre3 ==
1.7.10-pre3 is the third pre-release for Java Edition 1.7.10, released on June 10, 2014, to fix some bugs and a crash in 1.7.10-pre2 related to the disconnect screen.


=== Fixes ===
1 issue fixed
Other fixes

Fixed a bug related to the disconnect screen.


== 1.7.10-pre4 ==
1.7.10-pre4 is the fourth and final pre-release for Java Edition 1.7.10, released on June 18, 2014, which provides further upgrades to Realms.


=== Additions ===


==== General ====
Minecraft Realms

The ability to upload worlds to Realms.
An option to allow the Realms owner to enable command blocks.


=== Fixes ===
2 issues fixed
From the previous development version

MC-56572  Realms player activity screen squashed to bottom.
Other fixes

Fixed some possible future-compatibility issues.
