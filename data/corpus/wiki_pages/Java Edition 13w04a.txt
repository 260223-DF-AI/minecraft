13w04a is the sixth snapshot for Java Edition 1.5, released on January 24, 2013.
An experimental version of 13w04a was released by Dinnerbone on Mojira to try and fix a white line bug.


== Additions ==


=== Blocks ===
 Smooth sandstone

This block acted as a double sandstone slab. Breaking this block gives you 2 sandstone slabs.
Currently unobtainable.
 Smooth quartz

Currently unobtainable.


=== Mobs ===
General

Mobs now show the name given to them, either by renaming their spawn egg on an anvil or by external editors, above their head when the player looks directly at them within a circular 6 block radius.


=== Command format ===
Scoreboard

A new command feature.
Not 100% done yet.
The scoreboard can be used to track deaths, players killing another player, players killing something a mob, or players activating commands.
The complicated system uses a combination of name, display name, and criteria, and track an integer number of points for players.


== Changes ==


=== Blocks ===
Command blocks

Now use their name instead of @ in the chat.
Dispensers

Now can face in all directions.
Now equip players with armor from one block away.
They now activate TNT.
They now use bone meal.
They now use flint and steel.
This accurately decreases the durability of the tool.
Droppers

Now can face in all directions.
Redstone comparators 

Can now be used with jukeboxes.
Output signal strength relates to the disk number.
Slabs

43:8 is now the "smooth stone full half-slab block".
In general, if the top bit is set, the top texture is used on all sides, also allowing for smooth sandstone full half-slab blocks (43:9).


=== Items ===
Armor

Held armor can be equipped by right-clicking.
In this particular snapshot, this functionality is very faulty, prone to crashing and incorrect slot usage.
It will be fixed in the next snapshot.
Bone meal

No longer gives instant growth – instead, each use advances one growth stage:
Wheat seeds require seven bone meal.
Saplings and cocoa beans require two bone meal to grow into trees and cocoa pods, respectively.
Brown and red mushrooms still grow into huge mushrooms instantly.
Gives off green particles when used.
Note: Dinnerbone has stated that he still has to balance out this new bonemeal growth system.


=== Mobs ===
General

Mobs don't actively path over rails.


=== Gameplay ===
Creative

The player can now hit mobs/players from five blocks away.
This does not apply to Survival and Adventure mode, retaining the normal range.
