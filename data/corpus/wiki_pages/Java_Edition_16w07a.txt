16w07a is the fifty-fifth snapshot for Java Edition 1.9, released on February 15, 2016.


== Additions ==


=== Items ===
 Maps

Added sounds for making maps.


=== Command format ===
Gamerules

Added disableElytraMovementCheck.
When set to true, disables the server checking whether the player is moving too fast (cheating) while wearing elytra.
Sound events

Added block sound events:

chorus_flower.death and grow.
comparator.click.
furnace.fire_crackle.
iron_trapdoor.close and open.
Added enchant sound event:

Added entity sound events:

cow.milk.
horse.eat.
itemframe.add_item, break, place, remove_item, and rotate_item
leashknot.break and place.

== Changes ==


=== Items ===
Elytra 

Are now only activated when jumping mid-air.
The cape option in the options menu now also disables custom elytron textures.


=== World generation ===
Obsidian pillars

Removed the iron bars that were protecting the ender crystals during the dragon fight.
