== 18w30a ==

== 18w31a ==
18w31a is the third snapshot for Java Edition 1.13.1, released on August 1, 2018, which adds some new features, changes some existing features, optimizes performance, and fixes many bugs.


=== Additions ===


==== General ====
Chunks

Added a concept of force-loaded chunks to the game, and a command (/chunk) to toggle force-loading on and off.
Block tags

Added the tag minecraft:underwater_bonemeals.


=== Changes ===


==== Blocks ====
Conduits

Are now properly waterlogged.


==== Mobs ====
Fish

Now have a 5% chance of dropping bone meal when killed.
Squid

Now spawn only in rivers, oceans, or beaches.


==== Items ====
Bone meal

When used in water of warm ocean biomes, it now has a chance to grow coral.


==== Command format ====
Scoreboards

The scoreboard operator %= was changed from using Java's native % to using Math.floorMod. This causes the operator to use floored modulo instead of truncated modulo, so now -1 % 7 wraps to 6 instead of returning -1.
Functions

Line numbers in function parse error logs now start with 1 (was 0).


==== General ====
Performance has been optimized.
