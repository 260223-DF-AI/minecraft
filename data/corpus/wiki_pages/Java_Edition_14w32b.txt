14w32b is the forty-third snapshot for Java Edition 1.8, released on August 7, 2014, to fix some bugs and crashes from 14w32a.


== Additions ==


=== Blocks ===
Fences

New types of fences for all the different types of wood.
Different types can connect to each other.
Original fence renamed to oak fence.
New crafting recipe using 2 sticks and 4 planks to make 3 fences.
Fence gates

New types of fence gates for all the different types of wood.
Different types can connect to each other.
Original fence gate renamed to oak fence gate.
Can now catch fire.


=== Command format ===
/entitydata

Works similarly to /blockdata but can be targeted to entities.
Does not work on players.
Usage: /entitydata <entity> <dataTag>.


== Changes ==


=== Blocks ===
Armor stand

Can now use pick-block on armor stands (gives the stand, not the armor equipped).
Now craftable using only stone slabs instead of any stone-type slab.
Beacon

Beacon beam changes color instantly.
Beam now goes through all blocks that don't completely block light: ice, water, and leaves.

== Trivia ==
14w32b was previously removed from the launcher for some time but was added back in September 2018.
Due to a bug, jungle fences and gates are uncraftable in 14w32b. Instead, it causes the spruce variants to be crafted.
