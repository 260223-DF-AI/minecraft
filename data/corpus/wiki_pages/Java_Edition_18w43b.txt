18w43b is the second snapshot for Java Edition 1.14, released on October 24, 2018, which fixes bugs in 18w43a.


== Changes ==
Items
    

Changed textures of Bucket of Fishes and Pufferfish.
Effects
          

Removed the background and fixed some incorrect textures.
General

Cleaned up memory leaks in the client.


== Fixes ==
8 issues fixed
From released versions before 1.14

MC-117030  Forest ground too dark when pillaring up
MC-117094  Bright spots below /fill-ed platform
MC-124241  Score and Selector text components finalize with the result of the first instance the command runs as.
From the previous development version

MC-137443  A few effect icons are swapped/missing.
MC-137454  Opening Loom GUI doesn't darken surrounding background.
MC-137457  Crash upon opening the F3 menu.
MC-137492  Spruce trees drop leaves instead of saplings.
MC-137530  Inserted text for %s in a translation string shows 'null'.

== Trivia ==
This snapshot, being the second for 1.14, was released in the same week of the year as the second snapshot of the previous major update, 17w43b.
As with 18w43a, 18w43b was released exactly 6 years after the Minecraft issue tracker.
