0.0.14a_08 is a version of Java Edition Classic released on May 28, 2009, at 20:40 UTC, which made changes. This is the last version of Early Classic (i.e. the last Classic version without multiplayer functions).


== Changes ==


=== Blocks ===
Saplings

Now appear as full bright when in shade, and turn invisible when selected while in shade.


=== World generation ===
 Trees

Model changed from .
Now have a (+) shape for the top layer.


=== General ===
Light

Shadows now show up on water and lava surfaces.
Fog

Fog is now white instead of blue.
Fog now gets applied to shadow surfaces.


== Technical ==


=== Performance ===
Improved framerate.


== Trivia ==
The terrain.png file of this version seems to have an early texture for wool, clouds, or snow blocks.
