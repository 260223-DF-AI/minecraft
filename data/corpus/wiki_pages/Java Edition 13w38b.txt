13w38b is the sixth snapshot for Java Edition 1.7.2, released on September 20, 2013, to fix some of the bugs and crashes in 13w38a.


== Additions ==


=== General ===
Options

"Super Secret Settings"
Added bumpy shader.


== Changes ==


=== Blocks ===
Air

Redefined the concept of air.
Is now an actual block instead of the absence of a block (null).
It can be kept in the inventory.
When in the inventory, dropped, and inside an item frame, it uses the missing texture, but when held in the player's hand, it appears entirely invisible.
Drops from entities that would normally drop nothing, including villagers, bats, silverfish, and the ender dragon.


=== World generation ===
Swampland

Removed marsh-like areas (later readded in Java Edition 14w26a).


== Fixes ==
7 issues fixed
From the previous development version

MC-31613 – Hopper: java.lang.NullPointerException - Clicking in video settings screen.
MC-31631 – Main Menu does not display correctly.
MC-31651 – Block break animations no longer display.
MC-31754 – Map disappearing when leaving world.
MC-31777 – Scoreboard does not display sidebar anymore.
MC-31782 – Chatbar background doesn't show when typing.
MC-31846 – Placed tripwire string does not show its texture.


== Trivia ==
13w38b was previously removed from the launcher for some time, but was added back in September 2018.
As with 13w38a, this snapshot was released on the same day as 1.6.4.
