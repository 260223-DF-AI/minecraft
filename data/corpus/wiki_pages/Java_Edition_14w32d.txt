14w32d is the forty-fifth snapshot for Java Edition 1.8, released on August 8, 2014, which adds new doors and fixed a bug in 14w32c that kicked the client when joining dedicated servers.


== Additions ==


=== Blocks ===
Wooden doors

         Added new types of doors for all the different wood types: spruce, birch, jungle, acacia and dark oak doors.
Door recipes now require all six planks to be of the same type.
