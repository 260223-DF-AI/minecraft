1.6.4 Build 3 is the third build for MinecraftEdu 1.6.4, released in October 2013. It is based on Java Edition 1.6.4.


== Additions ==


=== General ===
Rearranged servertool buttons.
Very slight GUI size changes.
You can now include spaces and these special characters into your saved world name: . - _
Made Enable Animals and Enable Villagers to be separate settings. You can now toggle them separately.


== Changes ==


=== General ===
Made all minecraftedu console commands to be lowercase for consistency. All console commands have been added in the Commands article.
Updated translations.


== Fixes ==
3 bugs fixed

Teacher name now always renders as gold.
Users teacher menu no longer popups up when pressing the freeze students hotkey.
Fixed a bug where students were permanently frozen if they joined a server where freeze for all had been once enabled.
