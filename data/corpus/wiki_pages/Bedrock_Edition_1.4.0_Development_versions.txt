This is a list of Bedrock Edition 1.4.0 development versions.


== beta 1.2.14.2 ==
Beta 1.2.14.2 is the first beta version for Bedrock Edition 1.4.0, released on March 28, 2018, which adds more Update Aquatic features and fixes bugs.


=== Additions ===


==== Blocks ====
Blue ice

Generates in icebergs.
Version exclusive: Emits a light level of 4.
Buttons, pressure plates, and trapdoors

Now come in all 6 wood types.
Coral

Come in 5 variants: brain, bubble, fire, horn, and tube.
Can only be placed underwater.
Naturally generate in coral reefs.
Can only be obtained with a Silk Touch tool.
Coral blocks

Come in 5 variants: brain, bubble, fire, horn, and tube.
Naturally generate in coral reefs.
Must be mined with a Silk Touch pickaxe to drop itself, otherwise they drop a dead coral block.
Coral fans

Come in 5 variants: brain, bubble, fire, horn, and tube.
Naturally generate in coral reefs.
Version exclusive: Have a dead variant.
Version exclusive: Coral fans and its dead variants can be placed at the top of an underwater block.
Can only be obtained with a tool enchanted with Silk Touch.
Dead coral blocks

Come in the same 5 variants as coral: brain, bubble, fire, horn, and tube.
Obtained by mining a coral block without a Silk Touch pickaxe.
Naturally generate in coral reefs.
Dead coral fans

Come in the same 5 variants as coral: brain, bubble, fire, horn, and tube.
Naturally generate in coral reefs.
Can only be obtained with a tool enchanted with Silk Touch.
Dried kelp block

Crafted with 9 dried kelp.
Can be used as a fuel in furnaces, smelting 20 items per block.
Kelp

Generates in oceans.
Can only be placed in water.
Can grow multiple blocks high.
Has animated textures.
Can be smelted into dried kelp.
Version exclusive: Kelp in the inventory has a unique texture.
Seagrass

Generates in oceans, rivers, and swamps.
Can additionally be generated when using bone meal on blocks underwater.
Has a tall variant.
Has animated textures.
Version exclusive: Has a darker green color.
Version exclusive: Seagrass in the inventory uses the texture of normal grass.
Water

New water behavior
A water source block can now be placed inside of most non-solid blocks.
Flowing water will still not flow into non-solid blocks.
Blocks can now be placed underwater without creating pockets of air around them.


==== Items ====
Bucket of Fish

Come in 4 variants: cod, salmon, pufferfish, and tropical fish.
Obtained by using a water bucket on a fish mob.
Version exclusive: Bucket of pufferfish has a different texture than in Java Edition.
Dried kelp

Obtained from smelting kelp.
Can be used to craft dried kelp blocks.
Can be eaten, restoring 1 () hunger point.
Spawn eggs

Cod
Pufferfish
Salmon
Tropical Fish
Version exclusive: The spawn eggs look like roe with the respective color.


==== Mobs ====
Drowned

Added a baby drowned variant.
Fish mobs

Cod
Spawn in cold, normal, lukewarm oceans, and their variants.
Pufferfish
Spawn only in warm oceans.
Inflate themselves when a player gets near.
Inflict a nearby player with Poison for 7 seconds.
Salmon
Spawn in cold, normal, frozen, lukewarm oceans, and their variants.
Tropical Fish
Spawn only in warm and deep warm oceans.
Come in 14 different colors and patterns.


==== World generation ====
Biomes

Added cold, lukewarm, warm oceans, and their deep variants.
Buried treasure

A structure that consists of a buried chest with loot in it.
Treasure maps can lead the player to them.
Coral reefs

Naturally generate in warm oceans.
Version exclusive: Coral fans, dead coral blocks, and dead coral fans naturally spawn as parts of them.
Icebergs

Naturally generate on frozen oceans.
Shipwrecks

Can be found in oceans and beaches.
Contain 13 loot chests containing different types of loot depending on the ship.
Can generate upright, sideways or upside down.
Version exclusive: Stripped logs naturally spawn as parts of them.
Underwater ravines

Naturally generate in oceans.


==== General ====
Marketplace

Added an Inventory section to manage all of the player's acquired content.


=== Changes ===


==== Blocks ====
Pumpkins

Can now be sheared into carved pumpkins.


==== Items ====
Fish

Added new textures for raw fish, cooked fish, raw salmon, cooked salmon, tropical fish, and pufferfish.
Leads

Can now be attached to boats.


==== Mobs ====
Drowned

Drowned faces now glow in the dark.


==== Non-mob entities ====
General

Items now float in water.


==== Gameplay ====
General

Tweaked the directional movement of the gameplay cursor for more predictable movement.


=== Fixes ===
Improved load times in the Skin Picker and when applying resource packs.
'Delicious Fish' achievement will now unlock after cooking and eating salmon.
Fishing will now give the proper chance of catching a stack of 10 ink sacs.
When riding a boat and hitting a mob with the Knockback enchantment, the mob will no longer be always knocked toward the east.
Snow covered leaves no longer appear red when Smooth Lighting is disabled.
Commands in chains of command blocks will now run sequentially when other chains are running nearby.
Fixed the 'Disconnected from server' prompt being too large on Gear VR.
End portals and end gateways no longer drop themselves as items if they are replaced with water using a bucket.

== beta 1.2.20.1 ==
Beta 1.2.20.1 is the third beta version for Bedrock Edition 1.4.0, released on April 10, 2018, which adds more Update Aquatic features and fixes bugs. This update also includes the Chemistry Update features from Minecraft Education.
Although labeled as a beta for the respective update, an update labeled 1.2.20 never came, similar to how Java Edition Beta 1.9 was never released. Instead, these features were later postponed to be fully implemented into the release of 1.4.0.


=== Additions ===


==== Blocks ====
Sea pickles

Naturally generate in warm oceans, usually around coral reefs.
Up to 4 can be placed on a block.
Each one adds 3 to the light level, but only when placed underwater.
Can be smelted into lime dye.


==== Items ====
Heart of the Sea

Can be found inside buried treasure chest loot.
Currently has no use.
Version exclusive: Has a unique texture.
Nautilus shells

Currently have no use.
Version exclusive: Has a unique texture.
Tridents

Added a new animation for tridents enchanted with Riptide.
Spawn Egg

Dolphins.


==== Mobs ====
Dolphins

Spawn in any ocean that is not frozen.
Are neutral mobs.


==== World generation ====
Underwater ruins

Generate in normal, cold, and frozen ocean biomes, as well as their deep variants.
Come in different shapes and sizes.


==== Chemistry Update features ====
Compound creator

Used to create compounds from elements.
Colored torch

Have 4 colors: blue, green, purple, and red.
Element constructor

Used to create elements.
Hardened glass and hardened glass pane

Have 16 different colors.
Heat block 

Can melt snow and ice, similar to torches.
Lab table

Used to create new items or blocks by combining elements and compounds.
Material reducer

Used to create chemical materials from blocks.
Underwater torch

Torch that can be placed underwater.
Underwater TNT

TNT that can destroy blocks underwater.
Balloons

Entity which can be tied to fences and mobs, similar to leash knots.
Will make mobs float in the air when used on a mob.
Bleach

Used as a crafting ingredient to turn wool, carpets, banners, and beds white.
Glow sticks

Have 16 colors.
Ice bombs

Projectile that can transform water into ice.
Medicine potions

Antidote
Brewed by adding silver to an awkward potion.
Removes the Poison status effect.
Elixir
Brewed by adding cobalt to an awkward potion.
Removes the Weakness status effect.
Eye drop
Brewed by adding calcium to an awkward potion.
Removes the Blindness status effect.
Tonic
Brewed by adding bismuth to an awkward potion.
Removes the Nausea status effect.
Sparklers

Can be placed in the off-hand.
Have 5 colors: orange, purple, blue, red, and green.
Super fertilizer

Used to grow plants maturely in single uses.

World options

Education Edition toggle
Can be toggled in Cheats
Activates Chemistry Update features when toggled
Added various chemical elements and compounds.

== beta 1.2.20.2 ==
Beta 1.2.20.2 is the fourth and final beta version for Bedrock Edition 1.4.0, released on April 20, 2018, which fixes bugs.


=== Additions ===


==== Mobs ====
Dolphins

Added sounds.


==== General ====
User interface

Added Pocket UI screens for the compound creator, element constructor, lab table, and material reducer.


=== Changes ===


==== Items ====
Heart of the Sea and Nautilus shells

Changed textures to match Java Edition.


==== Mobs ====
Dolphins

Updated model.


=== Fixes ===
Fixed a crash that occurred when creating a world when Minecraft Education toggle was enabled.
Fixed a crash that occurred when deleting a world or resource pack through the Storage screen.
Attempting to import Minecraft Education world files no longer resets worlds.
Reduced the delay when placing or destroying blocks.
Curse of Binding and Vanishing books can no longer be acquired by trading with a librarian villager.
All Chemistry items now appear in Creative inventory if Education Edition toggle is enabled.
Reduced the amount of magma blocks that generate in underwater ravines.
Bucket of fish items now appear in the Creative Inventory.
Dolphins can successfully get back to water now.
Fixed several of the top crashes that occurred during gameplay.
Fixed some toggles and buttons staying highlighted in Settings when they shouldn't.
Fixed the crafting recipes of prismarine bricks and dark prismarine being switched.
Guardians will now attack cod.
Guardians can no longer jump out of the water and suspend themselves in the air while firing their beam.
Water can once again be removed with a bucket if there is a block above it.
The paper doll is no longer running on the main menu.
It is once again possible to fill an empty bucket using touch controls.
Water around chests in ocean ruins no longer appears if the Ruin generates above water.
Charging a bow or trident and switching items no longer causes blocks to not drop when broken.
Ravines once again generate correctly.
Crouching and flying up no longer causes the player to jump in Creative mode.
Fixed an instance of flat worlds not generating any new chunks.
