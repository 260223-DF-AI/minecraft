kz.png is a texture atlas used for storing textures of paintings. Paintings added in Bedrock Edition 1.21.0 and later versions are stored in their own texture files, but this atlas is still used for older paintings and the back texture.
paintings_kristoffer_zetterstrand.png was a texture atlas found inside minecraft.jar used for storing textures of paintings in Java Edition.


== Format ==

kz.png is a fixed-width 256256-pixel file that is split into different 1616 squares which correspond to the texture of each painting. A separate file, kz_mers.tga, stores the MERS values of individual pixels on all paintings, used in Vibrant Visuals.
Like with terrain.png, items.png and particles.png, paintings were originally stored on a main texture atlas rather than being separate individual files. Empty slots were filled in with the magenta grid texture.


== History ==


== Trivia ==
The "Kz" in the name "Kz.png" stands for "Kristoffer Zetterstrand", the artist of the majority of the Minecraft paintings, indicated by the fact that the file was renamed "paintings_kristoffer_zetterstrand.png" in Java Edition 1.6.
