This is a list of Java Edition 1.4.6 development versions.


== 12w49a ==
12w49a is the first snapshot for Java Edition 1.4.6, released on December 7, 2012, which added nether brick slabs, enchanted books and fireworks.


=== Additions ===


==== Blocks ====
 Nether brick slabs

New kind of slab.
Made by using 3 nether bricks blocks in a horizontal line.
Came from a suggestion at Minecon 2012.


==== Items ====
 Firework rocket

New kind of shootable projectible.
Sound effects will come soon.
Fireworks are customizable by crafting.
 Firework star

Used for crafting firework rockets.
 Enchanted books

Used for enchanting items.
In this snapshot due to a bug, any item could be given any enchantment in Survival mode using an anvil.


=== Changes ===


==== Blocks ====
General 

When holding sneak, the player can place blocks and items (like redstone, repeaters, and levers) directly onto blocks such as crafting tables, furnaces, and dispensers.
Cobwebs

Can be harvested utilizing Silk Touch shears.


==== Items ====
Shears

Can now have the Silk Touch enchantment by using the anvil.


==== Mobs ====
Ghasts

The blast radius of fireballs a specific ghast shoots can be customized with an editor.


=== Removals ===


==== General ====
Distance effects

Falling block behavior has changed - they are no longer capable of infinitely producing dropped items at high coordinate values.

== 12w50a ==
12w50a is the second snapshot for Java Edition 1.4.6, released on December 13, 2012, which added new Christmas chest textures and the Thorns enchantment.


=== Additions ===


==== Blocks ====
Chests

Added 3 new textures: xmaschest.png, largexmaschest.png, and skis.png.
xmaschest.png changes the chest texture to resemble a red gift.
largexmaschest.png changes the large chest texture to resemble a green gift.
skis.png is unused, dummy file. Dinnerbone said he added this file as a red herring.
Chests use these textures on December 25.


==== Items ====
Tools

Added tool highlight when switching items in the hotbar.
Renamed items display the changed name.
To disable it, player needs to open options.txt and change heldItemTooltips to false.


==== Mobs ====
Villagers

Enchanted books are now available through villager trading.


==== World generation ====
Dungeons

Enchanted books can be found in dungeons with special enchantments normally not obtainable.


==== Gameplay ====
Enchanting

Added Thorns enchantment for armor.
When player wearing such armor gets hit by a mob, the mob receives damage.


=== Changes ===


==== Items ====
General 

Item entities are rendered as 3D objects in the world when in fancy graphics mode.
This includes items placed in item frames.
Items now show their enchantments when dropped on the ground.
Firework rocket

Tweaked the effect the item gives and sounds.


==== Gameplay ====
Creative

Removed the firework rocket from the Creative inventory.
Enchanting

Efficiency now has no effect when using tools on the wrong block types.
Protection is no longer as effective as all the other Protection-type enchantments combined.
Unbreaking can be applied to anything with durability using enchanted books.


==== General ====
Controls

Pressing CTRL+Q by default (Or the "Drop" button if changed in controls), the player drops a full stack instead of one item.
Notices

Added a notice/warning for users still playing on PowerPCs or Java 5 (Java 1.5).
Mojang plans to drop support for Java 5 by version 1.6. This is due to the most recent versions of LWJGL no longer supporting Java 5, with PowerPC also no longer supported by Apple to be made compatible with Java 6 or newer.

== 12w50b ==
12w50b is the third and final snapshot for Java Edition 1.4.6, released on December 14, 2012, to fix some of the bugs and crashes in 12w50a. This is the final snapshot released in 2012.

== 1.4.6 (pre-release) ==
Pre-release 1.4.6 is the first and only pre-release for Java Edition 1.4.6, released on December 17, 2012.
This version was re-uploaded on the same day to fix an issue with mobs spinning.
The release version of 1.4.6 differs from this reupload by fixing a crash issue.
This is the final pre-release released in 2012.

Bows

A high pitched sound is played when a player hits another player from far away with a bow.
