12w05a is the third snapshot for Java Edition 1.2.1, released on February 2, 2012, which makes minor changes to mobs.
12w05a was reuploaded later on February 2, 2012, to fix a bug where the game would crash when right-clicking a sitting cat empty-handed, as well as fixing racist translation griefings.


== Changes ==


=== Blocks ===
 Dead bushes

Can be obtained by breaking them using shears.


=== Items ===
 Fire charges

Can be crafted using charcoal.


=== Mobs ===
 Creepers

Have new AI: they now stalk the player.
 Ghasts

Updated hitbox to incorporate their whole body, prior to this ghasts could only be injured by hitting their tentacles and lower body.
 Ocelots

Are now easier to tame, and tamed ocelots can now be told to sit.
Ocelots and cats have now become immune to fall damage.
Ocelots and cats now frighten creepers.
Villagers

Try to detect village houses and live in them.
Now have the ability to open and close doors.


=== World generation ===
Mineshafts

Now bridge over ravines.


=== General ===
Enchanting

No longer requires experience in Creative mode.
Languages

Updated language files.
Sounds

Lava sounds now function.


== Fixes ==
2 bugs fixed

Corrected a duplication bug involving Nether portals.
Removed racial slur from the title screen when using the Afrikaans language.
