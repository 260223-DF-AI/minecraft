Alpha v1.2.3 is a version of Java Edition Alpha released with server 0.2.5 on November 24, 2010.


== Additions ==


=== General ===
Multiplayer

Added visible hurting animations for players and mobs in multiplayer.
Added the ability to attack entities (mobs, boats, minecarts, and players) in multiplayer.


== Changes ==


=== Blocks ===
 Leaves

No longer have a darker color.
Now decay again.


=== General ===
Debug screen

The player's current coordinates are now shown.


== Fixes ==
Fixed a crash related to lighting.
Players in boats and minecarts now receive terrain updates.
