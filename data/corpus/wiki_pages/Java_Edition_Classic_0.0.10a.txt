0.0.10a is a version of Java Edition Classic released on May 17, 2009, at 11:53 UTC, to increase stability and add the ability to unhook the mouse. This is the last Classic private alpha version before the public availability of the game with version 0.0.11a.
This version was originally planned to be released to the public, but a bug with the start and destroy functions made the game unplayable. It was quickly replaced by 0.0.11a after only 4 minutes, which was then released publicly.
