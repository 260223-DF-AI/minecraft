Map markers are entities exclusive to the Bedrock Editor.


== Spawning ==
Map markers do not spawn naturally. Found only in Bedrock Editor, spawn eggs exist in the spawn egg group in the Creative inventory. They can also be summoned using the /summon command.


== Behavior ==
Map Marker: Continues rotating counterclockwise. At this time, triggers the particle editor:selected_marker.
Location Pointer Selected: Continues rotating slowly counterclockwise.
Location Pointer Idle: No particular behavior.
Ruler Pointer Selected: No special behavior.
Ruler Idle: No special behavior.


== Usage ==
In Bedrock Editor, it is used as a marker when using tools.


== Official server ==
In the official Capture the Flag test server, map markers can be seen without using Bedrock Editor.
