14w11b is the eighteenth snapshot for Java Edition 1.8, released on March 13, 2014, to fix some of the bugs and crashes in 14w11a.
This snapshot was reuploaded.[more information needed]


== Changes ==


=== Blocks ===
Block models

Generic blocks now load models and not predefined block shapes.


=== Mobs ===
Endermites

No longer despawn when named, though PersistenceRequired is still ignored.
