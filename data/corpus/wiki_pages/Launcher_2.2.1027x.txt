2.2.10271 (Windows), 2.2.10272 (Windows, via the Microsoft Store/Xbox app), 2.2.10274 (macOS), or 2.2.10273 (Linux), is an update to the launcher released on February 7, 2022.


== Changes ==
Fixed some links from the Launcher which were pointing to the wrong places.
Updated textures of the ore icons in Installation screen.


== Fixes ==
Fixed some user launch issues, crashes, and other critical bugs.
MCL-13519 - Launcher fails to launch game and then incorrectly claims that the game is already running.
MCL-17983 - Outdated ore textures in installation icon dropdown.
MCL-21061 - Instead of using saved password users now have to enter two passwords.
