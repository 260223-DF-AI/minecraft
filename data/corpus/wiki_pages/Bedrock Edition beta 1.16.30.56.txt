Beta 1.16.30.56 is the sixth RenderDragon beta version for Bedrock Edition 1.16.200, released on October 20, 2020, which fixes bugs.


== Changes ==


=== General ===
Updated fog settings
New format for PBR materials - Texture Sets
These changes will break existing worlds in the RTX Beta. Updating the RTX world templates from the Marketplace will get the latest changes to continue playing
Added glint effect to held and dropped enchanted items
Added support for Blindness effect
Support for solid colored banners
Added cracking animation when player is above water, but mining blocks underwater
Improved appearance of sand and ice during day-night cycle


== Fixes ==
General

Various performance and stability improvements
Reduced mouse and keyboard latency
Fixed several blue squares appearing on some textures (MCPE-98780)
Fixed a variety of shimmering item issues, including items in hand, in frames, on armor stands, and bobbing on the ground


== Trivia ==
Beta 1.16.30.56 has the same protocol version as beta 1.16.100.59, which means these versions are cross-compatible with each other.
