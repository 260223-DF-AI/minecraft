rd-161348 is a pre-Classic version made on May 16, 2009, at 11:48 UTC, which adds saplings, makes changes and fixes a bug. This is the last archived version of pre-Classic.
This version is available in the launcher; however, the copy in the launcher was last modified on August 6, 2013 and contains the terrain.png from Classic 0.0.13a, as well as signature files from 2013, making it technically not original.
This version is duplicated in the launcher, as attempting to play rd-20090515 instead downloads rd-161348.

=== General ===
Objects in the distance appear a tiny bit darker.


=== Gameplay ===
The game now runs in windowed mode again.
Particles now continually face the player.
Player facing directly up or down sees the full particle rather than a sliver of the side.
Controls

The crosshair (+) is made thicker.
The block preview in the top-right corner of the screen has been flipped.


== Fixes ==
2 bugs fixed

Fixed rendering glitches where the world became invisible at certain angles.
Fixed a performance issue where players experienced extreme lag when close to the ground after booting up the game.


== Trivia ==
The launcher variant of this version uses textures from later versions.
The terrain.png file used in the launcher build of this version is dated May 21, 2009 03:02 UTC+2 (in Sweden) and is bit-for-bit identical to the one used in Classic 0.0.13a.
The "rd" before the version number stands for RubyDung, a game Notch was working on before Minecraft, whose codebase was later reused for Minecraft.
This is the last version to officially use this naming scheme.
This is the first version that has been duplicated in the launcher.
