Beta 1.4 is a version of Java Edition Beta released on March 31, 2011, which added wolves, cookies, locked chests, and a new Minecraft logo as well as other changes and fixes.
Beta 1.4 was reuploaded an hour and a half after release to remove debug code.


== Additions ==


=== Blocks ===
 Locked chest

Randomly generates in the world if the date is set to April 1.
Originally linked to the Minecraft store, but the page now redirects to the official download page for Minecraft.
Can only be broken with explosions.


=== Items ===
 Cookie

New food item, which restores 1HP.
Can be stacked up to 8, making it the first stackable food item.
Crafted with wheat and cocoa beans.


=== Mobs ===
 Wolves

Neutral mobs.
Common in taiga biome.
Players can tame them using bones.
Once tamed, a wolf does not accept any more bones.
The number of bones required is random: each bone has a 13 chance of taming the wolf.
Tamed wolves defend their owners against attackers and can assist in combat.
When wolves are hit by the player or the player's tamed wolves, they become angry.
A tamed wolf's tail rises and lowers depending on its health.
Raw porkchops can be given to wolves.
When killed, they drop nothing.
Has 10HP.


=== General ===
Missing content

 Added the missing texture.
Used if a texture is absent from the game files.
Only visible through modding (including removing an image file from the jar).
Previously the game would just outright crash if a texture was absent.


== Changes ==


=== Blocks ===
Bed

Sleeping in a bed now resets spawn position, though does not work for most users.
At the moment the spawn point unintentionally only changes if "Leave bed" is clicked and the function is limited to multiplayer.
Ladders

Sneaking while climbing hangs on to the ladder.
Wheat crops

Now grow more quickly.


=== Mobs ===
Sheep

Can now spawn with pink or brown wool (very rarely, however).
Spiders

No longer trample crops.
Have their walking sounds removed.


=== World generation ===
Monster room

Added cocoa beans to chests.


=== Gameplay ===
Achievements

Lots of infrastructure has been implemented.
Among these are opening the inventory, mining wood, and building a crafting table.
Controls

The new default hotkey for the inventory is E.


=== General ===
Multiplayer

Servers can now have a new world seeded with an option in the server.properties file.
Options

GUI scale may now be changed in options.txt located in the .minecraft folder.
Statistics

Lots of infrastructure for statistics lists have been implemented.
Title screen

Changed Minecraft logo from  to .


== Fixes ==
4 bugs fixed

Zombies and spiders now hurt the player in all instances.
Before this version, zombies and spiders only hurt the player if they were 1 block above their location.[more information needed]
Client-sided slime spawn on multiplayer has been fixed.
Arm no longer swings when player cannot place a block.
Opening and closing the inventory quickly while swinging no longer removes the character 'swinging' animation, which previously made destroying blocks faster (Multiplayer only).


== Reupload ==
Beta 1.4 was reuploaded around an hour and a half after its initial release. The later version removed a small unused section of code added in the initial version which tracks the player's walking distance in memory, presumably testing code accidentally left in the game for Beta 1.5, which added statistics  namely, a statistic that tracks how many meters the player has walked.
