Build 0033 (or Build 1.66.0033.0) is a development version of Xbox 360 Edition. This version is based off Java Edition Beta 1.6.6.
The only archived copy of this build (labelled "Minecraft Reloaded") was tampered with, containing an edited terrain.png using textures from the ModernCraft texture pack. Despite what is often claimed, these textures were added to the game by a third party, and were not used officially by 4J Studios. The mipmapped versions of terrain.png were not edited, so distant blocks still use the vanilla textures. The modified Xbox dashboard description states "Minecraft with custom textures and stuff. v0.1a".

This graphic still remains in the game in TU0.


== Trivia ==
This build comes from Xbox PartnerNet - an internal Xbox LIVE service used by developers to test games before release.​[more information needed]
Partnernet has been the source of many XBLA leaks, especially around this time.
