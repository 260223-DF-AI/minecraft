0.29_02 is a version of Java Edition Classic released on October 30, 2009. This version is not available in the launcher, but has been archived elsewhere.


== Fixes ==
1 bug fixed

Fixed bug that caused most people in servers to disconnect.


== Trivia ==
0.29_02 can be changed to Survival mode by editing the Minecraft.class file in minecraft.jar.
0.29_02 came out exactly one year before the Halloween Update.
