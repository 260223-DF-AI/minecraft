0.4 was the first major MinecraftEdu Launcher version, released on July 23, 2011.[verify]


== Additions ==


=== General ===
Ability to launch a MinecraftEdu server.[verify]


=== Options ===
In the 'Select assignment' tab, the teacher can give their class an assignment and a map to complete it on, and start the MinecraftEdu Server using that map.
In the 'Select saved map' tab, the teacher can select a saved map, give it a description and change several gamerules, and start the server using the saved map.
The cog button allows the teacher to configure the server's render distance and port.
