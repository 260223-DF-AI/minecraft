18w48a is the ninth snapshot for Java Edition 1.14, released on November 29, 2018, which overhauls plains villages.


== Changes ==


=== Blocks ===
Barrels, bells, blast furnaces, brewing stands, cartography tables, fletching tables, lecterns, looms, smokers, smithing tables, and stonecutters

Can now generate naturally inside plains villages.
Beds

Now have an accurate hitbox.[verify for Java Edition]
Grindstones

Changed texture.
Now craftable using two sticks, two planks, and one stone slab.
Now have a GUI with 2 input slots and 1 output slot.
Can now combine two damaged items into one and/or remove non-curse enchantments.
For each enchantment removed, some experience is returned.
Can be found at plains village weaponsmiths.


=== Mobs ===
Villagers

Slightly modified trades.


=== Command format ===
/locate

Can locate the new villages, using the New_Village parameter.


=== Generated structures ===
Plains villages

Village houses now have a drastically different structure and there are many more types.
Use jigsaw blocks for generation.
Added new structure and loot table files for the new village houses.


=== General ===
Crafting

Recipes for combining damaged items have been removed; these are now done using the grindstone.
Dedicated servers

server.properties
difficulty and gamemode settings now accept string names; integer values are still allowed as legacy option.
Server now exits faster if eula.txt is not set.
Command line:
Added new option --help.
Added new command line option --initSettings.
Initializes eula.txt and server.properties with defaults, then exits.
