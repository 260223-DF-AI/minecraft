2.2.5519 (Windows), 2.2.5518 (macOS), or 2.2.5516 (Linux), is an update to the launcher released on September 21, 2021.


== Changes ==
Changed the process of launcher installation, in order to address user installation issues.
For an impacted user, after loading the Launcher, the user sees a short series of popups walking user through an installation.
Various improvements that improve security.
Changed how News checkboxes work.
State is saved if tab away.
If all checkboxes are unselected, shows all News items.


== Fixes ==
MCL-17721 - Dropdown windows in Launcher do not respond.
MCL-18924 - I have to log in every time the launcher is opened.
