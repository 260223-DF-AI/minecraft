2.2.911 (Windows), 2.2.910 (macOS), or 2.2.909 (Linux) is an update to the launcher released on December 8, 2020.


== Additions ==
Added Account migration FAQ
Added Open game directory button to Minecraft: Java Edition installations
Added disclaimer toast about account migration login screen


== Changes ==
Default "Modded" checkbox to checked
Various accessibility fixes for dialogs throughout the launcher


== Fixes ==
Fixed issue with adding new users to the launcher
Fixed issue with opening directories on macOS
MCL-15921  Cannot scroll in Installation Editing
