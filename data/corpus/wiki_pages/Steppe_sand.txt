Steppe sand was a block in MinecraftEdu. It was a purely decorative soil block, with no functionality.[verify]


== Obtaining ==
Steppe sand could be obtained through the teacher menu or by using command blocks. It was not available in the Creative inventory.


== Usage ==
It can be used as a building material.


== Data values ==
The last ID of this block was 3728.
