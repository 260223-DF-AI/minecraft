12w30c is the seventeenth snapshot for Java Edition 1.3.1, released on July 24, 2012 to fix some of the bugs and crashes in the 12w30a and 12w30b snapshots.


== Changes ==
Removed 2×2 crafting grid from the Survival inventory tab in Creative mode.


== Fixes ==
5 bugs fixed

Fixed piston block animation bug.
Fixed rain particles in certain biomes.
Fixed monster spawners set to spawn FallingSand entity crashing the game.
Fixed minor text bugs.
Fixed portal placement.


== Trivia ==
This is the first ever "c" snapshot.
