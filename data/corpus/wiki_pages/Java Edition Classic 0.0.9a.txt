0.0.9a is a version of Java Edition Classic released on May 16, 2009, at 19:15 UTC, which adds a new building mode and a counter for displaying frame rate and chunk updates, changes the resolution and fixes bugs.
Notch did not release versions 0.0.4a through 0.0.8a, if they even existed to begin with.


== Additions ==


=== Gameplay ===
Controls

Right-click switches between placing and destroying modes.
Selecting a block now shows a white overlay on all sides of the block.


== Changes ==


=== General ===
Changed the game's resolution from 1024×768 to 640×480.[is this the correct version?]


== Fixes ==
2 bugs fixed

Fixed a graphical glitch with preview tiles in the newly added build mode.
Players can no longer build inside of themselves.


== Bugs ==
1 bug reported

The white screen issue from 0.0.3a still affects the game for some testers.
