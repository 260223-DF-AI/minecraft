Many of the game's textures are compiled into one of several atlases upon resource loading. These are considered by the game to be dynamic textures.


== Summary ==
As of Java Edition 1.21.11, there are 15 atlases which the game generates when loading resources.
All atlases can be saved as an image file via F3 + S to a directory titled "debug" in .minecraft's screenshots directory.
