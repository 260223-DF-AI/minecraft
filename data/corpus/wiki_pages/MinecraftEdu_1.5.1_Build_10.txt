=== General ===
Added new editing capabilities for Information blocks. This includes line-by-line editing, ability to change caret position, ability to open links by clicking them.
If visual fill is enabled in MinecrafftEdu options, selected cuboid worldedit areas are now highlighted in the client.
Added file allowedstudentocommands.ini to server settings folder which can be used to define what commands are students allowed to use.
Added our own custom splash logo for the client.
Added few splash texts into main menu.
Added ability to add external translations for the software directly into minecraftedu folder instead of having to modify translation files inside the jars.
