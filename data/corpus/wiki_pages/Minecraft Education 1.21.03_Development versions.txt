This is a list of Minecraft Education 1.21.03 development versions.


== Preview 1.20.60.0 ==
1.20.60.0 is the first Preview version for Minecraft Education 1.21.03, released on February 6, 2024.


=== Additions ===


==== General ====

Cloud World Backup - Upload

Worlds are automatically uploaded to OneDrive upon world close as a backup.
Future Preview updates will add the ability to download and open worlds from OneDrive.
IT Admins will need to ensure that the following URLs are allowlisted through the network:
https://graph.microsoft.com
https://*.sharepoint.com
"Recipe unlocking" and recipe book search improvements when playing in Survival mode.
Logo

 Added logo for Preview versions.


=== Removed features ===
Darwin support was not provided in this version. This does not confirm that Minecraft Education as a whole would remove support for macOS, iOS, and iPadOS.
The desktop version of the game for Windows was not provided in this version. This version can only be obtained through Microsoft Store.
The sign-in procedure switched from Microsoft Authentication Library (MSAL) to a web authentication broker based authentication (Windows.Security.Authentication.Web).


=== Preview ===
The Minecraft Education Preview app can be installed at the same time as the main Minecraft Education app.
Current support: Windows (Microsoft Store), ChromeOS, and Android.


=== Fixes ===
Improvements to performance and product stability.


=== Known issues ===
Cloud is upload only. Download and open will come in a future Preview update.
Spectator mode has been accidentally included as an available game mode.
Some functions while coding in Python Notebooks will result in a Syntax Error: Unexpected 0 error message.
The How to play Guide has references to some features, like Realms, that only exist in Bedrock Edition. Those features are not included in Minecraft Education and the guides will be updated soon for the features in Minecraft Education.
Due to the new changes to how worlds are loaded, Some users may experience immediate crashes moments after loading into a world if IPv4 support isn't detected. This mandates that the user must have IPv4 enabled on their host operating system.
On Windows 11 23H2, Since Web Authentication Broker no longer relies on Microsoft's Windows.UI.Xaml library, Microsoft Edge is required to be installed for the login procedure to function correctly. Otherwise, the user might be met with a 1001 error.


== Preview 1.20.70.0 ==
1.20.70.0 is the second Preview version for Minecraft Education 1.21.03, released on April 4, 2024.


=== Additions ===


==== General ====

More features from the Trails & Tales update.
The 'How to Play' screen has been completely revamped into the 'Encyclopedia' screen.
"Recipe Unlocking" and recipe book search improvements when playing in Survival mode.
Cloud World Backup - Upload & Download

Worlds which are automatically uploaded to OneDrive upon world close as a backup can now be downloaded from OneDrive from the My Worlds screen.
IT Admins will need to ensure that the following URLs are allowlisted through the network:
https://graph.microsoft.com
https://*.sharepoint.com


=== Fixes ===
Improvements to performance and product stability.


=== Known issues ===
