0.1.0 is the first release for the New Nintendo 3DS Edition which came out on September 13–14, 2017.


== Features ==
All features from and prior to Pocket Edition v0.15.4 alpha, with a few exceptions.

Map items are replaced with a permanent map on the touch screen.
There is a Nether panorama on the bottom screen.


== Trivia ==
0.1.0 is the only New Nintendo 3DS Edition version to be prefixed with the number 0.
