This is a list of Java Edition 1.12.1 development versions.


== 17w31a ==
17w31a is the first and only snapshot for Java Edition 1.12.1, released on August 1, 2017.


=== Changes ===


==== General ====
Game credits

Updated game credits to include more Mojang staff.
Credits scroll 50% faster when viewed directly from the title screen than through the End Poem.
Splashes

Removed "Better than Prey!"
Skeleton

No longer picks up items based on their enchantments.


=== Fixes ===
1 issue fixed
Private issues

MC-119011 – Security issue with recipe book.


== 1.12.1-pre1 ==
1.12.1-pre1 is the first and only pre-release for Java Edition 1.12.1, released on August 2, 2017, which includes minor optimizations and fixes bugs.


=== Fixes ===
3 issues fixed
From released versions before Java Edition 1.12.1

MC-86016 – Some Mojang staff not mentioned in credits.txt.
MC-117087 – Class.getSimpleName performance issue with large number of block entities.
From the Java Edition 1.12.1 development versions

MC-119840 – Diamonds get moved into crafting slots on any chat message.
Other

Minor optimizations.
