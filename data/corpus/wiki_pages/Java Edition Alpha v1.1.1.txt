Alpha v1.1.1, also known as Seecret Saturday, is the tenth and the final seecret update, which added sneaking, new paintings and fishing rods. It was released with server 0.2.1 on September 18, 2010, and is the only Seecret Saturday update; all other nine seecret updates were released on Fridays.
This version included debug code relating to the gamma slider, which would set the display's gamma and contrast values to zero, sometimes resulting in the screen going completely gray. This code was removed in Alpha v1.1.2, which was released a few hours after v1.1.1.

== Trivia ==
Despite this version being available for approximately 3 hours and 25 minutes, considerably longer than many other archived versions such as the pre-reupload of Alpha 1.2.3_01 (which was available for 2 minutes), there were no known archives of it until June 25, 2021 (10 years and 9 months after the update's initial release) when X (formerly Twitter) user LunaSorcery provided an archived jar file.
There was a bug with crouching that caused the outer layer of custom skins render as if the player wasn't crouching.
This version is identical to Alpha v1.1.2, with the exception of the gamma slider debug code, which caused the infamous "gray screen bug".
Contrary to popular belief, calm4.ogg does not exist in this version's files.
