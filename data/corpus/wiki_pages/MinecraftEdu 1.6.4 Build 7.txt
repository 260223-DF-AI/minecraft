1.6.4 Build 7 is the seventh build for MinecraftEdu 1.6.4, released on December 22, 2013. It is based on Java Edition 1.6.4. This is the last build released in 2013.


== Additions ==


=== General ===
Mod Manager in Launcher, allows you to customize your MinecraftEdu experience.
Proxy Settings panel in Launcher.
Better support for world templates (fetches all new world template information).
Ability to upload your own world templates from the web.
Crosshairs for build mode states.
Welcome message in server interface that gets saved with the current world.
Setting to hide mods button in launcher: disable-launcher-mods-button=true to launchersettings.ini


== Fixes ==
7 bugs fixed

Fixed free memory % amount in server gui.
Updated a few settings to use the new Villagers setting rather than Animals. This fixed the villagers disappearing problem.
Fixed hidden option "hidestartmenu" hiding quit button instead of select name and gender.
Teleport function dismounts entities before teleporting.
Students can no longer get edu blocks from creative inventory search.
If player rides something and crosses a border block player will be dismounted.
Students can now use anything on top of build allow blocks. (including mod items)
