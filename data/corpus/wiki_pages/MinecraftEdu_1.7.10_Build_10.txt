1.7.10 Build 10 is the tenth build for MinecraftEdu 1.7.10, released on October 23, 2014. It is based on Java Edition 1.7.10.


== Additions ==


=== General ===
Delete button to World Library in Server Launcher


== Fixes ==
7 bugs fixed

World Library in Server Launcher not working in build 9 because of the language reformat.
Some mods like Custom NPCs, HQM and Galacticraft not working properly.
Students seeing teachers who are in spectate mode.
Some animals like squids "flickering" if server was started with animals disabled.
Ban command not working.
1.7.10 stable builds not bolded in Launcher's update list.
Single player crashing the client.
