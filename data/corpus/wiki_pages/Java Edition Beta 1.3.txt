Beta 1.3 is a version of Java Edition Beta released on February 22, 2011, which added redstone repeaters, beds, three types of slabs, and a new splash screen, notably among other changes and fixes, the world save format was changed from Alpha level format to McRegion.
Beta 1.3 was uploaded at 17:13 UTC, this version contained debug code that spawns the player with 64 torches, a diamond pickaxe, a diamond shovel, a diamond axe, and 64 beds upon respawning or creating a world. 
Beta 1.3 was reuploaded at 17:33 UTC to remove the debug code that the player spawns with items, listed above in a new world. Beta 1.3 was then reuploaded again at 17:50 UTC to fix a crash bug when opening the world menu without a saves folder. This last version is listed in the launcher as Beta 1.3b (old_beta b1.3b).


== Additions ==


=== Blocks ===
 Beds

Crafted from planks and wool.
Different colors of wool can be interchanged to make a bed.

 Redstone repeater

Allows for redstone to be powered further.
Crafted from stone, redstone torch and redstone dust.

 Stone slab

Takes over the old recipe for the stone slab.
Breaking a block made of two slabs now drops 2 slabs.
Crafted from cobblestone.

 Wooden slab

Breaking a block made of two slabs now drops 2 slabs.
Crafted from planks.

 Sandstone slab

Breaking a block made of two slabs now drops 2 slabs.
Crafted from sandstone.

 Seamless stone slab

Breaking a block made of two slabs now drops 2 slabs.
Unobtainable.
 Crying obsidian

Added a texture for crying obsidian to terrain.png.
Texture went unused and removed in Beta 1.5, but re-added in 20w10a via the Programmer Art resource pack.


=== Non-mob entities ===
New painting

"Skull On Fire" 
Code to make this painting display was added in Beta 1.2_01, but the texture itself was not added until this version, resulting in the use of the painting placeholder texture.


=== Command format ===
/whitelist

Can turn on/off the whitelist.
Can add/remove players from the whitelist.
Can give a list of whitelisted players and reload that list.
/time

Changes or queries the world's game time
There types of time involved in this command:
time add <time>
Adds <time> to the in-game daytime.
time set (day|night|noon|midnight)
time set <time>
Sets the daylight cycle time.


=== General ===

Lighting

Smooth lighting engine added with the help of MrMessiah.
Options

New graphics setting screen.
Added "Smooth Lighting" option.
Splash screen

New Mojang logo.


== Changes ==


=== Blocks ===
Redstone dust

Gets darker the farther away it is from a source of power.
Unpowered redstone is purely black now.
For these changes, they now use a dedicated tint system and new textures:
          
          
Sandstone

Now found at the bottom of naturally generated sand.
Mine time for sandstone has been decreased.
Stone slabs

Are now created by using 3 stone instead of cobblestone.

Breaking a block made of two slabs now drops 2 slabs.
Stone pressure plate

Now require two of its materials, instead of 3.

Wooden pressure plate

Now require two of its materials, instead of 3.


=== Items ===
Gunpowder

Renamed from "Sulphur".


=== Mobs ===
General

Fewer passive mobs spawn when on harder difficulties.
 Cows

Texture changed from .
Slimes

Now spawn again.
Squid

Can no longer be milked.
Squid now have the ability to move in three dimensions (rather than only along the bottom of the body of water)


=== World generation ===
Create New World

Players can now choose the name of the world before creation.
Players can now choose the world seed before creation.
More than 5 worlds can now be created at once.
Delete world message now includes "(A long time!)" after "(world name) will be lost forever!".
Prior to this, it just says "'<world name>' will be lost forever!".


=== Gameplay ===
Third person mode

View bobbing now applies while in third person, if enabled.


=== General ===
Controls

F1 and F3 are now toggled when pressed.
When F3 is toggled on, mobs display their unique entity ID numbers over their heads.
F8 toggles mouse smoothing.
Options

Scrollbars added in the world selection screen.
Players can now change the names of existing worlds.
This doesn't change the world's save id.
McRegion

Worlds now use a generic McRegion chunk save format, with a built-in auto conversion.
Texture packs

Scrollbars added in the texture pack list.


== Fixes ==
10 bugs fixed

When breaking double slabs (one slab placed on another) it now drops two slabs instead of one.
Wood doesn't burn forever anymore.
Grassland/sand related biome border fixed (boundaries line up, rather than obviously show chunk borders).
No longer receiving Java Log error .txt file spam in the minecraft.exe folder when closing the Minecraft client program.
Players can no longer use the click-release to quickly mine blocks (referred to as "click mining") in multiplayer.
Players no longer glitch when standing on fences in multiplayer.
Players can no longer milk squids.
When right-clicking on a wooden door while holding a bow or fishing rod, arrows no longer are fired into the door, nor is the fishing hook cast into the door.
Hostile mobs can now be attacked properly in multiplayer.
Lapis lazuli ore now drops 4-8 lapis lazuli in multiplayer.
