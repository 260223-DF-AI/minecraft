1.18 Release Candidate 4 (known as 1.18-rc4 in the launcher) is the fourth and final release candidate for Java Edition 1.18, released on November 29, 2021, which fixes an issue with coal ore distribution.


== Fixes ==
1 issue fixed
Other

Fixed an issue where coal ore at lower heights (Y=0 to Y=192) is not distributed as intended since 1.18-pre1.


== Trivia ==
This is the only fourth release candidate for a version.
