1.18.2 Release Candidate 1 (known as 1.18.2-rc1 in the launcher) is the first and only release candidate for Java Edition 1.18.2, released on February 25, 2022, which fixes bugs. This is the first release candidate released in 2022.


== Fixes ==
3 issues fixed
From released versions before 1.18

MC-188086 – Decorations and amethyst geodes get cut off on full chunk boundaries upon relog.
From 1.18

MC-244682 – Some custom dimensions settings can cause the server to stop running, but not crash.
From the 1.18.2 development versions

MC-248764 – Some worldgen datapacks can kill the internal server (possibly related to density functions).
