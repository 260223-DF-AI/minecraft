1.18 Pre-release 8 (known as 1.18-pre8 in the launcher) is the eighth and final pre-release for Java Edition 1.18, released on November 24, 2021, which fixes crashes and bugs.


== Changes ==


=== Mobs ===
Ravagers

No longer attack baby villagers.


== Fixes ==
7 issues fixed
From released versions before 1.18

MC-227537  Crash: java.lang.NullPointerException: Cannot invoke "it.unimi.dsi.fastutil.objects.ObjectSet.remove(Object)" because "$$4" is null.
MC-230866  Eating whilst traveling through a Nether portal prints error in game log.
From the 1.18 development versions

MC-236783  Parity issue: Ravagers still attack baby villagers in Java Edition.
MC-241991  Game freezes without crash logs when upgrading from 1.12.2 to 1.18 pre-5.
MC-242375  Cats no longer scare phantoms away.
From the previous development version

MC-242646  Falling in the void and trying to respawn crashes the game.
MC-242647  Crash for blocks finding block states: Missing Palette entry for index 6.
