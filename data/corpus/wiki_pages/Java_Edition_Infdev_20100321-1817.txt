A version of Minecraft Infdev was released on March 21, 2010.


== Additions ==
 General 

Duplicate login detection added.


== Fixes ==
1 bug fixed

The game no longer kicks the player to the title screen after generating a world offline.


== Bugs ==
1 bug reported

When playing the game, a screen sometimes randomly pops up saying "Session conflict! Did you log in from another computer?". It might occur when pausing the game for too long.
