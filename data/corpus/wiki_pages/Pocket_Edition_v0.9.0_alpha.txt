v0.9.0 alpha is a major update to Minecraft: Pocket Edition that was released on July 10, 2014. It adds new blocks, mobs, generated structures, infinite worlds as well as other features, changes, and fixes.

=== Mobs ===
All mobs

New spawning algorithm
Lighter color when the mobs are hurt
Passive mobs 

Now spawn in little groups
Can spawn naturally in creative mode
Zombies

Mechanics changed
When damaged, zombies have a low chance to spawn more zombies.
Spiders

Now spawn from leaves
Animation is twice as fast.
Creepers

Now walk faster.
No longer float two pixels off the ground.
Other

The options.txt file now reads the render distance as raw blocks.
Support for immersive mode (hides the on-screen navigation bar) for devices running Android 4.4 or higher.
New LevelDB based saving format.
New entity handling.
Remade network code (mostly the same packages but new system).
Tile entities are saved differently.
New culling algorithm.
Removed the camera entity
Updated light generation
No more client-side messages when a player joins
"Edit" and "New" buttons no longer appear on the world select menu if "Play with touch" is disabled, making it impossible to create worlds without enabling the option.
The Xperia Play touchpad can no longer be used to control the camera and crouch. Instead, it is used to move.
In flight mode, the Xperia Play D-pad up and down buttons are always used to control the altitude, rather than only while the cross button is pressed.
This requires to use the touchpad to move forwards and backwards during flight.
Pause menu options on the Xperia Play can be selected with the D-pad and interacted with by pressing the cross button, though the selected option is not highlighted.


== Fixes ==
6 bugs fixed
From released versions before 0.9.0

Mipmapping makes fire look like a big orange blob
Minecart in multiplayer problems
Glitched name and description for baked potatoes
Crafting signs only giving one unit
Different wood types convert to same inside furnaces
Fixed an item duplication glitch that could occur if the game crashed or if the player exited out of Minecraft without saving the world.
