14w33a is the forty-sixth snapshot for Java Edition 1.8, which was released on August 13, 2014.


== Additions ==


=== Command format ===
NBT format

SelectedItem
Contains the item data of the item currently held in the player's hand.
Added to compact the process of testing if a player is holding a specific item.
Previous method required nine commands in command blocks; now requires only one command.


== Changes ==


=== Blocks ===
Wooden doors

Original wooden door was renamed to "Oak Door".
 The models of jungle and acacia doors no longer have inner faces in the holes in their models.
Stone slabs

No longer craftable from granite, diorite or andesite, or their polished variants.


=== Items ===
Wooden doors

    The item textures of the new doors have been changed to match the dimensions of oak wood doors.


=== Mobs ===
Iron golems

Now attack creepers.
Now gets attacked by skeletons, zombies, spiders, cave spiders, slimes and magma cubes.
Snow golems

Now attack creepers.


=== Command format ===
General

Tab-completing coordinate arguments now fill in the coordinates of the block currently being looked at.


=== General ===
Splashes

"This is good for realms." changed to "This is good for Realms."
