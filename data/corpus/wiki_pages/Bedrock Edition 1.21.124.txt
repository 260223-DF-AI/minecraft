1.21.124 is a hotfix to Bedrock Edition released on November 21, 2025, which fixes some bugs reported in previous releases.


== Fixes ==
5 issues fixed
MCPE-230711 – Unable to connect to Realms or Multiplayer worlds - Error code Boat
MCPE-230785 – CJK Input Methods aren't blocked at Game Launch triggering wrong input and allowing to block Player movement
MCPE-230832 – Mouse cursor moves in the background during gameplay, disrupting gameplay
Other

Fixed various crashes.
Fixed texture memory leak that occurred while users were on a server with multiple users joining and disconnecting.
