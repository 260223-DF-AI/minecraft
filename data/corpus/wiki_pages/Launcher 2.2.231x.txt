2.2.2311 (Windows), 2.2.2310 (macOS), or 2.2.2313 (Linux), is an update to the launcher released on March 10, 2021.


== Additions ==
Added the option to disable hardware acceleration for the Launcher. This should fix some issues related to hybrid graphics and offloading.


== Fixes ==
Fixed an issue where the installation button for dungeons could become unresponsive.
MCL-17220 - Launcher hard crashes when using a custom Java installation with modded instances.
MCL-17125 - Launcher window blank/black when using dedicated GPU.
MCL-17155 - Launcher doesn't respect HOME variable.
