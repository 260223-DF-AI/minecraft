The JavaScript version of Minecraft 4k was released on December 2, 2012, three years after the last version of 4k (4k-2). Unlike other versions, its code is openly accessible and easily editable.
It is the last known version of Minecraft 4k to be released.
The code was modified 8 minutes after release to fix an initialization bug.


== Additions ==


=== Blocks ===
 Water[unofficial name]

Replaces the ID (9) used by a placeholder variant of Dirt.


== Changes ==


=== World generation ===
World

The world is now generated in a tunnel shape which the camera endlessly moves through.


=== Gameplay ===
Camera

Now rotates automatically in a continuous circular motion.


== Removals ==


=== Gameplay ===
Controls

The player can no longer place blocks, break blocks, or move.
