2.1.5819 (Linux), 2.1.5816 (macOS), or 2.1.5815 (Windows) is a launcher beta version released on July 16, 2019, which fixed bugs.


== Changes ==
More information about how the launcher starts the game is now logged.


== Fixes ==
MCL-8702 - Launcher black screen - ERROR:gl_surface_egl.cc(711)] eglCreateWindowSurface failed with error EGL_BAD_ALLOC
MCL-10036 - minecraft launcher does not appear on main display if closed on secondary display which is then disconnected before launch
Fixed a bug that meant the log files were missing information on macOS.
Fixed an error where the system language was not properly used by the launcher.
Fixed a bug causing the Launcher Log window to use up performance when open.
Fixed a bug where the --workDir argument did not work on certain systems.
