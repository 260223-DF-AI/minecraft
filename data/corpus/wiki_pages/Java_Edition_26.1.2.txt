26.1.2 is a hotfix for Java Edition released on April 9, 2026, which tweaks UI checkboxes and fixes an exploit. It is compatible with 26.1 and 26.1.1 servers.


== Changes ==


=== General ===
UI

Now checkbox message tooltip is displayed only when text overflows past 2 rows.
Confirmation checkbox in Report Player screen now displays tooltip with same text as message.


== Fixes ==
Fixed players in Spectator mode being able to attack other players.
