1.8.2-pre1 is the first pre-release for Java Edition 1.8.2, released on December 18, 2014. This is the final pre-release released in 2014.


== Additions ==


=== General ===
Debug screen 

New frame time graph.
Activated by pressing Alt+F3 to display frame time bar chart in lower left of screen.

== Changes ==


=== Blocks ===
TNT

Reverted explosion physics changes to those before the 14w31a snapshot.
Explosion force is no longer directionally biased.


=== Gameplay ===
Spectator mode

Mobs no longer follow/attack the player in Spectator mode.

== Trivia ==
This version was previously removed from the launcher for some time, but was added back in September 2018.
1.8.2-pre1 was released on the second latest calendar date out of all pre-releases, on December 18, only beaten by the 1.4.7 pre-release on December 28.
