The early private singleplayer alpha of Java Edition Classic refers to the earliest released versions of Minecraft. These versions were released under the name "Minecraft alpha" for private testing for select users in IRC. These versions were not meant for public consumption. This phase succeeds pre-Classic and precedes Early Classic. Creative singleplayer is the only mode in the game, and there is a very limited number of blocks available, with no inventory or hotbar.


== Additions ==
Font

Custom Minecraft font.
Mouse

Y-axis swapping.
Caves

Added randomly generated caves.
GUI

In-game version number at the top left of the screen. Includes FPS (frames per second) and chunk updates.
Building swapping

Right-click switches between placing and destroying modes. Selecting a block now shows a white overlay on all sides of the block.
