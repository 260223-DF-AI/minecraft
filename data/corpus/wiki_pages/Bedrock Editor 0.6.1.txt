Bedrock Editor v0.6.1 is a minor alpha release for the Bedrock Editor released on April 18, 2024, which adds a SimpleToolWrapper framework and fixes bugs.


== Additions ==
Implemented a SimpleToolWrapper framework to simplify the boilerplate implementation of setting up an editor tool, and manage the window visibility and lifecycle of the tool components.


== Changes ==
Editor Export API can now export a playable world.
Updated Editor Extension Starter Kit to include samples for custom brushes and SimpleToolWrapper.


== Fixes ==
Fixed the UI layout drawer content to respond properly to changes in the visibility of the PropertyPane API.
Fixed a bug for when a player switched to Test World while in the Nether or the End, the world's dimension incorrectly changed to the Overworld.
Fixed a bug that caused crashes during deletion of Script Widget Groups.
