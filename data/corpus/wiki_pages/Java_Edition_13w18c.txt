13w18c is the sixth snapshot for Java Edition 1.6.1, released on May 2, 2013 to fix some of the bugs and crashes in the 13w18a and 13w18b snapshots.


== Changes ==


=== Blocks ===
 Block of Coal

Smelting time was changed from 720 to 800 seconds.
Smelts 80 items, the same number of items that 10 coal can smelt.


== Fixes ==
7 issues fixed
From the 1.6.1 development versions

MC-14632  Enchanted books don't show what tab it is in.
MC-15035  Skins not loading for players.
MC-15041  Breaking Item Frames causes a null pointer exception and crash.
MC-15058  Internal exception: java.lang.NullPointerException.
From the previous development version

MC-14979  Wrong particle texture when changing texture packs in-game.
MC-15095  Coal blocks and note blocks do not show up in creative tabs.
MC-15096  Game crashes on startup.


== Trivia ==
As with 13w18a and 13w18b, this snapshot was released on the same day as 1.5.2.
