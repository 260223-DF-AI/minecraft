1.21.9 Pre-Release 3 (known as 1.21.9-pre3 in the launcher) is the third pre-release for Java Edition 1.21.9, released on September 22, 2025, which fixes two bugs related to copper golems.


== Fixes ==
2 issues fixed
From the 1.21.9 development versions

MC-301963 – Copper golems cause huge lag spikes when checking chests.
MC-301990 – Copper golem statues cause culling issues when placed above certain blocks.
