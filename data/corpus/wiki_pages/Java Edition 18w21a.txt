18w21a is the thirty-eighth snapshot for Java Edition 1.13 released on May 23, 2018.


== Additions ==
NBT format

Added the TreasurePosX, TreasurePosY, TreasurePosZ, GotFish, and CanFindTreasure tags for dolphins.
Settings

New option in options.txt: mouseWheelSensitivity
Defaults to 1.0
Affects sensitivity of mouse wheel in-game


== Changes ==
Mobs

Dolphins now lead the player to treasures: when they are fed raw cod, they swim to the nearest shipwreck or underwater ruins if they can find one within their search radius.
Mob yaw (left-right) head movement is now smooth.
NBT format

The ench tag is now called Enchantments, and no longer has number IDs in each compound.
General

Added (and fixed) item rarity colored tooltips for certain items:
