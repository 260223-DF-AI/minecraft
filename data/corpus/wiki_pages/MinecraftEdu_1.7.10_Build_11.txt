1.7.10 Build 11 is the eleventh build for MinecraftEdu 1.7.10, released on November 7, 2014. It is based on Java Edition 1.7.10.


== Additions ==


=== General ===
MCEdu version info to debug logs.
If the server is started for the first time with "nogui" parameter, teacher password will be asked in console/terminal.


== Changes ==


=== General ===
Lowered default video settings on the client.
Removed "Open to Lan" button from MinecraftEdu single player.


== Fixes ==
6 bugs fixed

Info Block pages acting funny.
Server Give not finding edu blocks.
Not being able to give certain items like boats and potions.
Give crashing the client if mods were installed.
Server side resource packs not working.
Users sometimes getting stuck in ground after teleporting (might still happen if the server is lagging a lot).
