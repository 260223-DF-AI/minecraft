A block of coal is a block added in Java Edition 2.0. It was later added as an authentic block in 1.6.


== Obtaining ==


=== Breaking ===
Blocks of coal can be mined using any pickaxe, in which case it drops itself. If mined without a pickaxe, it drops nothing.


=== Crafting ===
Similar to other mineral blocks, it is crafted with 9 coal.


== Usage ==
Blocks of coal can be crafted back into coal but the coal will not be stackable with normal coal.


=== Crafting ingredient ===


=== Fuel ===
Blocks of coal can be used as fuel in a furnace. One block of coal lasts 800 seconds (16000 ticks),[verify] which smelts 80 items.
