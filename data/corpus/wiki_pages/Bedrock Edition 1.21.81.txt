1.21.81 is a hotfix for Bedrock Edition released on May 14-16, 2025, which fixes some bugs.


== Fixes ==
8 issues fixed
MCPE-201832 – Hitting blocks while swimming will slow you down.
MCPE-217599 – Lingering potions with unique models crash the game.
MCPE-218490 – Vibrant Visuals: Very pixelated distortion on specific iOS devices.
Other

Fixed an issue where Vibrant Visuals (Experimental) would sometimes apply to worlds that do not correctly support it.
Fixed a crash that could occur when loading a world with leashed entities in it.
Fixed a bug where block custom component event beforeOnPlayerPlace was not using read only privilege. Due to V1.X.X of scripting releasing without read only privilege, the fix is now only applied to V2.X.X and higher versions of scripting.
minecraft:leashable component values can once again be updated at runtime via component groups.
minecraft:leashable presets are now re-evaluated every tick instead of only when the entity is leashed.
