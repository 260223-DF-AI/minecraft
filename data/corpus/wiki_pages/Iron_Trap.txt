An iron trap is a block found in 26w14a.


== Obtaining ==


=== Breaking ===
Although the iron trap has a drop, no vanilla tool is suitable for breaking it, so breaking it with any tool will have no effect. Furthermore, attempting to use Attack / Mine to order a tool to mine it will not happen, as tools only mine blocks that they can break.
However, it is still possible to make an iron trap block drop a living iron trap with an explosion.
Using cheats, it's possible to create a tool capable of mining it by modifying minecraft:tool.rules data component. Can be add this block in #mineable/pickaxe so that it can be mined by any pickaxe (additionally, adding a tag like #needs_iron_tool prevents it from being mined by some pickaxes.)


=== Post-generation ===
If at least eight living iron bars come close together, a living iron trap will form.
It can be placed as an item in a container by approaching an alive container or using a hopper.
It can be placed as a block with build, or somehow remove the action items from the hotbar.


== Usage ==
Living attempt to pathfind towards the nearest player. When coming in contact with a player, it will spawn a cage of iron bars around them, trapping them.
The copper cage has 25 iron bars (17 more than that used to produce the copper trap) and removing all the blocks that were previously in the iron bars' position, including blocks like bedrock.

== Issues ==
Issues relating to "Iron Trap" are not maintained on the bug tracker because it is an April Fools' joke, and is therefore not in the newest stable version or snapshot. Issues reported there are closed as "Invalid".


== Trivia ==
According to one of the developers, iron and copper traps are a reference to the mod SecurityCraft.
