A version of Indev 0.31 was released on January 6, 2010, at 21:58 UTC, which added map settings. It was announced at 21:59 UTC.


== Additions ==


=== World generation ===
World options

Level type (island, floating, flat, original).
Island: The world is a large island, sand and gravel are common. The map is surrounded by water.
Floating: The world is made of floating islands, under which is void. Water is rare. The map is surrounded by air.
Flat: The surface is similar to modern superflat. Trees and flowers are generated. The map is surrounded by water.
Original: The terrain resembles that of late Classic. The map is surrounded by water.[verify for Java Edition]
Level shape (square, long, deep).
Level size (small, normal, huge). Map sizes were:

The bottommost layer of the world no longer has an infinite lava sea.[verify for Java Edition]


== Changes ==


=== Gameplay ===
Creative mode

Blocks are no longer infinite in Creative mode.
