13w07a is the tenth snapshot for Java Edition 1.5, released on February 14, 2013.


== Changes ==


=== Gameplay ===
NBT

FallingSand can be edited using an NBT now called TileEntityData.
The players can change the sand to other tile entities.


=== General ===
Lighting

Slight optimizations.
Minecraft Realms

Updated the (currently unused) Minecraft Online localization strings, changing the name of the service to Minecraft Realms.


== Fixes ==
3 issues fixed
From released versions before 1.5

MC-2196  Map item placed in item frame experiences Z-fighting.
From the 1.5 development versions

MC-7302  Nametag of certain mobs too low / too high.
MC-9212  Team colors/prefixes/suffixes are not displayed with @a/@p/@r.
