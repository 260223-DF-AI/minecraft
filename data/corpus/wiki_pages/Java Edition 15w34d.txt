15w34d is the thirteenth snapshot for Java Edition 1.9, released on August 21, 2015 to fix a memory allocation crash found in 15w34c.


== Changes ==


=== Items ===
Hoe

Now takes damage while attacking with it.


== Fixes ==
7 issues fixed
From released versions before 1.9

MC-55654 – Mobs transported from the Nether to the Overworld cannot be identified through commands.
MC-80228 – Major projectile bug.
MC-80629 – Changing spawner with SpawnData using spawn egg does not work.
From the 1.9 development versions

MC-85627 – Empty lines on signs rendered as "null" when upgrading from 1.7.10 -> 1.8.8 -> 15w32c.
MC-86744 – When player closes world and then open it back again, the attack indicator turns off.
From the previous development version

MC-86906 – No durability for hoe.
MC-86911 – Minecraft crashes with shield and ender pearl.
