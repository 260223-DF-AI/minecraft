20w13a is the eighth snapshot for Java Edition 1.16, released on March 25, 2020, which adds the strider mob, lodestone and fixes some bugs.


== Additions ==


=== Blocks ===
 Lodestone

Crafted from a netherite ingot and chiseled stone bricks.
Using a compass on one creates a lodestone compass, which points to that lodestone block.
Its functionality applies in all dimensions.


=== Items ===
 Warped fungus on a stick

Crafted with a fishing rod and warped fungus.
Can be used to control striders in the same way as a carrot on a stick with pigs.
Spawn eggs

 Strider spawn egg
Spawns striders.


=== Mobs ===
 Strider

A new passive mob that spawns in the Nether on lava oceans.
Can be saddled and controlled with warped fungus on a stick.
Is damaged by water and rain.
Can be bred with warped fungi.
Can spawn with zombified piglins or baby striders riding them.
If they spawn with a zombified piglin, they also have a saddle equipped.


=== Gameplay ===
Tags

Added the logs_that_burn block and item tags.
Contains the dark_oak_logs, oak_logs, acacia_logs, birch_logs, jungle_logs, and spruce_logs block and item tags, respectively.
Added the non_flammable_wood block and item tags.
Contains crimson and warped stems, hyphae, planks, and other wood blocks, as well as stripped variants where applicable.
Items in the item tag cannot be used as fuel in a furnace.
Added the strider_warm_blocks block tag.
Contains lava.
Unless a strider is in a block in this tag, it shivers.


== Changes ==


=== Blocks ===
Basalt

Can now be made if lava touches blue ice while flowing over soul soil.
Bee nests

Now appear yellow on maps.
Campfires

Can now be crafted with stems and hyphae.
Fences

Nether wood fences now connect to other wood fences.
Hyphae and stems

Now use the nether wood material.
Can no longer catch fire from lava.
Jigsaw block

Can have one of 12 orientations.
Added new property Joint type to describe if attached piece can be rotated (rollable) or not (aligned).
NBT field target_pool has been renamed to pool.
attachement_type has been split into name (on parent block) and target (on child block).
Nether gold ore

The texture has been changed.
Now drops experience when mined.
Nether planks, stairs, slabs, doors, buttons, trapdoors, pressure plates, and signs

Can no longer be used as fuel in furnaces, blast furnaces, or smokers.
Note block

Now plays the bass instrument when placed on blocks in the nether wood material.
Polished basalt

Can now be crafted from basalt in a 22 grid, similar to other polished blocks.
Can no longer be smelted.
Can now be crafted from basalt in a stonecutter.
Respawn anchors

Can be charged with a dispenser filled with glowstone.
Can no longer be destroyed by the ender dragon.
Smoker

Can now be crafted with stems and hyphae.


=== Items ===
Charcoal

Can now only be smelted from items in the logs_that_burn block tag.
This change does not affect gameplay, as the new tag only contains items charcoal could be smelted from.
Compass

Now points to the center of the spawn point block instead of its northwest corner.


=== Mobs ===
Villagers

Farmers can now use bone meal to make their crops grow faster.
Farmers now put excess seeds in their composter.
Farmers share excess wheat with other farmers, so more farmers can make bread.
Zombified piglins

Texture has been tweaked.


=== Gameplay ===
Advancements

Breeding striders can now be used to unlock "The Parrots and the Bats" advancement.
Bartering

Items enchanted with Soul Speed are now slightly more common.
Creative inventory

Moved bamboo, kelp, and sugar cane to the "Decoration Blocks" tab.
Fishing

Open water is now calculated differently. Each layer in the 545 area around the bobber must meet one and only one of the following criteria:
Have only water source blocks, waterlogged blocks without collision, or bubble columns.
Have only air or lily pads.
Recipe book

Charcoal now unlocks using the logs_that_burn item tag.
Tags

Added the respawn anchor to the dragon_immune block tag.
Changed the logs block and item tags to use the logs_that_burn, crimson_stems, and warped_stems block and item tags, respectively.
