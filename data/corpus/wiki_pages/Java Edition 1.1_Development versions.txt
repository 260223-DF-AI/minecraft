This is a list of Java Edition 1.1 development versions.


== 11w47a ==
11w47a is the first snapshot for Java Edition 1.1, released on November 24, 2011, which fixed a bug and optimized the game. This is the first snapshot released in 2011, the first snapshot for Java Edition in general, and the first to use this naming format.


=== Changes ===
Optimizations

Subtle code optimizations.


=== Fixes ===
1 bug fixed

Fixed a bug with double doors where one may not open correctly.


== 11w48a ==
11w48a is the second snapshot for Java Edition 1.1, released on December 1, 2011, which modifies farmland, apple drop chances, and void fog.


=== Changes ===


==== Blocks ====
 Farmland

Farmland can now only be trampled if the player jumps/falls on it; walking no longer affects it.
Mobs can similarly trample farmland by jumping on it, but not by walking.


==== Items ====
 Apples

Now have a 1⁄200 (0.5%)  chance of dropping from oak leaves.


==== World generation ====
Taiga

Changed temperature value from 0.3 to 0.05, making taigas generate with snow again.


==== Gameplay ====
Void fog

Void fog has been removed in Creative mode.

Superflat

Completely flat world composed of grass blocks, dirt, and bedrock; also generates villages.
