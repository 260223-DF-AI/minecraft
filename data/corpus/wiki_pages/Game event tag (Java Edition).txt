A game event tag is a group of game events. See below for the use of each tag.


== List of tags ==


=== allay_can_listen ===
Allays can listen and react to game events in this tag.


=== ignore_vibrations_sneaking ===
Game events in this tag are ignored by sculk sensors if the source entity is sneaking.


=== shrieker_can_listen ===
Game events in this tag trigger sculk shriekers.
Used in the #warden_can_listen tag.


=== vibrations ===
Game events in this tag trigger sculk sensors.


=== warden_can_listen ===
Wardens can listen and react to game events in this tag.
