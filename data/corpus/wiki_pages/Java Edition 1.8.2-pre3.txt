1.8.2-pre3 is the third pre-release for Java Edition 1.8.2, released on January 15, 2015, which fixes a bug with passive mobs.


== Fixes ==
2 issues fixed
From the previous development version

MC-76556 – Creepers, blazes, pigs, ocelots, rabbits, and iron golems take damage every 0.5 seconds.
From the current version, hotfixed

MC-76569 – Realms: [Client thread/FATAL]: Error executing task.


== Trivia ==
This version was previously removed from the launcher for some time, but was added back in September 2018.
