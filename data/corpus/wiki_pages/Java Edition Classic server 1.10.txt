1.10 is a Java Edition Classic server version released for 0.30 (Creative) on December 11, 2009.


== Additions ==
A warning was added for turning verify-names in server.properties off.
The warning reads:
######################### WARNING #########################
verify-names is set to false! This means that anyone who
connects to this server can choose any username he or she
wants! This includes impersonating an OP!

AND SINCE THIS IS A PUBLIC SERVER, IT WILL HAPPEN TO YOU!

If you wish to fix this, edit server.properties, and change
verify-names to true.
###########################################################

Added admin-slot to server.properties.
Setting this to true prevents players from logging in if the server is almost full and they're not admins.


== Fixes ==
Tried to fix grow-trees.
Fixed a bug where people with long usernames could connect.
