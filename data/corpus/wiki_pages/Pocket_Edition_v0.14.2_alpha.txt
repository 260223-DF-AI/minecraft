v0.14.2 alpha is a minor update to Pocket Edition and the Windows 10 Edition Beta and is the first version of Gear VR Edition, released on April 26-27, 2016.
This version of Windows 10 Edition was the base for the first development version of Education Edition.


== Additions ==
Generating or Loading in a world now displays the message "Done!".
Version exclusive: New player burning animation.
Version exclusive: New entering nether portal animation.
Version exclusive: New suffocation animation.


== Changes ==


=== Blocks ===
Nether Bricks 

Now has a brighter texture.
  Lava 

Now uses premade animation files instead of procedurally generated textures.
The new texture is distinct from both the old version and the version used in Java Edition 1.5 onwards, featuring a bubble-like animation on the top.
  Water 

Now uses premade animation files instead of procedurally generated textures.
  Fire 

Now uses premade animation files instead of procedurally generated textures.


=== Items ===
Camera 

Data value is now 498 and is no longer functional.


=== Gameplay ===
New Inventory button design; gray
Most likely added to prevent the confusion of the Inventory button being an item in the Hotbar.
There is a bug while opening an inventory, in which the Hotbar becomes small.
There is code in the game files that shows that the interact button would now say "Take Picture".
New loading screen when joining a server.
Chat is now slightly different (starting from the health bar)


=== Technical ===
The minimum version you can have to play Minecraft on iOS is now iOS 6.0.


=== Other ===
Settings

Removed "Animated Water" setting.
Version exclusive: Added Anti-Aliasing option to Windows 10.
Version exclusive: Added Fancy Leaves option to Windows 10.
Version exclusive: Added Particle Render Distance option to Windows 10.
Re-added "Generating world" message while loading a world and server.
