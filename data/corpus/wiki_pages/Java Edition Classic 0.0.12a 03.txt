0.0.12a_03 is a version of Java Edition Classic released on May 19, 2009, shortly before 23:49 UTC, which improves performance and fixed a bug making it hard to select blocks on some graphics cards. It was announced by Notch on May 20, 2009, at 00:14 UTC.
This version was reuploaded on May 20, 2009, at 00:18 UTC for unknown reasons.


== Technical ==


=== General ===
Improved performance.


== Fixes ==
1 bug fixed

Fixed a bug making it hard to select blocks on some graphics cards.


== Trivia ==
This is the earliest unmodified version of Java Edition to be archived, as well as the earliest which is not a developer build.
