Alpha v1.1.2 is a version of Java Edition Alpha released on September 18, 2010 to fix sound and graphical bugs. Shortly after this version's unannounced release, Notch made temporary, load-reducing changes to his web server. Since one such change was disabling payments, he decided to declare Minecraft free-to-play for the weekend.


== Technical ==
General

Removed debug code that set the display's gamma and contrast values to zero.
This fixes the "gray screen" bug from the previous version.


== Fixes ==
Attempted to fix a bug involving sounds.[verify for Java Edition]
