1.7.10 Build 4 is the fourth build for MinecraftEdu 1.7.10, released in 2014. It is based on Java Edition 1.7.10.


== Additions ==


=== General ===
Readded Worldedit.
"Time Slider" that replaces the Enable / Disable night option.
Server icon that is shown in the multiplayer server list.


== Changes ==


=== General ===
Students can now use Esc menu while frozen.
Updated Forge to 10.13.0.1208


== Fixes ==
8 bugs fixed

Saving in server window not working sometimes.
Versions not showing correctly if user tried to connect to a server with wrong client version.
Students being able to click "Connect as a Teacher" button.
Students not being able to break their own Home Blocks
Spectate mode bug where users joining the server could see users that were in spectate mode.
Lava starting fires even when fire was disabled.
Server window give.
Server sometimes not starting because of old ops.txt, white-list.txt, banned-players.txt and banned-ips.txt files.
