Alpha v1.1.0, also known as Seecret Friday 9, is a version of Java Edition Alpha released with server versions 0.2.0 and 0.2.0_01 on September 10, 2010, at 18:40 UTC. This version notably added a shared client-server inventory. It is the ninth and last Seecret Friday update.
Alpha v1.1.0 was reuploaded later on the day of release.[more information needed]
Alpha v1.1.0 was reuploaded on September 13, 2010,[verify for Java Edition] to fix a bug where the player could force the camera's movement inside walls.
The version found in the launcher is the first reupload version, but is recompiled differently and has a different checksum.


== Additions ==


=== Items ===
 Compass

Always points towards the world spawn point.
Crafted with iron ingots and redstone dust.


== Changes ==


=== Blocks ===
Monster spawner

No longer renders the model of the mob inside of them.


=== World generation ===
Other

Clay is more common when a world is generated.


=== Gameplay ===
Multiplayer

Experimental switch to turn on monsters in multiplayer.
The inventory is now saved on multiplayer servers.
Servers now save player locations upon leaving the server.


== Fixes ==
2 bugs fixed

Chests, signs, and furnaces now work in multiplayer.
Fixed a bug where the player could force the camera's movement inside walls.
