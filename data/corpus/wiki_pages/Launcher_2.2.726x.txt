2.2.7265 (Windows), 2.2.7264 (Windows, via the Microsoft Store/Xbox app), 2.2.7268 (macOS), or 2.2.7266 (Linux), is an update to the launcher released on November 2, 2021.
This version was the first released to the Windows Store and merged the three PC entries of the Minecraft franchise: Java and Bedrock editions and Minecraft Dungeons.


== Additions ==
Bedrock Edition now in the Launcher.
Minecraft Dungeons from Microsoft Store now in the Launcher.
Xbox Game Pass for PC subscribers who are on a Windows 10 or Windows 11 device and are using the new Launcher from the Microsoft Store/Xbox app can now play Java Edition, Bedrock Edition and Minecraft Dungeons under Game Pass.


== Changes ==
Updated how accounts work in the Launcher.


== Fixes ==
Fixed a bunch of UI bugs.
Fixes to the way the Launcher interacts with users' entitlements to various games, especially in relation to if the user is a Game Pass subscriber.
