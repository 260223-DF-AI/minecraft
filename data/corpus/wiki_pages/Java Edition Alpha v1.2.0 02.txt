Alpha v1.2.0_02 is a version of Java Edition Alpha, released on October 31, 2010, which adds some changes and fixes bugs.
This version is duplicated inside the launcher, as attempting to play Alpha v1.2.0 downloads and launches v1.2.0_02 instead.


== Changes ==


=== Blocks ===
Soul sand

Now generates in some of the positions previously taken by gravel in the Nether.


=== Mobs ===
Creepers

Now has new hurt sounds.


== Fixes ==
2 bugs fixed

Items no longer disappear when the player dies.
Fixed the player's hurt sound not playing.
