1.16 Pre-release 5 (known as 1.16-pre5 in the launcher) is the fifth pre-release for Java Edition 1.16, released on June 12, 2020, which fixes bugs.


== Additions ==


=== General ===

Options

Added a warning message for enabling Fabulous! graphics.
The message appears when the player's graphics driver is known to have rendering issues with the graphics option, or if the driver is detected as problematic.
The list of drivers affected can be changed using resource packs.


== Changes ==


=== Blocks ===
Water

The texture used by water when adjacent to glass and similar such blocks was darkened.


=== General ===
Options

Removed the quotes around "Fast" and "Fancy" in the descriptions for these graphics types.
Instances of "Fabulous!" in the description for fabulous graphics now appears in italics ("Fabulous!").
World creation

Changed the description of the doMobLoot and doTileDrops game rules from "Controls resource drops, including experience orbs" to "Controls resource drops from <mobs/blocks>, including experience orbs"
