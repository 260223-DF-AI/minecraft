1.8.5 is a minor update to Java Edition which was released on May 22, 2015 to fix a security related issue, a scoreboard related issue and some changes to splashes. 1.8.5 is compatible with 1.8 – 1.8.4 servers.


== Additions ==


=== General ===
Splashes

Added "GNU Terry Pratchett".


== Changes ==


=== General ===
Splashes

Removed the following splashes:
"Happy birthday Notch!"
"Happy Birthday ez!"
"Notch <3 ez!"
Changed "Cooler than Spock!" to "Not as cool as Spock!"


== Fixes ==
5 issues fixed
From 1.8

MC-50550 – Stacked boats floating.
From 1.8.4

MC-79632 – Phantom custom player in scoreboard/UUIDs of non-player entities on scoreboards truncated after relog.
Private issues

MC-80478 – Better server validation for movement packets.
MC-80479 – Invalid beacon effects are not validated.
MC-80480 – Certain attribute values are not bound.
