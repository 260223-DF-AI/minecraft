1.20 Release Candidate 1 (known as 1.20-rc1 in the launcher) is the first and only release candidate for Java Edition 1.20, released on May 31, 2023, which fixes bugs.


== Additions ==


=== General ===
Splashes

Added "Joule is neat too!"


== Changes ==


=== Blocks ===
Chiseled bookshelf

Knowledge books can now be placed inside them.


=== General ===
Tag

Added knowledge_book into the #bookshelf_books item tag.


== Fixes ==
3 issues fixed
From released versions before 1.20

MC-256477 – Knowledge books can't be placed in chiseled bookshelves.
MC-262325 – Ice/Slime blocks don't give speed when landing towards the very edge of the block.
From the 1.20 development versions

MC-262853 – Pitcher plant breaks and floats when growing without enough light.
