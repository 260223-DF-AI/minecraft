2.2.1862 (Windows), 2.2.1864 (macOS), or 2.2.1867 (Linux), is an update to the launcher released on February 16, 2021.


== Changes ==
Added transitions between all pages in the Launcher. To prevent motion sickness, one can enable (and disable) transitions under settings. Transitions come disabled by default.
The filter for historical versions has moved in under settings.
Patch notes content is now locally cached.


== Trivia ==
In the changes section, it contained the text "dn ps- uos u s s 'uop psdn o nul u no I" as an Easter Egg, which says "If you change language to upside down, is this then shown right-side up?".


== Fixes ==
Login toast message now respects account migration dates.
MCL-16797 - Sculk sensor changes in 21w05a patch notes are not properly formatted.
MCL-15385 - Immediate crash on run Debian.
