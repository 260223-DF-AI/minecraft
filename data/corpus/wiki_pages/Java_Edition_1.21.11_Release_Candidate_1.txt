1.21.11 Release Candidate 1 (known as 1.21.11-rc1 in the launcher) is the first release candidate for Java Edition 1.21.11, released on December 4, 2025, which tweaks spear mechanics.
A separate unobfuscated version of 1.21.11 Release Candidate 1 was released, titled 1.21.11 Release Candidate 1 Unobfuscated (or 1.21.11-rc1_unobfuscated when in the launcher). This is an experimental version released in preparation for obfuscation being removed from Java Edition clients and servers.


== Changes ==


=== Items ===
 Spear

Increased the maximum duration during which the spear can deal knockback while charging.
For wooden spears, changed from 120 to 200 ticks.
For stone spears, changed from 110 to 180 ticks.
For copper spears, changed from 100 to 165 ticks.
For iron spears, changed from 90 to 135 ticks.
For golden spears, changed from 110 to 170 ticks.
For diamond spears, changed from 80 to 130 ticks.
For netherite spears, changed from 70 to 110 ticks.
The charge attack animations now better correspond to the different phases of the spear.
