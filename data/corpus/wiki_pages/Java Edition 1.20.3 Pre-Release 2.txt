1.20.3 Pre-Release 2 (known as 1.20.3-pre2 in the launcher) is the second pre-release for Java Edition 1.20.3, released on November 22, 2023, which fixes bugs and adds new breakable blocks to the wind charge.


== Experimental ==
These changes only take effect when the Update 1.21 experimental data pack is enabled.


=== Changes ===


==== Non-mob entities ====
Wind charge

Now break decorated pots, chorus flowers, and pointed dripstone blocks upon collision.


==== General ====
Tags

Added wind_charge into the #impact_projectiles entity type tag.
