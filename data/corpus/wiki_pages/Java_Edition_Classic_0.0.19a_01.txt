0.0.19a_01 is a version of Java Edition Classic released on June 19, 2009, between 23:28 and 23:36 UTC to fix bugs relating to glass and sponge.


== Changes ==


=== Blocks ===
 Glass

Texture changed from .
Now displays in the hotbar: 
No longer semi transparent.
This means that glass no longer goes invisible when selected, when looked at through water or when underground.
Sponge

No longer decays.


== Fixes ==
2 bugs fixed

Fixed some bugs that caused water to flow back onto a sponge.[verify for Java Edition]
Fixed a bug causing sponges to block lava flooding.[verify for Java Edition]
