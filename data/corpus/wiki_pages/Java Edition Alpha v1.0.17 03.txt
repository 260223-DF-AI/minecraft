Alpha v1.0.17_03 is a version of Java Edition Alpha, released on August 23, 2010, which fixed an item duplication bug and improved performance.


== Fixes ==
3 bugs fixed

Fixed a duplication bug where closing the inventory while holding an item with the cursor would duplicate it, creating a glitched version that could be infinitely replicated.
Improved framerate.
Attempted to fix a bug with players getting stuck in blocks.
