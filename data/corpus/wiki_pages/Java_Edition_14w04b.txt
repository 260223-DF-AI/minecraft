14w04b is the seventh snapshot for Java Edition 1.8, released on January 24, 2014, to fix some of the bugs and crashes in 14w04a.
14w04b was reuploaded a few minutes later to fix issues with items and item frames.

== Trivia ==
14w04b is the last Java Edition version in which the missing texture cannot be encountered in the vanilla game without resource packs.
14w04b was previously removed from the launcher for some time, but was added back in September 2018.
