2.4.69 is an update to the launcher released on May 16, 2023.


== Changes ==
Improvements to game installation reliability.
Added support for native JRE on ARM64 devices on Windows.
Various usability updates & under-the-hood bug fixes.


== Fixes ==
MCL-21785 - On the Java Edition tab, there is a UI that is slightly lower than the others.
MCL-22699 - Username is not consistently aligned in the launcher.
MCL-23290 - MCL uses old Minecraft Dungeons logo.
MCL-23363 - Minecraft Launcher Showing Old Snapshot Icon.
