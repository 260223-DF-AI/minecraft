1.6.4 Build 6 is the sixth build for MinecraftEdu 1.6.4, released on October 31, 2013. It is based on Java Edition 1.6.4.


== Additions ==


=== General ===
Launcher

Stable versions are now bolded in list of updates at Launcher.
Added a lot of hidden settings to serverwizardsettings.ini that can be used to hide certain components from servertool starting screen:
disable-servertool-top-menu=true
disable-servertool-top-menu-file=true
disable-servertool-top-menu-open=true
disable-servertool-top-menu-settings=true
disable-servertool-top-menu-language=true
disable-servertool-button-tutorial=true
disable-servertool-button-new=true
disable-servertool-button-saved=true
disable-servertool-button-online=true
disable-servertool-button-local=true
disable-servertool-button-last=true


== Fixes ==
2 bugs fixed

Setting server RAM amount works through the servertool settings.
Server free RAM amount showing wrong in the server interface.
