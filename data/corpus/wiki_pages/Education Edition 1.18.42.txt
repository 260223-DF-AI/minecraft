1.18.42 is a minor update to Education Edition, released on October 25, 2022, which fixes some bugs.


== Changes ==


=== General ===
Multiplayer

Improved proxy support for multiplayer services.
Greater connection reliability in multiplayer sessions.
Library

Searching content across the entire in-game Library is easier than ever with the new Library Search Bar.
Coding

Coding window now defaults to full screen on mobile devices.


== Fixes ==
Various bug fixes in addition to performance and stability improvements.
