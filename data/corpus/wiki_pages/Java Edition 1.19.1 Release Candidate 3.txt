1.19.1 Release Candidate 3 (known as 1.19.1-rc3 in the launcher) is the third and final release candidate for Java Edition 1.19.1, released on July 26, 2022, which makes a change to chat, and fixes a bug.


== Changes ==


=== General ===
Chat

The chat input box no longer applies to custom font glyphs with negative advances, or glyphs with advances greater than 32.


== Fixes ==
1 issue fixed
From the previous development version

MC-254529 – Warning and information toasts can overlap one another.
