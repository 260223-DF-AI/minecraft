2.1.15167 (Windows and Dungeons), 2.1.15168 (macOS), or 2.1.15166 (Linux) is an update to the launcher released on June 3, 2020.


== Changes ==
Improve error logging.
Improve entitlement checking.
Clarify copy for launcher updates.
Microsoft Account login stability improvements.
Add launcher release notes to Dungeons launcher.
Only show disclaimer text about downloading the Dungeons launcher on the Windows version of the Minecraft: Java Edition launcher.


== Fixes ==
Fix bug so that log level tick boxes are now aligned vertically instead of horizontally.
Fix bug with fluctuating progress bar during download and installation process.
