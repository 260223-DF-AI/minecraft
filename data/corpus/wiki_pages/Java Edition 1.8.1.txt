1.8.1 is a minor update for Java Edition released on November 24, 2014 to improve performance and fix bugs. 1.8.1 is compatible with 1.8 servers. This is the final version released in 2014.


== Additions ==


=== Items ===
Maps

Added proper color display support for the new blocks.


=== Command format ===
Game rules

doEntityDrops - Works like doTileDrops and doMobDrops but applies to non-mob entities (minecarts, paintings, etc.) instead.


=== Gameplay ===
Credits

Ryan Holtz (TheMogMiner) and Michael Stoyke (Searge) are now mentioned in the credits.


== Changes ==


=== Blocks ===
Jack o'lanterns

Can be used to create iron golems and snow golems again.
Also able to be put on head to avoid endermen, and some other mobs.


=== Items ===
General

Now rotate on fast graphics.
Potion of Leaping

Can now be extended with redstone.


=== Mobs ===
General

No longer run from creepers that are about to explode.
Armor stands

Added Marker tag.


=== World generation ===
Witch huts

Newly generated witch huts can spawn witches on Y 64 to 71.
This allows for three spawning floors in witch farms.


=== Command format ===
Game rules

doTileDrops no longer affects entities.


=== General ===
Options

Renamed "Multiplayer Settings" to "Chat Settings".
Off/fancy/fast clouds.
Off/on entity shadows.
Fast/fancy now only controls leaves and rain particles.
Servers

Removed the client side TCP packet delay caused by Nagle's algorithm.
Performance

Various improvements.
Many optimizations.
