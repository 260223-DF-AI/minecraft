2.2.1259 (Windows), 2.2.1260 (macOS), or 2.2.1262 (Linux), is an update to the launcher released on January 14, 2021.


== Changes ==
Scaled Alex to be almost 3 blocks tall on the hero image


== Fixes ==
Made changes to account initiation, as a attempt to fix grey-screen issue.
Fixed spacing issue for key golem when using right-to-left languages
Added external opening icon for patch notes links
MCL-16374 – Minecraft opens twice
MCL-16349 – Launcher temporarily is offline after logging in and out of a Mojang account while a Microsoft account is logged in
