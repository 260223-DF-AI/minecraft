Alpha v1.0.15 is a version of Java Edition Alpha released on August 3, 2010. It introduced Survival multiplayer servers.
The server version 0.1.0 was not released at the same time, but instead was released one day later, on August 4, 2010.


== Additions ==


=== Command format ===
/wood

Drops 4 saplings.
Has a 5 minute cooldown.
/iron

Drops 4 iron ingots.
Has a 5 minute cooldown.


== Changes ==


=== Blocks ===
Lava

Sets blocks on fire.


=== Items ===
General

Changed the appearance of items from the third person once again.


=== General ===
Animations

Arm-swinging animation added, viewable in multiplayer and in third person view.
Multiplayer

Players can now enter the IP for a server other than Mojang's official server.
Title screen

Fixed the "M" in the Minecraft title screen logo, changing from  back to .


== Fixes ==
2 bugs fixed

The player can no longer move while on the inventory screen.
The player can no longer place blocks in the same space with entities if it is covered with snow.


== Trivia ==
The corresponding blog post is labeled as "Minecraft Alpha v1.0.14", which is a duplicate of a previous version released four days earlier, on July 30. The date of the jar and the timestamp of the post clearly show that it is Alpha v1.0.15 instead.
