Alpha v1.2.3_01 is a version of Java Edition Alpha released on November 24, 2010, to fix bugs.
This version was reuploaded two minutes after release, without gameplay changes.


== Fixes ==
2 bugs fixed

Fixed a bug in singleplayer mode where the game would crash when trying to load a newer version, but when the player was in the overworld.
Fixed infinite looping by disabling leaf decay.
