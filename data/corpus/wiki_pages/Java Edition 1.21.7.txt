1.21.7 is a minor hotfix update to Java Edition released on June 30, 2025, which adds a new music disc and painting referencing A Minecraft Movie, and fixes some critical issues from 1.21.6. It is not compatible with 1.21.6 servers.


== Additions ==


=== Items ===
Music disc

 Added the "Lava Chicken" music disc by Hyper Potions.
Obtained by killing a baby zombie while it is riding a chicken (a chicken jockey).
If the chicken is killed before the zombie is, the music disc is not dropped.
Has a comparator output of 9 when played in a jukebox.

== Changes ==


=== General ===
Data pack

The pack format version is now 81.
Resource pack

The pack format version is now 64.
Tags

Added dennis to #placeable painting variant tag.
