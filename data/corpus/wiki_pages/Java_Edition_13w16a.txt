13w16a is the first snapshot for Java Edition 1.6.1, released on April 18, 2013, which adds some new features including horses and their relatives, leads, and carpets. To run this snapshot, it was required to use a new launcher, the initial version of which was released at the same time.
The client of 13w16a was reuploaded three times: once 13 minutes after release and twice the day after release.[more information needed]
The server of 13w16a was reuploaded once.


== Additions ==


=== Blocks ===
 Carpets

Made using 2 of the player's desired colors of wool in a horizontal line.
 Hay blocks

Can be placed in all orientations, just like wood.


=== Items ===
 Horse armor

Exist in diamond, iron and gold variants.
Crafted with six iron/gold/diamond and a wool block.

 Leads

Can be used to bind mobs to fences or lead them around.
Right-click mob first, then right-click fence or right-click the mob again to get the lead back.
 Horse saddle

Stacks to 32.
Crafted with five leather and three iron ingots.

 Horse spawn egg

When used, a different horse breed is created each time.
Works like any other spawn egg.


=== Mobs ===
 Horses

Ridable by right-clicking them, move with usual movement keys, maximum speed, and jump height differ between horses.
Have various amounts of maximum health, use wheat and carrots to heal them.
Drop leather when killed.
Can be bred using wheat  baby horses can be grown using apples, golden apples, wheat, and hay bales.
Players can make them jump by pressing and holding the space bar.
They naturally step up over 1 block, which enables rider to walk up hills.
They can also use ladders, when steered towards them.
Creation assisted by DrZhark (creator of the Mo' Creatures mod).
 Donkeys

Like horses, but can carry chests on their back although no inventory is currently available.
Can be bred with horses to make a mule.
 Mules

Mules can be obtained by breeding a donkey and a horse.
 Skeleton horses

Require NBT editing to view in-game.
An unused mob.
 Zombie horses

Require NBT editing to view in-game.
An unused mob.


== Changes ==


=== Mobs ===
General 

Changed the idea of mob difficulty.
Regional difficulty: The longer the player stays in one area, the harder it gets.
Scales with difficulty  Items, enchants or AI improve with difficulty.


=== Gameplay ===
Enchanting

Improved Respiration.
Now also helps seeing underwater (depends on the enchantment level).
Riding

Add mob's health bar to the heads-up display when riding on it (both pigs and horses).
When riding something, like a mob or a boat, dismounting is only possible using the sneak key.


=== General ===
Credits

Added Mo' Creatures developers to credits.txt.
Game window

Now displays the client's version in the window title (for example: "Minecraft 13w16a").
Directory structure

.minecraft directory structure is changed:
assets folder; stores resources such as language packs and sounds.
All music discs now use .ogg format instead of .mus format.
library folder; stores Java libraries such as LWJGL and jinput.jar.
versions folder; there are files version.json (meta data) and folders that stores minecraft.jar for each versions (but its name is the same as its version; the jar for 13w16a is versions/13w16a/13w16a.jar).
Game directory

Both the game's main class (net.minecraft.client.main.Main) and the new launcher began to support an optionally specified game directory, where the game stores its files (/saves, options.txt, etc.). The main class' parameter for the game directory was named as (a misleading) --workDir, and it was later changed (in 13w24a) to a more adequate --gameDir.


== Trivia ==
This is the earliest snapshot available in the Minecraft launcher. Although earlier snapshots are not available in the launcher, almost all of them have been archived.
This is the first snapshot to require Java 6 and is also able to launch with Java 9 and up.
This is the first snapshot of the older snapshots and versions that are available in the Minecraft launcher to not require be launched by a helper-wrapper class (net.minecraft.launchwrapper.Launch) that was used for pre-1.6 versions to tweak the old code to the modern launcher's needs. This and the later snapshots and versions are being launched normally, by their main class (net.minecraft.client.main.Main).
