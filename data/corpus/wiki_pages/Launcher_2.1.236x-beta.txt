2.1.2366-beta (Windows), 2.1.2365-beta (macOS), or 2.1.2364-beta (Linux) is an update to the launcher released on February 11, 2019.


== Additions ==
The launcher will now display a notification of new launcher versions, even if they release while the launcher is already running.


== Fixes ==
Fixed a bug that caused the launcher to soft-lock when there was no launcher_profiles.json file on disk.
MCL-5436: Cannot launch directory junction files.
MCL-10280: Launcher can't save the game directory in profiles.
