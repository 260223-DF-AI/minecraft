1.9 (originally released as 1.8.3) is a Java Edition Classic server version that released with 0.28 on October 27, 2009.
This version was originally released as "1.8.3" (as labeled in the README), but the README for the next version retcons this label to "1.9"..


== Additions ==
Added server support for all features added in 0.28.
Added support for Survival multiplayer.
Still a work in progress, and was not publicly available on the website.
Was very buggy at this time.


== Fixes ==
Fixed exploit where someone logging in with an illegal name would cause players to disconnect.
