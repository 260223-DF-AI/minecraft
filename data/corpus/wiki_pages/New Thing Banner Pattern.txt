A new thing banner pattern is a joke banner pattern from Java Edition 23w13a_or_b. It is used to customize banners with a symbol that is based on an orientation of the M from the Mojang logo, which resembles an abstraction of a crewmate from Among Us.


== Obtaining ==
New thing banner patterns can be obtained only after voting for the rule minecraft:special_recipe with value minecraft:m_banner_pattern.


=== Crafting ===


== Usage ==


=== Loom ingredient ===
New thing banner patterns are used in looms to add customization to banners. The pattern must be combined with 1 banner and 1 dye. Upon usage in the loom, the new thing banner pattern is not consumed.

== Issues ==
Issues relating to "New Thing Banner Pattern" are not maintained on the bug tracker because it is an April Fools' joke, and is therefore not in the newest stable version or snapshot. Issues reported there are closed as "Invalid".


== Trivia ==
The pattern being crafted from suspicious stew is a reference to the memetic neologism “sus” (short for “suspicious”), used to indicate an Among Us player might be the imposter.
