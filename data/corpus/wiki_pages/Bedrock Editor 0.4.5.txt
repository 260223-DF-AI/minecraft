Bedrock Editor v0.4.5 is a minor alpha release for the Bedrock Editor released on September 20, 2023, which introduces a new way to test projects.


== Additions ==
Testing

Added a new way to test projects directly in the Editor:
Changes made in the test session will not be saved.
Marketplace and persona are disabled in the test experience, which is also standard within the Editor.
Currently only available in single-player.


== Fixes ==
Fixed a bug in the Selection tool where the Y axis of the Vector3 origin exceeded the dimension height range.
Fixed a bug where blocks couldn’t be placed in the last block placed location after an undo.
