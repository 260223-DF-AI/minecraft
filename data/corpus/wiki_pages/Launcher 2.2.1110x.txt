2.2.11103 (Windows), 2.2.11105 (Windows, via the Microsoft Store/Xbox app), 2.2.11104 (macOS), or 2.2.11106 (Linux), is an update to the launcher released on March 1, 2022.


== Additions ==
Added mouseover tooltip to "Clean up cached account tokens" advanced settings button.


== Changes ==
Made some error messages clearer and more useful.
Updated some outdated textures/blocks in the installations icon list.


== Fixes ==
Fixed many miscellaneous under-the-hood stability and reliability issues.
Fixed bugs related to the Mojang to Microsoft Account migration.
Fixed volume install dialog default logic.
MCL-19984 - Java Edition shows Demo with active Xbox Game Pass subscription.
MCL-20491 - Game stuck on "making things awesome" screen.
MCL-20768 - Setting username from within Launcher limits name to 13 characters.
MCL-21101 - Z-fighting can be seen where the legs and torsos of player models in the "Skins" tab intersect one another.
