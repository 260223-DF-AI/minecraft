1.19.1 Release Candidate 2 (known as 1.19.1-rc2 in the launcher) is the second release candidate for Java Edition 1.19.1, released on July 21, 2022, which makes tweaks and fixes bugs.


== Changes ==


=== General ===
Chat

Added a warning toast when connecting to a server that doesn't enforce secure chat.
Options

Tweaked the names of the chat preview options.
Renamed "On Modified" to "While Typing".
Renamed "On Send" to "When Sending".


== Fixes ==
5 issues fixed
From 1.19

MC-254355 – Key binds set to mouse buttons of number greater than 8 switch over by 1 when the game starts.
From the previous development version

MC-254405 – Debug messages aren't prefixed with gray color indicators.
MC-254417 – Screenshot messages aren't prefixed with gray color indicators.
MC-254419 – Local game hosting messages aren't prefixed with gray color indicators.
MC-254421 – Social interaction messages aren't prefixed with gray color indicators.
