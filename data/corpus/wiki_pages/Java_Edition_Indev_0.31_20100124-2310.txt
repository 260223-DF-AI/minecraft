A version of Indev 0.31 was released on January 24, 2010, at 23:10 UTC. Its release was mentioned the day after in the announcement of the next Indev version's release.


== Additions ==


=== General ===
Particles

New particle effect for harvesting blocks.
Added a particle bubble.


== Changes ==


=== Blocks ===
Bedrock

Blast resistance now equals 6,000,000.
Chests

The player can no longer make triple chests.
TNT

Explosions lose power when going through stronger materials.
Now produce smoke particles again.
Now deals more damage.
No longer destroys blocks when ignited in water or lava.[verify for Java Edition]
  Torch[is this the correct version?]

Texture changed from  
Shrank hitbox.
Added a top texture.


=== Items ===
General

The player can no longer drop whole stacks of items, and must instead drop one at a time.
Items now get destroyed by fire and lava.[verify for Java Edition]
Bows

Now use arrows when firing.
Firing an arrow underwater creates bubble particles.


=== World generation ===
Starting house

Now contains an extra chest with TNT in it.
World

All air directly above bedrock layer is turned into lava.[verify for Java Edition]


== Bugs ==
2 bugs reported

Game crashes during generating world when player is offline.
Sometimes The Game Always Lags.
