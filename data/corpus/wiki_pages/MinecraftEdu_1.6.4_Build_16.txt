1.6.4 Build 16 is the sixteenth build for MinecraftEdu 1.6.4, released on April 24, 2014. It is based on Java Edition 1.6.4.


== Additions ==


=== General ===
Completely new Assignment system where you can set multiple assignments and you can track the student progress. (Client side).
New User Control panel in server window. You can administrate alias protection and whitelist with it.


== Fixes ==
1 bug fixed

External translations path was wrong in client. Everything works now as described here.
