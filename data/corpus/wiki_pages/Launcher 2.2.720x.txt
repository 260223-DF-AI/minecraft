2.2.7206 (Windows), 2.2.7207 (Windows, via the Microsoft Store/Xbox app), 2.2.7205 (macOS), or 2.2.7204 (Linux), is an update to the launcher released on November 1, 2021, this is the first version of the launcher distributed to the Microsoft Store/Xbox app.


== Additions ==
Bedrock Edition now in the Launcher.
Minecraft Dungeons from Microsoft Store now in the Launcher.
Xbox Game Pass for PC subscribers who are on a Windows 10 or Windows 11 device and are using the new Launcher from the Microsoft Store/Xbox app can now play Java Edition, Bedrock Edition and Minecraft Dungeons under Game Pass.


== Changes ==
Updated how accounts work in the Launcher.


== Fixes ==
Fixed a bunch of UI bugs.
