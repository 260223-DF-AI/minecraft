Dune armor trim is a type of smithing template used in smithing tables to add a trim to armor. It is consumed when used, but can be duplicated using an existing template, sandstone, and seven diamonds.


== Obtaining ==
Dune armor trims are found in desert pyramids. Once obtained, they can be duplicated using the crafting recipe.

== Usage ==
It is an ingredient in its own recipe, making it possible to craft more after the initial item is found.


=== Crafting ingredient ===

In Bedrock Edition, dune armor trims can be duplicated with more blocks than in Java Edition. In addition to the normal sandstone, it can also be duplicated with smooth, cut, or chiseled sandstone.


=== Smithing ingredient ===

Using a smithing table, the dune armor trim can be applied to any armor piece, using any trim material.
