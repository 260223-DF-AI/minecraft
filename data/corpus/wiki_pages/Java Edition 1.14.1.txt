1.14.1 is a minor update to Java Edition released on May 13, 2019, which improves performance and fixes bugs in 1.14. It is not compatible with 1.14 servers.


== Additions ==


=== General ===
Controls

Pressing Esc while holding F3 pauses the game without bringing up the pause menu.
Splashes

Added the following splash texts:
"Awesome game design right there!"
"Ph1lza had a good run!"
"10 years of Mining and Crafting!"


== Changes ==


=== Blocks ===
Campfire

Can now be lit by flaming arrows.
Crafting table

Updated texture to fix incorrect planks texture.
Iron ore

Changed texture to fix miscolored pixels.


=== Items ===
Iron horse armor

Tweaked colors slightly.
Leather horse armor

Changed texture to be more consistent with the other types of horse armor.


=== World generation ===
Villages

Many structures are tweaked to have better lighting.
Village generation has slight differences from 1.14.


=== General ===
Controls

Pressing F3 + N when in Survival mode now changes the player into Creative mode, if cheats are enabled.
