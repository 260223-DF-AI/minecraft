1.6.4 is a minor update to Java Edition released on September 19, 2013, which fixed a bug where structures would not save with the world file.
The server version of 1.6.4 was reuploaded.[more information needed]


== Additions ==


=== General ===
server.properties

Added server.properties option player-idle-timeout.
Kicks players from a server when they are idle for a set amount of minutes
Defaults to disabled (0).


== Changes ==


=== World generation ===
Desert villages

Wells are now made of cobblestone and gravel instead of sandstone.


== Fixes ==
5 issues fixed
From released versions before 1.6.1

MC-1867  Double sound effect for fall damage.
MC-3464  Arrows do not stick in soul sand.
From 1.6.1

MC-21066  Game crashed on a server.
From the previous development version

MC-31065  Villages not generating.
Other fixes

Fixed a handshake exploit.
