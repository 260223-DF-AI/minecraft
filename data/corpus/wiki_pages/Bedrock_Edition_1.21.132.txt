1.21.132 is a hotfix to Bedrock Edition released for most devices on January 8, 2026, and for PlayStation 5 on January 9, 2026, which addresses issues discovered in the latest update. This is the first version released in 2026. This is also the final version to use the old version formatting.


== Fixes ==
7 issues fixed
MCPE-232933  Clocks and compasses are invisible after the new update
MCPE-233088  Xbox: Additional players in splitscreen log in with no items/the wrong user's items after 1.21.130 update
MCPE-233369  Add-on assets disappearing in split screen
MCPE-233598  Player is kicked out of world with "Block" error code if a command affects too many entities
Other

Fixed an issue where textures would not be correctly reloaded when changing between graphics modes in the main menu.
Fixed a freeze that could happen when buying a Realm on Nintendo Switch.
Fixed attachables not rendering correctly when the game is paused or when rendered in an NPC UI window.
