Beta 1.8 Pre-release 2 ;) [sic] (Beta 1.8-pre2) is the second and final pre-release for Java Edition Beta 1.8, released on September 13, 2011.
An earlier variant of this version, from September 12, 2011, has also been found. In the earlier version, dead bushes are missing their tooltip, but apart from that and some other minor code differences, this version is identical. It is unknown if the September 12 version was publicly released.


== Additions ==


=== General ===
Options

"Pick Block" can now be remapped.
"Toggle Fog" can be remapped once again.


== Changes ==


=== Blocks ===
 Cracked stone bricks

Cracked stone bricks now drop themselves instead of regular stone bricks.
 Dead bushes

Now have a tooltip.
 Grass block

Name changed from "Grass".
 Mossy stone bricks

Mossy stone bricks now drop themselves instead of regular stone bricks.
 Vines

Now spread infinitely down, but do not spread if there are more than four other vine blocks nearby.


=== Items ===
 Bone meal

Can now be used on pumpkin seeds and melon seeds.
Huge mushrooms can only be made if a mushroom is planted on grass or dirt.
 Swords

Blocking now works in multiplayer.


=== Mobs ===
 Cave spiders

Now apply Poison correctly instead of doing so regardless of difficulty.
 Squid

Now drop experience orbs when killed.


=== World generation ===
Mineshaft

Their density is now significantly decreased.
Generate about 2.5 times as often.
Stronghold

Increased stronghold count to three per map instead of one.


=== Gameplay ===
Controls

Fog key mapped to F.
F6 and F7 working as a result of left-in debug code not being removed.
⇧ Shift+ clicking while crafting makes as much as possible and moves it straight to player's inventory.
Creative

Some of the missing blocks are now on the creative selection screen.
The player can no longer eat in Creative.
Drops

Chickens and cows killed by burning now drop their cooked meat.
 Experience

Texture changed from .


=== General ===
Options

"Generated Structures" is now on by default.
Fog

Adjusted fog strength at low altitudes to be less extreme.


== Fixes ==
7 bugs fixed

Fixed endermen having client-side problems in multiplayer.
Sprinting with right-click or sneaking is now fixed.
Fixed furnace tile entity crash.
Fixed leaving the crafting table with items on the grid creating phantom blocks.
Fixed singleplayer chunk memory leak.
Fixed vines and leaves crashing the game when placed in the Nether.
Fixed farmland rejecting melon and pumpkin seeds.
