/transform is an command added Java Edition 23w13a_or_b, which can be used to transform the user in several ways: transforming them into a mob, giving them the skin of any other player, or changing their scale.
It is possible to combine different transformations, such as transforming into an Allay and then setting the user scale to 2.0, resulting in a large Allay.


== Syntax ==
transform into <entity>
Transforms the user into the entity specified.
transform into player <player>
Applies the skin of the specified player to the user.
Note that transform into minecraft:player does not give the same effect.
transform scale <scale>
Scales the user up or down to the specified scale. Must be between 0.1 and 16.0.
transform clear
Clears all current transformations.
