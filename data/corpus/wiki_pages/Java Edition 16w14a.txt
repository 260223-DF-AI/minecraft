16w14a is the first snapshot for Java Edition 1.9.3, released on April 7, 2016, which fixes an important memory leak related to pathfinding.


== Fixes ==
1 issue fixed
From version 1.9.2

MC-100382 – Pathfinding memory leak.

== Trivia ==
Unlike most of the first-of-the-week snapshots, this was released on a Thursday instead of Wednesday, similar to the snapshots before 14w26a.
16w14a is the first actual snapshot released on the 14th week of a year, as 15w14a is just an April Fools' joke.
