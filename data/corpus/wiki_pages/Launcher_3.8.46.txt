3.8.46 is an update to the Minecraft Launcher that was released on February 27th, 2025.


== Additions ==
Added new A Minecraft Movie section and button.


== Fixes ==
MCL-24786 - Launcher string missing space between "log out" or "log in"
MCL-24212 - Minecraft Launcher profiles do not update "LastUsed" data value in json file.
Bug fixes to Java crash dump flow & offline mode functionality.
Various other bug & accessibility fixes & under the hood improvements.
