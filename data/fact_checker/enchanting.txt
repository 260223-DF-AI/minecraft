Minecraft Enchanting Language (Standard Galactic Alphabet) — Fact-Check Knowledge Document

1. OVERVIEW
The enchanting language used in Minecraft’s Enchanting Table is a fictional script known as the Standard Galactic Alphabet (SGA). It appears as random symbols on enchantment options but does NOT correspond to the actual enchantment being selected.

Key fact:
- The text is decorative only
- It does NOT encode real enchantment information
- Enchantment outcomes are determined by RNG and seed mechanics, not the displayed language

4. ENCHANTMENT TABLE TEXT BEHAVIOR

Displayed text rules:
- Randomly generated per enchant option
- Not tied to actual enchantment effect
- Changes every time enchant list refreshes
- Purely cosmetic overlay

ASCII:

Option 1:  ⏃ㄥㄥ モ尺 Ꮆㄖ
Option 2:  乃ㄩ丂 ㄒ卄 丨ㄒ
Option 3:  卩ㄖ山 尺ㄖ ㄒ

(These do NOT translate to real enchantments)

6. ENCHANTMENT LANGUAGE MISCONCEPTIONS

Incorrect assumptions:
- SGA does NOT encode enchant names
- Reading symbols does NOT predict outcome
- Translation tools are not applicable

Correct understanding:
- Enchant text = visual filler system
- Enchant result = hidden logic system

8. ENCHANTMENT SYSTEM COMPONENTS

Required inputs:
- Enchanting Table
- Lapis Lazuli
- Experience Levels
- Item to enchant

Modifiers:
- Bookshelves (increase enchant power)
- Item type (determines available enchants)
- Random seed (world-based RNG)

10. KEY RULES

- Enchanting language is purely decorative
- It uses Standard Galactic Alphabet styling
- It has no functional decoding value
- Enchantment outcomes are determined independently
- RNG + seed + level determine results
